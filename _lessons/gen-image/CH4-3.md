## 教學流程（Teaching Flow）

### 看 / Observation（10 min）

<!--

#### 開場情境（90 秒，純口述）

主線 A 弄一下咖啡工作室老闆準備做**春夏秋冬 4 季 × 3 張新品廣告 = 12 張**的年度行銷系列。

他已經在 CH4-1 學會跨業種套用、CH4-2 學會多變體。他覺得自己準備好了——但跑出 3 張後、發現**老闆自己的朋友問：「這 3 張是 3 個不同品牌嗎？」**

- 春季抹茶拿鐵：鼠尾草綠 + 奶油白 + 抹茶粉盤道具
- 夏季柚子氣泡：黃橘 + 白 + 柚子皮道具
- 秋季焙茶拿鐵：深棕 + 米色 + 乾茶葉道具

3 張結構都對（骨架 4 商品中心 + 道具環繞）、主題都是咖啡、但**看起來像 3 個不同品牌**——因為**配色、字體、道具、語氣**每張都照當季「感覺」跑、沒統一規範。

**缺的是什麼？品牌小卡——你品牌的視覺規格書、給 AI 看的**。

本單元教你：
- 填一份 品牌小卡（5 段）
- 每次跑新廣告都吃這份 DNA
- 跑完用系列檢查 3 問 做跨張一致性檢核

學完後、你跑 3-5-10 張都能**看起來像同一個品牌的 3-5-10 個商品**、不是 3-5-10 個不同品牌。

---

#### 對比展示｜有 DNA vs 沒 DNA（8 min）

> 本段用文字描述讓學員腦內想像 2 組連續廣告的差異。

**情境 A｜沒 DNA（各張自由發揮）**

主線 A 老闆跑春夏秋 3 張廣告、每張都自由選配色字體：

| 張次 | 商品 | 配色 | 字體 | 道具 | 語氣 |
|---|---|---|---|---|---|
| 春 | 抹茶拿鐵 | **鼠尾草綠 + 奶油白 + 白** | 明朝體 | 抹茶粉 + 湯匙 | 療癒、手作 |
| 夏 | 柚子氣泡 | **黃橘 + 白 + 天藍** | 黑體 | 柚子切片 + 冰塊 | 清爽、活力 |
| 秋 | 焙茶拿鐵 | **深棕 + 米 + 咖啡棕** | 手寫感 | 乾茶葉 + 木匙 | 沉穩、職人 |

**視覺感受**：3 張放一起像 **3 個咖啡店的廣告**——綠的、黃的、棕的、每張一個世界。

**情境 B｜有 DNA（所有張共用一份規格）**

主線 A 老闆寫了 品牌小卡：

- 風格 preset：日系職人
- 色票：primary 鼠尾草綠 #5a7a5a / accent 奶茶棕 #8a6a4a / bg 奶油白 #f5efe3
- 字族：主標明朝體 / 內文 Noto Sans JP
- 語氣：手作、職人、季節感、療癒（4 詞固定）
- 禁用：漸層、純黑、霓虹色、快時尚口號

同樣春夏秋 3 張、但**強制吃這份 DNA**：

| 張次 | 商品 | 配色 | 字體 | 道具 | 語氣 |
|---|---|---|---|---|---|
| 春 | 抹茶拿鐵 | **鼠尾草綠 + 奶茶棕 + 奶油白** | 明朝體 + 黑體 | 抹茶粉 + 湯匙 | 手作、職人、季節感 |
| 夏 | 柚子氣泡 | **鼠尾草綠 + 奶茶棕 + 奶油白** | 明朝體 + 黑體 | 柚子切片 + 湯匙 | 手作、職人、季節感 |
| 秋 | 焙茶拿鐵 | **鼠尾草綠 + 奶茶棕 + 奶油白** | 明朝體 + 黑體 | 乾茶葉 + 湯匙 | 手作、職人、季節感 |

**視覺感受**：3 張放一起像**同一個咖啡店 3 個季節新品**——配色字體語氣全一致、只有商品內容換。

**決定性的 3 項一致變量**：**配色 + 字體 + 道具類型**。商品換、但視覺 DNA 不換。

**本段結論**（老師帶 30 秒）：

- **沒 DNA = 每張都重新決定 = 看起來像不同品牌**
- **有 DNA = 每張吃同份規格 = 看起來像同品牌**
- DNA 不是「限制創意」、是**建立長期可辨識的品牌資產**
- 後面 35 分鐘教你填 DNA + 用 DNA 跑 3 張系列 + 跨張一致性驗收

### 學 / Learn（20 min）

#### 品牌小卡 5 段詳解（10 min，扣 `_sources/gen-image/brand-dna-template.md`）

---

**第 0 段｜風格 preset**（1 min）

寫什麼：

- 11 種 preset 選 1（日系 / 韓系 / 美式復古 / 矽谷科技 / 北歐極簡 / 歐陸法式 / 美式海報 / 瑞士排版 / 手繪童趣 / 工業風 / 街頭潮流）
- 選擇理由（30 字內：為什麼這品牌適合這個風格）
- 示範品牌參考（該風格的 1-2 個知名品牌當標竿）

為什麼重要：

- 風格 preset 決定 1-4 段的調性、是**全 DNA 的底色**
- 一個品牌**只能選 1 個 preset**——不能「日系為主、偶爾韓系」

---

**第 1 段｜色票**（2 min）

寫什麼：

| 角色 | Hex | 用途 |
|---|---|---|
| primary | #______ | 主標、CTA 底、重點強調 |
| accent | #______ | 價格、徽章、次要點綴 |
| bg | #______ | 整體背景、大面積底色 |

**禁用色**：列出這品牌絕對不用的顏色（例：純黑 #000、螢光、霓虹）

為什麼重要：

- **3 色是硬限**——超過 3 色 = 雜亂、失去辨識度
- **禁用色比可用色更重要**——AI 預設會用訓練資料平均配色、有禁用清單才會守界線

---

**第 2 段｜字族**（2 min）

寫什麼：

| 角色 | 字族 | 字重 | 用途 |
|---|---|---|---|
| headline | {例：Shippori Mincho / Noto Serif JP} | 700 | 主標、大字堆疊 |
| body | {例：Noto Sans TC / Noto Sans JP} | 400-500 | 副標、內文、說明 |

**字階建議**：

- H1 主標：40-56pt
- H2 副標：20-28pt
- Body 內文：14-16pt

為什麼重要：

- **同品牌只用 2 種字族**——超過 2 種 = 雜亂
- **明確字族名稱**（不寫「明朝體」、寫「Shippori Mincho」）——AI 才能精確執行

---

**第 3 段｜語氣**（2 min）

寫什麼：

- 3-5 個情緒關鍵詞（例：手作、季節感、療癒、職人）
- **禁用語氣**（例：激烈、誇張、促銷轟炸、網路梗）
- **示範句對比**（至少 1 組好 vs 壞）：
  - ✅ 好：「香り高い抹茶と、なめらかなマスカルポーネのハーモニー。」
  - ❌ 壞：「CP 值爆表！抹茶控必喝神級飲品！」

為什麼重要：

- **語氣決定文案調性**——AI 產主標 / 副標時的遣詞造句
- **示範句對比**給 AI 具體錨點、讓它知道「好 vs 壞」的邊界

---

**第 4 段｜禁用清單**（2 min）

寫什麼：

**視覺禁用**：
- 漸層、純黑、霓虹色、寫實人像...（列 3-5 項）

**情境禁用**：
- 促銷轟炸風格、網路流行梗、假名人代言...（列 3-5 項）

**字體禁用**：
- Comic Sans 類手寫感、過瘦 Hairline 字體、同頁超過 2 種字族...（列 3 項）

為什麼重要：

- **禁用清單是本課 DNA 最關鍵的一段**——AI 若沒禁用清單、會用訓練資料平均值（漸層、高飽和、多字族...）、容易跑偏
- 「可用」清單其實是**前 3 段已經定義**、第 4 段用來**明確劃出不跨的界線**

---

**第 5 段｜主要骨架對應**（1 min）

寫什麼：

- **主力骨架**：這品牌最常用哪個骨架（骨架 1-5）
- **次要骨架**：第二常用骨架
- **選擇理由**：30 字內說明為什麼

為什麼重要：

- 一個品牌若主力骨架是 4、跑到骨架 5 就破壞辨識度
- 明確標記 = AI 新產物料時預設從主力骨架出發

---

#### 品牌繼承 prompt 模板（5 min）

扣 `prompt-templates.md` § 類型 4 品牌繼承。

```
吃下以下品牌 DNA：

[貼整份 品牌小卡]

---

結構沿用：{骨架代號，例：骨架 4 商品中心・道具環繞}

本次主題：
- 商品：{新商品}
- 季節 / 場景：{例：夏季限定}
- 徽章內容：{例：6/1-7/31 限定}
- 價格：{例：Regular ¥580 / Tall ¥630}
- 道具暗示：{例：柚子切片、冰塊、氣泡水玻璃杯}

請依以上 DNA + 骨架 + 主題產出 L4 完稿。

【強制自檢】
產完請自我檢查並回覆：
1. 色票是否只用了 DNA 指定的 3 色？（列出實際用色 hex）
2. 字族是否符合 DNA 指定？（列出實際字族）
3. 禁用清單是否全部遵守？（逐條比對）
4. 語氣與 DNA 是否一致？（用 DNA 的示範句對比比對）
5. 骨架是否為 DNA 指定的主力骨架？

若任一項不符、請自動修正重新輸出。
```

**關鍵鎖定**：

- **整份 DNA 貼上**——不是摘要、是完整 5 段
- **強制自檢**——讓 AI 自己對照清單、PM 只做最後判定
- **商品 / 季節 / 徽章 / 價格 / 道具 5 項** 是變量——其他全吃 DNA

---

#### 系列檢查 3 問｜品牌系列一致性檢核（5 min）

扣 `prompt-templates.md` § 類型 5 驗收 C。**單張驗收用 A（CH2-1）、系列驗收用 C（本單元）**。

拿到 3 張系列後、同對話內貼：

```
請對剛才產出的 3 張系列（A / B / C）做品牌一致性自檢、對照以下 DNA：

[貼 品牌小卡]

逐項比對 3 張：

1. 色票：3 張是否都只用了 DNA 指定的 3 色？（列出每張實際用色 hex）
2. 字族：3 張的字族是否都符合 DNA 指定？（列出每張實際字族）
3. 禁用清單：3 張是否都遵守 DNA 禁用清單？（逐條 × 每張比對）
4. 骨架：3 張的結構骨架是否一致（都是同骨架）？
5. 語氣：3 張的文案調性是否一致（對照 DNA 示範句）？

結論：跨張一致性通過 __/5 項。
若全部 5 項通過 = 可作為品牌系列資產收納。
若 3-4 項通過 = 指出哪些項在哪張不符、建議迭代哪張。
若 ≤ 2 項通過 = DNA 太弱、需要加強 brief 或 DNA 禁用清單。
```

**PM 判定**：

- **5/5 通過** → **收為系列資產**、進 PRAC4 / CH5-1 反覆使用
- **3-4/5 通過** → 迭代不符那張（保留通過 4 項 + 只修不過項）
- **≤ 2/5 通過** → 回去強化 DNA（通常是禁用清單填太少）

> **預期差異**：AI 自檢的跨張一致性判斷比肉眼更準——肉眼會被單張美感吸走、忽略跨張偏離。**以 AI 自檢為主要依據**、加你自己的人工複審為輔。

### 做 / Hands-on（15 min）

#### 活動｜填 品牌小卡 + 跑 3 張系列 + 驗收 C

**教學意圖**：學員為自家品牌（或假想品牌）建立完整 DNA、跑 3 張系列、做跨張一致性驗收——完成後就有**可長期使用的品牌視覺資產**。

---

**階段 1｜填 品牌小卡（5 min）**

打開 `_sources/gen-image/brand-dna-template.md` 的填空範本、填自家品牌：

- 品牌：你的自家品牌 / 客戶品牌 / 假想品牌
- 第 0 段：風格 preset（11 選 1）
- 第 1 段：色票 3 色（有具體 hex）
- 第 2 段：字族 2 種（有具體字族名）
- 第 3 段：語氣 3-5 詞 + 至少 1 組好 vs 壞示範句對比
- 第 4 段：禁用清單（視覺 3-5 項 / 情境 3-5 項 / 字體 3 項）
- 第 5 段：主力骨架 + 次要骨架

**若 15 分鐘填不完 5 段**：先填 0 / 1 / 2 / 4（最關鍵 4 段）、3 / 5 簡化

---

**階段 2｜跑 3 張系列（7 min）**

用品牌繼承 prompt 模板、跑 3 張系列：

- 3 張共享同骨架 + 同 DNA
- 只換「商品 / 季節 / 徽章 / 價格 / 道具」5 項
- 例：春抹茶 / 夏柚子 / 秋焙茶 3 個新品

每張約 2 分鐘（30 秒組 prompt + 1 分鐘跑 AI + 30 秒等待）。

---

**階段 3｜驗收 C 跨張檢核（3 min）**

3 張都跑完後、貼系列檢查 3 問 給 AI、讓它做跨張一致性自檢。

**判定**：

- **5/5 通過** → 存檔 3 張系列 + DNA = 你的**品牌資產雛形**
- **3-4/5 通過** → 找出不符項、迭代不合格張次
- **≤ 2/5 通過** → 回階段 1 補強 DNA 禁用清單

---

#### 完成標準（self-check）

- [ ] 填完 品牌小卡（至少 0/1/2/4 四段、3/5 可簡化）
- [ ] 跑出 3 張系列（同骨架 + 同 DNA + 3 個不同商品 / 季節）
- [ ] 做過驗收 C 跨張一致性自檢
- [ ] 驗收 C ≥ 3/5 通過（底線、5/5 最佳）
- [ ] 存檔 3 張系列 + 品牌小卡（PRAC4 會用）

### 備援配置

- **學員不熟自家品牌**：用主線 A 弄一下咖啡的 DNA（已在 `brand-dna-template.md` 提供完整示範）、當作假想品牌練習
- **DNA 填不出 3 段語氣**：用預設示範（手作 / 職人 / 季節感）

- **驗收 C < 3/5**：通常是 DNA 禁用清單太空、現場加 3-5 條禁用項重跑
- **教室時間壓縮至 30 min**：砍看段（改口述）、保留學段 + 做段

**角色**：主線 A 老闆（弄一下咖啡工作室）

**公司**：弄一下咖啡工作室

**任務**：

老闆要做 **春夏秋冬 4 季 × 3 新品 = 12 張**年度系列廣告。他試跑春夏秋 3 張發現問題：

- 跑完用驗收 C 做跨張檢核

**本單元要他學會**：

- PM 系列把關者身份——不跑單張、建立長期資產
- 品牌小卡 5 段填寫邏輯
- 品牌繼承 prompt 的「AI 強制自檢」機制
- 驗收 C 跨張一致性 5 項清單

**POV**：

主要走第二人稱「你」（學員為自家品牌建 DNA）。老闆的 4 季 12 張是案例、你是為自己做同樣的事。

## 動手練習題（Hands-on Exercise）

**題目**：

在 15 分鐘內：

1. **填 品牌小卡**（5 min）：自家 / 客戶 / 假想品牌任選、填完 5 段（至少 0/1/2/4）
2. **跑 3 張系列**（7 min）：同骨架 + 同 DNA + 3 個不同商品 / 季節
3. **驗收 C 跨張檢核**（3 min）：AI 自檢 5 項一致性

**預期成果**：

- 1 份 品牌小卡（至少 0/1/2/4 四段）
- 3 張系列 L4 完稿
- 1 份驗收 C 記錄（5 項打勾 + 判定）
- 1 張**獨立備份**的 DNA（PRAC4 / CH5-1 會反覆使用）

**完成標準**（self-check）：

- [ ] DNA 填寫具體（色票有 hex / 字族有名稱 / 禁用清單至少 5 項）
- [ ] 3 張系列都吃同份 DNA（不是每張重填）
- [ ] 驗收 C 跨張通過 ≥ 3/5（底線）
- [ ] 3 張放一起能被第三人辨識為「同品牌」
- [ ] 能說出「若 DNA 某段沒填、系列會變什麼樣」

**預期差異提醒**：

- 你的 DNA 跟同學完全不同、你的 3 張系列也不同——這是設計
- **5/5 通過不是目標**——3-4/5 已經比沒 DNA 好很多
- DNA 是**迭代品**——第一版不完美、用個 1-2 個月後會想修、這是正常的
- 今天填的 DNA + 3 張系列是**品牌資產雛形**、PRAC4 + CH5-1 會繼續用

## 常見錯誤 3 條（Common Pitfalls）

### 錯誤 1｜DNA 填得太抽象（只寫「日系風」）

**錯誤現象**：

學員填 DNA 時、第 0 段風格寫「日系風」、第 1 段色票寫「清新色調」、第 2 段字族寫「好看的字體」、第 3 段語氣寫「親切」。填完感覺 DNA 有寫、實際上 AI 看不懂。

**原因**：

**抽象詞 = AI 的猜謎題**。日系風的配色有 10 種可能、字族有 20 種可能——AI 只能猜一個組合。下次跑新系列時 AI 又猜一個、**兩次結果不會一致**。

**解法**：

**每段都要具體到「可執行」的程度**：

| 抽象（錯誤）| 具體（正確）|
|---|---|
| 日系風 | 日系職人（預設示範品牌：蔦屋書店 / MUJI）|
| 清新色調 | primary 鼠尾草綠 #5a7a5a + accent 奶茶棕 #8a6a4a + bg 奶油白 #f5efe3 |
| 好看的字體 | headline: Shippori Mincho (700) / body: Noto Sans JP (400) |
| 親切 | 手作、職人、季節感、療癒（4 詞）+ 示範句「なめらかな抹茶のハーモニー」|

**檢查方法**：填完每段、**換隔壁同學看**——他能不能照你的描述重現同樣的視覺？若他說「我要猜什麼顏色才對」——你的描述不夠具體。

---

### 錯誤 2｜3 張系列跑完沒做驗收 C 跨張檢核

**錯誤現象**：

學員跑 3 張系列、每張個別看都還 OK、存檔。**沒跑驗收 C 跨張檢核**。兩週後要做第 4 張系列、打開前 3 張才發現——第 1 張是鼠尾草綠主導、第 2 張不小心用了橄欖綠（跟鼠尾草綠不同 hex）、第 3 張徽章變矩形（不是 DNA 指定的圓形）——**三張已經偏離 DNA**。

**原因**：

單張驗收用 A、但**系列驗收需要跨張比對**——人肉眼比對跨張偏離容易漏判（特別是色彩相近如鼠尾草綠 vs 橄欖綠）。驗收 C 就是專門為了抓**跨張微偏離**設計的。

**解法**：

**3 張系列跑完後、必跑驗收 C**：

- 讓 AI 逐項比對 3 張的色票 / 字族 / 禁用 / 骨架 / 語氣
- AI 會列出每張實際用色 hex、人腦會發現「欸第 2 張的 #7a8a5a 不在 DNA 指定的 3 色裡」
- 驗收 C < 5/5 → 立刻迭代那張、不要存檔

**紀律**：**驗收 C 是系列化的最後守門員**——跳過就等於把偏離的資產存進品牌庫、未來每次新廣告都在累積偏離。

---

### 錯誤 3｜DNA 的「禁用清單」沒填

**錯誤現象**：

學員填 DNA 時、第 4 段禁用清單留空或只寫「沒特別禁用」。結果 AI 產系列時、**自己加了漸層背景、多了裝飾邊框、用了額外的亮橘色點綴**——這些 DNA 沒禁、所以 AI 合法加上、但**跟你的品牌調性不符**。

**原因**：

學員以為「可用清單夠了、禁用清單可有可無」。**錯**。AI 預設會從**訓練資料平均值**取用（訓練資料裡充滿漸層、多色、裝飾邊框）、**沒禁用清單 = AI 自由發揮**。

**解法**：

**禁用清單是必填、不是選填**：

至少列 3 類 × 每類 3-5 項：

**視覺禁用**（3-5 項）：
- 漸層（本品牌走平面）
- 純黑 #000（太冷硬）
- 霓虹色、螢光色
- 寫實人像（走靜物路線）
- 玻璃態 glassmorphism

**情境禁用**（3-5 項）：
- 促銷轟炸風（本品牌走價值訴求）
- 網路流行梗
- 假名人代言
- 快速消費感（走慢食）

**字體禁用**（3 項）：
- Comic Sans 類手寫感字體
- 過瘦 Hairline 字體
- 同頁超過 2 種字族

**心法**：AI 是「很會跟你訓練的員工」——你告訴它「不要做 X」比「做 Y」更重要、因為它會自己學會做 Y、但**你不說它就會做 X**。

## 檢核題 2 條（Quiz）

### Q1（概念驗證｜簡答）

請說明 品牌小卡 的 **5 段是什麼**、以及**為什麼每段都重要**（特別是解釋：為什麼禁用清單不是選填、是必填）。

**預期答案要點**（至少命中 5 點算通過）：

1. **5 段**：
   - 第 0 段：風格 preset（11 選 1）
   - 第 1 段：色票 3 色 + 禁用色
   - 第 2 段：字族 2 種 + 字階
   - 第 3 段：語氣 3-5 詞 + 示範句對比
   - 第 4 段：禁用清單（視覺 / 情境 / 字體）
   - 第 5 段：主力骨架 + 次要骨架對應

2. **禁用清單必填的理由**：
   - AI 預設從訓練資料平均值取用（充滿漸層、多色、裝飾）
   - 若只填「可用清單」、AI 會**自由發揮加東西**
   - 禁用清單 = **明確劃出不跨的界線**、AI 才會守
   - 「告訴 AI 不要做什麼」比「告訴 AI 做什麼」更重要

3. **每段重要理由**（加分項）：
   - 色票：3 色硬限、超過 = 雜亂、失去辨識度
   - 字族：2 種硬限、明確字族名 AI 才能精確執行
   - 語氣：示範句對比給 AI 具體錨點、知道「好 vs 壞」邊界
   - 骨架對應：避免今天用骨架 4、明天跑骨架 5、破壞品牌辨識度

4. **DNA 整體價值**：
   - 給 AI 看的品牌規格書、不是給設計師看的文件
   - **一次填、多年用**——每次新廣告都吃同份 DNA
   - 建立長期可辨識的品牌資產

**不及格的答案**：

- 只列 5 段沒說為什麼重要
- 「禁用清單看情況」（錯誤、必填）
- 「DNA 就是品牌風格」（太空泛）
