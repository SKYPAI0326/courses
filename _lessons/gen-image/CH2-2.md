## 教學流程（Teaching Flow）

### 看 / Observation（10 min）

<!--

#### 開場情境（90 秒，純口述）

主線 A 弄一下咖啡工作室老闆在 CH1-1 學到：他不是設計師、是 PM。現在他打開 Gemini 準備做春季抹茶拿鐵廣告——這次他不再說「幫我做一張好看的咖啡店廣告」，但他還是不確定**到底要說什麼**。

「要幾個字？」「主標寫什麼？」「商品要放哪？」「配色？」——腦子開始混亂，又回到卡關狀態。

這個卡點不是他不會用 AI，是**他還沒把腦中模糊想法結構化**。本單元教你一個固定模板——**brief 六件套**——把任何需求填空進去，就成為 AI 能穩定吃的指令。

六件套不是套路，是 PM 身份的**工作語言**——你給老闆報告、跟客戶討論需求、跟設計師溝通——**也都該用這套**。

---

#### 現場對比實驗（8 min）

> 學員當場跑 2 次 AI，自己看自己的兩張產出差異。

**步驟 1｜模糊指令版（2 min）**

學員打開 Gemini 或 ChatGPT，複製以下指令丟進去：

```
幫我做一張好看的咖啡店廣告，1:1 尺寸
```

等待約 20-30 秒產出。

**步驟 2｜結構化 brief 版（3 min）**

同一個對話（或新開對話皆可），複製以下指令丟進去：

```
請產出一張 1:1 日系商業橫幅、用「A 主視覺版」風格：

【業種】手沖咖啡店
【商品】春季限定 抹茶提拉米蘇拿鐵，Regular ¥680 / Tall ¥730
【客群】25-35 歲女性、注重生活風格
【訴求】手作、季節感、療癒
【必要元素】
- 畫面中央偏右：抹茶拿鐵透明玻璃杯
- 左上：季節限定 tag
- 主標（2 行）：抹茶ティラミスラテ
- accent 手寫字：New!
- 左下：價格 Regular ¥680 / Tall ¥730
- 左下 CTA：メニューをチェック
【風格 preset】
- 配色 ≤ 3 色：鼠尾草綠 + 奶油白 + 抹茶綠
- 字體：主標明朝體粗體、內文 Noto Sans JP
- 視覺調性：手作、季節感、職人

【尺寸】1:1
```

等待約 20-30 秒產出。

**步驟 3｜兩張並排比對（3 min）**

學員把兩張截圖並排放（手機截圖 / 螢幕分割 / 或印出來放一起），觀察：

- 模糊指令版：**哪些地方 AI 在猜？**（配色？字體？商品形狀？文字內容？）
- 結構化 brief 版：**哪些地方跟你指定的一致？哪些還是被 AI 自由發揮？**
- 整體哪張**更接近你心中的成品**？

> **預期差異**：模糊指令版每人都會不同、且都偏離「你真正要的」；結構化 brief 版會比較接近，但每個人的配色、字型、構圖細節仍會有差異——這是 AI 特性。重點是 **brief 版能被你作為 PM 驗收**（因為有清單對照），模糊版你只能說「感覺不太對」。

**本段結論**（老師帶 30 秒）：

- brief 不是模板作業、是 **PM 的工作語言**
- 六件套缺一個 → AI 就要猜一個 → 結果偏離
- 填好六件套的 2 分鐘，比跑 10 次猜猜樂省時間
- 本單元接下來 20 分鐘，逐欄拆解六件套怎麼填

---

### 學 / Learn（20 min）

#### brief 六件套｜逐欄拆解（15 min）

本課最重要的教學物。每欄含**填什麼 + 常見錯誤 + 填好的判準**。

---

**欄位 1｜業種（2 min）**

- **填什麼**：你的業種大類（最好對應常見業種，例：咖啡店 / 美妝保養 / App / 旅遊 / 保健品…）
- **為什麼重要**：業種決定 AI 腦中喚起的視覺詞彙庫與配色範圍
- **填寫範例**：
  - 好：`手沖咖啡店`、`職人手工皂`、`AI 筆記 App`
  - 差：`服務業`、`消費品`、`一般商品`（太抽象）
- **填好的判準**：能對應到 `10-industries-brief-library.md` 中 10 業種之一，或至少能被第三人理解

---

**欄位 2｜商品（2 min）**

- **填什麼**：具體商品名 + 規格 + 型態
- **為什麼重要**：商品型態決定 AI 畫面中「主視覺物件」長什麼樣（瓶、罐、杯、盤、mockup…）
- **填寫範例**：
  - 好：`春季限定 抹茶提拉米蘇拿鐵，Regular ¥680 / Tall ¥730`
  - 好：`LUNÉA モイストリペアセラム 30mL 精華液`
  - 差：`我的新商品`、`好東西`、`主打商品`
- **填好的判準**：第三人看完能畫出商品大致形狀與大小

---

**欄位 3｜客群（2 min）**

- **填什麼**：年齡 + 性別 + 生活風格 / 職業 / 痛點（至少 2-3 項）
- **為什麼重要**：客群決定 AI 產出的「情緒氣質」——25 歲女性 IG 族群 vs 45 歲男性企業主的廣告，配色、字體、文案風格完全不同
- **填寫範例**：
  - 好：`25-35 歲女性、注重生活風格、IG 重度使用者`
  - 好：`30-45 歲自由工作者、追求效率、願意為工具付費`
  - 差：`一般大眾`、`有消費能力的人`、`喜歡 AI 的人`
- **填好的判準**：能聯想到 2-3 個具體的人（朋友 / 親戚 / 客戶）

---

**欄位 4｜訴求（2 min）**

- **填什麼**：3-5 個關鍵調性詞（情緒關鍵字，不是功能描述）
- **為什麼重要**：訴求決定 AI 產出的「溫度」——療癒 vs 刺激、手作 vs 科技感、職人 vs 大眾化
- **填寫範例**：
  - 好：`手作、季節感、療癒`
  - 好：`現代、極簡、效率`
  - 差：`好用、划算、多功能`（功能描述、非調性）
  - 差：`美`、`酷`（太單一）
- **填好的判準**：3-5 個詞、每個都能對應到具體的視覺語言

---

**欄位 5｜必要元素（核心、5 min）**

- **填什麼**：對應你選的 **8 版型**（CH2-1 學過、見 `8-version-library.md`）、列出該版型結構的每個槽位要放什麼
- **為什麼重要**：**版型 + 必要元素**決定畫面的版面結構。選錯版型或漏填元素、畫面會垮、AI 也不知道你要哪一種廣告
- **8 版型 → 必要元素對照表**（這就是「必要元素」欄要怎麼填的核心對照）：

| 版型 | 必要元素清單模板 |
|---|---|
| **A 主視覺版** | 主角（商品）80-90% / 文字 ≤ 20%（主標 1 行 ≤ 12 字）/ 黃金視覺位置（中上偏左 1/3）放亮點 |
| **B 橫幅版** | 左 50% 商品 70%+ 高度 / 右 50% 文字（主標 + 短標語 + CTA）/ 主次分明 |
| **C 社群版** | 主角商品 70%+ / 大量留白 / 文字 < 30%（品牌 + 商品名 +（可選）catchphrase）|
| **D 系列版** | 上半 70% 3 款並列 / 下半 30% 每款 1 行特色 + 信任標籤 / 統一色調 |
| **E 禮盒版** | 禮盒外包裝 70%+（含緞帶 / 紋飾 / 標籤）/ 深色背景 / 金色點綴 / 品牌名 + 系列名 |
| **F 特寫版** | 商品 95%+ 極致特寫 / 動態瞬間（流心 / 拉絲 / 蒸氣）/ 光線強化質感 / 零文字或極少品名 |
| **G 促銷版** | 上半 60% 大字優惠 + 限時 / 下半 40% 商品 + CTA / 強對比色 / 急迫感視覺元素 |
| **H 生活版** | 商品 30-40%（配角）/ 人的痕跡（咖啡杯 / 書 / 毛毯）/ 自然光 / 情境文案（手寫感）|

- **填寫範例**（A 主視覺版）：

  ```
  - 主角：春季抹茶拿鐵透明玻璃杯、占畫面 80%、置中偏上
  - 文字：主標 1 行「春のはじまり」、置於下方 15%
  - 亮點：拉花線條從中上偏左 1/3 處延伸
  - 留白：背景米白漸層、無其他元素干擾
  ```

- **填寫範例**（G 促銷版）：

  ```
  - 上半 60%：大字「第 2 杯 5 折」（粗黑體、占整段視覺最強）
  - 副訊息：4/25-4/30 限定（小字置於主訊息下方）
  - 下半 40%：商品（拿鐵）+ CTA「點餐優惠」
  - 對比色：黑底 + 黃字 + 紅圓標
  ```

- **常見錯誤**：選了 D 系列版（3 款並列）卻只填 1 款商品——版型結構與商品數量不符；選了 H 生活版卻填「大字優惠」——版型訴求矛盾
- **填好的判準**：每個元素都有明確位置 + 占比 + 用途；元素清單對照 `8-version-library.md` 該版型結構

> 💡 **快捷做法**：直接翻 `8-prompt-examples.md` 對應版型段、複製預寫的必要元素清單、改變數即可。本檔範例都是從那邊抽出來的。

---

**欄位 6｜風格 preset（2 min）**

- **填什麼**：選 1 種 preset（教學基準：**日系**，演練延伸：韓系 / 美式復古 / 矽谷科技 / 北歐極簡 / 歐陸法式 / 美式海報 / 瑞士排版 / 手繪童趣 / 工業風 / 街頭潮流）+ 配色 3 色 + 字體 2 種 + 調性關鍵字
- **為什麼重要**：風格 preset 決定畫面的**視覺表象**——結構（版型）不變、表象隨 preset 切換
- **填寫範例**：

  日系 preset（預設）：
  ```
  - 類型：日系
  - 配色 ≤ 3 色：鼠尾草綠 #5a7a5a + 奶油白 #f5efe3 + 抹茶綠點綴
  - 字體：主標 Shippori Mincho、內文 Noto Sans JP
  - 視覺調性：手作、季節感、留白、職人
  - 禁用：漸層、純黑、霓虹色
  ```

  韓系 preset（替代）：
  ```
  - 類型：韓系
  - 配色 ≤ 3 色：莫蘭迪灰 + 粉色 + 米白
  - 字體：細黑體 + 英文襯線
  - 視覺調性：精緻、高級、簡約
  ```

- **常見錯誤**：只寫「極簡」「日系風」不寫具體配色 / 字體 / 調性——AI 只能猜
- **填好的判準**：配色有具體 hex 或色名、字體有具體名稱、調性有 3 個以上具體形容詞

> **預期差異**：同樣一份 brief、日系 preset 跟韓系 preset 跑出來的成品**完全不一樣**——這是預期行為。**你作為 PM 控制的是結構（版型）與風格（preset）**、AI 處理表象細節。表象細節每次微有不同屬 AI 特性、只要結構與風格符合指定就是成功。

---

#### 完整 prompt 模板套用（5 min）

把 6 欄組合成一份可丟 AI 的 prompt。模板如下（也可直接從 `8-prompt-examples.md` 對應版型段複製、變數位置已預寫好）：

```
請生成一張 {ratio} 商業圖、用「{版型代號 + 中文名}」風格：

【業種】...
【商品】...
【客群】...
【訴求】...
【必要元素】（對應你選的版型結構、見 `8-version-library.md` 或 `8-prompt-examples.md`）
- ...
【風格 preset】
- 配色 ≤ 3 色：...
- 字體：...
- 視覺調性：...

【尺寸】{ratio，預設 1:1}
【禁用】（依版型加、如 G 促銷版禁柔和、H 生活版禁大字促銷）
```

**使用順序**（兩條路、選一條）：

**路徑 A｜直接複製本課彈藥庫**（推薦、最快）：

1. CH2-1 用 3 問選版型（A / B / C / D / E / F / G / H 任一）
2. 翻 `8-prompt-examples.md` 對應段、複製「快速啟動 prompt」
3. 用六件套替換變數空格（`{業種}` `{商品}` 等）
4. 丟 AI、5 分鐘出圖

**路徑 B｜從零組裝**（適合進階 / 變化）：

1. 填欄位 1-6（先不丟 AI）
2. 自我檢查：有沒有欄位是空的？（缺一不可原則）
3. 全部填完 → 套入模板 → 丟 AI
4. 10 分鐘內產出一張完稿

> 💡 **Tip 1｜文字亂碼處理**
>
> AI 對中 / 日文字渲染仍有不穩定性，建議：
> - 主策略：讓 AI 產「空版 + 圖像元素」，文字交給 **Canva 後製補字**（Canva Free 字型庫免費、可控）
> - 若要 AI 直接產文字：限**主標 + CTA 兩個元素**（細節文字留白）
> - Fallback：產完若文字錯誤，同對話內說「請將圖中所有文字替換為 [新文案]」重試
> - 日文精準度：**ChatGPT GPT-image-2 通常優於 Gemini Nano Banana**

> 💡 **Tip 2｜版權邊界**
>
> AI 產出**僅作為參考稿**，不得直接商用。商用流程：
> 1. 用 **Canva 替換**為合法授權素材（Canva Free / Pro 內建庫）
> 2. 或用 **Adobe Firefly 重製**（Firefly 訓練資料版權乾淨）
> 3. 或**自家拍攝 / 購買**素材
>
> 講義示範與練習產出統一加註：「本頁圖片為 AI 生成練習作品，不可商用」。

### 做 / Hands-on（15 min）

#### 活動｜挑一個自己的實際案場、跑一張完稿

**教學意圖**：讓學員把六件套套到**自己真實工作情境**，不是練習題。上完課能直接帶一張可用的完稿走（雖然商用還需 Canva / Firefly 重製）。

---

**階段 1｜填 brief（5 min）**

學員挑一個**真實情境**（自家品牌 / 客戶品牌 / 假想品牌皆可），填完整六件套：

| 欄位 | 你的填寫 |
|---|---|
| 業種 | |
| 商品 | |
| 客群 | |
| 訴求 | |
| 必要元素（配對你選的版型 A-H） | |
| 風格 preset | |

**自我檢查**：

- [ ] 6 欄都填了、無空欄
- [ ] 商品欄具體到第三人能想像形狀
- [ ] 客群欄有 2-3 項描述
- [ ] 訴求欄有 3-5 個具體調性詞
- [ ] 必要元素對應到某個版型的結構清單（A-H 任一）
- [ ] 風格 preset 有具體配色 / 字體 / 調性

**若任一格空白或抽象**：這就是你作為 PM 的**需求不清問題**——補齊再丟 AI。

---

**階段 2｜組 prompt + 跑 AI（8 min）**

用 `prompt-templates.md` § 類型 1 模板組裝你的六件套 → 丟 Gemini 或 ChatGPT → 產出 1 張 1:1 完稿。

**工具選擇建議**：

- 零預算：Gemini Free（Nano Banana）
- 付費：Gemini Advanced（Banana Pro）品質較穩
- 文字精準需求：ChatGPT GPT-image-2（需 Plus）

---

**階段 3｜初步驗收（2 min）**

拿到完稿後，對照自己的 brief：

- **業種 / 商品**：AI 畫的商品像你要的嗎？
- **客群 / 訴求**：整體氣質吻合你預期的客群嗎？
- **必要元素**：你列的每個元素都到位了嗎？
- **風格 preset**：配色、字體有在你指定的範圍內嗎？

> **預期差異**：你跑出的圖跟隔壁同學完全不同——這是正常的。**本單元目標是練會「寫 brief」，不是「跑出最漂亮的圖」**。只要你的 brief 填完整、prompt 組裝正確、AI 產出符合大致結構，就是成功。
>

## 常見錯誤 3 條（Common Pitfalls）

### 錯誤 1｜brief 六件套缺欄位（最高頻，扣 anti-patterns § 症狀 5「需求不清」）

**錯誤現象**：

學員學完六件套，回家用的時候只填其中 3-4 欄就丟給 AI。例如：
- 填了業種、商品、客群——但沒填「訴求」「必要元素」「風格 preset」
- 或填了 6 欄但每欄都很短（業種：咖啡店；商品：新品；客群：一般人）

AI 產出後，學員發現：**怎麼跟我想的不一樣？**——因為沒填的欄位，AI 就自己亂猜。

**原因**：

這是 PM 最常犯的失誤——**以為「大概講一下」AI 就懂**。但 AI 不是讀心術師、是**結構化指令執行者**。缺一個欄位 = AI 要猜一個欄位 = 成品跟你想的偏離。

本課的 `anti-patterns.md` § 症狀 5 完整描述這個毛病的根因。

**解法**：

**強制六件套完整檢查**：每次下 prompt 前，先跑以下清單：

- [ ] 業種（具體到「咖啡店」「美妝保養」層級）
- [ ] 商品（具體到第三人能想像形狀）
- [ ] 客群（至少 2-3 項描述）
- [ ] 訴求（3-5 個具體調性詞）
- [ ] 必要元素（對應某個版型的結構清單、A-H 任一）
- [ ] 風格 preset（有具體配色 / 字體 / 調性）

**任一欄空白或抽象不到位 → 不按送出**。花 2 分鐘補齊，比跑 10 次猜猜樂省時間。

---

### 錯誤 2｜「必要元素」與「版型」不匹配

**錯誤現象**：

學員選了 D 系列版（3 款並列、商品頁 / 限動）、但必要元素只填 1 款商品——版型結構需要 3 款並列、學員只給 1 款、AI 不知道怎麼湊。

或選了 H 生活版（情境氛圍 / 情感連結）、但必要元素卻填「大字 85 折優惠」——這是 G 促銷版的元素、跟 H 版型訴求矛盾。

或選了 F 特寫版（極致特寫 / 提升食慾）、但必要元素填了「主標 + 副標 + CTA + 3 特徵條列」——版型要求文字 < 5%、學員塞了一堆文字。

結果 AI 陷入矛盾：**按版型畫 → 必要元素塞不下；按必要元素畫 → 版型就走樣了**。

**原因**：

學員把「版型選擇」跟「必要元素列表」當成兩件獨立的事。實際上兩者是**綁定關係**——版型決定必要元素的清單與占比、必要元素必須對應版型的結構槽位。

**解法**：

**填必要元素前、先確認版型與商品 / 訴求匹配**：

1. **商品數量**：1 款 → A / B / C / F / H；3+ 款 → D
2. **商品型態**：實體商品 → A-H 都可；App / 數位 → A / B / C / G（避開 D / E / F）
3. **訴求方向**：抓注意力 → A；快速傳訊 → B；輕量日常 → C；比較選擇 → D；拉高單價 → E；激發感官 → F；衝轉換 → G；情感連結 → H
4. **不確定**：回 CH2-1「目的 → 版型」3 問速查表

**然後**才列必要元素——直接從 `8-prompt-examples.md` 對應版型段複製預寫好的必要元素清單、改變數即可。**不要自己重新發明清單**——版型結構是固定的、變的只是商品內容。

---

### 錯誤 3｜風格 preset 寫得太抽象（AI 還是要猜）

**錯誤現象**：

學員填風格 preset 時只寫：
- `極簡風`
- `日系`
- `清新`
- `高級感`

AI 沒有具體資訊可以照做，還是要猜配色、字體、調性——成品不穩定。

**原因**：

「極簡」「日系」「清新」是**標籤**、不是**指令**。AI 需要**具體參數**才能穩定執行。

**解法**：

風格 preset **必須包含 3 項具體資訊**：

1. **配色** ≤ 3 色（有具體色名或 hex）：
   - 好：`鼠尾草綠 #5a7a5a + 奶油白 #f5efe3 + 抹茶綠點綴`
   - 差：`清新配色`、`暖色系`
2. **字體** 2 種（主標 + 內文）：
   - 好：`主標 Shippori Mincho、內文 Noto Sans JP`
   - 差：`好看的字體`、`日文字體`
3. **視覺調性** 3 個以上具體形容詞：
   - 好：`手作、季節感、留白、職人`
   - 差：`美`、`酷`、`高級`

**偷懶技巧**：如果不知道配色怎麼選，直接從 `10-industries-brief-library.md` § 跨風格演練段的 11 種 preset 複製——每種 preset 都有配色 / 字體 / 調性的完整組合，**選一個貼上就能用**。

## 檢核題 2 條（Quiz）

### Q1（概念驗證｜簡答）

請寫出 brief 六件套的 6 個欄位，並**用自己的話**說明：**為什麼六件套缺一不可**？若缺一個會發生什麼事？

**預期答案要點**（至少命中 4 點算通過）：

1. **六件套**：業種 / 商品 / 客群 / 訴求 / 必要元素 / 風格 preset
2. **缺欄位 = AI 要猜**——AI 不是讀心術師，空著的欄位它就自己發揮，結果偏離預期
3. **六件套是 PM 的工作語言**——跟設計師 / 客戶 / 老闆溝通需求也該用這套
4. **每欄有不可替代的作用**：業種 = 視覺詞彙庫；商品 = 主視覺物件；客群 = 情緒氣質；訴求 = 畫面溫度；必要元素 = 結構槽位；風格 preset = 視覺表象
5. **填 2 分鐘 vs 跑 10 次猜猜樂**——PM 思維的根本差異（**加分項**，扣回 CH1-1 PM 思維）
6. **可被驗收**（加分項）——有六件套對照清單才能在 CH2-1 學的驗收提問 A 時逐項打勾

**不及格的答案**：

- 只列 6 欄位沒說為什麼缺一不可
- 「都很重要」（空泛）
- 「缺一個 AI 就做不好」（對但沒具體說做不好在哪）
