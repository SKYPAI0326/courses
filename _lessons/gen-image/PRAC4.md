## 教學流程（Teaching Flow）

### 熱身 / Warmup（5 min）

<!--
講師備註（不進學員講義）：

本段重點是讓學員 5 分鐘內決定「演練組合」。
3 種組合對應 3 種實際業務情境、學員依自己最常遇到的情境挑——這才真能帶走。
避免學員花 10 分鐘決定「要跑什麼」。
-->

#### 快速決策：選演練組合 + 品牌（5 min）

你是**系列把關者**——這是 M4 最後一輪演練、為自己建立**完整品牌資產**（DNA + 3 張系列）。

**步驟 1｜挑 1 個品牌（2 min）**

三選一：

- **A 自家品牌**：你實際經營或服務的品牌（最有商業價值）
- **B 客戶品牌**：你最近接案的客戶品牌（最接近實戰）
- **C 主線 A 弄一下咖啡工作室**：課程預設範例（若 A/B 不適用）

**步驟 2｜從 3 種演練組合挑 1 組（3 min）**

| 組合 | 情境 | 3 張主題變量 | 適合誰 |
|---|---|---|---|
| **I 跨季節** | 年度行銷規劃 | 春 / 夏 / 秋 新品（同類商品、換季節）| 做快消品 / 餐飲 / 美妝 |
| **II 跨產品線** | 品牌陣列展示 | 主打 A / B / C 3 款不同產品 | 有多產品線品牌 / 手作店 |
| **III 跨活動週期** | 行銷日曆 | 新品上市 / 週年慶 / 會員專屬 3 個活動 | 做電商 / 服務業 / SaaS |

**快速判斷**：

- 你手上業務**一年會跑 4 季行銷** → **組合 I**
- 你品牌**有 3+ 核心產品** → **組合 II**
- 你每月都有**不同行銷主題** → **組合 III**
- 都不太符合 → 預設用**組合 I**（最通用）

**預期差異**：你挑的組合 + 品牌跟同學不會一樣——這是設計、每個人跑自己業務情境才有帶走價值。

### 演練 / Exercise（20 min）

<!--
講師備註（不進學員講義）：

20 分鐘分 3 階段：
- 階段 1 填 DNA（8 min）
- 階段 2 跑 3 張系列（10 min）
- 階段 3 驗收 C（2 min）

巡堂重點：
- DNA 是否具體（色票有 hex / 字族有名稱）——不具體立刻提醒補
- 3 張系列是否共享同 DNA——若看到某張配色偏離立刻提醒
- 驗收 C 跨張通過率——< 3/5 要回頭補 DNA 禁用清單
-->

#### 階段 1｜填 品牌小卡 6 段（8 min）

參考 `_sources/gen-image/brand-dna-template.md` 的填空範本、**必填 6 段**：

| 段 | 內容 | 硬性要求 |
|---|---|---|
| 0 | 風格 preset | 11 選 1、含示範品牌參考 |
| 1 | 色票 | 3 色 + 具體 hex + 禁用色 |
| 2 | 字族 | 2 種 + 具體字族名 + 字階 |
| 3 | 語氣 | 3-5 詞 + 禁用語氣 + 好 vs 壞示範句 |
| 4 | 禁用清單 | 視覺 3-5 + 情境 3-5 + 字體 3 |
| 5 | 主力骨架 | 1-5 選 1 + 次要骨架 + 30 字理由 |

**若真的來不及**：第 0 / 1 / 2 / 4 為底線必填；第 3 / 5 可簡化但不能空。

**自我檢查（填完立刻自查）**：

- [ ] 色票每個都有 hex（不是「鼠尾草綠」、是 `#5a7a5a`）
- [ ] 字族有具體名稱（不是「明朝體」、是 `Shippori Mincho`）
- [ ] 禁用清單至少 10 項（視覺 5 + 情境 3 + 字體 2 以上）

---

#### 階段 2｜跑 3 張系列（10 min，每張 3 min）

依你選的演練組合（I / II / III）、跑 3 張系列：

**組合 I 跨季節範例**：

- 第 1 張：{品牌}春季限定 {新品 A}
- 第 2 張：{品牌}夏季限定 {新品 B}
- 第 3 張：{品牌}秋季限定 {新品 C}

**組合 II 跨產品線範例**：

- 第 1 張：{品牌}主打 {產品 A}
- 第 2 張：{品牌}主打 {產品 B}
- 第 3 張：{品牌}主打 {產品 C}

**組合 III 跨活動週期範例**：

- 第 1 張：{品牌}新品上市
- 第 2 張：{品牌}週年慶優惠
- 第 3 張：{品牌}會員專屬活動

**每張 3 min 流程**：

- **貼品牌繼承 prompt**（30 秒）
- **跑 AI**（1.5 min）
- **等待產出 + 初步檢查**（1 min）

**品牌繼承 prompt 模板**（扣 `prompt-templates.md` § 類型 4）：

```
吃下以下品牌 DNA：
[貼完整 品牌小卡]

---

結構沿用：{你 DNA 指定的主力骨架}
本次主題：{當張主題、例：春季限定 {新品 A}}
- 商品：{商品描述}
- 季節 / 場景：{如春季限定 / 週年慶}
- 徽章：{限期 / 活動名稱}
- 價格：{含稅價}
- 道具：{對應主題的道具暗示}

請依以上 DNA + 骨架 + 主題產出 1:1 L4 完稿。

【強制自檢】產完請回覆：
1. 配色是否只用 DNA 指定 3 色？
2. 字族是否符合？
3. 禁用清單是否全遵守？
```

**3 張都要用同一份 DNA + 同一個骨架**——只換主題變量。

> **預期差異**：3 張每次跑結果不同、AI 對「禁用清單」的遵守程度也有波動——通常 80-90% 遵守、10-20% 偷偷加東西（多色 / 多字族 / 漸層）。**這是為什麼要跑驗收 C**。

---

#### 階段 3｜驗收 C 跨張一致性檢核（2 min）

3 張跑完後、**同對話內貼系列檢查 3 問**（扣 `prompt-templates.md` § 類型 5）：

```
請對剛才的 3 張系列（A / B / C）做品牌一致性自檢、對照以下 DNA：
[貼 品牌小卡]

逐項跨張比對：
1. 色票：3 張是否只用 DNA 3 色？（列每張實際 hex）
2. 字族：3 張字族是否符合？（列每張實際字族）
3. 禁用清單：3 張是否全遵守？（逐條 × 每張）
4. 骨架：3 張是否同骨架？
5. 語氣：3 張調性是否一致？

結論：跨張通過 __/5。
5/5 = 可作為系列資產收納。
3-4/5 = 指出哪張哪項不符、建議迭代哪張。
≤ 2/5 = DNA 需加強。
```

**判定**：

- **≥ 4/5 通過** → 本 PRAC 合格、存檔進**個人品牌資產庫**
- **3/5 通過** → 迭代 1 張不符的（保留 2 項 + 修 3 項）
- **< 3/5 通過** → 回階段 1 補強 DNA 禁用清單、重跑 1 張最差的

---

### 分享 / Share（5 min）

#### 互相展示（3 min）

2-3 人一組、每人快速展示：

- **自家 品牌小卡**（口述 5 段要點、30 秒）
- **3 張系列**（並排展示、30 秒）
- **驗收 C 結果**（通過幾項、30 秒）
- **挑其中最滿意的 1 張 + 最卡關的 1 張**（30 秒）

**同學回饋（互相給 30 秒）**：

- 你的 3 張看起來**像同品牌**嗎？（外部視角檢驗）
- DNA 有沒有**什麼段落可以更具體**？
- 若要再做第 4 張、**你建議什麼主題**？

#### 老師總結 + M5 預告（2 min）

- 統計全班**驗收 C 通過率** ≥ 4/5 的人數
- 挑 2-3 個**最完整的 DNA 範例**當全班示範
- **CH5-1 預告**：下個單元 30 分鐘跑完一次**完整閉環**——你今天產出的 DNA + 3 張系列就是 CH5-1 的起點。**全部存檔備份**。

## 商業情境案例（Case）

**角色**：主線 A 老闆（弄一下咖啡工作室）或學員自家品牌

**公司**：弄一下咖啡工作室（示範）或學員品牌

**任務**：

老闆在 CH4-3 學會填 DNA + 跑 3 張系列。現在他要為自己建立**第一份完整品牌資產**——作為未來所有行銷物料的基礎。

他挑了**組合 I 跨季節**——因為咖啡店最常做季節限定：

- 春：抹茶提拉米蘇拿鐵
- 夏：柚子氣泡美式
- 秋：焙茶拿鐵

目標：**3 張系列放一起、第三方認出「這是同個咖啡店的 3 個季節新品」**。

本單元教他：
- 填完整 品牌小卡 6 段
- 用 DNA + 骨架跑 3 張系列
- 驗收 C 跨張檢核
- 存檔為**個人品牌資產庫雛形**

**本單元要他學會**：

- M4 全技能整合應用（跨產業能力、多變體能力、品牌繼承能力合流）
- PM 系列把關者的實戰手感
- 建立長期可反覆使用的品牌資產（不是一次性）
- 與同學比對、讓外部視角檢驗系列感

**POV**：

主要走第二人稱「你」（學員為自家品牌建資產）。老闆示範是參照、你做的是自己的實戰。

## 動手練習題（Hands-on Exercise）

**題目**：

在 30 分鐘內完成**完整品牌系列演練**：

1. **熱身（5 min）**：挑 1 品牌 + 1 組合（跨季節 / 跨產品線 / 跨活動週期）
2. **演練（20 min）**：
   - 填 品牌小卡 6 段（8 min）
   - 跑 3 張系列（10 min，每張 3 min）
   - 驗收 C 跨張一致性（2 min）
3. **分享（5 min）**：互相展示 + 回饋

**預期成果**：

- 1 份完整 品牌小卡（硬性 6 段、可簡化到 4 段）
- 3 張系列 L4 完稿（共享同 DNA + 同骨架）
- 1 份驗收 C 記錄（5 項打勾 + 結論）
- 1 個**外部驗證結果**（同學能否辨認為「同品牌」）
- 1 組可進 CH5-1 整合實戰的**品牌資產雛形**

**完成標準**（self-check）：

- [ ] 品牌明確（自家 / 客戶 / 假想皆可）
- [ ] 演練組合明確（I / II / III 選 1）
- [ ] DNA 至少 4/6 段填完、全部具體可執行
- [ ] 跑出 3 張系列（共享同 DNA + 同骨架、只換 3 項主題變量）
- [ ] 驗收 C 跨張通過 ≥ 4/5
- [ ] 至少 1 位同學確認「看起來像同品牌」
- [ ] DNA + 3 張系列**獨立備份**（CH5-1 會用）

**預期差異提醒**：

- 你的品牌、組合、DNA、系列跟同學完全不同——這是設計
- **不追求「跑得最漂亮的系列」、追求「放一起能被認出是同品牌」**
- 驗收 C 4/5 ≥ 一致性已合格、5/5 是加分、不強求
- 今天產出是**個人品牌資產庫的雛形**——未來做新廣告從這份 DNA 開始、不再從零

## 常見錯誤 3 條（Common Pitfalls）

### 錯誤 1｜熱身花 10 分鐘決定演練組合

**錯誤現象**：

學員看到 3 種組合（跨季節 / 跨產品線 / 跨活動週期）、開始糾結「我到底該選哪個？」「跨季節會不會太簡單？」「跨產品線我只有 2 款產品怎麼辦？」——結果 10 分鐘過去了、還沒開始填 DNA。

**原因**：

**選擇癱瘓**。對初學者來說 3 選 1 已經太多。但演練的重點是「跑通流程」、不是「選最完美組合」。

**解法**：

**5 分鐘強制決策**（老師巡堂、超過 5 分鐘提醒）：

- 一年做 4 季行銷 → **選 I 跨季節**
- 品牌有 3+ 產品 → **選 II 跨產品線**
- 每月不同行銷主題 → **選 III 跨活動週期**
- **都不符合或選不定 → 預設 I**（最通用）

**心法**：**選哪個都會有收穫**——重點是跑完完整流程、體驗 M4 所有技能的整合。

---

### 錯誤 2｜DNA 填 4 段就跑系列（跳過禁用清單）

**錯誤現象**：

學員覺得時間不夠、填了 0/1/2/3 四段就開始跑系列——**跳過第 4 段禁用清單**。結果 3 張系列跑完、AI 自由發揮加了：

- 第 1 張：加了淡淡漸層背景
- 第 2 張：多了一種你沒指定的輔助色
- 第 3 張：徽章從圓形變成半透明玻璃態

3 張單看都不差、**合起來驗收 C 只通過 2/5**——因為禁用清單太空、AI 每張都偷加東西。

**原因**：

禁用清單是 **DNA 的守門員**。沒禁用 = AI 用訓練資料平均值（漸層、多色、裝飾邊框）自由發揮。

扣 CH4-3 錯誤 3 的 PRAC 實戰版——在時間壓力下最容易被跳過、但跳過代價最高。

**解法**：

**禁用清單是 PRAC4 必填**、時間不夠改第 3 / 5 段簡化、不能省第 4 段：

- 視覺禁用至少 5 項
- 情境禁用至少 3 項
- 字體禁用至少 2 項
- **共 10 項 = 底線**

**快速寫禁用**：對照 CH4-3 教的通用禁用項（漸層 / 純黑 / 霓虹 / 寫實人像 / 快時尚口號 / Comic Sans / 過瘦字體）直接抄、不用創新。

### 錯誤 3｜3 張系列主題變量設太多

**錯誤現象**：

學員挑組合 I 跨季節、但 3 張主題是：

- 第 1 張：春季抹茶**拿鐵**（杯裝飲品）
- 第 2 張：夏季柚子**蛋糕**（盤裝甜點）
- 第 3 張：秋季焙茶**禮盒**（包裝盒）

**3 張商品型態完全不同**——杯 / 盤 / 盒。結果：
- 第 1 張符合骨架 4（商品中心 + 道具環繞的杯子結構）
- 第 2 張難以套進骨架 4（盤子平面、道具環繞邏輯不同）
- 第 3 張更不適合骨架 4（盒子是靜物包裝、不是開放式飲品）

**驗收 C 第 4 項「骨架一致性」直接失敗**。

**原因**：

**變量失控**。系列演練的核心是**控制變量、換單一維度**：

- 組合 I 跨季節 → **商品類別保持一致、只換季節主題**（都是飲品類、或都是甜點類）
- 組合 II 跨產品線 → **3 產品結構相近**（都是杯裝、或都是盒裝）
- 組合 III 跨活動週期 → **商品本身一致、只換行銷主題**

**解法**：

**3 張主題的商品型態必須相近**：

- 組合 I 範例（飲品）：春抹茶拿鐵 / 夏柚子拿鐵 / 秋焙茶拿鐵（都是拿鐵類）
- 組合 II 範例（皂類）：玫瑰皂 / 薰衣草皂 / 檜木皂（都是單塊皂）
- 組合 III 範例（App）：notero 新品上市 / notero 週年慶 / notero 會員方案（都是同 App 不同訊息）

**心法**：**演練是控制變量測技能、不是炫技**。變量多 = 看不出 DNA 效果、失去演練意義。

### Q2（反思驗證｜系列資產價值）

請回答：

1. 你今天產出的 **品牌小卡 + 3 張系列**，**未來半年內**可以**重複使用多少次**？
2. 若下個月你要做第 4 張系列（例：春季 → 冬季），你**需要重新填 DNA 嗎**？還是**只換主題變量**？
3. 這種**「DNA 一次寫、多年用」**的模式，跟你之前做行銷物料的模式相比、**最大的時間節省在哪裡**？

**預期答案要點**（至少命中 3 點算通過）：

1. **重複使用次數**：每季新品 × 4 季 = 4 次 / 年、加上週年慶 / 會員活動 / 特殊主題 = **10-20 張 / 半年**都用同份 DNA
2. **下個月做第 4 張**：
   - **不用重填 DNA**——DNA 是品牌規格、半年內不動
   - 只換主題變量（商品 / 季節 / 徽章 / 價格 / 道具 5 項）
   - 直接貼同份 DNA + 新主題 → 跑一張 → 驗收 A + C
3. **時間節省來源**：
   - 過去：每次做新廣告都從零想（配色 / 字體 / 語氣 / 版面）、每張 30-60 分鐘
   - 現在：DNA 寫一次、後續每張只用 3-5 分鐘（跑 AI + 驗收）
   - **節省幅度 ≈ 80-90%**
4. **額外價值**（加分項）：
   - 系列一致性自動保證（每張都吃同 DNA）
   - 新員工 / 外包設計師來、**給他一份 DNA 就能接手**、不用教品牌風格
   - 品牌 **辨識度複利**——每次曝光都強化相同視覺記憶

**不及格的答案**：

- 「看情況」
- 「不確定能不能重複用」
- 沒算出具體使用次數
- 「還是要重畫設計」（違反本單元核心 — DNA 就是為了不用重畫）
