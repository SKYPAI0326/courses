## 教學流程（Teaching Flow）

### 破題 / Hook

你是弄一下工作室的行政會計。6 月底要做月結，桌上一包資料：6 份各部門傳來的報帳單，命名各不相同（`報帳.docx`、`六月-業務部.xlsx`、`費用單(最終版2).docx`…）；兩份來源不同、有重複廠商的供應商清單；一份交易明細，有幾格空白、一筆金額異常大。

以前處理這些要花一個下午：手動改檔名、逐份開檔複製貼上、肉眼抓重複。

**這個單元，你會看到 Codex 用一句話把這包資料變成兩個具體成果**：

1. 6 份檔名統一成「部門-月份-報帳單」的報帳單
2. 一張把 6 份資料併好的 Excel

大約 90 秒。看完你就知道，後面 CH1-2、CH1-3 要學的設定能換來什麼。

### 你會看到的示範

一個已經設定好的 Codex 專案，只給它一句話——

> 把這個資料夾裡的報帳單，檔名統一成「部門-月份-報帳單」格式，再彙整成一張 Excel。

結果：原本 6 個亂命名的檔案全部改成整齊一致的格式；多出一張把 6 份資料併好的 Excel。

你現在還不用會做——怎麼把專案設定成這樣（建工作資料夾、寫操作守則、建專案）是 CH1-2、CH1-3 要教的。這裡先讓你看到終點：你接下來學的每個設定，都是為了讓 Codex 能像這樣替你把事做完。

### 看示範時注意三件事

不必動手，但看的時候記住三件事，建立你自己的判斷：

1. 它是**真的改了**檔案，不是嘴上說說。
2. 它有沒有**自己亂補**不存在的資料？（CH2 會專門練怎麼防。）
3. 如果這是**真的帳**，哪一步你會想先攔下來、自己核一次再用？

看完寫一句話：**你最想讓 Codex 幫你做掉的一件月結雜務是什麼？**（後面單元會用到。）

### 概念 / Concepts

你剛剛看到的就是 Codex。把它跟 ChatGPT 放一起，三個關鍵差異：

- **它會動手做事**，不只回答——讀檔、改檔、產生新檔、寫文件、做表格（剛剛它就真的改了 6 個檔名、生出一張 Excel）。
- **三個關鍵差異**：
  1. **以你指定的資料夾為主要工作範圍**：它讀那個資料夾裡的真實檔案，不必你貼一段文字它才知道。（實際能讀寫的範圍取決於你設定的權限模式、有無啟用外掛與 Computer Use，不是絕對隔離。）
  2. **會產出檔案，不只給一段文字**：它會真的在你電腦裡生出 Excel、Word、PDF。
  3. **能照你給的規則重複做**：規則寫一次，下個月同一件事一句話就重跑。
- **責任邊界（這條全課回扣）**：Codex **不知道自己做不到**。規則沒講清楚，它會把錯的東西很順地做出來，而且出錯也不會自己停下來。所以先記一句：**凡是它產出的，最後都要人看過**——剛剛它做得很順，但做得順不等於做得對。

### 檢核 / Verification

- 你能用自己的話說出 **Codex 跟 ChatGPT 的 3 個差異**。
- 你能指出示範裡**哪一步「要人把關」**（例：採用 Excel 前先抽查金額）。

兩件都答得出來，就可以進 CH1-2 安裝 Codex。

## 動手練習題（Hands-on Exercise）

本單元尚未上機（學員此時還沒安裝 Codex）。以「觀察 + 一句話」替代：

**題目**：看完示範，寫下**一件**你每個月都要做、最想丟給 Codex 的月結／行政雜務。

**預期成果**：一句具體的話。好：「把 3 個部門的費用單併成一張、抓出重複的發票號」。不好：「做帳」（太籠統）。

**完成標準**（self-check）：
- [ ] 寫的是一件**具體、會重複**的雜務（不是一次性任務）
- [ ] 講得出這件事**現在**手動要花多久
- [ ] 講得出這件事裡**哪一步一定要你自己核**（不能全丟 AI）

## 常見錯誤 3 條（Common Pitfalls）

> 本單元學員尚未上機，這三條是「認知誤區」，先破除，避免後面踩雷。

1. **錯誤現象**：把 Codex 當成「換皮的 ChatGPT」，以為它只會回字，於是把要測試的檔案跟正式月結檔丟在同一個資料夾給它處理。
   **原因**：沒意識到 Codex 會**真的改／覆寫**資料夾裡的檔案。
   **解法**：正式檔永遠留一份副本不動；練習、測試用獨立資料夾。這也是 CH1-3 建工作資料夾時要強調的。

2. **錯誤現象**：看到示範一句話就生出 Excel，覺得「以後直接交出去就好」，跳過覆核。
   **原因**：把「做得順」當成「做得對」。Codex 文字流暢、表格漂亮，不代表金額對、沒漏列。
   **解法**：建立反射——**它產出、你覆核**。金額抽查、總額對回來源，是你的責任，不是它的。

3. **錯誤現象**：聽到「AI 助理、會寫程式」就先卻步，覺得「我不會 coding 用不了」。
   **原因**：誤以為要懂技術、要打指令。
   **解法**：本課全程用 Codex 桌面 App、講中文指令即可，不碰終端機、不寫一行程式。

## 檢核題 2 條（Quiz）

**Q1（概念驗證）**：Codex 和 ChatGPT 最關鍵的差異是哪一個？
- [ ] A. Codex 回答速度比較快
- [ ] B. Codex 會在你指定的資料夾裡動手改檔、產出檔案 ←（正確答案）
- [ ] C. Codex 完全免費、ChatGPT 要付費
- [ ] D. Codex 中文講得比較好

**Q2（應用驗證）**：示範裡 Codex 把 6 份報帳單彙整成一張 Excel。如果這是真的月結帳，你拿去用之前，**哪一步一定要先自己把關**？
- 預期答案要點：
  1. **抽查金額／分類**是否正確（它可能引用錯範圍、分類錯）
  2. 確認它**沒有自己亂補**不存在的資料（缺值該停而不是硬填）
  3. **總額對得回**原始來源（沒漏列、沒重複加總）
