## 教學流程（Teaching Flow）

### 破題 / Hook（3 分鐘）

咖啡館老闆上週末幫自己寫了一頁「關於我們」放在 IG 個人頁。他寫的是：「我們是一家用心經營的精品咖啡館，堅持優質豆源、專業沖煮、溫暖服務，希望給你最棒的咖啡體驗。」

這段話有什麼問題？**沒問題，但也沒用**。把「咖啡館」換成「拉麵店」「手搖店」「美容院」，一個字都不用改也通。客人看完三秒就忘了，因為他腦袋裡沒有任何一個鉤子可以掛住你。

這就是爛定位：**四海皆準、誰都可以說、說了等於沒說**。

好定位長什麼樣？試著比較這三句：

- 「不追 trend 的固定菜單，每款都做到 80 分以上」
- 「給加班族一杯像家的咖啡，7 點就開」
- 「台北少數三種沖煮手法都能現場切換的老闆」

這三句你只要聽一次，就能在腦袋裡生出一張**很具體的畫面**：什麼人會去、什麼時間去、為什麼不是去隔壁那家。這就是定位的功能——**一句話讓人立刻記得你是誰、不是誰**。

今天這 20 分鐘，我會帶你用 CH2-1 剛剛做好的 NotebookLM 品牌聲音摘要，跑一次 TK-02，讓 AI 同時給你三個不同角度的定位句，然後你挑一個**能真正過你家客人點頭關**的留下來。

> **手機操作重點**：品牌聲音摘要如果還在 CH2-1 的 NotebookLM 裡，直接從 Notebook 的「Notes」頁面長按複製；要跨 App 貼到 ChatGPT 的話，複製一次就好，不要來回切。

### 概念 / Concepts（4 分鐘）

這 4 分鐘把三個觀念講清楚，剩下時間全部留給實作。

- **品牌定位一句話** — 一句話讓人立刻記得你「是誰、賣什麼給誰、跟別人差在哪」，30 字內；不是口號、不是 slogan、不是廣告詞，是**你跟朋友解釋你家在做什麼時，那個最短版本**。
- **三角度（功能 / 情感 / 對立）** — 同一家店可以從三種切角被記得：**功能**（你提供了什麼別人沒有的具體東西）、**情感**（客人走進你家會感受到什麼）、**對立**（你刻意不是誰、反對什麼主流做法）。三個角度**沒有誰比較對**，只有「哪個最適合你家現在這批客人」。
- **為什麼要三版本不是一版本** — AI 給你一版你會全盤接受；AI 給你三版你會開始比較，而比較的過程就是**你自己把定位想清楚的過程**。AI 的角色是幫你**擴展選項**，不是幫你決定答案。

**為什麼「優質 / 用心 / 專業」這類詞等於沒講？**

因為所有同業都這樣寫。你說「用心」，隔壁也說「用心」，客人根本沒辦法用這個字區分你們。定位的作用是**把你從一堆「差不多的店」裡拉出來**，泛用形容詞做不到這件事。所以等下你會看到 TK-02 的 Prompt 最後一行寫：「不要產出『優質』『專業』『用心』這類泛用形容詞。」——這條限制是整則 Prompt 的命脈。

> **延伸提示**：三角度不是本課發明的，做過品牌工作的人會認得這是「功能利益 / 情感利益 / 差異化」的白話版。你不用記學術名詞，記得「我家可以從三個切角被看見」就夠了。

### 操作示範 / Demo（5 分鐘）

我現場用咖啡館老闆的資料跑一次 TK-02，你看完馬上自己跑一遍。

**輸入素材（接續 CH2-1 的 NotebookLM 產出）**：

咖啡館老闆在 CH2-1 把官網文案、菜單、客戶評論上傳到 NotebookLM 後，用 NB-01 萃取出這份品牌聲音摘要：

```
【品牌聲音摘要 — 咖啡館 A（CH2-1 產出）】

- 核心詞彙：單品、手沖、固定菜單、老闆挑豆、8 折熟客、下午客少
- 語氣基調：平實、不炫技、略帶自嘲
- 客戶最常讚美的 3 件事：
  1. 「拿鐵奶泡比連鎖店扎實」
  2. 「老闆會記得我上次點什麼」
  3. 「菜單不變很安心，不用每次研究」
- 客戶最常抱怨的 2 件事：
  1. 「下午常常找不到位子」
  2. 「沒有 IG 拍照點」
- 不會出現在品牌溝通裡的詞：網美、打卡、限定、賦能
```

**TK-02 Prompt 原文（建議工具：NotebookLM 延用 CH2-1 的 Notebook，或 ChatGPT 把上面這份摘要貼進去當背景資料）**：

```
我需要你協助我萃取品牌定位，濃縮成一句話。

我的業務：[一兩句話描述你做什麼]

我的客戶：[誰會買、他們為什麼買]

我與競爭對手的差異：[你有什麼別人沒有的]

我不想服務的客戶：[例：殺價型 / 要 24 小時回覆的]

請做三件事：

1. 先指出我上面哪些資訊太模糊，列出需要補充的問題

2. 等我補充後，產出 3 個不同角度的定位句：
   A. 功能導向（強調你提供什麼）
   B. 情感導向（強調客戶感受到什麼）
   C. 對立導向（明說你「不是」誰）

3. 每句控制在 30 字內

不要產出「優質」「專業」「用心」這類泛用形容詞。
```

**五零件拆解（把 TK-02 對回 CH1-1 學過的框架，一次對齊）**：

- **身份**：隱含的「品牌顧問 / 定位教練」（TK-02 靠任務描述帶出身份，沒明說但 AI 會抓到）
- **任務**：萃取品牌定位 → 三角度版本
- **格式**：A / B / C 三句，每句 30 字內
- **限制**：不用「優質 / 專業 / 用心」這類泛用詞
- **退路**：資訊模糊時先列補充問題，不硬產

**咖啡館老闆填完的版本（現場示範）**：

```
我的業務：台北單店咖啡館，開店 2 年，只賣單品手沖 + 義式拿鐵，菜單固定不換。
我的客戶：週間下午來寫東西 / 開會的一人上班族；固定菜單讓他們「不用研究」。
我與競爭對手的差異：老闆自己挑豆、記得熟客上次點什麼、菜單不變；不做季節限定、不追 IG 拍照點。
我不想服務的客戶：把我們當 IG 打卡點的、要求特調的、殺價比菜單價低的。
```

**預期輸出示例（三版本定位）**：

```
【需補充】
1. 「菜單不變」持續多久了？是 2 年都沒變還是會小調整？
2. 熟客回購頻率大約多久一次？

（補充後：2 年沒動過主菜單；熟客平均兩週回一次）

【A 功能導向】
「台北少數 2 年不改菜單的單品店，老闆挑豆、老闆記你」

【B 情感導向】
「下午來寫東西的那杯咖啡，不用研究菜單，老闆記得你上次點的」

【C 對立導向】
「不追 IG 拍照點、不做季節限定，只服務每兩週要來一次的你」
```

看到這三句的差別了嗎？

- **A** 放的是**事實鉤子**（2 年不改菜單、老闆挑豆、記熟客）——適合對同業比較、對新客解釋。
- **B** 放的是**情緒畫面**（下午、寫東西、不用研究、被記得）——適合 IG 置頂介紹、店內文案。
- **C** 放的是**分眾篩子**（不做 IG、不做限定、服務頻繁回訪者）——適合店門口 / 菜單首頁，篩掉不是你客群的人省麻煩。

**卡關備案 3 條**：

- **AI 產出太抽象**（例：「讓咖啡成為生活的陪伴」）：把「不要產出『優質 / 專業 / 用心』這類泛用形容詞」這條限制，追加「也不要『陪伴 / 生活 / 溫度 / 品味』這類詞」，再跑一次。限制越具體，AI 越會往實話走。
- **三版本讀起來都差不多**：通常是你在「我與競爭對手的差異」那格寫太模糊。把那格改成**具體名詞 / 數字**（例：不是「菜單不變」，而是「主菜單 2 年沒動過」），再跑一次就會拉開差距。
- **NotebookLM 免費配額用盡 / 上傳額度已滿**：把 CH2-1 產出的品牌聲音摘要**整段複製**，貼到 ChatGPT 的對話開頭當背景資料（「以下是我的品牌聲音摘要，等下請依此回答」），再貼 TK-02。結果不會比 NotebookLM 差太多，因為真正的資料已經在文字裡。

> **手機操作重點**：三版本產出後，馬上**長按複製整段**，貼到手機記事本或 Apple Notes 存起來；不要只留在 AI 對話裡，下次 session 過期就找不到了。

### 動手 / Hands-on（8 分鐘）

現在換你。這段 8 分鐘我不講話，你跟著做。

**步驟 1（2 分鐘）：把 CH2-1 的品牌聲音摘要打開**

- 如果你 CH2-1 已經建好 NotebookLM 品牌 Notebook，點進去，把 NB-01 的輸出（品牌聲音摘要）整段複製。
- 如果你還沒有，用這份咖啡館老闆的模板當背景資料練手感也可以——重點是**跑過一次流程**，回家再換成你自己的。

**步驟 2（2 分鐘）：填 TK-02 四格**

把下面四格在手機記事本或 ChatGPT 輸入框裡填完：

- **我的業務**：\_\_\_\_\_\_\_\_\_\_
- **我的客戶**：\_\_\_\_\_\_\_\_\_\_
- **我與競爭對手的差異**：\_\_\_\_\_\_\_\_\_\_
- **我不想服務的客戶**：\_\_\_\_\_\_\_\_\_\_（**這格不能空**——寫得出不要誰，才寫得出要誰）

**步驟 3（2 分鐘）：把 TK-02 整段貼進 AI，跑三版本**

- 建議工具：NotebookLM（延用 CH2-1 的 Notebook，AI 會自動把品牌聲音摘要當背景）或 ChatGPT（把 CH2-1 的摘要先貼一次，再貼 TK-02）。
- 跑完如果 AI 先問你補充問題，**老實補充**，不要跳過——這些問題本身就是定位診斷，問到你就是你沒想清楚。

**步驟 4（2 分鐘）：三選一 + 問自己一個問題**

AI 給你 A / B / C 三句後，**不要急著選最漂亮的那句**。依序問自己：

1. **這句話是我家客人會點頭的嗎？** ——想像你把這句話拿給最近五個有消費過的老客人看，他們會不會覺得「嗯，對，就是你家這樣」？如果其中有人會皺眉說「咦，我沒這種感覺」，這句就先放一邊。
2. **這句話是我做得到的嗎？** ——例如選了「老闆記得你上次點什麼」，但你自己其實記不起來，那這句就是**對外承諾 > 實際服務**，客人第二次來踩到會更失望。挑一句你**實際上就已經在做**的，不是你想做的。
3. **這句話能把不適合的客人篩掉嗎？** ——好定位會讓有些人看完轉頭走，那很好；四海皆準的定位等於沒定位。

選好一句後，把它**貼在手機備忘錄置頂**，或設成 IG 個人頁簡介——下一單元 CH2-3 產社群貼文時會直接用到它。

**預期成果（做完應該拿到的東西）**：

1. 一份填完的 TK-02 四格輸入（文字）
2. AI 產出的 A / B / C 三版本定位句（截圖或文字存檔）
3. 選定的 1 句最終版（貼在手機備忘錄或 IG 簡介）
4. 放棄另外兩句的理由**一行筆記**（例：「A 太像官網文案；C 太兇，我還不敢這樣對新客人」）

**完成標準（self-check）**：

- [ ] 四格都填了，沒有空格（尤其「不想服務的客戶」）
- [ ] AI 產出三版本，每句 30 字內，沒有「優質 / 專業 / 用心」這類詞
- [ ] 已選定 1 句當最終版，並寫下另外兩句不選的理由
- [ ] 最終版已存在手機記事本或貼到 IG 簡介

### 檢核 / Verification（3 分鐘）

確認你真的學會了，**不是聽懂了**。自己回答下面三個問題，有一個答不上來就回頭補。

1. **你選的那句定位，跟隔壁同行能不能互換？** ——把「咖啡館」換成「拉麵店」「美容院」，這句還通嗎？如果通，代表你挑的那句太泛，回去選另一版或再跑一次 TK-02 把差異欄位寫具體點。
2. **你選的那句，最近 5 個老客人看了會點頭嗎？** ——實體上可以真的傳給其中一兩個熟客問：「我這樣形容我家店，你覺得像不像？」他說「像」就過關；他說「咦？」就要想是客群不對還是定位不對。
3. **你選的那句話，能讓你篩掉想避開的客人嗎？** ——如果這句話寫在店門口 / IG 簡介，會不會讓你「不想服務的那群人」自動少來？會就過關；不會就代表這句話還是太通用。

這三題都點頭，你這 20 分鐘就成立了。**接下來 CH2-3 我們會拿這句定位當輸入，一次產一週七則社群貼文**——所以先把它存好，別搞丟。

**課程類型**：skill-operation

### Prompt 原始碼

- **TK-02 品牌定位一句話**（三角度版本）Prompt 全文：已如上面「操作示範」段附，含五零件拆解標註
- 填空提示：「不想服務的客戶」不能空；具體名詞 / 數字越多，三版本差異越大

### 情境素材（接 CH2-1 產出）

- **咖啡館 A 品牌聲音摘要**（假資料，接續 CH2-1 的 NotebookLM 輸出）：
  - 核心詞彙清單（單品、手沖、固定菜單、老闆挑豆...）
  - 語氣基調（平實、不炫技、略帶自嘲）
  - 客戶讚美 Top 3 + 抱怨 Top 2
  - 不會出現在品牌溝通的詞清單
- **咖啡館 A TK-02 填空範例**（四格已填）：業務 / 客戶 / 差異 / 不想服務——供講師現場投影示範
- **NotebookLM / ChatGPT 雙路徑操作截圖**：NotebookLM 內貼 TK-02 的流程 vs ChatGPT 先貼品牌聲音摘要再貼 TK-02 的流程，兩路徑各 2 張截圖

### 預期輸出示例（三版本定位）

- 【A 功能】「台北少數 2 年不改菜單的單品店，老闆挑豆、老闆記你」
- 【B 情感】「下午來寫東西的那杯咖啡，不用研究菜單，老闆記得你上次點的」
- 【C 對立】「不追 IG 拍照點、不做季節限定，只服務每兩週要來一次的你」
- 每句附「適合用在哪裡」一行註解（A：對新客解釋 / B：IG 置頂 / C：店門口）

### 卡關備案（Plan B 清單）

- **AI 產出太抽象**（「讓咖啡成為生活的陪伴」）：補限制詞黑名單，追加「陪伴 / 生活 / 溫度 / 品味」再跑
- **三版本讀起來都差不多**：改「差異」欄為具體名詞 / 數字再跑
- **NotebookLM 免費配額用盡**：切換 ChatGPT 路徑，把品牌聲音摘要貼進對話開頭當背景
- **學員選不出來 / 三句都不喜歡**：回頭檢查「不想服務的客戶」那格——大機率是那格太空，定位沒有錨點
- **AI 一次給了六版、八版**（超過三版）：Prompt 最後追加「只給三版，不要多不要少」；免費版有時會過度發揮

### 課中測試檢查點

- 講師示範時，**務必故意讓 AI 在第一輪產出「泛用形容詞版本」**（不貼禁用詞限制），再重跑一次對比——讓學員親眼看到限制詞這行的差別
- 檢查學員 8 分鐘動手時間內是否至少完成「填四格 + 跑一次 AI」；沒跑到的學員現場截圖旁邊同學的當範本帶走

## 商業情境案例（Case）

**主線**：A 咖啡館老闆（本單元鎖定這條，不跳 B）

**角色**：台北單店咖啡館老闆，一人兼老闆 + 店員 + 行銷，開店 2 年

**公司規模**：單店，週間日均 30 杯、週末 60 杯；熟客佔 6 成

**任務**：

CH2-1 上週已經把官網文案、菜單、30 則客戶評論上傳到 NotebookLM，用 NB-01 抓出品牌聲音摘要。這週想把那份摘要進一步濃縮成**一句話**，放在：

1. IG 個人頁簡介第一行（現在那行寫「用心沖煮每一杯」——老闆自己也知道沒用）
2. 店門口小黑板最上方（篩掉來錯的客人，省下雙方尷尬）
3. 週五要去跟一個新客戶（本地共同工作空間）談合作，需要 30 秒自我介紹

**本單元要他學會**：用 TK-02 把 CH2-1 的品牌聲音摘要，從功能 / 情感 / 對立三個角度各榨出一句 30 字內的定位，然後**挑一句他家客人真的會點頭的留下來**——不是挑最漂亮的，是挑**最真實的**。

**情境延伸（非本單元必做，供講師臨機發揮）**：如果學員本身不是咖啡館 / 餐飲，可以把咖啡館當示範跑完，再用學員自己 CH2-1 的 NotebookLM 素材跑第二輪。主線 A 是示範標本，不是限制。

## 動手練習題（Hands-on Exercise）

**題目**：

延續 CH2-1 你建好的品牌 Notebook（或咖啡館示範素材）。用 TK-02 跑一次三版本定位，然後挑一句當**你這個月的定位宣告**。

具體步驟：

1. 開啟 CH2-1 的 NotebookLM Notebook（或切換到 ChatGPT，先貼品牌聲音摘要當背景）
2. 填完 TK-02 的四格輸入（業務 / 客戶 / 差異 / 不想服務的客戶）——**四格都要有內容**，不想服務那格特別不能空
3. 把 TK-02 Prompt 整段貼進去，讓 AI 依序：先列補充問題 → 補充後產 A / B / C 三版本
4. 讀完三版本，問自己三件事：
   - **這句話是我家客人會點頭的嗎？**（想像拿給最近 5 個老客人看）
   - **這句話是我做得到的嗎？**（不是「想做」，是「現在就在做」）
   - **這句話能篩掉我不想服務的客人嗎？**
5. 挑 1 句當最終版，貼在手機記事本置頂 / IG 簡介

**預期成果**：

1. 一份填完的 TK-02 四格輸入
2. AI 產出的 A / B / C 三版本定位句（截圖或文字存檔）
3. **1 句最終版**（存在手機記事本 / IG 簡介）
4. **1 行筆記說明另外兩句為什麼不選**（例：「A 太像官網文案；C 太兇」）

**完成標準**（self-check）：

- [ ] TK-02 四格全部有內容，「不想服務的客戶」不是空的
- [ ] AI 給了三版本，每句 30 字內，沒有「優質 / 專業 / 用心 / 陪伴」這類泛用詞
- [ ] 已挑 1 句當最終版，並寫下另外兩句的淘汰理由
- [ ] 這句跟「隔壁拉麵店 / 手搖店」不能互換（通用性測試）
- [ ] 最終版已存入手機記事本或 IG 簡介，方便 CH2-3 直接呼叫

## 常見錯誤 3 條（Common Pitfalls）

1. **錯誤現象**：三版本產出後發現 A / B / C 都偏功能導向（「我家賣單品」「我家菜單不變」「我家老闆記你」），完全沒有情感或對立角度。
   **原因**：TK-02 四格輸入時，「我與競爭對手的差異」欄只寫了功能性事實（菜單、豆子、服務），沒提供任何**情緒線索**（客人走進來的感受）或**對立訊號**（你反對什麼主流做法）。AI 只能從你給的素材擴展，給不出你沒放進去的東西。
   **解法**：把 CH2-1 NotebookLM 產出的「客戶最常讚美的話」原句貼進差異欄（「奶泡比連鎖店扎實」「菜單不變很安心」——這些都是情感線索），再追加一行「我反對的業界做法：不做季節限定、不追 IG 拍照點」——對立訊號有了，C 版本才產得出來。重跑一次三版本就會拉開。

2. **錯誤現象**：AI 產出的句子讀起來很美，但學員看不懂——例如「將日常淬鍊為一杯有溫度的陪伴」「讓每一次相遇成為生活的一部分」。老闆自己念都卡，更別說客人。
   **原因**：TK-02 限制詞黑名單只禁了「優質 / 專業 / 用心」，沒禁「陪伴 / 溫度 / 淬鍊 / 相遇 / 生活」這類**文青但空洞**的詞。AI 免費版為了「不踩禁用詞」會往另一個方向飄——過度抒情。
   **解法**：把限制詞追加為「不要『優質 / 專業 / 用心 / 陪伴 / 溫度 / 淬鍊 / 相遇 / 品味 / 生活 / 時光』這類形容詞或抽象名詞；只用具體可驗證的事實詞（例：2 年、30 字、熟客、拿鐵、下午）」，重跑一次。測試標準：每個詞你都能當場說出「這個詞對應哪件事」。

3. **錯誤現象**：學員挑了三版本中**聽起來最威的那句**（例：「全台北唯一 2 年不改菜單的咖啡館」），但實際上老闆自己也知道去年其實小改過兩次冬天限定，不算 2 年完全沒動。結果這句一貼 IG，熟客留言吐槽「你去年冬天不是有出熱紅酒拿鐵？」尷尬收場。
   **原因**：動手練習「問自己三件事」的**第 2 題**（這句話是我做得到的嗎）沒有認真答。學員挑「AI 產出最威的那句」而不是「自己實際做得到的那句」——品牌定位一旦對外承諾超過實際服務，客人第二次就會踩到失望雷。
   **解法**：三版本到手後，**逐句標註「我現在真的做得到 / 還沒到位」**。做不到的那句直接淘汰，不管它聽起來多好。寧可選 B 情感版「下午來寫東西的那杯咖啡」（做得到——你就是下午客少、就是適合寫東西），也不要選 A 功能版「2 年不改菜單」（做不到——去年有小動過）。定位是承諾，不是廣告。

## 檢核題 2 條（Quiz）

**Q1（概念驗證）**：TK-02 為什麼要一次產三個不同角度（功能 / 情感 / 對立）的定位句，而不是直接叫 AI 給一句最好的？

- [ ] A. 因為免費版 AI 單次產出不準，要多版本互相對照
- [ ] B. 因為三版本可以分別放在不同通路（IG、店門口、簡報）
- [ ] C. 因為比較三個角度的過程，是老闆自己把定位想清楚的過程；AI 幫你擴展選項，不幫你決定答案 ←（正確答案）
- [ ] D. 因為品牌定位本來就需要功能、情感、對立三個都有，缺一不可

> 解析：C 最接近本單元核心。A 是工具論但不是本質原因；B 雖然成立但屬於延伸應用，不是為什麼要三版本的**根本理由**；D 錯在「三個都要有」——實際上你只會選一個當主定位，另外兩個是參考選項，不是「缺一不可」。

**Q2（應用驗證）**：你跑完 TK-02，AI 給你這三句咖啡館定位：

- A 功能：「嚴選單一產區精品豆，職人手沖每一杯」
- B 情感：「在城市裡，給你一杯像家的咖啡」
- C 對立：「不跟連鎖店拚速度，只服務願意等 5 分鐘手沖的熟客」

**你現在要挑一句印在名片背面**。依照本單元的判斷標準，你會怎麼選？預期答案要點寫出來：

- **預期答案要點**：
  1. **A 和 B 會先被淘汰**：A 有「嚴選」「職人手沖」——業界幾乎每家精品咖啡都這樣寫，無法區分你（通用性測試失敗）；B 有「像家的咖啡」「城市裡」這類泛用情感詞，把「咖啡」換成「茶館 / 餐酒館 / 書店」一樣通（隔壁可互換測試失敗）。
  2. **C 最可能中選**：「不跟連鎖店拚速度」是對立訊號，「願意等 5 分鐘手沖的熟客」是具體客群描述，兩者都是其他同業不太敢直白寫的——能把「趕時間、要立刻拿走」的客人篩掉，對雙方都好。
  3. **但選 C 前要過第二關（做得到嗎）**：如果你實際上希望擴張外帶、做上班族商務客，C 版本會自我設限不適合；要先確認「服務願意等的熟客」真的是你當下策略，再選它。如果不是，應該**回頭把差異欄寫具體點重跑一次**，而不是硬挑現有三版本。
