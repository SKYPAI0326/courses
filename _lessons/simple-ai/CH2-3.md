## 教學流程（Teaching Flow）

### 破題 / Hook（2 分鐘）

週日晚上十點，咖啡館老闆坐在店裡收完銀、洗完杯子，想起一件事：下週一早上要發 IG。

他打開 IG，看著空白的發文框，腦袋也是一片空白。上週發了新拿鐵、前天發了門口的貓、再上週發了…想不起來上週發了什麼。滑一下別家咖啡館的帳號：人家一週五則、排版整齊、每則都有主題。自己呢？想到什麼發什麼，一週頂多兩則，常常發完自己都覺得「這是在發什麼」。

「下週七天，每天一則，現在想不出來。」這句話是老闆每個週日晚上的固定旁白。

今天這 25 分鐘，要解掉的就是這個固定旁白。我們要做的事很簡單：用**一條龍**的方式，週日晚上花 20 分鐘，把下週 7 則貼文的骨架、其中一則的兩組標題、另外一則改寫成的短影音腳本，全部備好。禮拜一早上你只要打開手機、看一眼骨架、拍張照配上去，就能發。

用三則 Prompt，接著上一堂（CH2-2）你剛寫好的品牌定位，從頭做到尾。

> **為什麼不是一條 Prompt 全包？**
> 因為一週七則、A/B 標題、30 秒腳本是三種不同形狀的東西，一條 Prompt 塞進去，AI 會四不像。我習慣的做法是：先用 MK-01 把骨架鋪出來，看哪一則最有感，再用 MK-02 把那一則的標題打磨到想點，最後挑一則適合拍的用 MK-03 改寫成短影音。三件事分開做，每件事都做得好。

### 概念 / Concepts

- **骨架 vs 成稿** — 這一節產出的不是「可以直接發」的完稿，是每則貼文的**四件套**：主題 / 開頭吸睛句（鉤子）/ 核心訊息 / 行動呼籲（CTA）。禮拜一早上花 5 分鐘補照片和口吻，才變成可以發的東西。AI 負責想，你負責定稿。
- **一週節奏感** — 七則不是七則亂發，每一天有自己的角色：週一打招呼 / 週二產品 / 週三客戶 / 週四幕後 / 週五週末推薦 / 週六互動 / 週日品牌理念。這個節奏不是規定，是避免你七天都在推「來喝咖啡」的定錨。
- **鉤子（開頭吸睛句）** — 貼文第一句的唯一任務是「讓人停下滑動」。不是開場白（「大家好，今天要跟大家分享…」直接廢），是一個問句、一個反差、一個具體到不行的場景。
- **CTA（行動呼籲）** — 貼文最後一句，明確告訴讀者「看完這則你要做什麼」：留言分享一件事、點個愛心、私訊關鍵字、來店出示螢幕。沒有 CTA 的貼文，讀者看完就滑走了。
- **A/B 標題** — 同一則貼文寫兩版不同角度的開頭（不是改字、是改切入點），自己唸出來感受哪版比較想繼續看。不是要上廣告 AB test，是逼自己想過兩個角度，挑一個比較有信心的。
- **短影音三幕結構** — 30 秒腳本分成三塊：開頭 3 秒鉤子（抓住不滑走）/ 中段鋪陳 15-20 秒（核心訊息）/ 結尾 3-5 秒 CTA。每一塊對應一個拍攝畫面（shot），一支影片 3-5 個 shot。沒有這三塊，你拍出來會是日常 vlog，不是行銷影片。

### 操作示範 / Demo（MK-01 示範 + 動手，10 分鐘）

#### 情境設定

接著上一堂（CH2-2），我們為咖啡館老闆寫好的品牌定位是：

> **給加班族一杯像家的咖啡。**

（如果你 CH2-2 沒到，或還沒想好，先用「給 [誰]一個[什麼感受]的[什麼]」這個骨架湊一句就好。）

現在打開 ChatGPT（手機免費版就行，手邊如果已經在 CH2-1 建好咖啡館 Notebook，用 NotebookLM 效果會更好，因為它會讀你上傳的菜單與客戶評論，產出會對得上你的實際產品）。

#### MK-01 Prompt 範本（貼這個）

```
幫我產出一週（7 則）Instagram 貼文骨架，為品牌建立內容節奏。

品牌定位：給加班族一杯像家的咖啡。

目標客戶：下班後想喘 30 分鐘的 25-40 歲上班族，以及週末想找個安靜角落的一人客。

本週主題：新品上市（本週主打：肉桂燕麥拿鐵 $140）

七則貼文節奏：
- 週一：週一激勵或打招呼
- 週二：產品介紹
- 週三：客戶故事
- 週四：幕後花絮
- 週五：週末推薦
- 週六：互動問答
- 週日：品牌理念或一句話

每則格式：
1. 主題（一句話，我要知道這則在講什麼）
2. 開頭吸睛句（第一句要讓人停下滑動，禁用「大家好」「今天要分享」這類廢話開場）
3. 核心訊息（150 字內，要提到具體產品名、價格或場景）
4. 行動呼籲（留言 / 分享 / 點連結 / 私訊 擇一，要具體）
5. Hashtag（5-8 個，大中小混合，加入在地標籤如 #台北咖啡 #松山區）

風格：親切、有家的感覺、像跟老顧客聊天，不要推銷感。

禁用：限時、必買、錯過終生後悔等誇張行銷詞；不要用驚嘆號超過一次。
```

看到這張表，注意幾件事：
1. 每一則的開頭句都是**具體場景**，不是「大家好」。
2. 每一則都**點到品牌定位**（加班族 / 像家 / 30 分鐘）。
3. 有一則直接寫產品（週二），有一則完全不提產品（週日），不是七則都在推銷。
4. 有具體的產品名（肉桂燕麥拿鐵）、具體的價格（$140）、具體的桌號（窗邊 2 號桌）——這是 Prompt 有寫「要提到具體產品名、價格或場景」的結果。

#### 手機操作重點

- 手機上按輸入框右邊的**麥克風**，直接口述品牌定位和本週主題，比打字快 3 倍。
- AI 產出那張表如果太長，在手機上**按產出訊息右下角的分享**，存成 PDF 到檔案；或**長按→複製**到備忘錄，比滑回來找方便。
- 覺得某一則不對味，**直接回它**：「週四那則太普通，重寫一版，開頭要具體到時間和動作」——不用整串重跑。

#### 動手（3 分鐘）

拿出 CH2-2 你寫的品牌定位（如果還沒有，用「我賣 [什麼] 給 [誰]」湊一句）。

打開 ChatGPT，把上面那串 MK-01 Prompt 貼進去，**只換三個東西**：
- 品牌定位 → 你的那句
- 目標客戶 → 你想賣給誰
- 本週主題 → 你下週想主推的東西（新品 / 客戶見證 / 幕後 任選一）

按送出，等 20 秒。

### 動手 / Hands-on（MK-02 A/B 標題，6 分鐘）

#### 為什麼要 A/B 標題

拿到七則骨架之後，你會發現其中一兩則「心裡有感覺但開頭句還不到位」。以咖啡館週三「客戶故事」這則為例，AI 給的是：

> 上禮拜有個阿姐每天同一時間坐同一張桌。

這句還行，但不一定是最想點的版本。**A/B 標題的用意不是跑廣告測試**，是逼自己想過兩個不同角度，挑一個你比較有信心的發出去。

#### MK-02 Prompt 範本（接著上一個對話貼）

```
上面週三那則客戶故事，幫我用兩個不同的切入點，各寫一組標題 + 開頭第一句，共三組（6 句）。

每組要標明：
- 切入點：疑問型 / 衝突型 / 故事型 擇一
- 標題 + 開頭句（標題 15 字內、開頭句 25 字內）
- 這組最可能吸引誰（一句話描述那類讀者）

禁用：驚嘆號、省略號、「你一定要知道」這種老套句。
```

#### 預期輸出（示意）

| 組別 | 切入點 | 標題 | 開頭第一句 | 適合誰 |
|---|---|---|---|---|
| A1 | 故事型 | 窗邊 2 號桌，那位阿姐 | 她每天下午 3 點進來，坐 40 分鐘，不點第二杯。 | 喜歡慢節奏敘事、被故事帶入情緒的客群 |
| A2 | 故事型 | 一張桌子，一個習慣 | 我記得她第一次進來那天在下雨，她說想躲一下。 | 重感性、在意「被記得」這件事的熟客型客群 |
| B1 | 疑問型 | 你有沒有固定坐的位子 | 我們店有個阿姐，她的位子是窗邊 2 號桌。 | 喜歡互動、願意留言的活躍讀者 |
| B2 | 疑問型 | 為什麼她都坐同一張桌 | 因為「這裡的燈不會刺眼」——她這樣跟我說。 | 好奇心驅動、想看答案的路人型讀者 |
| C1 | 衝突型 | 我們不是網美店 | 我們是「讓你放心發呆 20 分鐘」的店。 | 對網美店反感、尋找安靜角落的上班族 |
| C2 | 衝突型 | 最好的桌子不一定是最美的 | 是那張燈剛好不刺眼、讓你不想離開的那張。 | 重實用感、不愛打卡風的熟齡客群 |

#### 怎麼選

不是哪組比較「漂亮」，是問自己兩件事：
1. **哪一句你唸出來不會尷尬？** 如果這句貼文下面要留你的名字，你會不會想改掉。
2. **哪一句最像「加班族」會被勾到？** 回頭對品牌定位，不是對你自己喜歡。

以這個例子來說，如果目標客群是加班族，C1「我們不是網美店」可能最勾——因為加班族對網美店本來就有點倦，看到「不是」會停一下。B1「你有沒有固定坐的位子」也不錯，適合當週希望多互動。

**挑一組就好。挑了就用。**其他五組存著下次用。

#### 動手（2 分鐘）

從你剛才 MK-01 產出的 7 則中，挑一則你最「有感覺但還不到位」的，把上面 MK-02 Prompt 貼進去，把「週三那則客戶故事」換成你那則的**主題+核心訊息**。

看產出的三組，唸出來，圈一組。

### 延伸 / Short Video Script（MK-03 短影音腳本，7 分鐘）

#### 為什麼要改成短影音

七則貼文發完，其中**一則可以同時拍成 30 秒短影音**，當天發到 Reels / Shorts。拍一次、發兩個管道，邊際成本最低。通常適合拍的是：有畫面感的那則（產品 / 幕後 / 客戶故事）。

以咖啡館本週為例，**週五「週末推薦」這則**適合拍——有產品特寫、有組合優惠、有窗邊座位畫面。

#### MK-03 Prompt 範本（接著貼）

```
把上面週五那則「週末推薦」改寫成 30 秒短影音腳本，可以在 Reels / Shorts 上發。

產品：肉桂燕麥拿鐵 + 司康 週末組合 $180
目標客群：下班後想犒賞自己、週末想獨處的 25-40 歲上班族
影片目的：導購（希望看完的人週末會來店裡點這個組合）
拍攝場景：咖啡館吧台 + 窗邊座位 + 產品特寫（我只有手機一支，沒有攝影團隊）

腳本要求：
- 第一句必須在 3 秒內抓住注意力（具體動作或反差，不要「嗨大家好」）
- 總長度約 80-100 字（30 秒念得完）
- 分 4-5 個鏡頭（shot），每個鏡頭 5-7 秒
- 最後 3 秒要有明確 CTA
- 每個鏡頭要寫：【時間】畫面描述 + 口白

風格：口語、像自言自語，不要廣告感。

禁用：「快來買」「限時優惠」「錯過就沒有」這種推銷詞。
```

#### 預期輸出（示意：30 秒腳本 + shot list）

```
【0-3s】畫面：手沖壺倒下熱水，特寫蒸氣升起。
口白：「禮拜五下午最適合做兩件事。」

【3-10s】畫面：鏡頭從吧台平移到窗邊空桌，桌上放好一杯拿鐵和一塊司康。
口白：「翹班。喝咖啡。我們這週幫你把兩件一起備好了。」

【10-20s】畫面：司康被掰開特寫，奶油流出來；切到拿鐵表面肉桂粉。
口白：「肉桂燕麥拿鐵加司康 $180，原價 $200。窗邊 2 號桌，我永遠幫你留著。」

【20-27s】畫面：回到吧台，老闆（不露臉，只拍手）把一張小卡放上桌面，寫著「給加班族的 30 分鐘」。
口白：「我們不賣最好的咖啡。我們賣你禮拜五下午的 30 分鐘。」

【27-30s】畫面：門口招牌 + 店外街景。
口白：「私訊『週末』，幫你預留位子。」
```

#### 怎麼看這個腳本

- **第一句 0-3 秒**是整支影片的存亡關鍵。「禮拜五下午最適合做兩件事」是個**鉤子**（開頭吸睛句）——觀眾想知道是哪兩件事，就會多停 1 秒。多停這 1 秒，後面才有機會。
- **5 個鏡頭**，每個 5-7 秒，剛好 30 秒。這個結構讓你「知道要拍什麼」——不是拿手機隨便錄一段。
- **shot list** 就是畫面描述那一欄，拍之前先把這五個鏡頭的**靜態照片**構圖好，錄的時候照著拍，不用邊拍邊想。
- **結尾 CTA**「私訊『週末』，幫你預留位子」是具體到不行的動作——比「快來喝」強一萬倍。

#### 手機操作重點

- 30 秒腳本產出後，**手機上按分享**存成備忘錄；拍攝當下對著備忘錄念即可，不用背稿。
- 拍攝用手機**直立拍（9:16）**，Reels / Shorts / TikTok 都吃這個比例。
- 口白不用一次錄完，**一個鏡頭一個鏡頭錄**，剪接時拼起來。免費剪接 App（CapCut）直接拼。

#### 動手（2 分鐘）

從你 MK-01 產出的 7 則中，挑**一則最有畫面感**的（通常是產品 / 幕後 / 客戶故事），把 MK-03 Prompt 貼進去，把咖啡館的設定換成你的。

看產出的腳本——如果可以，週末真的拍一支試試。不行的話，存著週日晚上再說。

### 檢核 / Verification

收尾 15 秒，自己檢查手邊有沒有這三件東西：

- [ ] **一張七則骨架表**（MK-01 產出）：有七天、每天四件套（主題 / 開頭句 / 核心訊息 / CTA）都不空，且其中有具體產品名/價格/場景（不是通用的「歡迎來喝咖啡」）
- [ ] **一則貼文的 A/B 標題**（MK-02 產出）：至少兩組切入點明顯不同（不是改幾個字），已圈出你會用的那組
- [ ] **一支短影音腳本**（MK-03 產出）：有 4-5 個鏡頭、第一句 3 秒內有鉤子、結尾有具體 CTA、你看著腳本能想像拍出來長什麼樣

這三件東西就是你下週社群排程的原料。週一早上你要做的只剩：看骨架 → 拍照 → 貼上去。

### 卡關備案（Plan B）

- **AI 產出的七則都長得很像，都是「好天氣來喝咖啡吧」這種通用句**
  → 回它一句：「這七則太通用，每則都不像我的店。重寫一次，每則都要具體到**我的產品名 / 我的街區 / 我的某個真實情境**」。通常第二輪會好很多。
- **A/B 標題產出兩組切入點看起來一樣**
  → 回它：「這兩組都是疑問型，幫我換成：一組故事型（像小說開頭）、一組衝突型（打破常見認知）」。強制它換角度。
- **短影音腳本不會拍 / 不知道畫面怎麼弄**
  → 把腳本截圖給 ChatGPT，回它：「我只有手機一支、沒有攝影助手、不會剪接。請把每個鏡頭改寫成『**站在哪裡 + 手機擺哪 + 拍什麼東西**』的拍攝指引，像在跟新手店員講話」。它會給你可執行的指示。
- **免費版配額用完（今天 ChatGPT 已經用很多次）**
  → 換 Gemini 繼續跑同一組 Prompt；或今天先做 MK-01，明天再做 MK-02 / MK-03。這三則 Prompt 本來就不必同一次產完，拆兩天做也可以。
- **NotebookLM 讀不到你的菜單檔**
  → 檢查檔案格式是不是 PDF / Google Docs / 貼上純文字；如果是 Excel 或圖片要先轉成 PDF。或者退回 ChatGPT，直接在 Prompt 裡把菜單前 5 樣產品 + 價格列出來。

## 常見錯誤 3 條（Common Pitfalls）

1. **錯誤現象**：MK-01 產出七則骨架，每則都是「陽光正好、來一杯咖啡吧」、「週末放鬆、歡迎光臨」這種通用句，換任何一家咖啡館都適用。
   **原因**：Prompt 沒給**具體輸入**——沒有具體產品名、沒有具體客戶情境、沒有本週主題；AI 只好用罐頭模板。有時候是品牌定位欄位填太抽象（「給客戶最棒的體驗」），AI 也會反射性給你最抽象的文案。
   **解法**：回去把 Prompt 的三個欄位補具體——品牌定位要有「誰 / 什麼感受 / 什麼東西」三件、目標客戶要有年齡和情境、本週主題要有具體產品或事件（例：「新品：肉桂燕麥拿鐵 $140」而不是「新品上市」）。然後在 Prompt 最後加一句「**重寫，每則都要具體到我的產品名和真實情境**」。通常第二輪會好很多。

2. **錯誤現象**：MK-02 產出三組 A/B 標題，三組唸起來都是同一個調性（「你知道 X 嗎 / 你會 X 嗎 / 你有 X 嗎」全是問句），沒有 A/B 的意義。
   **原因**：Prompt 沒強制 AI 切換切入點類型，它偷懶給你三個同類型的變體。或者內文本身太單薄，AI 無從切換角度。
   **解法**：在 Prompt 明確指定三組的切入點各不一樣——例如指定「**一組疑問型、一組衝突型、一組故事型**」，用分類強制它換角度。如果內文太單薄，先在 MK-02 前把那一則的核心訊息擴寫成 100 字以上再丟給它切標題。

3. **錯誤現象**：MK-03 產出 30 秒腳本，第一句是「嗨大家好，我是 XX 咖啡館的老闆，今天要跟大家介紹我們的新品」——這句用掉 8 秒，觀眾早在第 2 秒就滑走了。腳本沒有鉤子（開頭吸睛句）。
   **原因**：AI 的預設反射是「先自我介紹再進入主題」，這是長影片的邏輯，放到短影音完全不對。Prompt 如果沒特別強調「第一句 3 秒內抓住注意力」，或沒給具體範例，AI 就會套長影片模板。
   **解法**：Prompt 明確寫「第一句必須在 3 秒內抓住注意力，**禁用『嗨大家好』『我是 XX』『今天要介紹』**」，最好再給一句範例（例：「像『禮拜五下午最適合做兩件事』這種具體場景句」）。產出後如果還是看到「嗨大家好」，直接回它：「第一句砍掉重寫，用一個具體動作或反差句代替」。

## 檢核題 2 條（Quiz）

**Q1（概念驗證）**：MK-01 產出一週七則貼文骨架，每則的「四件套」是哪四件？

- [ ] A. 標題 / 封面圖 / 內文 / Hashtag
- [ ] B. 主題 / 開頭句（鉤子）/ 核心訊息 / CTA（行動呼籲） ←（正確答案）
- [ ] C. 產品 / 價格 / 優惠 / 連結
- [ ] D. 文案 / 排版 / 顏色 / 字體

> 解析：骨架聚焦在**文字邏輯**，不處理視覺（封面/排版那是發文當天才補的事）。四件套是：這則在講什麼（主題）/ 怎麼讓人停下滑動（鉤子）/ 想讓人接收什麼（核心訊息）/ 看完要做什麼（CTA）。C 混進了產品細節（是核心訊息的一部分，不是四件套層級）；D 是視覺設計範疇。

**Q2（應用驗證）**：你用 MK-01 跑完手工皂的一週七則，產出看起來都是「今天天氣好適合洗澡」「週末記得放鬆」這類任何店都能發的通用句，沒有你家的產品影子。下一步你該怎麼做？（選出最有效的兩個動作）

- 預期答案要點：
  1. **補具體輸入**：回去檢查 Prompt 的三個欄位——品牌定位有沒有具體到「誰 / 什麼感受 / 什麼產品」（不是「給客戶最好的皂」）；目標客戶有沒有具體年齡和情境（不是「女性」）；本週主題有沒有具體產品名 + 價格（例：「玫瑰手工皂 $250 新上架」，不是「新品介紹」）。
  2. **下指令重寫**：在同一對話追加一句：「這七則太通用，每則都不像我的店。重寫一次，每則都要具體到**我的產品名 / 我的真實情境 / 我家的味道或質地**」。AI 第二輪通常會明顯貼近。
  3. （可選加分）改用 NotebookLM：如果已上傳過產品表 + 客戶評論到 Notebook，改用 NotebookLM 跑 MK-01，它會直接引用你的實際產品名和評論金句，比 ChatGPT 憑空寫貼得多。
