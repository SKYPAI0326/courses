# 10 業種主題庫｜完整 brief + prompt + 建議骨架

> 本檔為 `gen-image` 課程的 PRAC2 / PRAC3 / PRAC4 核心教材。每業種含一份假想品牌 + 完整 brief + 可直接丟 AI 的 prompt + 建議骨架。
>
> **使用原則**：
> 1. 學員從 10 業種中挑 2-3 個最接近自己工作的跑，不必全跑
> 2. 所有假想品牌均為虛構，不得引用真實同名品牌
> 3. 骨架代號對應 `l2-skeleton-library.md`
>
> **🎯 預期差異聲明（重要）**：本課**不備固定示範圖**。學員跑本檔 prompt 時，每次輸出都會不同——同學之間、同一人不同時間、不同平台都會有差異。這不是 bug，是 AI 特性。**教學重點是結構判讀與 prompt 結構能力，不是複製某張「標準圖」**。每個 prompt 都附有「**結構檢核清單**」，跑完對照此清單、只要 ≥ 70% 命中就是成功。

## 跨風格演練｜風格 preset 選項清單

本課**日系為教學基準**（CH 示範段），**其他風格為演練延伸**（PRAC 段落學員自選）。每種風格都可以套任一骨架——**骨架是結構、風格是表象**，兩者正交。

### 風格 preset 對照表

| 風格 preset | 配色傾向 | 字體傾向 | 質感關鍵字 | 適合業種 |
|---|---|---|---|---|
| **日系**（預設）| 奶油白 / 鼠尾草 / 奶茶棕（柔和低飽和）| 明朝體 + 黑體 | 職人、季節感、留白 | 餐飲、美妝、手作、旅遊 |
| **韓系** | 粉色系 / 莫蘭迪灰（柔和高級）| 細黑體 + 英文襯線 | 精緻、質感、簡約 | 美妝、服飾、咖啡、甜點 |
| **美式復古** | 芥末黃 / 磚紅 / 墨綠（土色系）| 粗黑襯線 + 手寫字 | 懷舊、手作、溫度 | 咖啡、市集、手作、餐廳 |
| **矽谷科技** | 深藍 / 白 / 螢光點綴 | 無襯線 Sans + mono | 未來、效率、資料 | App、SaaS、B2B 科技 |
| **北歐極簡** | 米白 / 黑 / 一抹亮色 | 細黑體 | 冷靜、留白、秩序 | 居家、家飾、SaaS |
| **歐陸 / 法式** | 奶油黃 / 古董玫瑰 / 墨綠 | 襯線體（古典 Didot）| 優雅、古董、精品 | 奢侈品、香氛、家居 |
| **美式海報** | 紅 / 白 / 藍高飽和 | 粗大黑體 | 衝擊力、直給、量感 | 快時尚、體育、美食 |
| **瑞士排版** | 黑 / 白 / 一種主色 | 無襯線（Helvetica）| 幾何、網格、紀律 | 設計、展覽、文化 |
| **手繪童趣** | 馬卡龍色 / 高對比 | 手寫感字體 | 溫暖、童心、親和 | 寵物、兒童、甜點 |
| **工業風** | 灰黑 / 鐵鏽色 / 水泥灰 | 粗無襯線 | 粗獷、機械、硬派 | 健身、五金、音響 |
| **街頭潮流** | 螢光綠 / 紫 / 黑 | 變形字 + 塗鴉 | 叛逆、街拍、對比 | 潮牌、滑板、夜店 |

### 風格 preset 使用規則

1. **同 brief + 不同 preset = 不同成品**｜10 業種庫的每個 brief 都可以套不同 preset 跑多張，體驗「結構不變、表象切換」
2. **同 preset + 不同骨架 = 同品牌不同物料**｜同一品牌（例咖啡店）用日系 preset 可以套骨架 4（商品廣告）、骨架 5（活動招生），保持品牌一致
3. **不同 preset 同骨架 = 跨業種體驗**｜骨架 1 商品左，用日系套美妝、換韓系套保健品、換美式復古套伴手禮——每種都走得通
4. **學員 PRAC 段可自選任一 preset**，無強制日系

## 使用指南（給 course-designer）

### PRAC2 設計（多主題生圖演練）

- 學員挑 **2-3 業種**，每業種跑 **1 張 L4 完稿**
- 產出要求：
  - [ ] 版型正確（對應骨架代號）
  - [ ] 字階三層分明
  - [ ] 配色 ≤ 3 色
  - [ ] 必要元素齊全
- 可 Plan B：若 AI 無法正確處理日文，改用英文 prompt 或改中文後 Canva 補字

### PRAC3 設計（線稿反推演練）

- 學員挑 **2-3 張 PRAC2 完稿** 或 **找 3 張網路商業作品** 跑 L2 反推
- 產出要求：
  - [ ] 5 張 L2 線稿
  - [ ] 每張 100 字結構筆記（視覺重心 / 焦點層級 / 對齊網格）

### PRAC4 設計（品牌系列演練）

- 學員選 **1 個業種**，建立 brand-dna.md
- 用同骨架產 **3 張系列圖**（可以是春夏秋冬任 3 季，或不同產品線）
- 產出要求：
  - [ ] 3 張 L4 完稿
  - [ ] 1 份 brand-dna.md
  - [ ] 視覺一致度可被第三方辨識為同品牌（自評 + 互評）
