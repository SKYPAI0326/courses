# 8 版型 × 通用 prompt 範例集｜學員主用速查

> **這是課程的 prompt 彈藥庫**——學員不必自己想 prompt 怎麼寫、看到任務就翻這裡、複製對應版型的範例、改 6-10 個變數、5 分鐘跑出符合該版型的圖。
>
> **通用原則**：本檔範例**不限業種**——每個版型都有跨業種的 prompt 樣本（咖啡 / 美妝 / 手工皂 / App / 旅遊 / 課程 / 保健 / 寵物⋯⋯）、體現「同 1 個版型可以套到任何商品」。
>
> **配套**：本檔需與 `8-version-library.md`（版型結構認知）+ `prompt-templates.md`（基底模板）+ `10-industries-brief-library.md`（業種 brief 庫）合用。

---

## 怎麼用這個彈藥庫

### 3 步使用流程

**Step 1｜對照「目的 → 版型」表**（見 `8-version-library.md` § 速查總表）、選 1 個版型代號（A / B / C / D / E / F / G / H）

**Step 2｜翻到本檔對應版型段、複製「快速啟動 prompt」**（已含 6-10 個 `{變數空格}`）

**Step 3｜用你的 brief 六件套替換變數空格**（業種 / 商品 / 客群 / 訴求 / 必要元素 / 風格 preset）→ 丟 AI

→ **3-5 分鐘拿到符合該版型結構的 L4 完稿**

### 變數空格約定

| 符號 | 意思 | 填法範例 |
|---|---|---|
| `{變數}` | 必填、空著 AI 會猜 | `{商品}` → `春季抹茶布朗尼` |
| `{變數｜選項 A / 選項 B}` | 必填、給選項 | `{季節｜春 / 夏 / 秋 / 冬}` |
| `{變數＋預設}` | 可不填、有預設值 | `{尺寸＋1:1}` |

## A｜主視覺版 prompt

### 用途

抓住注意力、停留 3 秒、IG 廣告 / 廣告首圖 / 商品頁 hero。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 1:1 商業主視覺、A 版型「主視覺版」風格：

【業種】{業種，如：咖啡店 / 美妝 / 手工皂 / App / 旅遊}
【商品】{商品名 + 規格，如：春季限定抹茶拿鐵 Tall ¥730}
【客群】{年齡層 + 性別 + 風格，如：25-35 歲女性、注重生活風格}
【訴求】{3 個調性詞，如：手作、季節感、療癒}

【必要元素｜A 版型結構】
- 主角（商品）占畫面 80-90%、置中或偏上 1/3
- 文字僅占 10-20%、置於下方或角落
- 留白最少、視覺衝擊最強
- 黃金視覺位置（中上偏左 1/3）放亮點 / 流心 / 爆點
- 文字內容：{主標 1 行不超過 12 字，如：春のはじまり}

【風格 preset】
- 配色 ≤ 3 色：{配色組合，如：鼠尾草綠 + 奶油白 + 抹茶綠}
- 字體：{主標字體 + 內文字體，如：Shippori Mincho + Noto Sans JP}
- 視覺調性：{3 個關鍵詞，如：手作、季節感、職人}

【尺寸】1:1
【禁用】漸層、純黑、霓虹色、過多文字
```

### 跨業種範例（A 版型 × 5 業種）

| 業種 | 商品代換 | 主標代換 | 配色代換 |
|---|---|---|---|
| 咖啡店 | 春季抹茶拿鐵 ¥730 | 春のはじまり | 鼠尾草綠 + 奶油白 + 抹茶綠 |
| 美妝 | LUNÉA 精華液 30mL | Glow from within | 米白 + 莫蘭迪粉 + 玫瑰金 |
| 手工皂 | HANAKO 玫瑰皂 80g | 手のひらに花を | 米粉 + 米白 + 深棕 |
| AI 筆記 App | NoteFlow Pro 月費 | Think clearer. | 深藍 + 純白 + 電光藍 |
| 旅遊地方創生 | 花蓮石梯坪二日遊 | 一望無際的太平洋 | 海軍藍 + 沙白 + 珊瑚橘 |

## B｜橫幅版 prompt

### 用途

1 秒抓眼球、快速傳重點、網站 banner / EDM 標題 / 信件 hero。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 16:9 橫式商業 banner、B 版型「橫幅版」風格：

【業種】{業種}
【商品】{商品名 + 規格}
【客群】{年齡層 + 性別}
【訴求】{3 個調性詞}

【必要元素｜B 版型結構】
- 左 50%：商品主視覺（實體商品或情境圖）、占畫面高度 70%+
- 右 50%：文字區
  - 主標 1 行（粗體、不超過 14 字）：{主標}
  - 短標語 1 行（細體、不超過 20 字）：{短標語}
  - CTA 按鈕（如有）：{CTA 文字＋立即了解}
- 主次分明、主標與商品並列強化記憶

【風格 preset】
- 配色 ≤ 3 色：{配色組合}
- 字體：{主標字體 + 內文字體}
- 視覺調性：{3 個關鍵詞}

【尺寸】16:9 或 1200×400 horizontal banner
【禁用】文字塞滿、商品過小、漸層
```

### 跨業種範例（B 版型 × 5 業種）

| 業種 | 商品 | 主標 | 短標語 |

| 旅遊地方創生 | 花蓮 2 日遊 | 山海間的慢時光 | 假日限定、4 月開團 |
| 寵物用品 | PUPPY 凍乾零食 | 主子也愛的點心 | 凍乾鎖鮮、無添加 |

## C｜社群版 prompt

### 用途

少字強視覺、輕量易讀、IG 貼文 / Threads / 日常曝光。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 1:1 IG 貼文、C 版型「社群版」風格：

【業種】{業種}
【商品】{商品名}
【客群】{年齡層 + 性別 + 平台習慣，如：25-30 歲女性、IG 重度用戶}
【訴求】{3 個調性詞，重視「呼吸感」}

【必要元素｜C 版型結構】
- 主角（商品）占畫面 70%+、置中
- 大量留白（背景簡潔、單色或淺色漸進）
- 文字 < 30%、置於下方或側邊
  - 品牌名：{品牌}
  - 商品名：{商品名}
  - （可選）一句 catchphrase：{catchphrase}
- 流心 / 主體引導視線到視覺重點

【風格 preset】
- 配色 ≤ 3 色：{配色組合，建議低飽和、淺色為主}
- 字體：{細黑體 + 英文襯線}
- 視覺調性：{留白、質感、輕量、日常}

【尺寸】1:1（1080×1080）
【禁用】塞滿元素、強烈促銷文字、急迫感
```

### 跨業種範例（C 版型 × 5 業種）

| 業種 | 商品 | 留白氛圍 | catchphrase |

| 手工皂 | 單顆皂 + 麻繩 | 米粉布面 + 留白 | hand-made, hand-felt |
| AI 筆記 App | App icon + Mac | 米灰桌面 + 一杯咖啡 | think less, do more |
| 服飾配件 | 帆布包單品 | 淺木桌 + 一支花 | carry your day |

## D｜系列版 prompt

### 用途

完整呈現多款商品、讓客戶比較選擇、商品頁 / 限動 / Lookbook。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 1:1 商品系列圖、D 版型「系列版」風格、3 款並列：

【業種】{業種}
【系列名】{系列名，如：春季限定 3 款 / Mother's Day 禮盒組}
【3 款商品】
- 款 1：{品名 + 1 句特色}
- 款 2：{品名 + 1 句特色}
- 款 3：{品名 + 1 句特色}
【客群】{年齡層 + 性別 + 購買場景}

【必要元素｜D 版型結構】
- 上半 70%：3 款商品橫向並列、構圖節奏一致
- 下半 30%：每款下方 1 行特色 + 1 個信任標籤
- 整體視覺：色調一致、光線一致、留白一致
- 加上「特色標籤」（如：嚴選食材 / 無添加 / 手工製作）拉信任

【風格 preset】
- 配色 ≤ 3 色：{統一系列色調}
- 字體：{主標字體 + 內文字體}
- 視覺調性：{系列感、整齊、專業}

【尺寸】1:1
【禁用】3 款風格不一、構圖不對齊、色調差異過大
```

## E｜禮盒版 prompt

### 用途

質感包裝 × 精緻呈現、拉高單價、節慶禮盒 / 送禮市場 / 高價策略。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 1:1 禮盒形象圖、E 版型「禮盒版」風格：

【業種】{業種}
【商品】{禮盒名 + 內容物 + 價位，如：HANAKO 母親節禮盒 3 入 ¥2,400}
【節慶 / 場合】{母親節 / 中秋 / 聖誕 / 婚禮 / 企業送禮}
【客群】{送禮者畫像，如：25-40 歲女性、送媽媽}

【必要元素｜E 版型結構】
- 主角：禮盒外包裝（含緞帶、紋飾、標籤）、占畫面 70%+
- 拍攝角度：略低、產生「商品仰望感」
- 背景：深色 / 絨布感 / 大理石、襯托質感
- 金色 / 燙金元素：精品感（標籤 / 文字 / 細節）
- 文字：品牌名 + 系列名（簡潔、不搶戲）

【風格 preset】
- 配色 ≤ 3 色：{深色主調 + 金色 + 點綴色，如：深酒紅 + 金 + 米白}
- 字體：{襯線體優雅、避免無襯線粗黑}
- 視覺調性：{精品、質感、儀式感、送禮}

【尺寸】1:1
【禁用】明亮的促銷感、大字優惠、雜亂背景
```

### 跨業種範例（E 版型 × 4 節慶）

| 節慶 | 禮盒商品 | 主色調 | 附加元素 |
|---|---|---|---|
| 母親節 | 手工皂 3 入禮盒 | 米粉 + 金 + 深棕 | 緞帶 + 乾燥花 |
| 中秋 | 烘焙月餅 6 入 | 酒紅 + 金 + 米白 | 燙金紋飾 + 木盒 |
| 聖誕 | 美妝旅行組 | 墨綠 + 金 + 紅 | 雪松枝 + 緞帶 |
| 企業送禮 | 茶葉禮盒 | 深棕 + 金 + 米白 | 燙金 logo + 木紋 |

## F｜特寫版 prompt

### 用途

極致特寫 × 感官刺激、提升食慾、社群貼文 / 美食平台 / 餐廳菜單。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 1:1 商品極致特寫、F 版型「特寫版」風格：

【業種】{業種，建議：餐飲 / 烘焙 / 美妝 / 飾品}
【商品】{商品名 + 質感重點，如：流心抹茶巧克力布朗尼}
【感官重點】{要強化什麼，如：流心 / 油光 / 結露 / 紋路}

【必要元素｜F 版型結構】
- 商品占畫面 95%+、極致特寫（畫面填滿質感）
- 拍攝角度：45° 俯角或正面平視（依商品調整）
- 光線：側光或頂光、強化質感反光
- 動態瞬間：{動態描述，如：巧克力流心爆漿瞬間 / 起司拉絲 / 咖啡蒸氣}
- 文字：零文字或極少品名（占 < 5%）

【風格 preset】
- 配色：{依商品本色、不額外加裝飾色}
- 視覺調性：{食慾、誘人、質感、近距離}
- 鏡頭感：macro 微距、淺景深、背景模糊

【尺寸】1:1 或 4:5
【禁用】文字過多、構圖過小、商品比例 < 80%

## G｜促銷版 prompt

### 用途

強烈訊息 × 行動導向、快速轉單、檔期活動 / 廣告投放。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 1:1 促銷廣告、G 版型「促銷版」風格：

【業種】{業種}
【商品】{商品名}
【優惠細節】
- 主優惠：{優惠主訊息，如：2 盒 85 折 / 全館滿千折百}
- 限時：{時間範圍，如：4/25-4/30 / 最後 3 天}
- 限量：{數量限制，如：每人限 2 件 / 限量 100 組}
【行動指令】{CTA 文字，如：立即下單 / 私訊預訂}

【必要元素｜G 版型結構】
- 上半 60%：大字優惠訊息（**最重要、占畫面最強**）
  - 主訊息（特大、粗體）：{大字優惠}
  - 副訊息（中字）：{限時 / 限量}

【風格 preset】
- 配色：{對比色組合，如：黑底 + 黃 + 紅}
- 字體：{粗黑體、強烈視覺}
- 視覺調性：{急迫、強烈、行動導向}

【尺寸】1:1 或 9:16（IG 限動）
【禁用】留白、柔和配色、優美文案（這是促銷不是品牌片）

## H｜生活版 prompt

### 用途

情境氛圍 × 情感共鳴、拉近距離、品牌日常 / 生活提案 / 內容行銷。

### 快速啟動 prompt（複製這段、改變數）

```
請產出 1:1 生活情境圖、H 版型「生活版」風格：

【業種】{業種}
【商品】{商品名}
【情境】{時間 + 場景 + 行為，如：午後 + 書桌前 + 一杯咖啡 + 翻書}
【情緒】{要傳遞的感受，如：療癒 / 慢時光 / 與自己對話}
【客群】{客群畫像 + 生活風格}

【必要元素｜H 版型結構】
- 商品融入生活場景、占畫面 30-40%（不是主角、是配角）
- 加入「人的痕跡」：{選項，如：咖啡杯 / 翻開的書 / 毛毯一角 / 鑰匙 / 眼鏡}
- 自然光、不用棚拍（窗邊光 / 戶外光最佳）
- 情境文案（手寫感、像書信、避免命令句）：
  - 主文案：{1-2 行情感文，如：午後時光、來塊療癒的布朗尼}
  - 落款（可選）：{如：— 弄一下咖啡}
- 構圖：商品 + 1-2 件生活物件、有「人剛離開」的氛圍

【風格 preset】
- 配色 ≤ 3 色：{自然色系，如：米白 + 木色 + 微綠}
- 字體：{襯線 / 手寫感、避免粗黑}
- 視覺調性：{溫柔、慢、生活感、自然光}

【尺寸】1:1 或 4:5
【禁用】商品過大占主角、強促銷文字、棚拍燈感
```

### 跨業種範例（H 版型 × 5 場景）

| 業種 | 商品 | 情境 | 情境文案 |

## 7 步學員實作路徑｜對應範本指引

> 本課程的 7 步路徑、每步都有對應的 prompt 範例與用法。學員可從任一步起跳、不必照順序。

### Step 1｜素材 / 需求 → 精準 prompt

**情境**：你手上有 1 段需求描述（老闆口述 / 客戶 LINE 訊息 / 自己想做的東西）、需要轉成可丟 AI 的 prompt。

**轉換流程**：

```
原始需求（任意一段話）→
1. 抓出業種 / 商品 / 客群 / 訴求 4 件
2. 對照「目的 → 版型」表選 1 個版型
3. 翻本檔對應版型段、複製快速啟動 prompt
4. 把 4 件套進變數空格、補必要元素 + 風格 preset
5. 丟 AI
```

**範例**（任意需求轉 prompt）：

```
原始需求：「老闆說要做一張春季新品的 IG 廣告、抹茶拿鐵新品、年輕女性看的、要好看一點」

抓 4 件：
- 業種：咖啡店
- 商品：春季抹茶拿鐵
- 客群：25-35 歲女性、IG 重度用戶
- 訴求：手作、季節感、療癒（自己補）

選版型：IG 廣告 + 抓注意力 → A 主視覺版

複製 § A 段 prompt → 替換 5 個變數空格 → 跑
```

### Step 2｜prompt → 商業用圖片廣告

**情境**：你已有 prompt、要把它跑成可用的廣告圖。

**動作**：直接複製本檔對應版型段的 prompt、丟 Gemini 或 ChatGPT、20-30 秒拿到圖。

**驗收**：對照該版型結構 3 件事（主視覺占比 / 文字密度 / 整體感覺）、3 件都對 → 收。

### Step 3｜提示詞優化 → 切換 8 版型

**情境**：你有 1 個 prompt 跑出 1 張圖、但要再跑 7 張不同版型。

**動作**：保留 brief 六件套不變、**只換版型結構段**（從 A 換成 B 換成 G⋯⋯）。

**範例**：
- 原 prompt = § A 段（主視覺版）→ 同個商品換成 § G 段（促銷版）
- 同 1 張產品圖 + 不同版型 prompt = 8 張不同廣告

### Step 4｜廣告 → wireframe（圖片轉骨架）

**情境**：你拿到 1 張喜歡的廣告（自己跑的 / 同行的 / 客戶給的）、想把它的「結構」抽出來、之後套到別的商品上。

**動作**（CH3-1 教法）：丟圖給 AI、用以下 prompt：

```
請把這張商業廣告轉換成 L2 線稿 wireframe：
- 只保留構圖框架（哪裡放主角、哪裡放文字、哪裡留白）
- 不保留具體商品 / 品牌 / 配色
- 用黑線白底、註明每個區塊的功能（如：主視覺區 / 主標 / CTA）
- 標出比例（如：主視覺 60% / 文字 30% / 留白 10%）
```

**輸出**：1 張可繼承的線稿、之後可套任何商品。

### Step 5｜wireframe → 跨主題套用（設計繼承）

**情境**：你有 1 張喜歡的 wireframe、想拿去做別的商品（換業種、換客群、換場合）。

**動作**：

```
請依以下 wireframe 結構、產出新商業圖：

【wireframe 結構】（貼上 Step 4 拿到的線稿描述）

【新主題】
- 業種：{新業種，如：從咖啡店換成手工皂}
- 商品：{新商品，如：玫瑰手工皂}
- 客群：{新客群}
- 訴求：{新訴求}
- 風格 preset：{新風格}

請保留 wireframe 結構、替換所有商品 / 配色 / 文字內容。
```

**輸出**：結構繼承、主題替換的新廣告。**1 張好 wireframe 可以套 N 個商品**。

### Step 6｜現有商業圖 → wireframe（反推）

**情境**：你看到一張別人做得很好的廣告、想學它的結構。

**動作**：同 Step 4——把那張圖丟 AI、轉成 wireframe。**這就是「站在巨人肩膀上」**——別人花了大錢請設計師、你 30 秒抽出結構自己用。

### Step 7｜大量提示詞範例演練

**情境**：你想累積實戰手感、不知道從哪開始。

**演練菜單**（任選、由淺入深）：

| 演練 | 動作 | 預計時間 |
|---|---|---|
| **演練 1** | 同 1 個商品（自選）× 跑 A / B / C / G 4 個版型 | 30 min |
| **演練 2** | 跨 3 業種（咖啡 / 美妝 / 手工皂）× 同 1 個版型（如 A）| 20 min |
| **演練 3** | 從本檔挑 5 個跨業種範例、各跑 1 張、體驗範本通用性 | 30 min |
| **演練 4** | Step 4-5 反推：找 1 張喜歡的廣告、抽 wireframe、套自己商品 | 40 min |
| **演練 5** | 整合：對 1 個自選商品、做完整 8 版型行銷漏斗（A → H 全跑）| 90 min |

## 變數空格填寫指南

### 6 個基底變數的填法

| 變數 | 必含元素 | 好範例 | 差範例 |
|---|---|---|---|
| `{業種}` | 大類、第三人能對應 | 咖啡店 / 美妝保養 / 線上課程 | 服務業 / 一般商品 |
| `{商品}` | 名 + 規格 + 型態 | 春季抹茶拿鐵 Tall ¥730 | 我的新商品 / 主打 |
| `{客群}` | 年齡 + 性別 + 風格 / 痛點 | 25-35 歲女性、IG 重度用戶 | 一般大眾 / 有錢人 |
| `{訴求}` | 3-5 個調性詞、不是功能 | 手作、季節感、療癒 | 好用、划算、多功能 |
| `{必要元素}` | 對應版型結構、每元素有位置 | （照本檔每段範本） | 「就一張好看的」 |
| `{風格 preset}` | 配色 + 字體 + 調性 3 件 | 鼠尾草綠 + 奶油白；明朝體 + Sans JP；手作職人 | 極簡、日系（標籤不是指令）|

## 給 course-designer 的提醒

1. 本檔是 CH2-1 / CH2-2 學員的「**prompt 彈藥庫**」——學員在做段不必自己想 prompt、複製本檔對應版型 → 改變數 → 跑
2. CH2-1 § 做 / Hands-on 階段 3 應該改為「**從本檔複製對應版型範本 → 替換變數 → 丟 AI**」、不要學員從零想 prompt
3. CH2-2 brief 六件套的「**必要元素**」欄、應該對應 8 版型結構（從 A 段到 H 段查）、不是 5 骨架
4. PRAC2 / PRAC3 / PRAC4 演練可直接從本檔挑跨業種範例、體驗通用性
5. CH3-1 / CH4-1 / CH4-2（wireframe 段）可承接 Step 4-6 的反推流程
6. 本檔範例**不綁布朗尼 / 不綁咖啡店**——是跨業種通用範本、體現「8 版型 × 任何業種 = 任何廣告」
