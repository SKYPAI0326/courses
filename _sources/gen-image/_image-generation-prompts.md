# gen-image 課程示意圖｜Nano Banana 生圖 prompt 清單

> **使用流程**：
> 1. 把每段 prompt 複製貼到 **Nano Banana**（Gemini 2.5 Flash Image / Nano Banana Pro 任一）
> 2. 拿到圖後、用「**該段標註的檔名**」存檔（**檔名要完全一致、含副檔名**）
> 3. 把圖丟進資料夾 → `courses/gen-image/img/`（已建立）
> 4. 全部生完、回我「圖片已放好」、我會把 `<img>` 標籤接進對應 HTML 頁

> **資料夾路徑**（複製即用）：
> `/Users/paichenwei/Library/Mobile Documents/com~apple~CloudDocs/01-PROJECTS/課程專用網頁/courses/gen-image/img/`

---

## 全局風格基準（每張圖都遵守）

- **設計調性**：無印良品 / 日系極簡 / 留白多 / 自然光
- **主色階**（課程主題色）：暖奶茶棕 `#8a6a4a` + 米白 `#f5efe3` + 灰 `#a89c8d`
- **字體**：主標 Shippori Mincho（明朝體粗體）、內文 Noto Sans JP
- **禁用**：漸層、純黑、霓虹色、過度裝飾、emoji
- **尺寸**：除非另註明、預設 1:1（1080×1080）
- **檔案格式**：JPG（品質 90）或 PNG（透明區可不要）
- **若 AI 文字渲染失敗**：產出後再用 Canva 補字、或同對話請 AI 重產

---

## 統一商品基準（用於 Tier 1 同商品 × 8 版型）

- **商品**：春季限定 抹茶提拉米蘇拿鐵（Matcha Tiramisu Latte）
- **形狀**：透明玻璃杯、雙層分層（下抹茶綠、上奶霜）、頂層撒抹茶粉
- **配料道具**：抹茶粉、湯匙、茶葉、和紙
- **品牌名**（一致）：弄一下咖啡 / NORDIC NOMAD COFFEE
- **季節**：春、櫻花尾聲、淡綠
