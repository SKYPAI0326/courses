> 整合自 「品牌風格識別設計｜完整提示詞教學」（this.web 製作、2026-04 取得）。

> 進階產出規格：4:3 多頁網站 Lookbook（6-8 格拼貼）、用於 CH5-1 結業挑戰。

## 一、Meta-prompting 心法（CH2-2 進階法）

> **核心思維**：當你不知道怎麼寫 prompt、不要硬寫——**先讓 AI 幫你寫 prompt**、再把產出的 prompt 餵給生圖 AI。

### Meta-prompt 模板

```
我想為一家名為 [品牌名稱] 的 [產品 / 服務] 品牌、設計一份 [風格分類] 風格的多頁網站品牌識別（多張圖像）。
請根據以下指令模板、填入你認為合適的風格、需求、輸出最終的指令、讓我拿去給 AI 生成圖片。

**請直接生成中文文字指令、不用解釋。**

請為一家名為 [品牌名稱] 的 [產品 / 服務] 品牌打造一套 [整體風格] 風格的多頁品牌識別。

整體品牌應呈現出 [核心風格關鍵詞]、結合 [產業 / 產品類型] 的特性與 [品牌定位（如高端 / 親民 / 實驗性等）] 的質感。

風格需融合 [風格 A] 與 [風格 B]、並帶有 [附加氛圍（如未來感 / 復古 / 極簡 / 奢華等）]。

關注品牌情緒（如年輕 / 沉穩 / 前衛 / 溫暖等）。

視覺需同時呈現 [對比特質 A] 與 [對比特質 B]（如：趣味性 vs 專業感、親和力 vs 高端感）。

採用 [色彩策略（如高飽和 / 單色系 / 對比色 / 柔和色等）] 的視覺識別、搭配 [視覺元素（如幾何 / 有機形狀 / 流體 / 網格等）]、[字體風格（如現代無襯線 / 實驗字體 / 優雅襯線等）]、以及 [節奏或動態提示（如動畫感 / 節奏切換 / 模組化版面等）]。

影像風格 [影像語言（如超現實 / 寫實產品攝影 / 3D 渲染 / 拼貼等）]。

頁面需包含 [頁面結構（如：首頁、產品頁、品牌故事頁、其他頁面等）]、輸出比例為 4:3、以品牌 Lookbook 方式呈現、採用非對稱、解構式版型、模組錯位排列。
```

### 使用流程

1. 把上方 meta-prompt 貼進 ChatGPT / Gemini / Claude
2. 把 `[紅色文字區]` 替換成你的品牌資訊
3. AI 會幫你產出**最終生圖指令**（已含結構、配色、字體、版型等所有細節）
4. 把產出的指令複製貼給 Nano Banana / GPT Image / Imagen 跑

### 小技巧

- **指令不滿意**：直接跟 AI 說「請重新生成指令」即可
- **避免某風格**：在最後加「避免深色、霓虹風格」這類指令、防 AI 慣性走偏
- **適用平台**：ChatGPT GPT Image 2 / Nano Banana Pro 都吃這套；Gemini Imagen 一般版本也能跑

## 二、7 風格分類表（取代舊版「配色組合速查」）

每種風格 = 適合業種 + 視覺特徵 + 配色 + 字體 + 影像語言。

### 1. 簡約 minimal

- **適合品牌**：家具家飾、香氛、生活選品、美容服務、建築室內設計
- **視覺特徵**：留白多、結構乾淨、低飽和、單色系
- **配色**：米白 + 淺灰 + 淺木 + 微量點綴色
- **字體**：Bodoni / Didone（主標）+ Helvetica Neue（內文）
- **影像語言**：寫實產品攝影、自然光、淺景深
- **應用範例**：香水品牌官網、極簡家居選品

### 2. 活力 playful

- **適合品牌**：氣泡飲、零食、潮流服飾、年輕族群 App、遊戲娛樂平台
- **視覺特徵**：高飽和、對比強、有趣味元素、手寫感
- **配色**：黃 + 紫 + 綠（高飽和對比）/ 螢光色 + 白
- **字體**：圓黑體（粗）+ 手寫感字體
- **影像語言**：商品擬人化、表情貼圖、波浪型背景、illustrated
- **應用範例**：洋芋片品牌、年輕飲料、手遊官網

### 3. 科技 futuristic

- **適合品牌**：3C 產品、軟體平台、穿戴裝置、數據分析工具、雲端服務
- **視覺特徵**：精準、冷感、機械感、模組化
- **配色**：深灰 + 銀 + 螢光綠 / 深藍 + 白 + 電光藍
- **字體**：Inter / SF Pro（主標）+ JetBrains Mono（數字 / 等寬）
- **影像語言**：產品 3D 渲染、儀表板介面截圖、機械結構特寫
- **應用範例**：智慧錶官網、SaaS 著陸頁、AI 工具介紹

### 4. 自然 organic

- **適合品牌**：保養品、有機食品、健康飲品、瑜伽/冥想服務、永續環保產品
- **視覺特徵**：暖色調、植物元素、手作感、留白
- **配色**：米色 + 草綠 + 淺木 / 大地色系
- **字體**：Shippori Mincho（主標）+ Noto Sans JP / TC（內文）
- **影像語言**：自然光、植物配料、手作場景、生活情境
- **應用範例**：環保餐具品牌、有機保養、瑜伽品牌
- **本課程主推**：本課程主題色（暖奶茶棕 #8a6a4a）屬於此風格家族

### 5. 復古 retro

- **適合品牌**：餐酒館、復古相機、街頭潮牌、黑膠唱片、精釀啤酒
- **視覺特徵**：紙質紋理、手寫標籤、老照片濾鏡、畫框感
- **配色**：深棕 + 米黃 + 磚紅 / 暗綠 + 鮮黃
- **字體**：襯線粗體（主標）+ 手寫感字體（accent）
- **影像語言**：黑白 / 棕褐色照片、老建築、復古字體標籤、紙質背景
- **應用範例**：精釀啤酒網站、餐酒館品牌、唱片行

### 6. 奢華 luxury

- **適合品牌**：精品腕錶、高端珠寶、頂級香水、私人會所、豪宅地產
- **視覺特徵**：深色背景、燙金、低調反差、戲劇光線
- **配色**：黑 + 金 + 深酒紅 / 墨綠 + 金
- **字體**：Didone / Bodoni（主標、襯線優雅）+ 細襯線（內文）
- **影像語言**：戲劇光線、金屬反光、絲絨質感、極簡商品擺拍
- **應用範例**：頂級香水官網、精品錶、私人會所

### 7. 藝術 conceptual

- **適合品牌**：展覽 / 藝廊、數位藝術平台、創意工作室、時尚品牌
- **視覺特徵**：解構、非對稱、實驗性版面、強烈視覺衝擊
- **配色**：黑 + 螢光綠 + 純白 / 黑 + 紅 + 灰
- **字體**：Sans 粗體（主標）+ 等寬字體（標註）+ 實驗字體（accent）
- **影像語言**：抽象雕塑、實驗影像、玻璃裂痕、解構幾何
- **應用範例**：當代藝術館、創意工作室、實驗時尚品牌

## 四、進階產出格式｜多頁品牌識別 Lookbook（4:3）

> **產出規格**：1 張圖內含 6-8 格拼貼、模擬「品牌完整網站視覺」
> **適用情境**：CH5-1 結業挑戰、品牌 portfolio、提案 mockup

### 標準 Lookbook 結構（6 格）

| 格位 | 內容 |
|---|---|
| 1（左上、最大）| 首頁 hero（主視覺 + 主標 + 導覽列）|
| 2（右上）| 品牌故事頁（情境照 + 文案）|
| 3（中左）| 產品頁（單品特寫 + 規格）|
| 4（中右）| 產品系列頁（多款並列）|
| 5（左下）| 聯絡 / 訂購頁（表單 + 資訊）|
| 6（右下）| 形象照（門市 / 場景 / 工作室）|

### 6-8 格 Lookbook 拼貼 prompt 模板

```
請產出一張 4:3 多頁網站品牌識別 Lookbook、用 [風格分類] 風格：

【品牌】[品牌名稱]
【業種】[產品 / 服務類型]
【整體調性】[3 個關鍵詞]

【拼貼結構】6 格、模組錯位、非對稱排列：
- 格 1（左上、占畫面 35%）：首頁 hero（含導覽列 + 主視覺商品 + 主標 + CTA）
- 格 2（右上、占畫面 30%）：品牌故事頁（情境照 + 文案 + 「了解更多」按鈕）
- 格 3（中左）：產品特寫頁（單品 + 成分 / 規格說明）
- 格 4（中右）：產品系列頁（3 款並列 + 信任標籤）
- 格 5（左下）：聯絡資訊（地址 / 電話 / Email + 地圖 mockup）
- 格 6（右下）：形象照（門市 / 工作室 / 場景）

【風格 preset】（依選定的風格分類填）
- 配色：[該風格配色組合]
- 字體：[該風格主標 + 內文字體]
- 影像語言：[該風格影像特徵]

【尺寸】4:3
【版型】非對稱、解構式、模組錯位
【禁用】[依風格加避免項、如「避免深色、霓虹風格」]
```

## 給 course-designer 的提醒

1. 7 風格分類**不取代** CH2-1 的 8 版型、兩者是**正交概念**：
   - **版型** = 單張廣告的「結構」（A-H 8 種）
   - **風格** = 整體視覺的「調性」（7 種）
   - 任意風格 × 任意版型 = 完整 prompt
2. Meta-prompting 是 CH2-2 的進階變體、**不是替代**——學員初學還是要會自己寫 brief 六件套
3. 多頁 Lookbook 是 CH5-1 的進階作品、**不是必修**——學員若時間有限、做完 8 版型 + 品牌系列就達標
