# 8 種用途版型｜學員主用速查

> 本檔為 `gen-image` 課程的**前台主教學物**——每個版型 = 一種用途、一種場景、一張結構卡。
>
> 教學心法：**版型沒有最好、只有最適合**。學員拿到任務、3 秒內查表選 1 個版型、複製 prompt、跑出來。
>
> 🚀 **配套彈藥庫**：每個版型都有 `8-prompt-examples.md` 對應段落、含「複製即用」prompt 模板 + 跨業種範例 + 變數填寫指南。本檔重視覺結構認知、彈藥庫重 prompt 實作執行——兩者並用、學完即跑。

## A｜主視覺版

### 一句話：**最吸睛的視覺版型、用來抓住注意力**

> **記住這句話**：「畫面越強，訊息才會被看見。」

### 設計重點

- **畫面最大化**——讓商品占滿畫面、視覺衝擊最強
- **文字極簡化**——只留必要資訊、不干擾視覺
- **留白最少化**——減少空白、強化沉浸與張力

### 適用場景

- IG 貼文 / 廣告
- 廣告首圖主視覺
- 任何需要「第一眼留住人」的場合

### 版型結構

```
┌─────────────────────────┐
│                         │
│    主角（產品）         │
│    80-90%               │
│                         │
│                         │
├─────────────────────────┤
│ 極少文字（標題/賣點）   │ 10-20%
└─────────────────────────┘
```

### 小技巧

- 用光影強化質感
- 流心、亮點、爆點放在畫面**黃金視覺位置**（畫面中上偏左 1/3 處）
- 第一眼立即吸引目光、停留 3 秒、決定要不要看下去

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § A 段（複製即用、含 5 個變數空格）

**A 版型必填 5 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類 | 咖啡店 / 美妝 / 手工皂 / App |
| `{商品}` | 名 + 規格 | 春季抹茶拿鐵 Tall ¥730 |
| `{客群}` | 年齡 + 性別 + 風格 | 25-35 歲女性、IG 重度 |
| `{訴求}` | 3 個調性詞 | 手作、季節感、療癒 |
| `{主標 1 行}` | ≤ 12 字 | 春のはじまり |

## B｜橫幅版

### 一句話：**最適合橫式空間、用來快速吸引注意**

> **記住這句話**：「在橫幅中、1 秒抓住眼球就是關鍵。」

### 設計重點

- **橫向延展**——利用橫幅構圖、延展視覺吸引力
- **資訊精簡**——文字精簡有力、快速傳達重點
- **主次分明**——主標與畫面主體並列、強化記憶點

### 適用場景

- 網站 BANNER
- 廣告橫幅
- EDM 標題 / 信件 hero

### 版型結構

```
┌──────────────┬────────────┐
│              │ 主標題     │
│ 主角         │            │
│ （產品）     │ 短標語     │
│              │            │
└──────────────┴────────────┘
  橫向延展 90%+ │ 文字 < 30%
```

### 小技巧

- 將主體放大 + 搭配短標語
- 視覺高度 70% 以上、文字精簡到一行兩行
- 提升瞬間吸引力

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § B 段（複製即用、含 6 個變數空格）

**B 版型必填 6 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類 | 線上課程 / 保健品 / 旅遊 |
| `{商品}` | 名 + 規格 | AI 圖像課 5 週 / OmegaPlus 60 顆 |
| `{客群}` | 年齡 + 痛點 | 30-45 歲、追求效率 |
| `{主標 1 行}` | ≤ 14 字、粗體 | 5 週、學會用 AI 做廣告 |
| `{短標語 1 行}` | ≤ 20 字、細體 | 早鳥 8 折、4/30 截止 |
| `{CTA 文字}` | 簡短行動指令 | 立即了解 / 預約諮詢 |

## C｜社群版

### 一句話：**最輕量的視覺版型、讓產品自己說話**

> **記住這句話**：「少即是多、讓產品自己說話。」

### 設計重點

- **文字少**——只留重點、不干擾畫面
- **畫面大**——產品占視覺焦點
- **留白多**——呼吸感更強、質感更高級

### 適用場景

- IG 貼文
- Threads 貼文
- 任何「日常曝光、不打擾用戶」的社群場合

### 版型結構

```
┌─────────────────────────┐
│                         │
│    主角（產品）         │
│    70%+                 │
│                         │
├─────────────────────────┤
│ 少量文字                │ 30%-
│ （品牌/品名）           │
└─────────────────────────┘
```

### 小技巧

- 讓流心、主體**引導視線**到重點
- 大量留白、不要塞滿
- 適合日常社群、不適合促銷檔期

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § C 段（複製即用、含 5 個變數空格）

**C 版型必填 5 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類 | 咖啡店 / 美妝 / 手工皂 |
| `{商品}` | 單品 | 拿鐵 + 拉花 / 精華液小瓶 |
| `{客群}` | 平台習慣 | 25-30 歲女性、IG 重度 |
| `{品牌名}` | 品牌簡稱 | LUNÉA / 弄一下咖啡 |
| `{catchphrase}` | 一句話 | a small joy in your morning |

## D｜系列版

### 一句話：**完整呈現多款產品、讓客戶比較選擇**

> **記住這句話**：「系列感、就是讓喜歡變成全都想要。」

### 設計重點

- **系列整合**——統一風格與構圖、強化品牌辨識度
- **比較引導**——並列呈現、讓差異一目了然
- **價值強化**——加入食材 / 製程資訊、提升信任感

### 適用場景

- IG 貼文 / 限動（系列推薦）
- 電商商品頁
- 品牌目錄 / Lookbook

### 版型結構

```
┌─────┬─────┬─────┐
│產品1│產品2│產品3│  主視覺區 70%+
│     │     │     │
│     │     │     │
├─────┼─────┼─────┤
│口味 │口味 │口味 │
│特色 │特色 │特色 │  資訊區 30%-
└─────┴─────┴─────┘
   賣點 / 品質保證
```

### 小技巧

- 用**一致的色調與構圖節奏**、讓系列看起來更專業
- 每個商品的拍攝角度、光線、留白都對齊
- 加上「特色標籤」（嚴選食材 / 無添加 / 手工製作）拉高信任

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § D 段（複製即用、含 8 個變數空格）

**D 版型必填 8 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類 | 咖啡店 / 美妝 / 保健品 |
| `{系列名}` | 整體名 | 春季限定 3 款 / 早晚保養三件 |
| `{款 1}` | 名 + 1 句特色 | 抹茶拿鐵（手作職人）|
| `{款 2}` | 名 + 1 句特色 | 櫻花拿鐵（季節限定）|
| `{款 3}` | 名 + 1 句特色 | 草莓拿鐵（在地小農）|
| `{客群}` | 購買場景 | 25-35 歲女性、咖啡愛好者 |
| `{特色標籤}` | 信任元素 | 嚴選食材 / 無添加 / 手工製作 |
| `{統一色調}` | 系列色 | 鼠尾草綠 + 奶油白 |

## E｜禮盒版

### 一句話：**質感包裝 × 精緻呈現、用來拉高單價**

> **記住這句話**：「包裝感、就是溢價的視覺證明。」

### 設計重點

- 利用包裝與情境營造**價值感**
- 適用節慶、送禮、高價商品策略
- **質感優先於資訊量**——學員可參考精品包裝攝影風

### 適用場景

- 節慶禮盒（中秋、母親節、聖誕）
- 送禮市場（婚禮、企業禮品）
- 高價商品策略

### 版型結構

```
┌─────────────────────────┐
│                         │
│   禮盒（外包裝為主角）  │
│   含緞帶、紋飾、標籤    │
│                         │
├─────────────────────────┤
│ 品牌名 + 系列名         │
└─────────────────────────┘
```

### 小技巧

- 拍攝角度低一些、產生「商品仰望感」
- 深色背景或絨布感、襯托質感
- 加入金色 / 燙金元素提升精品感

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § E 段（複製即用、含 6 個變數空格）

**E 版型必填 6 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類 | 美妝 / 烘焙 / 茶葉 / 保健 |
| `{禮盒名}` | 名 + 內容物 + 價位 | 母親節禮盒 3 入 ¥2,400 |
| `{節慶}` | 場合 | 母親節 / 中秋 / 聖誕 / 婚禮 |
| `{客群}` | 送禮者畫像 | 25-40 歲女性、送媽媽 |
| `{深色主調}` | 主背景色 | 深酒紅 + 金 / 米粉 + 深棕 |
| `{附加元素}` | 質感物 | 緞帶 + 乾燥花 / 燙金紋飾 |

## F｜特寫版

### 一句話：**極致特寫 × 感官刺激、用來提升食慾**

> **記住這句話**：「越近越想吃。」

### 設計重點

- 放大細節與質感、強化視覺衝擊
- 快速激發食慾或感官想像
- 適合食物、飾品、化妝品等「質感為王」的商品

### 適用場景

- 社群貼文（IG / Threads）
- 美食平台（Foodpanda / UberEats / Google Maps 商家頁）
- 餐廳菜單視覺

### 版型結構

```
┌─────────────────────────┐
│                         │
│                         │
│   產品極致特寫          │
│   （流心、質感、紋路）  │
│                         │
│                         │
└─────────────────────────┘
   零文字或極少品名
```

### 小技巧

- 拍**流動瞬間**（巧克力流心、起司拉絲、咖啡蒸氣）
- 光線讓質感發亮（油光、糖霜、結露）
- 文字幾乎不要、讓畫面說話

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § F 段（複製即用、含 4 個變數空格）

**F 版型必填 4 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類（建議餐飲 / 烘焙 / 美妝 / 飾品）| 烘焙 / 拉麵 / 戒指 |
| `{商品 + 質感重點}` | 商品 + 想突顯的部分 | 流心抹茶巧克力布朗尼 |
| `{動態瞬間}` | 拍什麼瞬間 | 流心爆漿 / 起司拉絲 / 咖啡蒸氣 |
| `{光線}` | 打光方向 | 頂光 + 側補光 / 暖黃側光 |

## G｜促銷版

### 一句話：**強烈訊息 × 行動導向、用來快速轉單**

> **記住這句話**：「優惠不亮、訊息白給。」

### 設計重點

- 突出**優惠與行動指令**（折扣、限時、限量）
- 搭配大字 + 對比色、加速決策與下單
- 不追求美感、追求**轉換率**

### 適用場景

- 檔期活動（雙 11、母親節、會員日）
- 廣告投放（FB / IG / LINE 廣告）
- 限時優惠 banner

### 版型結構

```
┌─────────────────────────┐
│ 限時優惠                │
│ 商品名                  │
│ 2 盒 85 折              │  大字優惠
│ 即日起-4/30             │
├─────────────────────────┤
│   產品                  │
└─────────────────────────┘
```

### 小技巧

- **數字越大越好**（85 折比 「優惠中」 強 10 倍）
- 對比色（黑底金字、白底紅字）
- 限時感（即日起至 X/X、最後 N 天）

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § G 段（複製即用、含 7 個變數空格）

**G 版型必填 7 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類 | 咖啡店 / 美妝 / 線上課程 |
| `{商品}` | 名 | 春季拿鐵 / 全館商品 / 早鳥課 |
| `{主優惠}` | 強訊息 | 第 2 杯 5 折 / 全館 8 折 |
| `{限時}` | 時間範圍 | 4/25-4/30 / 最後 3 天 |
| `{限量}` | 數量限制（可不填）| 每人限 2 件 / 限量 100 組 |
| `{CTA}` | 行動指令 | 立即下單 / 私訊預訂 |
| `{對比色}` | 配色 | 黑底 + 黃 + 紅 |

## H｜生活版

### 一句話：**情境氛圍 × 情感共鳴、用來拉近距離**

> **記住這句話**：「不賣商品、賣『與商品一起的時光』。」

### 設計重點

- 融入**生活場景**（午後、睡前、辦公、約會）
- 營造情感連結、建立品牌好感與忠誠度
- 文字溫柔、像朋友推薦不像廣告

### 適用場景

- 品牌日常（IG 動態、品牌官網 banner）
- 生活提案（季節物語、節氣文案）
- 內容行銷 / 品牌故事

### 版型結構

```
┌─────────────────────────┐
│ 「午後時光、              │  情境文案
│   來塊療癒的布朗尼。」    │  （手寫感）
│                         │
│                         │
│   產品 + 生活道具       │  情境主視覺
│   （咖啡、書、餐巾）    │
│                         │
└─────────────────────────┘
```

### 小技巧

- 加入**人的痕跡**（咖啡杯、翻開的書、毛毯一角）
- 自然光、不用棚拍燈
- 文字像書信一樣柔軟、避免命令句

### 快速啟動 prompt

📋 **完整 prompt 模板**：見 `8-prompt-examples.md` § H 段（複製即用、含 7 個變數空格）

**H 版型必填 7 變數**：

| 變數 | 填什麼 | 範例 |
|---|---|---|
| `{業種}` | 大類 | 咖啡店 / 美妝 / 旅遊 / App |
| `{商品}` | 名 | 拿鐵 + 馬芬 / 精華液 + 化妝水 |
| `{情境}` | 時間 + 場景 + 行為 | 午後 + 書桌 + 翻書 |
| `{情緒}` | 想傳遞的感受 | 療癒 / 慢時光 / 與自己對話 |
| `{人的痕跡}` | 1-2 件生活物 | 咖啡杯 / 翻開的書 / 眼鏡 |
| `{主文案}` | 1-2 行情感文 | 午後時光、慢慢來 |
| `{自然色系}` | 配色 | 米白 + 木色 + 微綠 |

## 版型選擇 3 問

PM 拿到任務、用這 3 問挑版型、不必死記 8 種：

1. **這要給誰看？**（目標受眾在什麼平台、什麼情緒）
2. **要他們做什麼？**（看 / 記住 / 下單 / 轉發）
3. **預算與資源？**（有沒有時間做精緻、還是快出檔期）

3 問答完、對照「目的 → 版型」表、選 1 個。**版型沒有最好、只有最適合**。

## 同產品 × 多版型搭配（系列行銷思維）

一個商品上市、可以**同時做 3-4 種版型**搭配行銷檔期：

| 階段 | 用什麼版型 | 用途 |
|---|---|---|
| 預熱期 | **A 主視覺版** | IG 廣告抓注意力 |
| 上市期 | **B 橫幅版** + **F 特寫版** | 網站 banner + 社群激發購買慾 |
| 推廣期 | **D 系列版** + **H 生活版** | 多款比較 + 情感連結 |
| 衝刺期 | **G 促銷版** | 檔期優惠快速轉單 |
| 節慶期 | **E 禮盒版** | 拉高單價 |

**這就是「同產品 × 不同版型 × 不同場景」的策略價值**——一個商品做 8 張、覆蓋整個行銷漏斗。

## 附錄｜後台知識：5 種版面骨架

> 此段給「想深入理解版型結構為什麼這樣排」的學員 / 進階使用者。**初學不必看**——8 版型已夠用。

每個用途版型背後、有對應的「結構骨架」（PM 在寫 prompt 時可選用）：

| 用途版型 | 對應結構骨架 |
|---|---|
| A 主視覺版 | 商品中心 / 大畫面占比 |
| B 橫幅版 | 商品左 + 文字右（橫向）|
| C 社群版 | 商品中心 + 留白 |
| D 系列版 | 多列分割（橫向 3-4 欄）|
| E 禮盒版 | 商品中心 + 質感包裝 |
| F 特寫版 | 滿版特寫 |
| G 促銷版 | 大字疊文 + 商品輔助 |
| H 生活版 | 風景 / 情境底 + 商品 |

PM 寫 prompt 時可以直接寫「請用 A 主視覺版」、AI 會理解；要更精準時、加上骨架描述（如「商品中央 80% 占比」）。
