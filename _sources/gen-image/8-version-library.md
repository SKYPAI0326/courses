# 8 種用途版型｜學員主用速查

> 本檔為 `gen-image` 課程的**前台主教學物**——每個版型 = 一種用途、一種場景、一張結構卡。
>
> 教學心法：**版型沒有最好、只有最適合**。學員拿到任務、3 秒內查表選 1 個版型、複製 prompt、跑出來。

---

## 速查總表

| 版型 | 一句話用途 | 主場景 | 文字密度 | 主視覺占比 |
|---|---|---|---|---|
| **A 主視覺版** | 抓住注意力、停留 3 秒 | IG 廣告 / 廣告首圖 | 10-20% | 80-90% |
| **B 橫幅版** | 1 秒抓眼球、快速傳重點 | 網站 banner / EDM 標題 | < 30% | 70%+ |
| **C 社群版** | 少字強視覺、輕量易讀 | IG 貼文 / Threads | < 30% | 70%+ |
| **D 系列版** | 完整呈現、比較選擇 | 商品頁 / IG 限動 | 30%+ | 多商品並列 |
| **E 禮盒版** | 拉高單價、提升價值感 | 節慶禮盒 / 送禮市場 | < 30% | 包裝主視覺 |
| **F 特寫版** | 提升食慾、激發感受 | 社群貼文 / 美食平台 | < 20% | 極致特寫 |
| **G 促銷版** | 快速轉單、促進行動 | 檔期活動 / 廣告投放 | 大字優惠 | 商品 + 強烈訊息 |
| **H 生活版** | 拉近距離、建立情感連結 | 品牌日常 / 生活提案 | 小字情境 | 情境氛圍 |

## A｜主視覺版

### 一句話：**最吸睛的視覺版型、用來抓住注意力**

> **記住這句話**：「畫面越強，訊息才會被看見。」

### 設計重點

- **畫面最大化**——讓商品占滿畫面、視覺衝擊最強
- **文字極簡化**——只留必要資訊、不干擾視覺
- **留白最少化**——減少空白、強化沉浸與張力

### 適用場景

- IG 貼文 / 廣告
- 廣告首圖主視覺
- 任何需要「第一眼留住人」的場合

### 版型結構

```
┌─────────────────────────┐
│                         │
│    主角（產品）         │
│    80-90%               │
│                         │
│                         │
├─────────────────────────┤
│ 極少文字（標題/賣點）   │ 10-20%
└─────────────────────────┘
```

### 小技巧

- 用光影強化質感
- 流心、亮點、爆點放在畫面**黃金視覺位置**（畫面中上偏左 1/3 處）
- 第一眼立即吸引目光、停留 3 秒、決定要不要看下去

---

## B｜橫幅版

### 一句話：**最適合橫式空間、用來快速吸引注意**

> **記住這句話**：「在橫幅中、1 秒抓住眼球就是關鍵。」

### 設計重點

- **橫向延展**——利用橫幅構圖、延展視覺吸引力
- **資訊精簡**——文字精簡有力、快速傳達重點
- **主次分明**——主標與畫面主體並列、強化記憶點

### 適用場景

- 網站 BANNER
- 廣告橫幅
- EDM 標題 / 信件 hero

### 版型結構

```
┌──────────────┬────────────┐
│              │ 主標題     │
│ 主角         │            │
│ （產品）     │ 短標語     │
│              │            │
└──────────────┴────────────┘
  橫向延展 90%+ │ 文字 < 30%
```

### 小技巧

- 將主體放大 + 搭配短標語
- 視覺高度 70% 以上、文字精簡到一行兩行
- 提升瞬間吸引力

---

## C｜社群版

### 一句話：**最輕量的視覺版型、讓產品自己說話**

> **記住這句話**：「少即是多、讓產品自己說話。」

### 設計重點

- **文字少**——只留重點、不干擾畫面
- **畫面大**——產品占視覺焦點
- **留白多**——呼吸感更強、質感更高級

### 適用場景

- IG 貼文
- Threads 貼文
- 任何「日常曝光、不打擾用戶」的社群場合

### 版型結構

```
┌─────────────────────────┐
│                         │
│    主角（產品）         │
│    70%+                 │
│                         │
├─────────────────────────┤
│ 少量文字                │ 30%-
│ （品牌/品名）           │
└─────────────────────────┘
```

### 小技巧

- 讓流心、主體**引導視線**到重點
- 大量留白、不要塞滿
- 適合日常社群、不適合促銷檔期

---

## D｜系列版

### 一句話：**完整呈現多款產品、讓客戶比較選擇**

> **記住這句話**：「系列感、就是讓喜歡變成全都想要。」

### 設計重點

- **系列整合**——統一風格與構圖、強化品牌辨識度
- **比較引導**——並列呈現、讓差異一目了然
- **價值強化**——加入食材 / 製程資訊、提升信任感

### 適用場景

- IG 貼文 / 限動（系列推薦）
- 電商商品頁
- 品牌目錄 / Lookbook

### 版型結構

```
┌─────┬─────┬─────┐
│產品1│產品2│產品3│  主視覺區 70%+
│     │     │     │
│     │     │     │
├─────┼─────┼─────┤
│口味 │口味 │口味 │
│特色 │特色 │特色 │  資訊區 30%-
└─────┴─────┴─────┘
   賣點 / 品質保證
```

### 小技巧

- 用**一致的色調與構圖節奏**、讓系列看起來更專業
- 每個商品的拍攝角度、光線、留白都對齊
- 加上「特色標籤」（嚴選食材 / 無添加 / 手工製作）拉高信任

---

## E｜禮盒版

### 一句話：**質感包裝 × 精緻呈現、用來拉高單價**

> **記住這句話**：「包裝感、就是溢價的視覺證明。」

### 設計重點

- 利用包裝與情境營造**價值感**
- 適用節慶、送禮、高價商品策略
- **質感優先於資訊量**——學員可參考精品包裝攝影風

### 適用場景

- 節慶禮盒（中秋、母親節、聖誕）
- 送禮市場（婚禮、企業禮品）
- 高價商品策略

### 版型結構

```
┌─────────────────────────┐
│                         │
│   禮盒（外包裝為主角）  │
│   含緞帶、紋飾、標籤    │
│                         │
├─────────────────────────┤
│ 品牌名 + 系列名         │
└─────────────────────────┘
```

### 小技巧

- 拍攝角度低一些、產生「商品仰望感」
- 深色背景或絨布感、襯托質感
- 加入金色 / 燙金元素提升精品感

---

## F｜特寫版

### 一句話：**極致特寫 × 感官刺激、用來提升食慾**

> **記住這句話**：「越近越想吃。」

### 設計重點

- 放大細節與質感、強化視覺衝擊
- 快速激發食慾或感官想像
- 適合食物、飾品、化妝品等「質感為王」的商品

### 適用場景

- 社群貼文（IG / Threads）
- 美食平台（Foodpanda / UberEats / Google Maps 商家頁）
- 餐廳菜單視覺

### 版型結構

```
┌─────────────────────────┐
│                         │
│                         │
│   產品極致特寫          │
│   （流心、質感、紋路）  │
│                         │
│                         │
└─────────────────────────┘
   零文字或極少品名
```

### 小技巧

- 拍**流動瞬間**（巧克力流心、起司拉絲、咖啡蒸氣）
- 光線讓質感發亮（油光、糖霜、結露）
- 文字幾乎不要、讓畫面說話

---

## G｜促銷版

### 一句話：**強烈訊息 × 行動導向、用來快速轉單**

> **記住這句話**：「優惠不亮、訊息白給。」

### 設計重點

- 突出**優惠與行動指令**（折扣、限時、限量）
- 搭配大字 + 對比色、加速決策與下單
- 不追求美感、追求**轉換率**

### 適用場景

- 檔期活動（雙 11、母親節、會員日）
- 廣告投放（FB / IG / LINE 廣告）
- 限時優惠 banner

### 版型結構

```
┌─────────────────────────┐
│ 限時優惠                │
│ 商品名                  │
│ 2 盒 85 折              │  大字優惠
│ 即日起-4/30             │
├─────────────────────────┤
│   產品                  │
└─────────────────────────┘
```

### 小技巧

- **數字越大越好**（85 折比 「優惠中」 強 10 倍）
- 對比色（黑底金字、白底紅字）
- 限時感（即日起至 X/X、最後 N 天）

---

## H｜生活版

### 一句話：**情境氛圍 × 情感共鳴、用來拉近距離**

> **記住這句話**：「不賣商品、賣『與商品一起的時光』。」

### 設計重點

- 融入**生活場景**（午後、睡前、辦公、約會）
- 營造情感連結、建立品牌好感與忠誠度
- 文字溫柔、像朋友推薦不像廣告

### 適用場景

- 品牌日常（IG 動態、品牌官網 banner）
- 生活提案（季節物語、節氣文案）
- 內容行銷 / 品牌故事

### 版型結構

```
┌─────────────────────────┐
│ 「午後時光、              │  情境文案
│   來塊療癒的布朗尼。」    │  （手寫感）
│                         │
│                         │
│   產品 + 生活道具       │  情境主視覺
│   （咖啡、書、餐巾）    │
│                         │
└─────────────────────────┘
```

### 小技巧

- 加入**人的痕跡**（咖啡杯、翻開的書、毛毯一角）
- 自然光、不用棚拍燈
- 文字像書信一樣柔軟、避免命令句

## 版型選擇 3 問

PM 拿到任務、用這 3 問挑版型、不必死記 8 種：

1. **這要給誰看？**（目標受眾在什麼平台、什麼情緒）
2. **要他們做什麼？**（看 / 記住 / 下單 / 轉發）
3. **預算與資源？**（有沒有時間做精緻、還是快出檔期）

3 問答完、對照「目的 → 版型」表、選 1 個。**版型沒有最好、只有最適合**。

## 同產品 × 多版型搭配（系列行銷思維）

一個商品上市、可以**同時做 3-4 種版型**搭配行銷檔期：

| 階段 | 用什麼版型 | 用途 |
|---|---|---|
| 預熱期 | **A 主視覺版** | IG 廣告抓注意力 |
| 上市期 | **B 橫幅版** + **F 特寫版** | 網站 banner + 社群激發購買慾 |
| 推廣期 | **D 系列版** + **H 生活版** | 多款比較 + 情感連結 |
| 衝刺期 | **G 促銷版** | 檔期優惠快速轉單 |
| 節慶期 | **E 禮盒版** | 拉高單價 |

**這就是「同產品 × 不同版型 × 不同場景」的策略價值**——一個商品做 8 張、覆蓋整個行銷漏斗。

## 附錄｜後台知識：5 種版面骨架

> 此段給「想深入理解版型結構為什麼這樣排」的學員 / 進階使用者。**初學不必看**——8 版型已夠用。

每個用途版型背後、有對應的「結構骨架」（PM 在寫 prompt 時可選用）：

| 用途版型 | 對應結構骨架 |
|---|---|
| A 主視覺版 | 商品中心 / 大畫面占比 |
| B 橫幅版 | 商品左 + 文字右（橫向）|
| C 社群版 | 商品中心 + 留白 |
| D 系列版 | 多列分割（橫向 3-4 欄）|
| E 禮盒版 | 商品中心 + 質感包裝 |
| F 特寫版 | 滿版特寫 |
| G 促銷版 | 大字疊文 + 商品輔助 |
| H 生活版 | 風景 / 情境底 + 商品 |

PM 寫 prompt 時可以直接寫「請用 A 主視覺版」、AI 會理解；要更精準時、加上骨架描述（如「商品中央 80% 占比」）。
