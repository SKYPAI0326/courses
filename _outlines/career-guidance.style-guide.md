# Style Guide — career-guidance

> 本檔覆蓋 `_style_guide_template.md`，course-designer / course-lesson-writer 讀本檔，不讀 template。
> 本課定位：**職涯定位工作坊** / 8h 一日 / 探索型在職者（工作 3-10 年）/ 上午判斷方向 + 下午四工具三 lane / AI 是輔助不是主軸。
> fork 自 career-pivot-mid（轉職實戰課），原課保留。本課差異：鎖「**還沒決定要不要換、先把方向想清楚**」而非「已決定轉職、要拿 offer」。
> **核心心法（通用版，主）**：方向不是越早決定越好，是越清楚越好。年資是經歷，不是答案；適性是線索，不是命運。
> **核心心法（資深 callout，輔）**：年紀不是優勢也不是劣勢，是市場會問的事實——用實力站著，不用年紀辯護。

---

## 受眾稱呼

- 稱呼學員：**你**（禁用「您」）
- 講師自稱：**「我」**（對話式，像「**會陪你把方向想清楚的職涯教練**」，不是激勵演講、不是前輩訓話）
- 不對學員預設年齡上下限——3 年到 10+ 年都可能在場，語氣對「還在長、但已不是新人」的人說話

---

## 案例主角（三主線示範角色 = 三條 lane）

本課最重要的差異化：下午四工具一律「**三 lane 共用**」。每個工具、每個範例都要能對應到三條路徑，學員依 CH1-6 選的方向找最像自己的那條。**主角強調年資（工作事實）+ 路徑選擇，不強調年齡**。

### 深耕線：小宇（企劃 4 年，最後選「深耕現職 + 內部爭取資源」）

- **背景**：B2B 軟體公司行銷企劃，工作 4 年，年薪 NT$ 70 萬
- **狀態**：做得還行但失去成長感，懷疑要不要換，盤點後發現「問題是缺舞台不是缺方向」
- **方向結論**：深耕現職——爭取主導一個跨部門專案、向主管要資源與更大職責
- **PDP × MBTI**：孔雀 I + ENFP（影響型 + 探索型）
- **三 lane 角色**：示範「深耕」如何用四工具（能力證據包→績效面談爭取資源）

### 內轉線：阿哲（工程 8 年，選「內部轉任技術管理」）

- **背景**：軟體公司資深後端工程師，工作 8 年，年薪 NT$ 130 萬
- **狀態**：技術做到頂、想帶人但公司沒缺，猶豫要內轉管理還是跳槽
- **方向結論**：內轉——在現公司爭取 team lead 機會（先試 30 天小組長代理）
- **PDP × MBTI**：貓頭鷹 C + INTJ（分析型 + 規劃型）
- **三 lane 角色**：示範「內轉」如何用四工具（內部轉任訪談 + 轉任提案）

### 轉職線：惠姊（行政 12 年 → 行銷企劃，沿用 career-pivot-mid 範例調整）

- **背景**：國貿公司行政主管，工作 12 年資，年薪 NT$ 78 萬
- **狀態**：行政天花板、想往內容/行銷走
- **方向結論**：外部轉職——投 B2B 行銷企劃，30 天先投 3 個試水溫
- **PDP × MBTI**：孔雀 I + ENFP（與小宇同型不同路徑，正好示範「同型可走不同路」）
- **三 lane 角色**：示範「轉職」如何用四工具（履歷能力證據包 + 外部面試）

### 使用原則

- **下午每個工具都要三 lane 並陳**（深耕小宇 / 內轉阿哲 / 轉職惠姊各一個範例）——這是本課核心，**留任 lane 不可只是轉職 lane 的影子**
- 上午單元擇一主示範即可（CH1-4 可遷移技能用小宇；CH1-6 三路徑決策三人各示範一條收斂）
- 用具體角色帶觀念：「阿哲是貓頭鷹 INTJ + 想內轉管理，所以他的關鍵對話是『內部轉任訪談』版本...」
- **不貼年齡標籤**：寫「工作 4 年小宇」不寫「26 歲小宇」
- **小宇與惠姊同為孔雀 ENFP 但走不同路**：刻意設計，用來教「適性是線索不是命運，同型可深耕也可轉職」

### 情境禁用清單

- **不討論「斜槓」「自由工作者」「數位遊牧」**：本課鎖正職路徑的三選擇
- **不討論創業**：30 天行動實驗不含創業（副業僅限「驗證方向」不是創業）
- **不討論移民 / 海外工作**：鎖台灣就業市場
- **不做「履歷急救 + 立即衝刺投遞」**：那是 career-pivot-mid 範圍；本課履歷=能力證據盤點，服務三條路
- **不深入單一產業**：可舉例不可深入

### 核心命名（本課鎖定，每單元引用同一組，不要換說法）

1. **PDP × MBTI 雙測**（不是「人格測驗」）
2. **職涯適性對應表**（不是「工作質性」、不是「適合什麼工作」——本課改用「職涯適性」，每格含「留在現職怎麼調」）
3. **可遷移技能盤點**
4. **三路徑決策**（深耕 / 內轉 / 轉職——固定這三個詞，不要寫成「留任/異動/離職」）
5. **能力證據包**（不是「履歷」——下午抽象工具名，服務三 lane）
6. **關鍵對話腳本**（三版本：績效面談 / 內部轉任訪談 / 外部面試）
7. **條件談判**（不是「薪資談判」——含加薪/資源/職責/職稱/offer）
8. **30 天行動實驗**（不是「入職計畫」——低風險驗證，含驗證指標 + 喊停條件）

## 語氣基準

- **務實沉穩、教練感**：陪你想清楚，不替你決定，不灌雞湯
- **重量感**：不用「機會」「夢想」「人生」大字眼，用「下一步」「未來幾年」「先試一個月」具體詞
- **直接不廢話**：每句有意義，不墊場
- **不諂媚**：誇要具體（「你過去 4 年帶過的 3 個專案 = 你想深耕的證據」）
- **承認侷限**：每段方法說「什麼時候不適用」
- **不販賣焦慮**：用事實降溫，不用「再不換就晚了」
- **重定位「下一步是驗證、不是決定」**：全課反覆強調探索型學員 CH1-6 後得到的是「比較想先試 A」，下午是教怎麼低風險驗證，不是逼你立刻換工作

## 每單元必要元素

### CH 頁面（本課 10 個 CH + 2 個 PRAC）

| 元素 | 要求 |
|---|---|
| 開場情境一句話 | 上午單元用三主線之一開場；下午用三 lane 對照開場 |
| 三路徑連結 | 下午每單元至少一次「深耕的你 / 內轉的你 / 轉職的你，這個工具怎麼用」三分述 |
| 紙本 / 表單 | 每單元至少 1 份紙本或 fill-form 當場動手 |
| 「為什麼」連結 | 每個方法步驟附「為什麼這樣做」一句話 |
| 反例展示 | 每單元至少一個常見誤區反例 |
| 量化數據 | 涉及市場 / 薪資地方引具體數字（CH1-5 必須實際調研）|
| 結尾 1 分鐘反思 | 每單元結尾「寫下這 X 分鐘對你最有用的 1 件事」|

### PRAC 實作頁

- 課後 1 週內完成，不擠進 8h
- PRAC1：**職涯方向地圖**（適性結論 + 四路徑/三路徑評估 + 下一步排序 + 雙測截圖）
- PRAC2：完整能力證據包 PDF + 自己路徑的關鍵對話腳本 + 30 天行動實驗計畫 + 1 次 ChatGPT 演練自查
- 講師批改回饋（不在課堂內）

### 補充頁面

- **index.html**：總覽 + 上午判斷方向/下午四工具路徑圖 + 紙本入口 + 雙測入口 + 決策痛點招生文案
- **module1.html / module2.html**：各自導覽
- **personality-test.html**：PDP × MBTI 雙測獨立頁（判讀去轉職特化，加職涯適性 + 現職適配度）

## Designer / Lesson-Writer 寫作優先級（硬性）

1. **CH1-2 PDP × MBTI 雙測是命脈**：30 題 + 16 題 + 計分腳本 + **16 型 × 職涯適性完整對應表**。每格 4 條 = 職涯適性 2 條 + 不適合情境 1 條 + **「留在現職怎麼調」1 條**（取代原轉職特化提醒）。16 型都要寫到位。
2. **CH1-5 是 P0 最大調研工時**（G1 Q1 標示最沒把握）：必須實際調研 2026 通用就業現況，引具體數字，交付 4 清單（成長產業 / 萎縮領域 / AI 衝擊不到職位 / 薪資結構與技能需求）。**全去轉職特化**（不寫「年資友善企業」「轉職要降薪幅度」）。來源：104 / LinkedIn Workforce Report / 勞動部統計處 / 主計總處 / 1111 / Cake。
3. **CH2 四工具三 lane 範例齊備是核心風險解法**：能力證據包 / 關鍵對話腳本 / 條件談判 / 30 天行動實驗，每個都要深耕（小宇）+ 內轉（阿哲）+ 轉職（惠姊）三版本範例。**留任 lane 必須是獨立有料的內容，不是轉職的影子**。
4. **CH1-6 三路徑決策**要給可操作評分維度（成長性 / 風險 / 時間成本 / 適性吻合度），不是空談。
5. **AI 只在 CH2-1 / CH2-2 出現**，其他單元嚴禁帶 AI 操作。
6. **重定位「下一步是驗證不是決定」**：30 天行動實驗強調「低風險、可驗證、不需先離職、含喊停條件」。
7. **退路 / 救援設計**：雙測矛盾、CH1-6 選不出方向、能力證據包寫不出量化、30 天想不到實驗——每個命脈段都要卡關備案。
8. **核心心法貫穿**：通用版（方向越清楚越好）為主；涉及資深學員年齡話題時才用 callout 版（年紀是市場會問的事實）。
