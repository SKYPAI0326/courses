## Part 5：前端與部署實戰（20h）

> **路徑設計**：本 Part 提供兩條路徑——進階路徑 CH5-1~4（共 10h，學原理 + Git + 後端）與傻瓜模式主路徑 CH5-5（1.5h，純拖拉部署、不開 Terminal）。學員依「想學會原理」或「只要結果」自選一條，兩條最後都銜接 PRAC5-1~4 + PRAC5-13~15。
>
> **主角設計**：CH5-1~4 每章用不同小型企業工作者作為案例主角（建宏 / 宜蓁 / 志豪 / 怡君），刻意不貫穿——每章獨立的小案例比連續成長線更貼近真實工作現場，且能擴大案例情境涵蓋面（避免單一主角帶來的代入限制）。CH5-5 走「自學包」結構不綁特定主角，所有路徑學員皆適用。
>
> **待對齊**：CH5-1 建宏與 CH5-3 志豪職能皆設為「行銷企劃」，案例情境雖不同但職能重疊；未來補 `gen-ai-140h.style-guide.md` 時可調整志豪職能（如改為 PM / 工程師 / HR）擴大職能光譜。

- CH5-1：HTML / CSS / JavaScript 最低必要知識 *(課堂 3h / 自學 2h)*
- CH5-2：AI 協作開發：從需求描述到完整頁面 *(課堂 3h / 自學 2h)*

- PRAC5-1：工具改版（v1 → v2 精準改版） *(課堂 2.5h / 自學 1h)*
