## 各單元可插入的迷你練習切入點

### CH1-1「AI 不只是聊天工具」

1. **工具對標練習**
   - 動作：對標自己 3 個常用工作，用 Ch 提供的表判斷 ChatGPT / Claude / Gemini 哪個最適合
   - 插入位置：第 2 小節「工具比較」後
   - 成果：「我的工具選擇清單」（任務 → 最佳工具 + 理由）

2. **使用型態分類練習**
   - 動作：列舉自己 5 個日常任務，判斷該用 Web / Desktop / API
   - 插入位置：第 3 小節「三種使用型態」後
   - 成果：「工作場景使用型態對應表」

3. **成本估算練習**
   - 動作：對照 Ch 成本說明，估算本週實驗的 token 花費
   - 插入位置：第 4 小節「免費版、付費版、Credits」後
   - 成果：「預算規劃表」（API 預估月成本）

### CH1-2「黃金提示詞公式」

1. **四要素改寫練習**
   - 動作：拿一個自己的模糊指令（如「幫我寫文案」），套四要素改寫
   - 插入位置：第 2 小節「四要素公式」後
   - 成果：v1（模糊）vs v2（精確）Prompt 對比

2. **輸出格式實驗**
   - 動作：同一任務分別用表格 / 條列 / JSON 各請 AI 一次，比較差異
   - 插入位置：第 3 小節「六種輸出格式」後
   - 成果：「格式效果比較表」

3. **Few-shot 對照練習**
   - 動作：同一任務做 zero-shot 與 few-shot（2 例）各一次，觀察差異
   - 插入位置：第 4 小節「Few-shot 範例法」後
   - 成果：「zero-shot vs few-shot 實驗紀錄」

### CH1-3「Prompt 除錯」

1. **問題診斷練習**
   - 動作：給 3 個失敗 Prompt，判斷是「太空泛 / 格式錯 / 不照角色」
   - 插入位置：第 1 小節「五種常見問題」後
   - 成果：「診斷答案卡」

2. **多輪修正演練**
   - 動作：選一個工作任務，做 3 輪修改（先對→再好→再快），記錄每輪改動
   - 插入位置：第 2 小節「多輪修正節奏」後
   - 成果：「三輪修改紀錄」（v1→v2→v3 + 改進說明）

### CH1-4「System Prompt 設計」

1. **四元素拆解練習**
   - 動作：拆解 Ch 範本 System Prompt，標出 Persona / Task Boundary / Output Lock / Refusal
   - 插入位置：第 2 小節「四個結構元素」後
   - 成果：「範本拆解筆記」

2. **跨輪穩定性測試**
   - 動作：用自建 System Prompt 對話 3 輪，觀察格式是否依然穩定
   - 插入位置：第 3 小節「跨輪對話穩定性」後
   - 成果：「對話記錄 + 穩定性評估」

## 總結建議

### 密度評估
- **CH1-1 / CH1-2：足**（3–4 操作 + 充足實驗空間）
- **CH1-3 / CH1-4：中**（理論足，互動少，需補診斷與測試類練習）
- **PRAC1-1～PRAC1-4：足**（均為完整互動工具，涵蓋「寫→診→做→配」技能鏈）

### Part 1 擴充量
- 合計建議插入 **10–12 個迷你練習**（全部落在 4 個 CH 內）
- PRAC 維持現狀，不新增、不改寫

### 核心發現

1. **CH + PRAC 已形成「講→看→做」閉環**，PRAC 無需動，迷你練習全塞 CH
2. **迷你練習插入節奏**：每小節後 1 個，5–10 分鐘可完成
3. **PRAC1-4 驗收清單缺「長對話測試」環節**，可於階段三補強（或在 CH1-4 新增的跨輪穩定性迷你練習中涵蓋）
4. **視覺風格統一**（米白 + 無印 + 卡片式），迷你練習插入無阻礙
5. **不建議新增 PRAC1-5**：會打破 4CH + 4PRAC 對稱結構，也與其他 Part 不一致

### 風險點
- 若每個 CH 插 3+ 個迷你練習，頁面可能從 2,000 字漲到 3,500+ 字，需注意長度平衡
- 迷你練習若缺「是否完成」的可視化勾選，會退化成裝飾 → 階段二需設計統一的 checkbox / 提交區塊組件

## 給階段二的建議

- **設計統一的「迷你練習」組件樣式**（卡片 + 步驟 + 成果欄 + 自我檢核），於 CH1-1 先做原型、經你確認後批量套用其餘 3 個 CH
- **標記每個迷你練習的預估時間**（5 / 10 / 15 分鐘），方便講師現場節奏控制
- **優先順序**：CH1-2（最核心）→ CH1-3（密度最稀）→ CH1-4 → CH1-1
