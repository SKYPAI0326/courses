個人AI資產管理系統

Prompt、工具、模板、設定——這些都是你的 AI 資產。學會系統性整理，讓每一份努力都能複用、累積、持續增值。

什麼是 AI 資產？
Tier 1 Prompt 庫
經過測試、穩定產出好結果的指令模板。這是最直接可以複用的資產，每次使用都能節省反覆嘗試的時間。
例：週報 Prompt、會議摘要 Prompt、客訴回覆模板
Tier 2 HTML 工具庫
在 Part 3 建立的計算器、流程工具、查詢工具等。放在固定資料夾，需要時直接雙擊開啟。
例：費用計算器、SOP 檢查表、工作日計算工具
Tier 3 個人 Profile 系統
在 CH1-3 建立的角色設定、語調偏好、常用背景資訊。貼入新對話即可讓 AI 快速了解你的背景。
例：職位、產業、常用術語、偏好語氣
Tier 4 學習筆記與踩坑記錄
使用 AI 過程中發現的規律、失敗原因、有效做法。這些記錄本身就是你的 AI 使用能力沉澱。
例：某類任務 AI 容易犯的錯、需要人工驗證的情境
資產管理三原則
📁
統一存放位置
建立一個固定資料夾（如「AI資產庫」），所有資產都放在這裡，不散落在不同地方。
🏷️
清楚命名規則
檔名包含：用途 + 類型 + 版本，例如「週報摘要-prompt-v2」。一眼看出是什麼。
🔄
定期維護更新
每季檢視一次，淘汰過時的資產，記錄新的有效做法。資產要活用才有價值。
命名規則建議
資產類型命名格式範例
Prompt[情境]-prompt-v[版本]週報摘要-prompt-v3
HTML 工具[功能名稱]-tool差旅費計算-tool
文書模板[文件類型]-template客訴回覆-template
Profileprofile-[角色/場景]profile-業務主管
學習筆記note-[主題]-[YYYYMM]note-圖表生成-202501
建立資產庫的五步驟
1
建立資料夾結構
在桌面或雲端硬碟建立「AI資產庫」主資料夾，下面建 Prompts / Tools / Templates / Profile / Notes 五個子資料夾。
2
盤點現有資產
把課程中建立的 Prompt、HTML 工具、Profile 全部找出來，依命名規則重新命名後放入對應資料夾。
3
建立索引清單
用一份試算表或 Notion 頁面列出所有資產：名稱、用途、最後更新日期。讓你快速找到需要的資產。
4
設定觸發機制
決定什麼時候會用到哪些資產，例如「每週一上午 → 打開週報 Prompt」，建立習慣性使用的流程。
5
設定季度維護時間
在行事曆設定每季一次的「AI資產盤點」，30 分鐘，淘汰過時資產，整理新收集的有效 Prompt 與工具。
AI 資產庫 vs 傳統工具箱
Prompt 庫 = 常用工具（錘子、螺絲起子），拿起來就能用
HTML 工具 = 特製治具，解決特定重複性工作
Profile = 工作証明，讓 AI 知道你是誰、需要什麼
學習筆記 = 使用手冊，記錄每個工具的使用訣竅
