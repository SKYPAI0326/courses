單元 7-2

GA4 × GTM 整合：正確安裝追蹤代碼

用 GTM 安裝 GA4 是目前最推薦的做法。這節課帶你完成 GA4 設定代碼的安裝、驗證基礎事件、以及確認 GA4 即時數據正確進入，建立可靠的數據基礎。

學習目標 OUTCOMES
能在 GTM 中建立 GA4 設定代碼，並設定觸發條件為「所有頁面」
能使用 GTM 預覽模式和 GA4 即時報告，驗證追蹤代碼是否正確運作
能識別並說明 GA4 的六個增強型評估事件（Enhanced Measurement）
INSTALLATION STEPS · 安裝步驟

七步驟完成GA4 × GTM 整合

1
取得 GA4 測量 ID
在 GA4 中：管理 → 資料串流 → 選擇你的串流 → 複製測量 ID（格式：G-XXXXXXXXXX）
GA4 → 管理 → 資料串流
2
在 GTM 新增代碼
GTM 左側選單 → 代碼 → 新增 → 代碼設定 → 選擇「Google Analytics：GA4 設定」
GTM → 代碼 → 新增
3
填入測量 ID
在代碼設定中，貼上剛才複製的 GA4 測量 ID（G-XXXXXXXXXX）。代碼命名建議：GA4 - 設定代碼
4
設定觸發條件：所有頁面
觸發條件 → 選擇「All Pages（所有頁面）」。確保 GA4 代碼在網站每個頁面都會觸發。
5
開啟增強型評估（Enhanced Measurement）
回到 GA4 → 管理 → 資料串流 → 選擇串流 → 開啟增強型評估。這會自動追蹤捲動、外連點擊、搜尋、影片等行為。
6
GTM 預覽模式驗證
GTM 右上角 → 預覽 → 輸入你的網站 URL → 開始預覽。在真實網站上操作，確認 GTM Debug 面板顯示代碼已觸發（Tags Fired）。
7
GA4 即時報告驗證 → 發布
在 GA4 → 報告 → 即時，確認有看到「1 個使用者」在 30 分鐘內。驗證無誤後回 GTM 發布版本。
ENHANCED MEASUREMENT · 增強型評估

免費的自動追蹤：六個增強事件

GA4 的增強型評估功能在安裝後自動追蹤六種常見用戶行為，不需要額外設定。這是 GA4 相比 UA 最大的進步之一。

事件名稱 追蹤什麼 行銷用途
scroll 用戶捲動到頁面 90% 判斷內容完讀率，找出跳脫點
click 點擊外部連結（離開網站） 了解用戶去哪個外部資源
view_search_results 站內搜尋 了解用戶找什麼，補充內容
video_start / progress / complete 嵌入 YouTube 影片的播放行為 影片完看率、影片對轉換的貢獻
file_download PDF / Excel 等檔案下載 追蹤資源下載的行為，評估 Lead Magnet 效果
form_start / form_submit 表單互動（部分網站有效） 評估表單填寫流程
⚠️
注意：增強型評估的 form_submit 並非對所有表單都有效（取決於網站的表單實作方式）。若追蹤重要表單，建議用 GTM 設定自訂觸發條件以確保準確。
