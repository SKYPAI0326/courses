單元 2B-3

行銷專屬 Prompt 庫

這節課的目標是你離開教室時，手上已經有一套可以立即使用的 Prompt 模板庫——涵蓋文案、企劃、競品分析、受眾研究四大類。更重要的是，你學會了如何持續擴充這個庫，讓它成為你的 AI 工作資產。

學習目標 OUTCOMES
能直接套用文案、企劃、競品、受眾四類 Prompt 模板，完成當日行銷工作
能根據模板結構自行修改填入品牌變數，不需要每次從零寫 Prompt
能建立個人的 Prompt 庫管理系統，持續累積有效的模板
CONCEPT · 為什麼要建庫

每次從零開始，是在浪費你最有價值的資源

如果你每次需要 AI 協助時，都重新思考怎麼寫 Prompt，你就是在用「思考 Prompt」的時間替代本應「思考行銷策略」的時間。這不是提升效率，是換了一個不同的低效率。

Prompt 庫的概念很簡單：把那些「你曾經試過、有效的 Prompt」儲存起來，填入變數後直接使用。時間久了，你的庫會越來越精準，越來越符合你的品牌和工作場景——這是一個會增值的工作資產。

類比
就像好的設計師不每次都重新建立色彩系統和元件庫，好的行銷人也不每次都重新寫同類型的文案。Prompt 庫是你的 AI 工作的「設計系統」——一致、可複用、可迭代優化。
TEMPLATE LIBRARY · 文案類

文案 Prompt 模板

文案 IG 貼文文案生成器
你是一位擅長社群文案的品牌寫手，熟悉 [目標受眾] 的語言和生活情境。 請為 [品牌名稱] 寫一篇 IG 貼文文案。 產品/主題：[產品或話題] 品牌語氣：[例：溫暖平靜 / 活潑直接 / 知識型] 本次訴求重點：[主打什麼價值或情境] 格式：繁體中文，[字數] 字以內，三段式（場景開場 → 核心訊息 → CTA），末尾附 5 個 hashtag。 限制：不使用 [禁忌詞]；語氣不過度促銷；不捏造數字。
使用場景：日常貼文、新品上市、節日行銷、品牌訊息傳遞
文案 廣告標題批量生成（A/B 測試用）
你是資深廣告文案師，熟悉數位廣告的點擊率優化邏輯。 請為以下廣告生成 10 個不同切角的標題： 產品：[產品名稱與核心功能] 目標受眾：[受眾描述] 廣告平台：[Google 搜尋廣告 / Meta / IG] 字數限制：每個標題 [字數] 字以內 請涵蓋以下切角（各至少 2 個）： - 問題/痛點切角（讀者的困擾） - 結果/利益切角（使用後的改變） - 好奇心切角（讓人想點進去的懸念） - 社會認同切角（他人的使用經驗） - 緊迫感切角（限時或稀缺）
使用場景：廣告投放前的 A/B 測試素材準備，需要快速生成多個變體
文案 Email 主旨行優化
你是 Email 行銷專家，了解台灣消費者的開信行為。 以下是一封 Email 的現有主旨行： 「[現有主旨]」 請改寫成 5 個版本，分別套用： 1. 好奇心驅動（讀者想知道後面說什麼） 2. 個人化感受（感覺是寫給我的） 3. 數字/具體利益（清晰的價值承諾） 4. 緊迫感（有時間限制） 5. 問句（引發讀者自我對話） 每個版本在 [字數] 字以內，附上一句說明這個版本適合哪種受眾狀態。
使用場景：EDM 活動、電子報、再行銷郵件的主旨行 A/B 測試
TEMPLATE LIBRARY · 企劃類

企劃 Prompt 模板

企劃 社群內容月曆草稿
你是社群行銷策略師，擅長為 [產業/品牌類型] 規劃內容策略。 請為 [品牌名稱] 規劃 [月份] 的 IG 內容月曆草稿。 品牌資訊： - 受眾：[受眾描述] - 本月行銷目標：[品牌知名度 / 新品推廣 / 促銷轉換] - 本月重要節點：[節日/活動/產品上市日期] 請規劃 12 篇貼文（每週約 3 篇），每篇包含： - 發文日期 - 內容類型（教育/娛樂/產品/互動/UGC） - 貼文主題描述（一句話） - 視覺建議（圖片風格提示）
使用場景：月初的內容規劃，作為後續詳細製作的骨架
企劃 行銷活動企劃框架
你是有 [N] 年經驗的整合行銷企劃，熟悉台灣市場的消費者行為。 請為以下活動草擬企劃框架： 活動名稱/主題：[活動名稱] 目的：[品牌目標] 預算規模：[大/中/小] 執行期間：[時間區間] 請依序完成： 1. 活動核心概念（一段話，說明 why 和 what） 2. 目標受眾與洞察（他們為什麼會參與這個活動） 3. 渠道組合建議（各渠道扮演的角色） 4. 執行時程大綱（週為單位） 5. 預期 KPI 設定建議（3 個可量化指標） 6. 潛在風險與備案 請用條列式呈現，重點突出，方便後續展開。
使用場景：活動企劃初期的框架搭建，作為向主管或客戶提案的草稿基礎
TEMPLATE LIBRARY · 競品 / 受眾類

分析 Prompt 模板

競品 競品定位分析
你是品牌策略顧問，擅長競品分析與市場定位。 請分析以下 [N] 個競品在 [市場/產業] 的定位： 競品：[品牌A、品牌B、品牌C] 請依序完成： 步驟1：各品牌的核心定位訴求（各 2-3 句） 步驟2：三品牌的共同點（市場共識是什麼） 步驟3：三品牌各自的差異化角度 步驟4：市場空缺分析（哪個受眾或訴求目前沒有人在做？） 步驟5：對我方品牌的策略建議方向 ⚠ 請注意：以下分析僅基於公開資訊推斷，請不要捏造具體數字或未公開的市場份額。
使用場景：品牌策略規劃前的市場掃描，注意驗證 AI 輸出的事實內容
受眾 Persona 生成器
你是消費者研究專家，擅長建立有深度的受眾人物誌。 請為以下品牌建立 2 個不同的 Persona： 品牌：[品牌名稱與產品類型] 大致受眾範圍：[年齡 / 性別 / 地區 / 職業] 每個 Persona 請包含： - 姓名與基本資料（虛構但具體） - 一天的典型生活場景（從起床到睡前，100 字） - 與我們品牌相關的痛點或需求（3 條） - 他/她如何搜尋和發現新產品 - 對我們品牌的潛在疑慮是什麼 - 打動他/她的一句核心訊息 請讓兩個 Persona 呈現不同的消費動機和生活樣貌。
使用場景：新品上市前的受眾研究、廣告受眾設定、內容策略規劃
BUILD YOUR OWN · 建庫方法

如何打造你自己的 Prompt 庫

1
選定工具：Notion / Google Docs / Obsidian 任一皆可 重點是分類清楚且隨時能找到。建議用「任務類型」分類（文案/企劃/分析/研究），而不是按時間排列。
2
每個 Prompt 存三個欄位 ① Prompt 模板本身（含 [變數] 標記）② 適用場景說明 ③ 上次產出的滿意輸出範例。有了例子，下次一眼就知道這個模板適不適合。
3
只儲存真正用過且有效的 「聽說這個 Prompt 很厲害」不算。必須是你親自測試、調整、得到滿意輸出的版本。收藏太多沒用的 Prompt 比沒有庫還糟——你不會用到它們的。
4
定期更新：每個月整理一次 AI 工具在進化，你對 Prompt 的理解也在進化。每個月花 15 分鐘回顧庫——刪掉不再有效的模板，把近期迭代出的更好版本替換進去。
5
品牌化：填入你的固定變數 把你品牌的常用資訊（受眾描述、品牌語氣、禁忌詞）直接填入模板，建立「品牌版 Prompt」——這樣每次使用時只需要填入任務相關的變數，不需要每次重新描述品牌背景。
