M1 · 消費者心理學
單元 1-1

快思慢想：System 1/2 與情緒決策先於理性

你寫了一份功能齊全、論據充分的產品說明，消費者卻沒被說服——因為購買決策不是在你溝通的那個系統裡發生的。這節課拆解 Kahneman 的雙歷程理論，讓你看懂消費者真正怎麼做決定。

學習目標 OUTCOMES
能說明 System 1 與 System 2 的分工，並指出大多數購買決策發生在哪個系統
能辨識一則行銷訊息是「System 1 取向」還是「System 2 取向」
能設計「先觸發 System 1、再用 System 2 合理化」的訊息結構
CONCEPT · 雙歷程理論

人有兩套思維系統，同時在運作

Daniel Kahneman 在《快思慢想》中提出的雙歷程理論（Dual Process Theory）是現代行為經濟學的核心。簡單說：你的大腦裡有兩個「決策者」隨時待命，一個快、一個慢。

System 1 · 快思
自動、直覺、情緒驅動
幾乎不耗費精力
依賴捷徑（heuristics）
對視覺、情緒、社交訊號敏感
大多數日常決策都在這裡完成
System 2 · 慢想
審慎、邏輯、需要集中注意力
消耗大量認知資源
能覆蓋 System 1 的直覺
對數字、論據、條件比較敏感
通常用來「事後合理化」System 1 的決定

關鍵不在於哪個系統「比較好」，而在於：System 2 很懶。它消耗資源，所以大腦預設讓 System 1 接管。只有當 System 1 處理不來（例如算一道兩位數乘法），System 2 才會被叫醒。對行銷人而言，這代表你大多數時候面對的，是一個運行在 System 1 的消費者。

INSIGHT · 購買的真實時序

消費者用 System 1 買，再用 System 2 合理化

行銷人最常犯的錯誤，是用 System 2 的語言設計溝通內容——詳盡的功能清單、理性的比較表格、技術規格說明——試圖影響一個主要運行在 System 1 的購買決策。結果是：訊息正確，但打不到人。

真實的購買決策通常是這樣發生的：

1
看到（System 1 觸發情緒反應）一張照片、一句標題、一個社交訊號，在零點幾秒內產生「想要」或「無感」。
2
產生購買衝動這個衝動是情緒性的，還沒有理由，但已經傾向「要」。
3
用 System 2 尋找合理理由「它的材質確實比較好」「分期下來其實不貴」「我本來就需要」——理性在這裡上場，但它是來找藉口的，不是來做決定的。
4
購買，並再次用 System 2 說服自己這個決定是對的降低購買後的認知失調，讓自己對這次決定感覺良好。
◉
你的行銷任務有先後順序：先觸發 System 1 的「想要」，再用 System 2 的論據幫消費者「合理化」他們已經想要的東西。順序反了，再好的論據也只是對著一個沒被打開的腦袋說話。
APPLICATION · 設計含義

這對你的素材設計意味著什麼

理解雙系統後，行銷素材的優先順序就清楚了。一則完整的訊息應該同時餵兩個系統，但順序與比重要對：

前
第一眼給 System 1主視覺、情緒、第一句標題——負責「停下來」與「想要」。這部分若失敗，後面的論據根本沒人看。
中
中段給 System 2規格、比較、評價、保證——負責「找理由」。這部分讓已經心動的人有台階下，也降低猶豫。
後
結尾再回 System 1行動呼籲（CTA）要簡單、低摩擦、帶一點急迫感——讓衝動不要在猶豫中冷掉。
⚠
常見錯誤：把素材做成一份「規格說明書」，整篇都在餵 System 2。這對「已經決定要買、只差比價」的人有效，但對「還沒被打動」的多數人無效——而後者才是你大部分的潛在客戶。
AI 接口 · 文案系統檢測

用 AI 檢查你的文案打到哪個系統

寫完一段行銷文案後，自己很難客觀判斷它是 System 1 還是 System 2 取向。把它丟給 AI 做一次「系統歸類」，能快速看出失衡。

AI 接口 Prompt：雙系統文案檢測
你是行為經濟學顧問。請分析我這段行銷文案： 「[貼上你的文案]」 依 Kahneman 雙歷程理論判斷： 1. 逐句標記每句主要訴諸 System 1（情緒/直覺/視覺）還是 System 2（邏輯/數字/比較） 2. 整體比重大約是 System 1 幾成、System 2 幾成 3. 第一句是否能在 3 秒內觸發 System 1 的「想要」？若不能，給 2 個改寫版本 4. 結尾 CTA 是否低摩擦？ 請具體、不要客套。
PRACTICE · 動手練習

換你試試

本單元任務
挑一個你正在經營（或熟悉）的產品，找出它目前的主力行銷素材（一則貼文、一張 banner 或一頁產品頁）。

1. 標出哪些部分在餵 System 1、哪些在餵 System 2。
2. 判斷第一眼的元素能不能在 3 秒內讓人「想停下來」。
3. 若整篇偏 System 2，改寫第一句與主視覺概念，讓它先觸發情緒。
4. 用上方 AI Prompt 驗證改寫後的版本。
