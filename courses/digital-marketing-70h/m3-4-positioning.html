M3 · 市場洞察與策略定位
單元 3-4

品牌定位工作坊

定位不是公司說自己是什麼，而是顧客心裡記住了什麼。這節課用 4P、4C、USP 三個工具，幫你把 M3 前三節課學到的市場洞察，濃縮成一句能指引所有行銷決策的核心定位。

學習目標 OUTCOMES
能用 4P 和 4C 框架分析品牌的行銷組合，找出顧客視角和企業視角的落差
能套用 USP 公式，寫出一句話的價值主張
能對定位陳述進行「三個測試」，確認定位是否有力且可執行
FRAMEWORK · 4P vs 4C

從企業視角到顧客視角

4P 是企業的行銷組合工具；4C 是對應的顧客視角。兩者並用，能幫你找出「你以為在做的事」和「顧客實際感受到的事」之間的落差。

4P — 企業視角
Product 產品
我們提供什麼功能和規格？
4C — 顧客視角
Customer Value 顧客價值
顧客獲得了什麼解決方案和感受？
4P — 企業視角
Price 價格
我們定價多少？
4C — 顧客視角
Cost 顧客成本
顧客付出的不只是金錢，還有時間、學習成本、風險感。
4P — 企業視角
Place 通路
我們在哪裡銷售？
4C — 顧客視角
Convenience 便利性
顧客能多輕鬆買到？購買過程有多少摩擦？
4P — 企業視角
Promotion 推廣
我們說了什麼訊息？
4C — 顧客視角
Communication 溝通
顧客聽到了什麼？他們能雙向互動嗎？
TOOL · USP 提煉

一句話定位：USP 公式

USP（Unique Selling Proposition）是你的品牌在市場上的獨特主張——不是所有優點的清單，而是「為什麼選你而不是競品」的單一最有力理由。

USP 填空公式
對於 [目標受眾]，
我們是唯一能夠 [獨特功能或解決方案] 的品牌，
因為 [可信的理由]。
範例：
對於 35-50 歲、注重成分的女性，
我們是唯一能夠 在不犧牲成分品質的前提下提供平價保濕方案 的品牌，
因為 我們直接與台灣原料商合作，省去代理商層層加價。
VALIDATION · 定位三測試

寫完 USP，用三個問題驗證

01
差異化測試：競品能說同樣的話嗎？把你的 USP 替換成競品的名字，如果同樣成立，你的定位不夠獨特。繼續挖掘更深層的差異點。
02
相關性測試：目標受眾在乎這件事嗎？USP 說的優點必須是目標受眾的核心決策標準之一。「獲獎無數」如果顧客不在乎獎項，就沒有意義。
03
可信度測試：你有辦法證明這件事嗎？每一個 USP 主張都需要可驗證的支撐——成分標籤、第三方認證、顧客見證、具體數字。說不出來，受眾不會信。
→
完成 M3 的四個單元，你已有了環境分析（競品在哪）、受眾研究（顧客是誰）、旅程設計（接觸點在哪）和品牌定位（你要站哪裡）。這四個元素加在一起，就是進入 M4 社群行銷之前必備的策略底盤。
AI 接口 · 用 AI 提煉 USP 與價值主張

三行品牌資料 → AI 拆出 USP 候選與定位陳述

M3 收尾的最後一個示範：把你的品牌資料 + 競品差異交給 AI，它幫你產出 3 個 USP 候選 + 一句定位陳述。重點不是 AI 寫得多神，是它強迫你把「差異化」說到一句話內可被傳遞。

1
從 M3-2 拿出你的 Persona 一句話描述就是那句「對 [目標受眾]，[品牌名] 是…」的 STP Positioning 草稿。沒跑過 M3-2 演練？先回去補，這個 prompt 需要那份輸入。
2
列出 3 個競品各自的核心主張（1 句話就夠）去看他們的官網首頁或廣告，找「他們說自己是什麼」。不需要深入分析，一句話抓重點。
3
用三個測試檢查 AI 給的 USP 候選每個 USP 候選出來後，套上方「定位三測試」的三個問題。過不了差異化測試的，直接刪掉，不要帶進下一階段。
把這段貼進 ChatGPT 或 Claude
請扮演品牌策略顧問，幫我提煉 USP（Unique Selling Proposition）與一句定位陳述。 【品牌】[名稱 + 主力商品 + 價格區間] 【主 Persona 一句話描述】[填入，從 M3-2 拿] 【3 個直接競品的核心差異點】[一一列出每個競品「主打什麼」] 【我認為的優勢】[3 點，越具體越好] 請輸出： (1) 3 個 USP 候選（不同切入角度：功能 / 情感 / 場景） - 每個 USP：一句話 + 對應 Persona 的哪個痛點 + 為何競品難複製 (2) 一句定位陳述（套「對 [Persona] 而言，[品牌] 是 [類別] 中 [差異化] 的選擇，因為 [理由]」） (3) 4P 與 4C 對齊建議 - 產品 / 價格 / 通路 / 推廣 各列 1 點要怎麼配合這個定位 - 顧客需求 / 成本 / 便利性 / 溝通 各列 1 點 最後請標：哪個 USP 風險最低、哪個短期最有效、哪個長期建立護城河最強。
ChatGPT 大概會回（以「山徑咖啡」為例）

3 個 USP 候選

(1) 功能切入：「2024 年實測穩定度排名前 3 的台製手沖壺」
　對應痛點：1-3 年新手怕踩雷｜競品難複製：他們是進口商，無自家實測數據

(2) 情感切入：「給走在手沖路上、想升級下一把壺的人」
　對應痛點：新手卡關自我懷疑｜競品難複製：他們訴求「專家」而非「同路人」

(3) 場景切入：「永康店實體可試手感再買的精品咖啡器具」
　對應痛點：怕網購買錯｜競品難複製：他們無實體店

一句定位陳述

「對於走在手沖路上 1-3 年想升級下一把壺的咖啡愛好者而言，山徑咖啡是手沖器具中『可實體試手感 + 有實測數據可信賴』的選擇，因為我們同時擁有自家設計團隊與永康實體店。」

4P / 4C 對齊建議

• 產品（4P）：每款產品頁面附實測數據卡（壺嘴角度、穩定度、出水秒數）
• 價格：定價中段（不打低價戰）
• 通路：強化永康實體店體驗 + 自家官網（不上蝦皮稀釋品牌）
• 推廣：每月 1 場現場試手感工作坊

• 顧客需求（4C）：解決「不知道哪把適合自己」
• 成本：除金錢外、加「實體試用 = 時間成本變零」
• 便利性：永康店 14:00-20:00 隨時可走進去摸
• 溝通：用「升級」「同路人」取代「專家」「精品」語彙

USP 風險矩陣

• 風險最低：(3) 場景切入（事實基礎最穩、無法被否認）
• 短期最有效：(2) 情感切入（最容易在社群引共鳴）
• 長期護城河最強：(1) 功能切入（數據累積 3 年後競品無法追趕）

實際輸出每次略有差異，但「3 USP + 定位句 + 4P/4C + 風險矩陣」四件套結構穩定。AI 給的是初稿，必須拿 1-2 個小型測試（廣告 A/B / 落地頁 / 線下訪問）真實驗證哪個 USP 在你的市場最有 traction。
M3 收尾原則：USP 不是一句話寫漂亮、是一句話讓 Persona 在 3 秒內知道「這個牌子是給我的」。一句話過不了，4P/4C 都白布局。
