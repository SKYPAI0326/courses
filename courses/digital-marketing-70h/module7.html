MODULE 7 · 5 小時

GTM：數據追蹤工程

你看到的 GA4 數據，是由追蹤代碼收集來的。GTM（Google Tag Manager）讓行銷人不用修改網站程式碼，就能自己部署 GA4、Meta Pixel、廣告轉換追蹤。掌握 GTM，才是真正的數據自主。

時數：5 小時
單元：4 單元
AI 接口：GTM 設定 Debug / 事件規劃
UNITS · 單元導覽

從概念到實作，四個層次

7-1
GTM 核心概念
容器 / 代碼 / 觸發條件 / 變數四元素，版本管理與預覽模式。
7-2
GA4 × GTM 整合
用 GTM 安裝 GA4 設定代碼、驗證基礎事件、連結 Google Ads。
7-3
自訂事件追蹤
按鈕點擊 / 表單送出 / 捲動深度 / 外部連結點擊 實作。
7-4
進階追蹤
Meta Pixel 安裝、電商 Data Layer 設定、GTM 除錯技巧。
AI 接口

本模組的AI 輔助實作

M7 AI 接口任務
  • 用 AI 規劃追蹤計畫：描述你的網站類型和轉換目標，讓 Claude 列出需要追蹤的事件清單與 GTM 設定步驟
  • GTM Debug 協助：把 Preview 模式的事件截圖描述給 AI，請它判斷是否正確觸發，或找出設定問題
  • Data Layer 設計：描述電商產品資料結構，讓 AI 生成 dataLayer.push() 的程式碼草稿給開發人員參考
