## M1 消費者心理學

### M1-1 System 1/2 雙歷程文案拆解
```
你是行為經濟學顧問。請分析我這段行銷文案：「[貼上文案]」
依 Kahneman 雙歷程理論判斷：
1. 逐句標記每句訴諸 System 1 還是 System 2
2. 整體比重大約 System 1 幾成、System 2 幾成
3. 第一句是否能 3 秒內觸發 System 1？若不能給 2 個改寫版本
4. 結尾 CTA 是否低摩擦？
請具體、不要客套。
```

### M1-2 Cialdini 六原則拆解
```
你是說服心理學顧問。我貼上一個產品頁/廣告的文字：「[貼入]」
請依 Cialdini 六原則：
1. 逐一判斷該頁用了哪些原則，引用對應的句子
2. 每個原則標「強 / 中 / 弱」
3. 指出完全沒用到的原則
4. 評估有沒有疑似「虛假訊號」
5. 建議優先補強哪 2 個原則、怎麼補
務必具體引用原文。
```

### M1-3 4 個行為偏誤框架改寫
```
你是行為經濟學文案顧問。產品事實：「[寫下]」
請各寫一個版本（每個 1-2 句）：
1. 錨定效應框架
2. 損失趨避框架
3. FOMO 框架
4. 中性事實陳述（對照組）
每個版本後標：在漏斗哪個階段最適用？最後提醒：哪個版本有「訊號造假」風險。
```

### M1-4 JTBD 訪談分析
```
你是需求研究顧問，熟悉 Jobs-to-be-Done 方法。
逐字稿：「[貼上]」
請分析：
1. 真正「雇用」產品要完成的 Job（用「在___情況下，我想___，以便___」）
2. 真實痛點（用他們的原話）
3. 考慮過的替代方案 → 推導真正競爭對手
4. 痛點翻成行銷切入點
請標注哪些是逐字稿明確說的、哪些是你的推論。
```

## M2 生成式 AI 體系

### M2A-1 LLM 原理：上下文差異實測
```
（無上下文版）幫我寫一段品牌定位描述，是賣手沖咖啡器具的台灣品牌。
（有上下文版）
事實資料：
- 品牌名：山徑咖啡（Trail Coffee）
- 主力商品：自製竹節手沖壺、不鏽鋼分離濾杯，2024 年上市
- 對象：剛入門 1-3 年的咖啡愛好者，預算 1,500-4,000 元
請只用以上事實寫定位，不要自行加入沒給的數字或合作對象。
```

### M2A-2 AI 工具堆疊建議
```
我是 [職位]，主要工作場景如下：
1. [常做的 3-5 件任務]
2. [我目前用的工具]
3. [每月可投入 AI 工具的預算]

請依以下框架建議我的 AI 工具堆疊：
【Tier 1 核心對話模型】1 個（ChatGPT / Claude / Gemini）
【Tier 2 專業任務工具】2-3 個
【Tier 3 整合層】1 個（如 Make.com）

針對每個工具：為何適合我、入門門檻、預估每週可省的時間。
最後排除：跟現有工具重複的、或預算明顯不夠的。
```

## 使用建議

1. **挑出你最常用的 5-8 個 Prompt**，存到 Google Keep 或 Notion，加標籤
2. **每次使用後微調**，把實際用得順的版本寫回 .md 檔
3. **記錄使用次數**，找出 ROI 最高的 5 個 Prompt 持續優化
4. **不要每次都從頭打 Prompt**，從庫複製比從 AI 對話歷史翻快 10 倍
