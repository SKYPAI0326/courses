Module 1 · 4 單元 · 6 小時
消費者心理學

行銷的作業系統

工具會過時，人性不會。這個模組從消費者決策的底層邏輯出發——快思慢想、六大影響力原則、行為偏誤、需求挖掘——建立一套讓後續所有工具都變得更有效的思維框架。

學習目標 OUTCOMES
能解釋 System 1/2 的差異，並舉出 3 個數位行銷場景中被 System 1 驅動的購買決策
能識別一個品牌的行銷策略中用到了哪些 Cialdini 影響力原則，並指出缺失之處
能用 Jobs-to-be-Done 框架重新定義一個產品的核心價值主張，並連結到痛點轉換點
CONTEXT · 為什麼第一個模組是心理學

大多數行銷課程從錯誤的地方開始

學社群行銷→學平台規則。學 SEO →學關鍵字工具。學廣告→學出價邏輯。這些都沒錯，但缺了一個更根本的問題：人為什麼點擊？為什麼停留？為什麼購買？為什麼推薦？

如果你不理解消費者的決策機制，那 SEO 只是排名遊戲，廣告只是燒錢競賽，社群只是發文頻率的比拼。反過來，一旦你理解了人性，同樣的工具就會產生截然不同的效果。

◎
本課程的核心主張
行銷策略的品質，取決於你對消費者心理的理解深度。工具是放大器，放大的是你對人性的洞察——或你對人性的誤解。

PREVIEW · System 1 vs System 2

消費者用 System 1 買東西，再用 System 2 合理化

Daniel Kahneman 的雙歷程理論（Dual Process Theory）是現代行為經濟學的核心。簡單說：人有兩套思維系統在同時運作。

System 1 · 快思
自動、直覺、情緒驅動
幾乎不耗費精力
依賴捷徑（heuristics）
對視覺、情緒、社交訊號敏感
大多數日常決策都在這裡完成
System 2 · 慢想
審慎、邏輯、需要集中注意力
消耗大量認知資源
能覆蓋 System 1 的直覺
對數字、論據、條件比較敏感
通常用來「事後合理化」System 1 的決定

行銷人最常犯的錯誤，是用 System 2 的語言設計溝通內容——詳盡的功能清單、理性的比較表格、技術規格說明——試圖影響一個主要運行在 System 1 的購買決策。

◉
實際的購買時序往往是：
看到（System 1 觸發情緒反應）→ 有購買衝動 → 用 System 2 尋找合理理由 → 購買 → 再次用 System 2 說服自己這個決定是對的。

你的行銷任務：先觸發 System 1，再用 System 2 的邏輯幫消費者「合理化」他們已經想要的東西。
PREVIEW · Cialdini 影響力六原則

六個可預測地影響人類行為的槓桿

Robert Cialdini 的《影響力》研究了數十年的人類說服行為，歸納出六個在跨文化、跨情境中穩定有效的原則。以下是它們在數位行銷中的直接應用：

互惠
Reciprocity
先給予，對方會感到有義務回報。先提供價值，消費者更容易轉換。
→ 免費試用、電子書下載、免費診斷服務、有價值的 YouTube 教學
稀缺
Scarcity
越難得到的東西感覺越有價值。限量或限時能顯著提升行動意願。
→ 「剩餘 3 席」、「優惠截止倒數計時」、「限量商品」
社會證明
Social Proof
當不確定如何選擇時，人傾向跟隨其他人的行為。
→ 星級評分、使用人數、用戶見證、UGC、「熱銷中」標籤
權威
Authority
人傾向信任具備專業知識或地位的人。展示資歷或背書能降低疑慮。
→ 專家推薦、媒體報導、認證標章、KOL 合作、創辦人背景
喜好
Liking
人更容易被自己喜歡的人說服。品牌個性與形象直接影響轉換率。
→ 品牌語氣設計、創辦人故事、使用接近目標受眾形象的代言人
承諾與一致性
Commitment
人傾向維持與過去行為一致的選擇。小承諾帶動大承諾。
→ 免費訂閱 → 付費升級、填問卷 → 接受推薦、試用 → 購買
PREVIEW · 行為偏誤

四個行銷人必須掌握的認知捷徑

錨定效應 Anchoring
人對第一個接收到的數字（錨點）產生強烈依賴，後續判斷都以此為基準。
原價 $3,000 劃掉，現在只要 $1,200——$3,000 就是錨點，讓 $1,200 感覺超划算。
損失趨避 Loss Aversion
失去某樣東西的痛苦，約是獲得同樣東西的快樂的兩倍強烈（Kahneman）。
「不要錯過」比「得到更多」有效。「您有一個未使用的優惠」比「快來領優惠」轉換率更高。
FOMO
Fear Of Missing Out。看到他人的正面體驗或機會時，感到焦慮與遺憾的心理狀態。
「已有 5,000 人參加這個工作坊」、「你的朋友 A 也在用這個工具」——觸發不想落後的衝動。
峰終定律 Peak-End Rule
人對一段體驗的記憶，主要由「最高峰」與「結束時」的感受決定，而非整體平均值。
客服糟糕但結帳超順暢，整體評價可能還不錯。設計「最後一哩路」的體驗至關重要。
⚠
倫理紅線：影響力原則與行為偏誤是強大的工具，但有明確的倫理邊界。製造虛假稀缺（庫存其實很多）、偽造社會證明（假評論）、利用脆弱族群的損失趨避——都不只是道德問題，長期而言也是品牌最大的風險。課程中的所有應用討論，都以「真實、透明、對消費者有益」為前提。
PREVIEW · 需求挖掘

從「賣什麼」到「解決什麼」

Maslow 的需求層次告訴我們，人的需求從生理安全到自我實現層層遞進。但行銷的挑戰在於：消費者買的往往不是產品本身，而是產品帶來的「狀態改變」。

Clayton Christensen 的 Jobs-to-be-Done（JTBD） 框架把這個洞察具體化了：

◎
核心問句：「在什麼情況下，消費者雇用這個產品來完成什麼『工作』？」

麥當勞早晨的奶昔不是被「飢餓的人」雇用的——它被「開車通勤、需要一個能撐到午餐、單手操作、不會弄髒手的夥伴」的人雇用。理解這個 Job，行銷策略就完全不同了。

JTBD 的實務應用：與其問「你為什麼買這個？」，問「上一次你決定買這個是什麼時候？當時你在做什麼？想解決什麼問題？考慮過哪些替代方案？」——這五個問題能挖出遠比問卷調查更真實的需求洞察。
