公文模板套用練習

把 PRAC1 填好的三張交辦單升級：在第四欄加入具體格式規格，對比升級前後的差距。25 分鐘，三個任務。

完成後你能做到 · LEARNING OBJECTIVE

比較加入前後的 AI 輸出品質差異

任務說明 · SETUP

你在 PRAC1 填的那三張交辦單，現在要升級。

上一次，你學會把四個欄位填滿——你是誰、你要做什麼、有什麼背景、要什麼樣子。填完之後，AI 的輸出品質比什麼都不填好很多。但你可能還是覺得「語氣不太像我們機關的」或「格式跟我們平常發函的不大一樣」。

原因很單純：你在「要什麼樣子」那欄填的格式要求還不夠具體。CH4-1 教了正向指令的三招；CH4-2 給了三種公文格式的完整模板。現在，把這兩件事疊加進你已有的交辦單，感受一下有指定格式模板跟沒有的差距。

操作要點提醒 · CH4-1 + CH4-2 重點複習
  • 正向指令三招：指定分幾段、指定語氣（公文體 / 白話）、指定字數範圍；不用「不要太長」「不要太正式」等否定形式
  • 格式規格的位置：格式規格寫在「要什麼樣子」的第四欄，不要寫在「你要做什麼」
  • 三種格式模板：陳情回函（敬覆者 + 事實說明 + 處理結果 + 結語）、開會通知（主旨 + 說明 + 請各機關配合事項）、簽呈（主旨 + 說明分點 + 擬辦）
  • 字數要跟格式一起寫：格式模板要搭配字數上限才有效；只給格式不給字數，AI 仍可能輸出 600 字
需要準備的東西 · BEFORE YOU START
  • PRAC1 填好的三張交辦單（情境 1–3），放在手邊對照
  • 開啟 ChatGPT 或 Gemini 網頁版（不需要付費帳號）
  • 本頁下方各任務有「升級操作」說明，直接照著加入第四欄即可

常見錯誤 · COMMON PITFALLS
錯誤 1格式規格填在錯誤的欄位
現象：格式規格和任務描述混在「你要做什麼」那欄，導致 AI 輸出格式有時符合、有時不符合
原因：學員把「格式」理解為「任務的一部分」，所以填在第二欄
解法：格式規格屬於「要什麼樣子的回答」，永遠寫在第四欄。第二欄只說「起草一封陳情回函」，第四欄才說「格式：三段式，敬覆者 + 事實說明 + 處理結果 + 結語」
錯誤 2加了格式模板但字數仍然超標
現象：加了格式模板後，AI 輸出的字數比指定範圍多出 50–100 字
原因：只給格式模板沒給字數上限，或字數上限寫在最後，AI 優先照格式、把字數要求當次要
解法：字數上限要緊跟在格式說明後面，格式和字數一起寫。例如「格式：三段式……；全文 200 字以內」，不要把字數另起一行或放在最後
錯誤 3格式對了但語氣變成罐頭文件
現象：升級後 AI 輸出格式正確，但語氣變得過於制式，讀起來像罐頭文件
原因：指定了格式模板後，AI 傾向套標準格式，語氣彈性降低
解法：在語氣要求那行加一個修飾詞。例如「語氣正式但避免套話」或「用公文體，但結語段用白話說明」，讓 AI 知道格式和語氣可以分開調整
檢核題 · QUIZ
Q1 你要起草一封陳情回函，下面哪個「要什麼樣子」的填法最能確保 AI 輸出直接符合機關格式？
正確答案：C。A 和 D 未指定段落結構，AI 會自行決定幾段；B 是否定指令，沒有具體格式說明；只有 C 同時指定了結構名稱、段落順序、字數上限和語氣，AI 的輸出空間被充分限縮。
