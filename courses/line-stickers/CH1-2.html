角色設定 + AI 角色一致性的 3 種手法

這堂只做兩件事：寫一份讓 AI 看得懂的角色設定卡，再學 3 種手法把這份設定卡用出來。設定卡沒寫好，下一堂你產 8 張表情會全是不同的貓。

(01) · 場景切入

8 隻不同的貓

為什麼這很重要
設定卡是彈，這堂沒裝好，下一堂你真的去產 8 張時就是空槍。

想像你要做一隻「橘貓上班族」的 8 張表情貼圖。第一張「笑」畫得超可愛——圓滾滾橘色虎斑貓戴著白領巾，剛剛好。接著第二張「哭」——AI 畫了一隻花紋比較斜、領巾變成藍色、體型也瘦了一圈的橘貓。第三張「生氣」——這次直接變成花貓，連橘色都沒了。

如果你什麼都不做，只是用 AI 隨便產 8 張，你會得到 8 隻不同的貓。LINE 審核員看到這個套組，會直接退件，理由就一條：「角色不一致」。

—
這堂課只做兩件事：教你寫一份「角色設定卡」（把你腦中的角色寫到 AI 看得懂），以及教你 3 種手法把這份設定卡用出來。等到 CH1-3，你才會真的去產 8 張表情。這堂課是「上戰場前裝彈」。
(02) · 關鍵概念

✓寫不出來 = 你還沒想清楚，不是 AI 的錯

角色一致性
目標
同一個角色出現在 8 張不同情境的圖時，外觀辨識度仍然成立。

常見現象

✓這是 AI 的常態，不是你哪裡設錯

(03) · 操作示範

角色設定卡 + 4 類起手式 + 3 種一致性手法

1
角色設定卡：5 格填空模板

打開記事本，把這 5 格抄下來。每格至少寫一句完整的話，不要只寫「可愛」「療癒」「卡通風」這種一個詞了事。

【角色設定卡】

1. 種族 / 物種：________________
   （是什麼生物 / 物品 / 人？例：橘色虎斑貓 / 粉紅水母 / 戴眼鏡的女生 / 咖啡杯擬人）

2. 體型 / 比例：________________
   （Q 版圓滾滾 / 寫實正常比例 / 二頭身 / 大頭小身）

3. 配色（最多 3 色 + 1 配件色）：________________
   （主色：________ / 副色：________ / 點綴：________ / 配件：________）

4. 個性 / 表情常態：________________
   （溫和 / 活潑 / 慵懶 / 暴躁；眼睛常態：圓眼 / 半瞇 / 細線；嘴巴常態：微笑 / 嘟嘴 / 大口）

5. 風格參考（畫風 + 線條 + 色調）：________________
   （扁平卡通 / 漸變水彩 / 馬克筆塗鴉 / 像素風；粗線條 / 細線條 / 無線條；暖色調 / 冷色調 / 高彩度）

示範：橘貓上班族的設定卡

1. 種族 / 物種：橘色虎斑貓（站立兩腳走，會講話的擬人貓）
2. 體型 / 比例：Q 版圓滾滾，二頭身，肚子微凸
3. 配色：主色橘 #E89B5A / 副色奶油白（肚子和臉頰）/ 點綴深橘色虎斑紋 / 配件白色領巾
4. 個性：溫和但偶爾無語的上班族；眼睛常態半瞇圓眼，嘴巴常態微微嘟嘴
5. 風格參考：扁平卡通、粗線條（外輪廓 3px）、無漸層、暖色調（橘 + 米白 + 淺褐）

為什麼這 5 格非寫不可——少哪一格，AI 就自己亂補那一格：

沒寫種族
→ AI 隨機選貓 / 狗 / 狐狸
沒寫體型
→ 第 2 張變寫實貓、第 3 張變二頭身
沒寫配色
→ AI 隨便挑橘的色號，每張都不同
沒寫個性
→ AI 隨機選表情常態，溫和的貓被畫成兇狠
沒寫風格
→ 8 張可能 5 張卡通 3 張水彩
2
4 類起手式 Prompt 模板（選 1 類對號入座）

不同角色適用不同起手式。以下 4 類覆蓋大部分情境，每類都附填空版與示範版。

類型 A：寵物擬人
零授權問題 · 最推薦
適合：你家寵物 / 朋友家寵物 / 虛構的貓狗倉鼠。新手首選。
✓橘貓上班族 / 柴犬店長 / 倉鼠房東
✓輪廓清楚、無透明度，去背最容易
類型 B：真人擬卡通
注意肖像授權
適合：把自己或家人畫成 Q 版。練習階段建議只選自己，避免處理授權問題。
✓自己授權自己，沒問題
✓伴侶 / 小孩要上架，需取得書面同意
類型 C：物品擬人
視覺結構簡單
適合：你身邊任一物品——馬克杯 / 行李箱 / 鬧鐘 / 包包 / 手機。
✓物品本體保持原形，加五官和四肢
✓背景一律純白，視覺結構清楚
類型 D：幻想生物
進階 · 外觀要極具體
適合：完全原創一隻——雲朵 / 水母 / 小精靈 / 自創怪獸。
✓外觀要寫到極具體，AI 不認識「水母」這個詞
✓水彩漸層風去背難度高，入門建議先選 A 或 C
⚠
真人授權與肖像紅線（類型 B 必讀，不可跳過）

自己 → 可以，你授權自己。
伴侶 / 小孩 / 父母 → 自用送他本人 OK；若要上架可下載，等同公開使用對方肖像，必須事先取得口頭或書面同意。
朋友 / 同事 → 不要做。上架等同公開散布肖像。LINE 審核可能直接退件。
公眾人物 / 名人 → 絕對禁止。違反 LINE 條款 + 肖像權，帳號可能被處分。

類型 A 填空起手式

請畫一隻擬人化的【種族】，
外觀：【體型】，主色【主色號】+ 副色【副色】，配件是【配件】。
個性是【個性形容詞】，眼睛【眼睛狀態】、嘴巴【嘴巴狀態】。
風格：【畫風】，【線條粗細】，【色調】。
背景：純白色。
這是 LINE 貼圖角色設定圖（第一張，正面站姿，比讚）。

橘貓上班族示範填法：

請畫一隻擬人化的橘色虎斑貓，
外觀：Q 版圓滾滾二頭身，主色橘 #E89B5A + 副色奶油白（肚子和臉頰），配件是白色領巾。
個性是溫和但偶爾無語的上班族，眼睛半瞇圓眼、嘴巴微微嘟嘴。
風格：扁平卡通，粗線條（外輪廓 3px），暖色調無漸層。
背景：純白色。
這是 LINE 貼圖角色設定圖（第一張，正面站姿，比讚）。

類型 B 填空起手式（先擬卡通化，不要太像真人）

請把這張照片裡的人物畫成 Q 版卡通角色（不是寫實肖像）。
保留：【保留特徵：髮型 / 髮色 / 眼鏡 / 標誌性配件】。
卡通化：【體型】，五官簡化（眼睛變圓點或半月、嘴巴變一條線）。
配色：主衣著【顏色】，背景純白。
風格：【畫風】，【線條】，【色調】。
這是 LINE 貼圖角色設定圖（第一張，正面 Q 版，揮手）。
不要畫成寫實人像，要明顯卡通化、辨識度低於原照片，但保留特徵 1–2 個。

類型 C 填空起手式

請畫一個擬人化的【物品】（物品本體保持原形，但加上五官和四肢）。
物品本體：【顏色】+ 【材質：陶瓷/塑膠/布/金屬】，【其他特徵：把手/拉鍊/按鈕】。
擬人五官：【眼睛位置：上半部 / 中央】，【眼睛形狀】，【嘴巴形狀】。
四肢：【手腳設計：細線手腳 / 圓潤短手短腳 / 完全沒有四肢只有眼睛】。
個性：【個性形容詞】。
風格：【畫風】，【線條】，【色調】。
背景：純白色。
這是 LINE 貼圖角色設定圖（第一張，正面，舉手打招呼）。

愛抱怨的咖啡杯示範填法：

請畫一個擬人化的馬克杯（杯子本體保持原形，但加上五官和四肢）。
物品本體：白色陶瓷 + 一條藍色橫線裝飾，有把手，裝著咖啡（看得到液面）。
擬人五官：眼睛在杯身上半部，半瞇圓眼，嘴巴微微嘟嘴像是在抱怨。
四肢：細線手腳，從杯身兩側伸出。
個性：愛抱怨但其實很可靠的咖啡杯。
風格：扁平卡通，粗線條，暖色調。
背景：純白色。
這是 LINE 貼圖角色設定圖（第一張，正面，舉手打招呼）。

類型 D 填空起手式

請畫一隻原創的【幻想生物名稱】，沒有對應現實生物。
外觀：【主形狀：圓 / 水滴 / 雲朵 / 蘑菇】，【體型】，【特徵：觸手 / 耳朵 / 角 / 翅膀 / 尾巴】。
配色：主色【主色號】，半透明 / 不透明，【點綴色】。
五官：【眼睛數量+位置+形狀】，【嘴巴形狀】。
個性：【個性形容詞】。
風格：【畫風】，【線條】，【色調】。
背景：純白色。
這是 LINE 貼圖角色設定圖（第一張，正面，比愛心）。

粉紅水母示範填法：

請畫一隻原創的粉紅水母，沒有對應現實生物。
外觀：水滴型半透明身體，下方有 6 條柔軟觸手。
配色：主色粉紅 #FFB6C1，半透明可看穿，點綴色淺紫色。
五官：兩個圓眼睛在頭頂上方，嘴巴是一個小三角形微笑。
個性：活潑可愛、容易害羞。
風格：漸變水彩風，細線條，冷色調。
背景：純白色。
這是 LINE 貼圖角色設定圖（第一張，正面，比愛心）。
難度提醒（類型 D）：粉紅水母這種「半透明 + 漸變水彩」的角色，去背時透明度容易失真、APNG 壓縮較吃資源。如果你是第一次做，建議先選類型 A 或 C（輪廓清楚、無透明度），水彩風留到熟練之後再挑戰。
3
3 種角色一致性手法的優劣對比

填好設定卡、寫好起手式之後，「同一角色 8 張」實際上有 3 種做法。每種各有優劣，你根據情境選，不是死記哪個最強。

手法 操作方式 優點 缺點 適用情境
A：上傳參考圖 先有 1 張正解圖，上傳給 AI 說「照這隻畫」 一致性最強（AI 直接看圖比對） 第一張要先有；每張都重新讀圖，額度最高 角色已定型、想做完整套組
B：同對話多輪 開一個對話，先用起手式產第 1 張；然後在同一個對話繼續說「換成哭的表情」 省額度（同對話 context 不重算）；快 連續到 4–5 張會「漂移」（毛色變淡 / 體型變瘦） 額度緊、想快速跑完 8 張
C：一次產多張 一個 Prompt 裡直接說「請產 4 張同角色不同表情」 速度最快、額度最省 風格容易跑掉；無法精準控制每張情境 探索期、快速看「這個角色好不好看」

建議的組合策略：

第 1 張
→ 用起手式 + 設定卡直接產（這張是「校準範例」）
第 2–4 張
→ 用手法 B（同對話多輪）省額度，快速跑
第 5–8 張（漂移後）
→ 仍在同對話內，切到手法 A——把第 1 張上傳當參考圖校準，不要重開新對話
手法 C
→ 不在 8 張流程裡用；留給「想快速試風格」時用
省額度提醒：CH1-1 已經請你同時註冊 Gemini + ChatGPT 兩家。手法 B 最省、手法 A 最耗。Gemini 免費版每天有產圖上限，用光了就切到 ChatGPT 繼續。

(05) · 完成驗證

  
設定卡驗收清單（5 條，全勾才合格）


  
注意：這是檢查你的設定卡寫得夠不夠具體，跟 CH1-3 的「角色一致性檢查清單 5 條判準」是不同的兩張表。


  

    

      
第 1 格（種族）

      
不是「貓」，而是「橘色虎斑貓 + 兩腳站立 + 會講話」

    

    

      
第 2 格（體型）

      
有量化：不是「可愛」，而是「Q 版二頭身 + 圓滾滾 + 肚子微凸」

    

    

      
第 3 格（配色）

      
寫了顏色名稱或色號：不是「暖色」，而是「主色橘 #E89B5A + 副色奶油白 + 配件白色領巾」

    

    

      
第 4 格（個性）

      
有「常態表情」：不是「個性溫和」，而是「溫和但偶爾無語，眼睛半瞇圓眼，嘴巴微微嘟嘴」

    

    

      
第 5 格（風格）

      
有「畫風 + 線條 + 色調」三段：不是「卡通風」，而是「扁平卡通 + 粗線條 3px + 暖色調無漸層」

    

  


  
5 條全勾才算合格。少 1 條，下一堂 CH1-3 你產 8 張時 AI 就會在那一格自己亂發揮。如果你只勾了 2–3 條：不用重來，先把橘貓的設定整段複製過去當骨架，再把你的角色名稱和配色塞進去。CH1-3 會帶著你走完整流程，那時候會自然補齊。

這四種狀況最常發生。對號入座，照著做就能繼續。


  

    

      
AI 拒答

      
換「擬人化卡通角色，用於 LINE 貼圖」這種更白話的詞重講；還是拒答就換 ChatGPT 試試。

    

    

      
設定卡寫不出第 4–5 格

      
直接拿橘貓設定卡當骨架，5 分鐘內讓自己開始產第一張，不要追求一次到位。

    

    

      
想做真實人物但不確定授權

      
先用自己或虛構角色完成這堂的練習，授權問題之後再決定，不要卡在這裡。

    

    

      
第一張明顯不像設定卡

      
回去補第 5 格「風格參考」，特別是「線條粗細 + 色調 + 是否漸層」這三個。

    

  










  
(07) · 常見錯誤

  
這 3 種錯誤你最容易踩


  

    

      
錯誤 1：設定卡只寫「可愛 / 療癒」

      

        現象：第 4 格寫「可愛」、第 5 格寫「療癒風」。第一張 OK，第二張就完全換了個臉。

        原因：「可愛」和「療癒」是對結果的形容詞，不是對畫面的描述。AI 看到「可愛」會自由發揮，看到「Q 版二頭身 + 圓眼睛 + 粗線條外輪廓 3px」才會固定。

        解法：「可愛」→「Q 版二頭身、圓滾滾、大眼睛、嘴角微微上揚」；「療癒」→「暖色調（橘 + 米白 + 淺褐），無強對比，柔和線條」。寫完設定卡，蓋住「種族」那格，只看其他 4 格——你的家人朋友讀完能想像出同一隻嗎？
      

    

    

      
錯誤 2：每張都重開新對話

      

        現象：產完第 1 張後覺得不錯，關掉頁籤、開新對話，重貼設定卡產第 2 張。第 2 張配色變了、體型瘦了。繼續關、繼續開。8 張產完得到 8 隻不同的貓。

        原因：AI 在「同一個對話」內有 context 記憶，前面產過的圖會被當作參考。開新對話 = 清空記憶 = AI 從零重畫。每次找到的解略有不同，8 張自然漂移 8 個方向。而且每開一個新對話就燒一次完整額度。

        解法：產完第 1 張後不要關對話，直接在同一個對話框輸入「同一隻角色，換成哭著流眼淚的表情」。等到第 4–5 張開始漂移，才切到手法 A 上傳第 1 張當參考圖校準——仍在同對話內，不重開。
      

    

    

      
錯誤 3：設定卡種族欄寫「皮卡丘 / 哆啦 A 夢 / 史努比」

      

        現象：你為了快，第 1 格寫「皮卡丘風格的小怪獸」。AI 畫得非常像，你覺得超成功，產完 8 張上傳。LINE 審核直接退件：含他人 IP。更嚴重的情況：帳號被警告或處分。

        原因：皮卡丘 = 任天堂 / The Pokémon Company 著作權；哆啦 A 夢 = 小學館著作權。「我只是參考風格」不成立。LINE IP 審核連「明顯類似」都退件，AI 不會告訴你畫出來像誰，責任在你。

        解法：種族欄禁止寫任何已存在角色名；風格欄禁止寫「吉卜力風 / 迪士尼風 / 三麗鷗風」。想做類似可愛動物：寫生物本身（黃色老鼠 / 藍色機器貓）加上你自己定義的特徵，讓 AI 從零畫你的角色。
      

    

  










  
(08) · 自我檢核

  
確認核心觀念有沒有記住


  

    
Q1：下列哪一條，是把「療癒風 Q 版可愛貓咪」翻譯成 AI 能穩定守住的角色設定卡描述？

    

      
      
      
      
    

    

      顯示答案
      
正確答案：C。A、B、D 全部用「療癒 / 可愛 / 溫暖」這種對結果的形容詞，AI 看到會自由發揮。C 把所有形容詞都翻譯成可量化的具體描述（顏色名稱 + 色號、體型量化、線條粗細、是否漸層），AI 看了能穩定產出。這是本單元最核心的能力。

    

  


  

    
Q2：你用手法 B（同對話多輪）在 Gemini 產了 4 張橘貓不同表情：笑、哭、生氣、得意。第 1–3 張很一致；但第 4 張「得意」毛色變淡了、肚子奶油白範圍縮小了。還有 4 張要產。請問下一步應該怎麼做？（開放題，列出 2 點即合格，3 點優秀）

    

      顯示參考答案
      

        
合格（2 點）：

        
    
          
  • 不要直接繼續在同對話產第 5 張——AI 漂移已出現，再產下去越來越偏，後面全報廢。
    • 
          
  • 切到手法 A，仍在同對話內：把第 1 張下載，在同一個 Gemini 對話視窗上傳這張圖，附 Prompt「請照這張畫的角色，畫一個求拜託的表情」。AI 重新從圖校準。不要重開新對話——重開 = AI 忘記前 4 張 context。
    • 
        

        
優秀（+1 點）：

        
    
          
  • 先回去用手法 A 重產第 4 張「得意」（把漂移的那張替換掉），再繼續第 5 張。不要把漂移的圖留在套組裡——LINE 審核會抓到角色不一致。
    • 
        

      

    

  










  
(09) · 課堂紙本素材

  
4 張紙本素材——可單張列印帶走

  
下面這 4 張就是課堂會發給你的紙本素材。每張右上有「📄 列印這張」按鈕——按下去會開新分頁印出 A4 一張，自己印或讓我們幫你印都行。CH1-2 + CH1-3 都會用到，請收好。


  
  

    

      

        
素材 1 · 這堂用

        
角色設定卡填空表（5 格手寫版）

      

      
    

    

      
5 格寫滿才算合格。每格寫具體一點，不要只寫「可愛」「療癒」「卡通風」這種一個詞了事。寫不出第 4–5 格 → 翻背面抄橘貓示範當骨架。

      

        
欄位你的答案

        
          
1. 種族 / 物種
是什麼生物 / 物品 / 人		　

          
2. 體型 / 比例
Q 版 / 寫實 / 二頭身		　

          
3. 配色（最多 3 色 + 配件）
主色 / 副色 / 點綴 / 配件，附色號最佳		　

          
4. 個性 / 表情常態
含眼睛、嘴巴常態描述		　

          
5. 風格參考
畫風 + 線條 + 色調三段		　

        
      

      
— 背面 · 填寫範例：橘貓上班族版 —

      

        
欄位橘貓上班族示範

        
          
1. 種族橘色虎斑貓（站立兩腳走，會講話的擬人貓）

          
2. 體型Q 版圓滾滾，二頭身，肚子微凸

          
3. 配色主色橘 #E89B5A / 副色奶油白（肚子和臉頰）/ 點綴深橘色虎斑紋 / 配件白色領巾

          
4. 個性溫和但偶爾無語的上班族；眼睛半瞇圓眼，嘴巴微微嘟嘴

          
5. 風格扁平卡通、粗線條（外輪廓 3px）、無漸層、暖色調（橘 + 米白 + 淺褐）

        
      

      
寫完蓋住「種族」那格，只看其他 4 格——你的家人朋友讀完能想像出同一隻嗎？能 = 合格；不能 = 那一格還太空。

    

  


  
  

    

      

        
素材 2 · 這堂用

        
4 類起手式範本卡（對號入座，選 1 類）

      

      
    

    

      
設定卡寫完，從這 4 類選 1 類起手式照填空格。從最接近你的角色那類起手，不要 4 類都試。

      

        
類型適合誰起手式範本

        
          

            A 寵物擬人
零授權 · 最推薦		
            你家寵物 / 朋友家寵物 / 虛構貓狗倉鼠
            「請畫一隻擬人化的【種族】，外觀【體型】+ 主色【主色號】+ 配件【配件】，個性【個性】，風格【畫風 + 線條 + 色調】，背景純白色，這是 LINE 貼圖角色設定圖。」
          

          

            B 真人擬卡通
注意肖像授權		
            把自己畫成 Q 版（建議只選自己）
            「請把這張照片裡的人物畫成 Q 版卡通，保留【特徵 1–2 個】，五官簡化（眼睛變圓點、嘴巴變一條線），背景純白色——不要寫實，要明顯卡通化。」
          

          

            C 物品擬人
視覺結構簡單		
            身邊任一物品（馬克杯 / 行李箱 / 鬧鐘）
            「請畫一個擬人化的【物品】（本體保持原形 + 加五官四肢），物品本體【顏色 + 材質】，五官【眼睛位置 + 形狀】+【嘴巴】，個性【個性】，背景純白色。」
          

          

            D 幻想生物
進階 · 外觀要極具體		
            完全原創（雲朵 / 水母 / 自創怪獸）
            「請畫一隻原創的【生物名稱】，沒有對應現實生物。外觀【主形狀 + 體型 + 特徵】，配色【主色 + 透明度 + 點綴色】，五官【眼睛 + 嘴巴】，背景純白色。」
          

        
      

      
對號入座，從最接近你的那類起手。新手強烈建議從 A 或 C 開始（輪廓清楚、無透明度，去背最容易）；類型 B 一定先看素材 4 再做；類型 D 留到熟練後再挑戰。

    

  


  
  

    

      

        
素材 3 · 隨身帶

        
3 種角色一致性手法對照表

      

      
    

    

      
下一堂 CH1-3 產 8 張表情會反覆切換這 3 種手法，先把優劣記起來。漂移時要切，不是死守一種。

      

        
手法一致性強度額度耗用何時用

        
          

            A 上傳參考圖
先有 1 張正解，叫 AI 照著畫		
            ★★★ 最強
            ★★★ 最高
            角色已定型、想做完整套組；漂移後校準用
          

          

            B 同對話多輪
同一個對話接著產，只說「換 X 表情」		
            ★★ 中（4–5 張會漂）
            ★ 最省
            額度緊、想快速跑完 8 張的主力
          

          

            C 一次產多張
一個 Prompt 直接要求「產 4 張不同表情」		
            ★ 最弱
            ★ 最省
            探索期試風格用；不在 8 張流程裡
          

        
      

      
本課推薦組合：第 1 張用起手式 + 設定卡產出（手法基礎）→ 第 2–4 張用手法 B同對話接著跑省額度 → 第 5 張開始漂移時切手法 A，把第 1 張上傳當參考圖校準（仍在同對話內，不要重開新對話）。

    

  


  
  

    

      

        
素材 4 · 隨身帶

        
真人授權與肖像紅線速查（類型 B 必讀）

      

      
    

    

      
類型 B 真人擬卡通起手前一定先看這張。「自用」和「公開上架」是兩件事——自用 OK，上架等同公開使用對方肖像，紅線完全不同。

      
    
        
  • ① 自己 → 綠燈：你授權自己，做什麼都可以（自用 / 上架販售 / 免費發送都 OK）。
    • 
        
  • ② 伴侶 / 小孩 / 父母 → 黃燈：自用送他本人 OK；若要上架（即使免費）= 公開使用對方肖像，必須事先取得口頭或書面同意。小孩需法定代理人同意。
    • 
        
  • ③ 朋友 / 同事 → 紅燈：不要做。即使對方口頭說好，上架後散布範圍與時間無法收回，LINE 審核可能直接退件。
    • 
        
  • ④ 公眾人物 / 名人 / 偶像 → 嚴禁：絕對禁止。違反 LINE 條款 + 肖像權 + 著作權（藝人合約限制），帳號可能被警告或處分。
    • 
      

      
關鍵分界：自用送朋友 OK；上架販售（即使免費下載）就等於公開使用，必須先取得授權。練習階段建議：第一隻角色先選自己或虛構，授權問題之後再決定，不要卡在這裡。
