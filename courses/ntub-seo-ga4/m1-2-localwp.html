SECTION 1.2

LocalWP 本地站建置與 WP 基礎

用 LocalWP 一鍵在自己的筆電上開出 WordPress 網站，當作後 9 週 SEO 改寫、GA4 埋碼、GTM 佈建的演練場。

為什麼這很重要 · WHY THIS MATTERS
真實網站動一個字都怕崩，學員不敢練。LocalWP 讓你在本機開出獨立的 WP 環境，可以隨意改 Title、刪文章、裝外掛、實驗結構化資料——崩了就重建一個。本週建好之後，整門課的 SEO/GA4/GTM 演練都在這個站上完成。
STEP BY STEP · 操作步驟
1
下載並安裝 LocalWP
到 localwp.com，點 Download，選擇 Mac 或 Windows 版本（檔案約 700MB）。安裝過程一路按 Next，不需要建立 Local 帳號，可選「Skip Sign Up」。
提示：學校教室電腦若有還原卡，請裝在「自備筆電」或外接硬碟；裝在系統碟一重開就消失。
2
建立第一個本地站
開啟 LocalWP，點左下「+」→「Create a new site」。三個欄位這樣填：
Site name: ntub-demo Local domain: ntub-demo.local Site path: 保留預設即可 Environment: Preferred (PHP 8 / nginx / MySQL 8) WP user: admin WP password: 自設一組（記下來） WP email: 你的 email
提示：網址使用 .local 結尾是 LocalWP 的慣例，不會跟真實網域衝突。
3
改成繁體中文介面
點右上「WP Admin」進入後台 → Settings / 設定 → General → Site Language 選「繁體中文」→ Save。整個介面會變成中文，後續操作說明才對得起來。
提示：若 WP Admin 按鈕灰掉，先確認 LocalWP 主畫面右上角「Start site」綠燈亮著。
4
套用一個簡單佈景主題
外觀 → 佈景主題 → 安裝「Twenty Twenty-Four」（WP 內建）→ 啟用。這是官方主題，乾淨無多餘設定，SEO 結構清楚，適合做演練。
提示：不要一開始就裝花俏的視覺主題；複雜主題會讓我們之後檢查 H1/H2 結構時看不清楚。
5
發表一篇測試文章
文章 → 新增文章 → 標題打「Hello NTUB」→ 內文隨便輸入兩段 → 發表。這篇文章之後會被你不斷改寫——加 Meta、改 H1、塞關鍵字、測 Yoast 分數。
提示：固定網址（Permalink）建議先到「設定 → 永久連結」改成「文章名稱」格式，這樣 URL 會是 /hello-ntub/ 而非 /?p=1。
6
確認站台前台可瀏覽
回 LocalWP 主畫面，點「Open site」→ 應該開啟 http://ntub-demo.local，看到剛剛發的文章。這就完成了。
提示：若瀏覽器顯示「無法連線」，回 LocalWP 看「Trust」按鈕，按下後輸入電腦密碼即可。
完成驗證 · VERIFY
能在瀏覽器打開 http://ntub-demo.local 看到首頁與「Hello NTUB」文章；能登入 /wp-admin 看到繁中後台。兩件事都成立才算完成。
常見問題 · TROUBLESHOOT
.local 網址打不開？
多數是 LocalWP 沒有取得系統信任。回 LocalWP → 點站台 → 上方分頁「Tools」→「Trust」，輸入電腦密碼即可。
忘記 WP 後台密碼？
LocalWP 主畫面有「Open site shell」，可以執行 wp user update admin --user_pass=新密碼 重設。或最快：刪掉站台重建一個。
這個站 Google 找得到嗎？
找不到。本地站只有你自己的電腦能連，Google 爬蟲進不來。所以本課程的 SEO 驗證不靠 GSC，而是用 Yoast 分數、PageSpeed Insights、結構化資料測試器代替。
