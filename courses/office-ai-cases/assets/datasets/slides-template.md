# 簡報組裝模板（5-8 頁版型）— 給 CH2-2 案例 ④ 使用

## 簡報結構建議

5-8 頁的數據簡報結構（給主管 / 客戶用）：

### 頁 1：標題 + 一句話結論
```
標題：[主題] · [期間]
副標：[一句話結論 — 例：「Q1 業績 +18%、毛利率持平、需注意 4 月斷崖」]
製作者 / 日期
```

### 頁 2：本期重點（3-5 條 bullet）
```
本期重點
- 重點 1：[具體數字 + 變化]
- 重點 2：[具體數字 + 變化]
- 重點 3：[具體數字 + 變化]
（建議數字優先於形容詞）
```

### 頁 3：主要圖表 1（趨勢類）
```
建議用圖：折線圖（時間趨勢）/ 柱狀圖（類別比較）
關鍵：標清楚 X 軸 / Y 軸 / 單位 / 圖例
```

### 頁 4：主要圖表 2（結構類）
```
建議用圖：圓餅圖（佔比）/ 堆疊柱狀圖（多維比較）
關鍵：突顯前三大、其他歸併
```

### 頁 5：異常 / 風險點
```
本期注意事項
- 異常 1：[具體事項 + 影響 + 建議行動]
- 異常 2：[同上]
（避免「整體還好」「需要持續觀察」這種空話）
```

### 頁 6（選）：下期行動建議
```
下期行動建議
1. [具體行動 + 負責人 + 期限]
2. [同上]
3. [同上]
```

### 頁 7（選）：附錄資料來源
```
資料來源
- [系統 / 報表名稱 / 時間區間]
- [計算公式 / 取樣方式說明]
```

### 頁 8（選）：Q&A
```
Q&A
（留空白頁面給簡報結束後使用）
```

## 簡報 prompt 5 大骨架（通用 wrapper）

組裝簡報的 prompt 結構：

```
角色：你是資深商業分析顧問
資料：[貼上你的分析結論摘要 200 字]
任務：產生 5-8 頁的數據簡報內容大綱，每頁含「標題 + 3-5 bullet + 圖表類型建議」
輸出格式：Markdown 表格（頁碼 / 標題 / 主要內容 / 圖表建議），每頁不超過 80 字
驗證：請列出你引用的數字依據（從上面的資料哪幾筆推論出來）
```

## 簡報承諾邊界（重要！）
