CH 2 · 3  ·  35 min

條件談判

加薪 / 資源 / 職責 / offer 議價——三條 lane，各一套劇本。不論你選深耕、內轉還是轉職，談判的核心問題只有兩個：我要爭取什麼？我能拿什麼交換？

學習成果
能用「我要爭取什麼、能拿什麼交換」設計三條 lane 的條件談判
能用「3 個量化貢獻」回應「為什麼值這個價」
能完成一張條件談判卡（lane 選擇 / 核心訴求 / 開場句 / 結尾句草稿）
(01)

三個人，同一種沉默

為什麼這很重要
條件談判不是要你「強硬爭取」，是讓你把一個自己清楚、對方也說得通的交換方案放上桌。沉默不等於安全——不談也是一種選擇，但要清楚它的代價。

三個人，分別在 CH2-2 後的那個下午，遇到了同一種沉默。

深耕線 · 小宇
績效面談快到了。他知道自己要留、要深耕，但想爭取的不只是「辛苦了繼續加油」——他要主導一個跨部門專案，或至少要到資源讓他能証明自己。問題是：「要求資源」跟「抱怨沒有發展空間」的邊界在哪裡？
內轉線 · 阿哲
技術主管口頭說「有意願讓他試 team lead」。CH2-2 腳本寫好了，但他還沒想清楚：職稱怎麼談？職責範圍怎麼框？如果主管說「先代理試試看，沒有職稱沒有加薪」，他要不要接？
轉職線 · 惠姊
收到第一家 B2B 行銷企劃的 offer，年薪 70 萬，比現薪 78 萬低。她知道轉職初期降薪很可能——但：這個數字該不該談？怎麼談才不會毀掉 offer？除了月薪，還有什麼可以爭取？

這三個卡點，本質上是同一件事：條件談判。

「條件談判」不是「薪資談判」。薪資只是眾多可談的條件之一。你能談的包括：加薪（現職或 offer 議價）、資源（預算 / 工具 / 人力支援）、職責範圍（管什麼、不管什麼）、職稱（市場定位，不只是名片抬頭）、時間安排（試用期、檢視機制、回升路線）。

核心問題只有兩個：我要爭取什麼？我能拿什麼交換？

(02)

進劇本之前：4 個共通理解

01
條件談判 vs 薪資談判
薪資談判是「爭取月薪 / 年薪數字」這個單一維度；條件談判是「爭取整個交換結構」。月薪談不動時，職稱 / 職責 / 學習預算 / 檢視機制都是合法籌碼。擴大可談範圍 = 擴大達成協議的可能性。
02
3 個量化貢獻 = 你的定價依據
不論哪條 lane，「為什麼值這個價」的有效格式只有一種：具體成就 + 數字 + 對象。「我做了很多」「我有 X 年經驗」不是定價依據，對方聽到的是「平均水準」。（承接 CH2-1 能力證據包產出）
03
談判是設計，不是爭吵
好的條件談判不是「我要更多、你給不給」的拉鋸，是「我提出一個對雙方都說得通的交換方案」。對方（主管 / HR）也需要有東西去跟上面交代。幫對方找到一個能交代的理由，是條件談判最重要的技術。
04
不談的代價也是選擇
「不談比較安全」是常見第一反應。不談的代價是：你接受了一個自己不完全滿意的條件，然後每天在那個條件下工作，把「不甘心」轉成「對工作失去動力」。不談也是一種選擇，但要清楚它的代價。
◎
三 lane 共用的 3 心法

1. 先定義「我要爭取什麼」——談判前先排序：月薪 / 職稱 / 職責 / 資源 / 學習預算 / 時間彈性，哪個對你最重要？最高優先的那個才是核心訴求，其他是籌碼或讓步空間。用整張清單去談，通常什麼都談不到。

2. 量化貢獻是你的籌碼，不是求情——用「3 個量化貢獻」開口的人，姿態是「我有依據相信這個交換是合理的」。求情讓你被動，依據讓你對等。

3. 每次談判結尾要有明確的下一步——「我再想想」「我考慮看看」都是讓談判死掉的結尾。好的結尾句要有：(1) 具體時間 (2) 對方也要做的事 (3) 下次對話的錨點。雙方都有任務的談判才會繼續走。
(03)

三條 lane 談判劇本（14 min · 每劇本約 4.5 min）

選最像你情境的那條，完整看一遍開場 / 反問 / 結尾 / NG——然後等下動手練習時，用你自己的版本替換。

劇本 A · 深耕線 · 小宇
加薪 + 爭取資源（年度績效面談）

情境：小宇（B2B 軟體公司行銷企劃，4 年，年薪 NT$ 70 萬）。目標：(1) 年薪從 70 萬調到 75 萬 (2) 主導 2026 下半年一個跨部門行銷專案。

3 個量化貢獻（來自能力證據包）：(1) 主導 3 場 B2B 客戶活動，活動後進入商機清單的潛在客戶比前一年提升 40% (2) 獨立寫了 12 份每月行銷報告，主管說「幾乎不用再看細節」 (3) 業務提案素材協助成案 3 個

開場句
「謝謝這次面談機會。在討論績效結果之前，我想先說一件事：我對現在的工作有一個明確的期待方向，我想讓你知道，不只是等回饋。過去 4 年我做了 3 件我認為對部門有具體貢獻的事——（唸 3 個量化貢獻）。基於這個基礎，我今天想討論兩件事：一是 2026 年薪調整，我的期待是從 70 萬調到 75 萬；二是下半年能不能給我一個主導跨部門行銷專案的機會。這兩件事我想一起談，因為它們對我來說是一個整體。」
核心結構：先讓主管知道你不只在「接受評估」而是「主動提方向」→ 3 個量化貢獻 → 兩個具體訴求（月薪 + 機會）並陳，不是只談薪水
反問句
「我想先確認一下：你對我在下半年主導跨部門專案這件事，在原則上有什麼想法？我不是要求馬上拍板，而是想知道這件事在你評估範圍內嗎？有了這個前提，我們再談薪資調整會更有方向。」
為什麼這樣問：把加薪跟「更大職責」綁在一起談判，比只談薪水更有籌碼。若主管認可「讓你主導更大的事」，薪資調整的合理性也更容易成立。
結尾句
「我知道薪資調整和專案授權都需要你評估和確認。我不需要今天拍板——我的希望是這週內，你能告訴我這兩件事的初步方向：薪資調整可以朝 75 萬討論嗎？跨部門專案的機會下半年有沒有空間？我會根據你的方向，想清楚自己能怎麼提出更具體的方案。」
核心結構：給主管時間 + 設定具體回覆期限「這週內」+ 讓主管知道你不是要馬上答案，但要有方向
NG 答法
❌「我覺得我今年表現還不錯，希望能加薪。公司有沒有這個空間？」
為什麼 NG：(1)「還不錯」不是定價依據，主管無法幫你跟 HR 爭取 (2)「公司有沒有這個空間」是把決定權全交給主管，你的姿態是被動等決定 (3) 沒有提出量化貢獻，主管給你的參考點就是「行業中位數」
劇本 B · 內轉線 · 阿哲
職責 + 職稱談判（接 team lead）

情境：阿哲（資深後端工程師，8 年）。技術主管已口頭表達「有意願讓他試試看」。今天要把「試試看」談成有架構的方案：職稱確認、職責範圍書面化、薪資調整方向。

3 個量化貢獻（來自能力證據包）：(1) 核心支付模組上線後穩定運行 18 個月無重大事故，每日處理 peak 交易量 40 萬筆 (2) 協助 2 名新進工程師上線，兩人 3 個月後都能獨立接需求 (3) 主導資料庫遷移，0 downtime 完成，節省 6 週手動維護工時

開場句
「謝謝你上次說的『有意願讓我試試看』。我想把這個從口頭確認，談成一個對我們雙方都清楚的安排。我過去 3 年做了 3 件我認為能支撐這個角色的事——（唸 3 個量化貢獻）。我今天想確認 3 件事：第一，team lead 這個角色的具體職責範圍是什麼；第二，職稱怎麼走；第三，如果職責範圍確認了，薪資是不是也會對應調整。這 3 件事我希望今天能有一個清楚的方向，不是全部要拍板，是要有框架。」
核心結構：接「口頭有意願」但要把它轉成有架構的方案 → 3 個量化貢獻 → 三件事並陳（職責 / 職稱 / 薪資），不單獨談任何一件
反問句
「我想先聚焦職責範圍：team lead 這個角色，你期待我管幾個人？現在 team 的 3 人工程師，是不是都會 report 給我？績效評估這件事，你希望我介入到什麼程度？我問這些，是因為職責範圍定清楚了，職稱和薪資的討論才有依據——一個管 3 人的 lead 和一個只負責技術方向的 lead 在市場上是不同的定位。」
為什麼這樣問：把「職責 → 職稱 → 薪資」的邏輯順序說清楚，讓主管理解為什麼要先定職責，不是在刁難。
結尾句
「我這週可以把我對 team lead 職責範圍的理解整理成一份 1 頁的提案——不是要求全照我說的，而是讓我們下次對話有一個具體的東西可以討論和修改。你覺得這週五我傳給你、下週一我們花 30 分鐘對齊，可以嗎？」
核心結構：我主動提案「1 頁職責範圍文件」→ 讓主管帶有具體東西去跟更上面討論 → 設定下次對話的具體時間
NG 答法
❌「你說有意願讓我試試，我很願意試。你覺得怎麼安排比較好？」
為什麼 NG：(1) 把所有設計權交給主管，最後很可能得到「代理期、無職稱、先試三個月看看」這種對你最沒保障的方案 (2) 職稱 / 薪資 / 職責沒有一件被你主動放上談判桌，談判在「你開心就好」的模糊氛圍裡結束了
劇本 C · 轉職線 · 惠姊
offer 議價（薪資 + 非薪資條件）

情境：惠姊（國貿行政主管，12 年，年薪 NT$ 78 萬）。收到 B2B 行銷企劃 offer，年薪 70 萬。評估後決定可以接受短期降薪，但要把「降薪 + 回升路線 + 非薪資補償」談成整體方案。

3 個量化貢獻（來自能力證據包）：(1) 統籌 3 次公司搬遷 + 2 次 ISO 認證流程，0 次延期 (2) 製作 20 份 SOP 手冊，後任行政同仁上手時間從 3 週縮短到 1 週 (3) 協助業務提案準備，過去 2 年 4 個大客戶提案使用她製作的素材，成案 3 個

開場句
「謝謝這個 offer。我想告訴你為什麼我選擇這份工作：不是因為現在的行政做不下去，而是我過去 12 年累積了 3 件我認為跟行銷企劃高度相關的事——（唸 3 個量化貢獻）。基於這個背景，我可以接受職涯轉換初期的薪資調整，但我希望我們把這個安排設計成「有條件、可回升的」，而不只是一個固定數字。我的提案是：70 萬 base + 12 個月後薪資檢視機制——如果在 12 個月內我達到你認可的具體成果，薪資調整到 75 萬。同時我也想確認非薪資部分：這個職位有學習預算嗎？」
核心結構：說明「為什麼這次轉換有依據」→ 3 個量化貢獻（行政到行銷企劃的可遷移性）→ 接受降薪，但設計回升機制 → 非薪資條件也放上桌
反問句
「12 個月檢視機制，貴公司這個職位有沒有先例？如果沒有，我們可以在 offer letter 裡附上 3 個具體的量化成果指標（例如：獨立企劃執行 2 個行銷活動、撰寫 12 份以上對外內容、協助 1 個業務案件進入提案流程），達標後薪資調整到 75 萬。這樣對你也有一個跟主管交代的具體依據。」
為什麼這樣問：把「12 個月檢視」從「口頭承諾」變成「寫進 offer letter 的具體成果指標」。HR 需要一個跟主管交代的故事——你給他了。
結尾句
「我需要 24 小時考慮這個 offer 的整體方案。明天這個時間之前給你回覆。同時想請你回去確認兩件事：一是 12 個月薪資檢視機制公司原則上可以嗎？二是學習預算的具體額度。有這兩個資訊，我明天的回覆會更確定。」
核心結構：24 小時規則 + 讓 HR 帶具體任務回去 + 設定明確回覆時間
NG 答法
❌「我接受 70 萬，我知道轉職可能要降薪，沒關係的。希望我進去之後表現好，公司會看到。」
為什麼 NG：(1)「沒關係的」是把談判籌碼全部放棄——你有現職、不急，這本身是談判優勢，但你自己消掉了 (2)「希望公司會看到」是把回升路線押在「對方主動做好事」，沒有寫進任何具體機制 (3) 沒有提出量化貢獻，HR 不知道為什麼 70 萬是個合理的起點
(04)

動手練習：填一張條件談判卡

從三條 lane 中選最像你情境的那條（深耕 / 內轉 / 轉職），用條件談判卡填寫你自己的版本。選「最近 3 個月最可能發生哪場談判」的那條，不是選「最想練的」。

1
我要爭取的最重要一件事（1 行）
月薪 / 職稱 / 職責 / 資源 / 學習預算——只選一個。三件都很重要時，問自己「哪一件如果沒拿到，我最不甘心」。
2
我的 3 個量化貢獻（每個 ≤ 1 行）
格式：動詞 + 對象 + 數字或成果。直接從 CH2-1 能力證據包拿過來用。
3
我的開場句草稿（5-8 行）
包含：3 個量化貢獻 + 核心訴求。不是只說「我想加薪 / 我想要這個機會」。
4
我的結尾句草稿（3-5 行）
強制框架：(1) 我 ___ 天內給你回覆 (2) 同時請你確認：___ (3) 我們 ___ 時間對齊，可以嗎？
5
兩兩演練開場句 30 秒
你口說演練，對方給 1 條回饋：「3 個量化貢獻夠不夠具體？」換邊。
完成標準（self-check）
  • 從三條 lane 中選了 1 條，並能說出「為什麼這條最像我最近的情況」
  • 寫了 3 個量化貢獻，每個都有「動詞 + 對象 + 數字或成果」至少 2 個元素
  • 開場句包含「量化貢獻 + 核心訴求」，不是只說「我想加薪 / 我想要這個機會」
  • 結尾句有具體時間（幾天內 / 明天 / 週五）+ 對方要做的 1 件事
  • 口說演練了 1 次開場句（30 秒內講完）
  • 收到對方 1 條具體回饋（不是「整體不錯」這種空話）
(05)

1 分鐘反思

寫下這兩個句子
在接下來最可能發生的那場談判裡，我要爭取的最重要一件事是：＿＿＿＿＿
我能拿來交換的，是：＿＿＿＿（至少 1 個量化貢獻 + 1 個我能讓步的地方）
方向不是越早決定越好，是越清楚越好。條件談判也一樣——不是越早開口越好，是越清楚自己要什麼、越清楚自己值什麼，越好。

  • 選不出 lane：「最近 3 個月最可能發生哪場談判」決定，三 lane 不互斥
  • 「沒有談判機會」：任何一次 1-on-1 都可以提，不用等績效面談
  • 量化貢獻想不出：回 CH2-1 能力證據包直接轉用；或「過去 1 年最有成就感 3 件事 → 動詞 + 對象 + 結果」5 分鐘快速版
  • 「對方就是不肯動」：目標不是每次都拿到，是「讓對方知道你有依據、留下後續空間」
(07)

常見錯誤3 條

錯誤 01
核心訴求欄填了 3 件事——每件都說得不清楚
學員把「談判清單」誤解成「談判目標」，把加薪 + 更大職責 + 職稱升一格全部並列。對方聽完不知道你今天最重要的訴求是什麼，心理反應是「這個人要很多」。
解法：先問「如果今天只能爭取一件事，你選哪個？」填卡時「核心訴求」欄限定 1 件事，其他列為「補充籌碼」。講師巡場看到填了 3 件事的學員，引導：「哪一件如果沒拿到你最不甘心？那個就是你今天的核心訴求。」
錯誤 02
開場句只有年資、沒有任何量化貢獻
學員知道要講可遷移技能，但在談判場景下腦子還是回到「我有年資 + 我有意願 + 請給機會」的框架。「做了 12 年行政」不是定價依據——對方聽到的是「他待很久了」，不是「他能做什麼、值多少」。
解法：劇本展示時，NG 版（「做了 12 年行政，希望給機會」）vs 正確版（3 個量化貢獻）並排對照，讓學員感受「對方聽到的感覺完全不同」。卡頓的學員引導回 CH2-1 能力證據包：「把那張 STAR 卡拿出來，3 條最強的直接照唸。」
錯誤 03
結尾句「我回去想想」——沒有時間、沒有對方任務
學員把「禮貌」跟「有效結尾」搞混了。「我回去想想」在學員感覺是不施壓，但在主管 / HR 角度是「應徵者 / 下屬沒有主張」，而且沒有時間錨點代表談判可以無限期拖延。
解法：談判卡「結尾句草稿」欄加 3 個強制框架：(1) 「我 ___ 天內 / ___ 之前給你回覆」(2)「同時請你確認：___（對方的任務）」(3)「我們 ___ 時間對齊一下，可以嗎？」——照格式填，不需要原創。演練時觀察結尾句沒有時間錨點的學員，提示：「加一個具體時間——不是在催對方，是讓對方知道你有準備。」
(08)

檢核題（2 條）

Q1 · 概念驗證：以下四個開場句，哪一個最符合「條件談判」的正確姿態？
A. 「我今年表現還不錯，希望主管能考慮幫我加薪，不知道可不可以？」
B. 「我做了 3 件對部門有具體貢獻的事：(1) 主導 3 場客戶活動、活動後進商機清單的潛在客戶比前一年提升 40% (2) 獨立產出 12 份月報免主管複審 (3) 業務提案素材協助成案 3 個。基於這個貢獻記錄，我想討論年薪從 70 萬調到 75 萬，以及下半年能不能給我主導一個跨部門專案的機會。」
C. 「我在其他公司看到同職位薪資比我高，所以想請公司調整。」
D. 「我已經在這裡做了 4 年了，我覺得是時候應該升薪了。」
查看解析
正確答案是 B。A 是「求情姿態」，沒有定價依據。C 是「比較姿態」，把談判基礎放在外部數據而不是自己的具體貢獻，容易讓主管感到防禦。D 是「年資姿態」，「做了 4 年」對方聽到的是「他覺得他待夠久了」，不是「他能貢獻什麼」。只有 B 同時做到：(1) 3 個量化貢獻作為定價依據 (2) 具體訴求（薪資 + 資源並陳）(3) 主動提出而不是等對方給，是「談判設計」的姿態而不是「求情」或「比較」。
Q2 · 應用驗證：阿哲跟技術主管談「接 team lead」時，主管說：「你願意先以代理 team lead 的方式試 3 個月，期間職稱不變、薪資不變，3 個月後再評估，你覺得怎樣？」請依「談判三心法 + 反問句策略」，給阿哲 2 條具體可用的應對方向。
查看參考答案
方向 1：接受代理期，但要設計回升機制
可以接受代理期，但要把代理期談成「有結構的過渡方案」。應對句：「代理期我可以接受，但我想確認：3 個月後的評估標準是什麼？如果我達標，職稱和薪資調整在原則上是確定的嗎？能不能把這幾個條件先在這次對話裡確認，然後我整理成書面 + 你確認後，我才開始這個代理期？」不要在不清楚職責的情況下接受代理期，否則 3 個月後主管可以說「再延 3 個月」，你沒有任何依據反駁。

方向 2：用反問探清職責範圍，再決定要不要接受
「代理 team lead」是個模糊名詞，實際職責可能差很多。在接受之前，先用反問句釐清：「代理期間，我對這 3 個工程師的職責是什麼程度？工作分派 / PR review / 績效 input 這 3 件事，哪些是我負責的？」職責明確了，才能判斷「這 3 個月代理期值不值得做、做完之後有沒有轉正的真實路徑」。
