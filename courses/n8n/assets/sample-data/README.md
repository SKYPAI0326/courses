# Sample Data · 課程練習素材

《AI 資料工廠》課程各演練單元使用的標準練習資料。學員照著實作時可直接拿這裡的檔案測試，避免「卡關時不知道是資料問題還是節點設定問題」的灰色地帶。

> **內容原則**：所有素材皆**自製、無版權風險、無真實個資**。若你要新增素材，請遵循同樣原則，勿放入真實客戶資料、真實發票、真實履歷。

## 資料夾結構

```
sample-data/
├── invoices/          # Module 3 PDF 批次處理用：模擬發票
│   └── invoice-sample-001.txt   （文字版範本，學員可自行轉成 PDF 練習）
├── daily-report/      # Module 3 日報生成用：會議紀錄、客戶回饋
└── forms/             # Module 4 Google Form 整合用：欄位定義範例
```

## 對應課程單元

| 單元 | 用到的 sample-data |
|------|---------------------|
| `m3-2-batch.html` / `m3-2-rename.html` | `invoices/` |
| `m3-3-generate.html` | `daily-report/` |
| `m4-1-remote.html` / `m4-1-data.html` | `forms/` |

## 預期輸出

每個演練頁的「✅ 完成檢查」區塊會列出該練習應產生的具體結果（節點亮綠燈、output 欄位、本機檔名、外部服務狀態）。本 sample-data 旨在讓「輸入端」可重現，輸出端的驗證請對照單元頁。

## 增補素材時的 SOP

1. 自製內容、無版權、無真實個資
2. 命名格式：`{用途}-sample-{流水號}.{副檔名}`
3. 在本 README 補一行說明該檔對應哪個演練

## 不要放在這裡的東西

- 真實客戶/學員的發票、履歷、合約掃描
- 含個資的對話紀錄
- 任何受著作權保護的內容
- 未經當事人同意的對話、信件
