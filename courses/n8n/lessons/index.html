課程總覽：兩條路徑任選

本課程承接 Make 學員，升級到 n8n 拿到「本機檔案不用上傳就能處理」的核心能力 — 不再需要把 PDF / CSV / 客戶檔案先傳 Drive 中轉。先決定「我想多快上手」：要 60-90 分鐘下載一個 zip 雙擊就裝完 14 個 workflow？走 M0 快速路徑。要徹底搞懂為什麼這樣設計、回家自己從零做？走 Module 1-4 完整路徑。

想知道本課程為什麼選 Docker 自架而不是 n8n Cloud？看 補充章節：n8n Cloud vs Docker 自架誠實對比（11 維度 + 隱私分層 + 學員情境決策表）。
