Module 01
SECTION 1.1

n8n Desktop 安裝

在這節課，你將完成 n8n Desktop 的安裝與啟動，認識基本操作介面，並建立第一個可接收 Webhook 的工作流程。

⏱預估時間：15–20 分鐘
📋7 個步驟
🔧需要：電腦、瀏覽器
1
前往官網下載 n8n Desktop

打開瀏覽器，前往 n8n 官方網站。找到 Desktop 版本的下載連結，根據你的作業系統選擇對應的版本並下載安裝包。Mac 用戶下載 .dmg 檔案，Windows 用戶下載 .exe 檔案。

🖥
n8n 官網首頁，找到「Download Desktop App」按鈕，選擇對應作業系統版本
💡 Mac 下載 .dmg，Windows 下載 .exe。若官網找不到 Desktop 下載，可搜尋「n8n desktop app download」。
2
執行安裝程序

下載完成後，執行安裝檔。Windows 用戶直接雙擊 .exe，按照提示完成安裝。Mac 用戶打開 .dmg 後，將 n8n 圖示拖曳到應用程式（Applications）資料夾中。

📦
Mac：拖曳 n8n 圖示到 Applications 資料夾的畫面
3
Mac 用戶：解除系統安全性限制

Mac 第一次開啟 n8n 時，系統可能出現「無法驗證開發者」的警告而拒絕啟動。請前往「系統設定」→「隱私權與安全性」→「安全性」，找到 n8n 的警告訊息，點擊「仍要打開」或「強制打開」。

🔐
Mac 系統設定 → 隱私權與安全性 → 安全性，出現允許 n8n 的選項
⚠️ Mac 用戶必做：跳過此步驟的話 n8n 將無法啟動。Windows 用戶可直接跳到步驟 4。
4
第一次啟動 n8n Desktop

點擊 n8n 圖示啟動應用程式。第一次啟動需要稍等片刻（約 10–30 秒），程式會在背景準備本機服務。完成後，應用程式視窗會顯示 n8n 的主介面，或自動在瀏覽器中開啟。

🚀
n8n Desktop 啟動後的主介面，左側有 Workflows 導覽
5
認識 n8n 三大區域

n8n 介面主要分三塊：左側的 Workflows 是建立和管理流程的地方（對應 Make 的 Scenarios）；Credentials 管理服務連線帳號（對應 Make 的 Connections）；Settings 是應用程式設定。

🗂
n8n 介面，左側導覽欄標示出 Workflows、Credentials、Settings 三個位置
🔄 對應 Make 的概念：n8n 的 Workflow = Make 的 Scenario，n8n 的 Node = Make 的 Module，概念一致、操作邏輯相近。
6
建立並命名第一個 Workflow

點擊左側「Workflows」→「New Workflow」建立空白流程。點擊上方預設名稱（如 My workflow），輸入自訂名稱例如「n8n 測試流程」，按 Enter 確認。這個命名習慣有助於日後管理多個流程。

📝
點擊 New Workflow 後，進入空白編輯畫面，並在頂端命名 Workflow
7
新增 Webhook 節點並測試接收

在空白流程中，點擊畫面中央的「+」或「Add first step」，搜尋並選擇「Webhook」節點作為觸發器。點擊節點後選擇「Test step」，n8n 會產生一個測試用的 Webhook URL。複製這個網址，貼到瀏覽器新分頁並按 Enter，回到 n8n 確認節點是否出現綠色的「成功接收」提示。

🔗
左：Webhook 節點設定畫面，複製 Test URL；右：節點顯示綠色「Waiting for test event」後收到資料的狀態
✅ 為什麼先測 Webhook：這個動作確認 n8n 本機服務正常運作。下一節的 Cloudflare Tunnel 就是要讓外部服務（Make）也能送資料到這個 Webhook，而不只是你自己的瀏覽器。
完成驗證 CHECKPOINT
你應該看到這個畫面才算成功

n8n 的 Webhook 節點在你用瀏覽器訪問它的 URL 後，顯示綠色的成功提示，並在節點下方出現接收到的資料結構（即使只是空物件 {}）。這代表 n8n Desktop 安裝正常，本機服務可以接收 HTTP 請求。

常見問題
n8n 無法啟動（Mac）
再次確認「系統設定 → 隱私權與安全性 → 安全性」中是否已允許 n8n 執行。有時需要先嘗試開啟一次被拒絕，才會在設定頁面出現「允許」按鈕。
Webhook 節點沒有收到測試資料
確認已點擊節點後選擇「Test step」（或「Listen for test event」），讓節點進入等待狀態（出現「Waiting for test event」提示）後，再用瀏覽器訪問 URL。若仍無反應，嘗試重新啟動 n8n Desktop。
介面顯示英文，看不懂
n8n 目前不支援中文介面，但操作邏輯與 Make 高度相似。本課程所有步驟均會附上對應的介面截圖與說明，對照操作即可。
