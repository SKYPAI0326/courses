PART 7 · 分析與交付
7.2

Looker Studio：三平台合成一張表

GA4 報表給分析師看、Google Ads 給操盤手看、Meta Ads Manager 給投手看。客戶 / 老闆只想看一張彙整過的，且每週自動更新。Looker Studio（原 Data Studio）就是為這個場景生的。

學完這節，你會
串接 GA4、Google Ads、Meta（透過第三方 connector）資料源
寫計算欄位做 Blended ROAS、CPC、Funnel conversion rate
套用一套可重複使用的「五頁交付模板」
(01)

資料源三種接法

資料源接法注意
GA4原生 connector（免費）快但欄位受限，某些維度（hostname、某些 item_*）要用 BigQuery
Google Ads原生 connector（免費）直接拉 campaign / ad_group 層級費用
Meta / TikTok / LINE付費 connector：Supermetrics / Funnel.io / Windsor.ai每個資料源大約 $29–99/月。或用 n8n / Make 自接 API 寫入 Sheet
Sheets（合併媒體花費）Google Sheets connector最便宜的跨平台整合：每週一手動貼廣告費到 Sheet，Looker 拉這張
✓
省錢解：小預算客戶（月廣告費 < 15 萬）用 Google Sheets 手動彙整。投手每週填一次廣告花費，Looker 自動算 Blended ROAS。付 Supermetrics 等自動方案一年省下的錢未必打平。
(02)

計算欄位（Calculated Field）常用公式

圖表右上「新增欄位」輸入公式：

-- ROAS SUM(purchase_revenue) / SUM(ad_cost) -- Blended ROAS（跨 Sheet 整合） SUM(total_revenue_from_ga4) / SUM(meta_spend + google_spend + line_spend) -- CPC 自算（Meta 平台已經有，但不同來源對齊用） SUM(spend) / SUM(clicks) -- Funnel 轉換率 SUM(purchase_count) / SUM(view_item_count) -- 營收占比（各渠道） SUM(revenue) / SUM(revenue) TOTAL -- CASE 分組 CASE WHEN source_medium = 'google / cpc' THEN 'Google Ads' WHEN source_medium CONTAINS 'facebook' THEN 'Meta' WHEN source_medium = '(direct) / (none)' THEN 'Direct' ELSE 'Others' END
⚠
除以 0 陷阱：ROAS 公式在花費為 0 時會錯誤。加 NULLIF：SUM(revenue) / NULLIF(SUM(spend), 0)。
(03)

交付用五頁模板

可以當作固定交付結構，每個客戶都 clone 一份改配色 / Logo 即可：

頁內容受眾
1. Overview四大 KPI scorecard（營收、訂單、ROAS、CPA） + 日趨勢折線老闆、客戶
2. Media Performance各平台花費 vs 營收 bar + ROAS 表格投手、老闆
3. FunnelSankey / 漏斗圖：view → cart → checkout → purchase優化團隊
4. ProductTop 20 商品銷量、毛利、庫存警示採購、商品部
5. Campaign Detail活動 × 廣告組 × 素材 drill-down投手自己看
ℹ
每頁加 日期篩選器（預設「過去 28 天」） + 活動 / 品牌篩選器。頂部寫資料更新時間（用 CURRENT_DATE() 或「Data last refreshed」原生欄位）。
(04)

自動化分發與權限

Looker Studio 報表做完後兩種發送方式：

  • 排程電郵（Schedule delivery）：每週一早上 9 點把 PDF 寄給 CEO + 業主 + 客戶。設定 → 檔案 → 排程傳送
  • 共享連結：客戶要即時看最新數據時，設「擁有連結者可檢視」。但注意 — 連結內的資料源權限若是你的帳號，客戶看到的是你看得到的資料
◆
交付清單（SOP）：(1) 把報表複本存給客戶自己的 Google 帳號當 owner；(2) 連資料源連到客戶自己的 GA4 / Google Ads（用服務專案也可）；(3) PDF 與 Live Link 都寄；(4) 附一份「五張表怎麼看」的說明文件 —— 客戶才知道哪個數字代表什麼。省略這步，客戶第二週開始就不看了。
