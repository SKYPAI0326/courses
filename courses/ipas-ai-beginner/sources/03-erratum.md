### 4. p.3-27 假說檢定名詞表 — Type I / Type II Error 表格 ⭐⭐⭐

**原版表格錯誤**（拒絕 H₀ 寫成「Type II 錯誤(α)」），**更正後正確表格如下**：

| 真實狀況 | 接受 H₀ | 拒絕 H₀ |
|---|---|---|
| H₀ 為真 | 正確 | **Type I 錯誤（α）** ← 拒絕為真的 H₀ |
| H₀ 為假 | Type II 錯誤（β） | 正確 |

**記法**：
- α = Type I = 「冤枉好人」（拒絕為真的 H₀）
- β = Type II = 「縱放壞人」（接受為假的 H₀）

### 5. p.3-31 模擬考題第 3 題答案修正 ⭐⭐⭐

**原版答案 (B) 錯誤，更正為 (A)**

```
題目：下列何者「並非」K 平均數（k-means）集群法的特點？
(A) 原理相對其他集群法較為複雜  ← 正確答案（K-means 原理其實簡單）
(B) 可結合其他方法，使用上較為彈性
(C) 在特定情況下，能將集群的任務處理得足夠好
(D) 不適合非球形、數據密度變化大或有離群數據的集群問題
```

**解析**：K-means 原理相對簡單（隨機選中心 → 分配 → 更新中心 → 收斂），常與 PCA、Elbow method 等結合使用，對球形且大小密度接近的群體表現良好。

### 6. p.3-32 模擬考題第 7 題（題目+答案敘述+解析） ⭐⭐⭐

**題目修正**：「以下哪一個敘述是正確的」→「以下哪一個敘述**最合乎統計檢定的意義**」

**選項 (B) 敘述修正**：
- 原：(B) 我們有 95% 的信心拒絕虛無假設
- 更正：(B) **我們在 95% 的信心水準下拒絕虛無假設**

**選項 (D) 敘述修正**：
- 原：(D) 我們有 5% 的機率犯型一錯誤
- 更正：(D) **我們犯型一錯誤的機率為 5%**

**正確答案**：(B)

**解析重點**：
- 顯著性水準 0.05 = **容許最多 5% 的機率犯型一錯誤**（α = Type-I Error 上限）
- ⚠️ **不代表實際犯錯機率是 5%**（這是常見誤解）
- p = 0.03 < α = 0.05 → 拒絕 H₀
- 「在 95% 信心水準下拒絕 H₀」表達是嚴謹的統計推論用語
