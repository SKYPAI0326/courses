## 評鑑統計

- 兩科目共 **7 大評鑑主題**
- 共 **17 個評鑑細項**（科一 9 個 + 科二 8 個）
- 鑑別式 AI 與生成式 AI 主題佔比相對重，多橫跨多節

## 課程結構建議（給 course-pm）

對齊上述 7 大主題 → 7 章 + 1 個導讀章（總共 8 章）+ 中級附錄 1 章 + 模擬考 1 章 = **10 章**

```
CH0  導讀（認證介紹、學習方法、考試格式）
CH1  人工智慧概念
CH2  資料處理與分析概念
CH3  機器學習概念
CH4  鑑別式 AI 與生成式 AI 概念
CH5  No code / Low code 概念
CH6  生成式 AI 應用領域與工具使用
CH7  生成式 AI 導入評估規劃
CH8  邁向中級之路（附錄，非考試教材）
CH9  模擬測驗（題庫互動）
```

## 0 基礎學員設計原則

- 每章開場「這章學完你能回答的日常問題」
- 每個專業名詞首次出現必附白話定義 + 日常舉例
- 章末對照「考題長相」連回官方樣題 1-2 題
- 公式類概念（Z-score、IQR、Loss、ROI）以「為什麼要算這個」帶入
