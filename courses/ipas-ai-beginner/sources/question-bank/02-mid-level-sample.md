## 科目一：人工智慧技術應用與規劃（15 題）

**核心考點**：NLP、CNN、RNN/Elman、AutoML、Regression、CV 任務分類、模型壓縮、供應鏈攻擊、多模態 Early Fusion、Transformer、Word2Vec、Z-score、系統整合測試。

### 重點題目

**Q1** [A] 何者並未使用 AI 或 ML 技術？
- (A) 使用一組預定義規則來確定最佳移動做法的象棋遊戲 ✓（純規則 = 非 AI）
- (B) 深度神經網路語音辨識
- (C) ⚠️ 題目陷阱：感測器 + 預定義導航規則的自駕車（看似規則但仍歸類為 AI）
- (D) NLP 聊天機器人

**Q2** [C] 文本資料處理中「將接續的文本轉換為詞彙單位」= **斷詞（Tokenization）** ⭐
- 對比：初級只談 NLP 概念；中級談具體技術名詞

**Q4** [D] 深度學習錯誤敘述：
- (D) **Elman 神經網路適合處理影像辨識** ← 錯（Elman 屬 RNN，適合序列）
- 對比：初級只談「CNN 適合影像/RNN 適合序列」；中級考細節變體名稱

**Q7** [B] 電腦視覺 4 大任務分類圖辨識（含實際圖片）：
- (a) Image Classification（圖像分類）→ 整張圖一個標籤
- (b) Semantic Segmentation（語義分割）→ 像素級分類
- (c) Object Detection（物件偵測）→ Bounding box + 類別
- (d) Instance Segmentation（實例分割）→ 區分同類別不同個體
- 對比：初級沒考這個層級；中級要會看圖辨識

**Q8** [A] 模型壓縮（Model Compression）技術：**參數剪枝（Pruning）** ⭐
- 對比：初級不考；中級談部署優化

**Q9** [B] 供應鏈攻擊（Supply Chain Attacks）：第三方開源模型/數據可能植入惡意內容
- 對比：初級「資料隱私」泛談；中級具體攻擊面

**Q10** [B] 詞嵌入（Word Embedding）：**Word2Vec** ⭐
- 對比：初級不考；中級談文字向量化

**Q11** [B] 多模態 Early Fusion：在「模型輸入階段或特徵提取階段」整合不同模態資料
- 對比：初級談多模態應用；中級談技術整合機制

**Q12** [D] 多模態醫療整合：**Transformer 架構** ⭐
- 對比：初級泛談 Transformer；中級指明實際應用架構選擇

**Q14** [B] 特徵尺度差異極大時，最有效的預處理：**Z-score 標準化**
- 對比：初級已考 Z-score 應用；中級加上「梯度下降敏感度」考量

**Q15** [B] 系統整合測試：驗證模型服務與資料平台、前後端介面協同
- 對比：初級「POC」泛談；中級進入測試階段細節

## 科目二：大數據處理分析與應用（15 題）

**核心考點**：t/F 檢定、ROC/AUC、特徵工程、PCA 降維、IQR 法、差分隱私、即時串流、CIFAR-10 numpy 操作、社群網路圖論分析、混合精度訓練。

### 重點題目

**Q1** [D] 統計檢定不正確敘述：「檢測一年級身高是否等於 170 公分用卡方檢定」（應用 t 檢定）
- 對比：初級不考檢定方法選擇；中級需區分 t/F/卡方適用

**Q2** [A] ROC 曲線正確敘述：繪製真陽性率 vs 假陽性率的關係 ⭐
- 對比：初級泛談模型評估；中級進入 ROC 細節

**Q4** [D] 預測下一季產品銷售數量 = 線性迴歸（連續值）
- 對比：與初級樣題第 20 題類似，但情境更複雜

**Q6** [D] 非監督式降維技術 = **PCA** ⭐
- 對比：初級已考 PCA；中級放在更多選項中辨識

**Q7** [D] IQR 法異常值判定範圍：低於 Q1 - 1.5×IQR 或高於 Q3 + 1.5×IQR ⭐
- 對比：初級泛談 IQR；中級需會具體公式

**Q8** [A] 差分隱私（Differential Privacy）目的：**添加隨機噪聲掩蓋個體訊息** ⭐
- 對比：初級已介紹差分隱私概念；中級考細節機制

**Q9** [B] PCA 降至兩個主成分代表「將原始高維資料轉換為兩個主成分表示」
- 對比：初級泛談；中級需精準理解

**Q10** [C] Z-score 計算：成績 90 分，平均 70、標準差 10 → Z = (90-70)/10 = **2** ⭐
- 對比：初級泛談 Z-score；中級需實際計算

**Q12** [C] 即時監控交易異常 → **串流處理系統**
- 對比：初級不考；中級談架構選擇

**Q13** [C] 分析社群網路使用者互動結構 → **圖論分析**

**Q14** [C] 加速大數據訓練 → **混合精度訓練（Mixed Precision Training）**

**Q15** [B] CIFAR-10 程式碼閱讀：x_test 共 10000 筆 ⭐
- 對比：初級**完全不考程式碼**；中級需讀 Python/sklearn/numpy
- ⚠️ **這是初→中級的最大跳躍點**

## 科目三：機器學習技術與應用（14 題）

**核心考點**：MapReduce、CNN 變體（R-CNN/Inception/ResNet/VGG）、過擬合、評估指標 R²/F1/AUC/Accuracy、Adagrad/Adam、特徵縮放、損失函數、GDPR 被遺忘權、sklearn pipeline 排序。

### 重點題目

**Q1** [A] MapReduce：Map = 一組資料映射成另一組；Reduce = 統合與歸納 ⭐
- 對比：初級不考；中級談分散式運算架構

**Q2** [B] CNN 將卷積層加寬而非加深 = **Inception**（其他選項：R-CNN/ResNet/VGG19 是加深方向）
- 對比：初級泛談 CNN；中級需辨識變體架構名稱

**Q3** [B] 訓練誤差低、測試誤差大 = **過擬合**
- 對比：初級已考；中級保留為基礎題

**Q4** [A] 迴歸模型評估指標 = **R²**
- 對比：初級已知有 R² 但中級指明用途

**Q6** [C] 分類任務深度模型輸出函數 = **Sigmoid 或 Softmax** ⭐
- 對比：初級不考；中級談激活函數選擇

**Q9** [B] 標準化（Standardization）作用：均值 0、標準差 1 ⭐
- 對比：初級已考；中級保留

**Q10** [B] 準確率公式：(TP + TN) / (TP + FP + TN + FN) ⭐
- 對比：初級泛談 Accuracy；中級需會公式

**Q11** [D] 損失函數功能：衡量模型預測與真實值之間的誤差
- 對比：初級已考

**Q12** [B] GDPR 被遺忘權：在符合條件下請求刪除其個人資料 ⭐
- 對比：初級泛談 GDPR；中級具體權利條款

**Q13** [A] 類別不平衡調整 = **資料重抽樣**
- 對比：初級不考；中級談 ML 工程細節

**Q14** [B] 自適應學習率 + 適合稀疏資料 = **Adagrad** ⭐
- 對比：初級泛談優化器；中級需區分 Momentum/Adagrad/Adam/SGD 適用情境

**Q15** [A] sklearn pipeline 排序題（c→a→b→d）⭐⭐⭐
- 對比：**初級完全不考程式碼**；**中級進入 Python 實務**
- ⚠️ 第二大跳躍點

## 初→中級難度跳躍分析（給 CH8 附錄用）

### 概念深度提升

| 初級 | 中級 |
|---|---|
| 「CNN 適合影像」 | 區分 VGG / ResNet / Inception / R-CNN |
| 「資料隱私重要」 | 差分隱私機制、GDPR 被遺忘權細節 |
| 「過擬合要避免」 | Early Stopping / L1/L2 / Dropout 機制細節 |
| 「假說檢定」 | t / F / 卡方 / ROC AUC 適用區分 |
| 「Z-score 概念」 | Z-score 實際計算 |
| 「IQR 概念」 | IQR 法 1.5×IQR 範圍實作 |
| 「優化器存在」 | Momentum / Adagrad / Adam / SGD 適用情境 |
| 「Transformer 架構」 | Self-Attention 機制、Word2Vec 詞嵌入 |

### 技能跳躍

| 初級不考 | 中級開始考 |
|---|---|
| 程式碼閱讀（Python / numpy / sklearn） | ✓ 基本 sklearn pipeline、CIFAR-10 操作 |
| 統計實作（檢定方法選擇） | ✓ t / F / 卡方 |
| 模型架構辨識（圖示題） | ✓ 電腦視覺 4 大任務 |
| 模型壓縮、邊緣部署 | ✓ Pruning、Quantization |
| 系統整合測試 | ✓ 多模型協同測試 |
| 大數據架構 | ✓ MapReduce、串流處理 |

### 學習路線圖建議

**初級通過後想考中級，建議學習方向**：

1. **Python 基礎**（必備）
   - NumPy / Pandas 基本操作
   - sklearn 工作流（fit / predict / pipeline）

2. **進階統計**
   - 假設檢定（t / F / 卡方）
   - ROC / AUC / 混淆矩陣計算

3. **深度學習實作**
   - CNN 變體（VGG / ResNet）
   - RNN / LSTM 應用
   - Transformer 與 Self-Attention

4. **AI 工程化**
   - 模型壓縮（Pruning / Quantization）
   - 模型服務化（API / 邊緣部署）
   - 系統整合測試

5. **資料工程**
   - MapReduce 概念
   - 串流處理架構
   - 圖論分析

6. **AI 安全**
   - 對抗性攻擊
   - 供應鏈攻擊
   - 差分隱私實作
