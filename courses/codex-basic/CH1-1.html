(01)

破題：6 月底，桌面上一包亂檔

情境
你是弄一下工作室的行政會計。6 月底，月結要做。各部門傳來的報帳單命名方式都不一樣（報帳.docx、六月-業務部.xlsx、費用單(最終版2).docx…）；兩份不同來源的供應商清單裡有重複廠商；一份交易明細有幾格空白、有一筆金額大得不尋常。

以前你會花一個下午：手動改檔名、一份份打開複製貼上、肉眼抓重複。今天先不教你怎麼做——先看 Codex 怎麼在 90 秒做完這件事。看完你再決定值不值得學。

!
本單元是純示範，你不需要安裝任何東西。Codex 安裝在下一頁 CH1-2，這頁只要看、只要感受「終點長相」就好。
(02)

Codex 是什麼，和 ChatGPT 的 3 個差異

Codex 是 OpenAI 推出的 AI 助理 App。和 ChatGPT 最大的不同：它不只「回答」，還會動手做事——讀檔、改檔、產生新檔案、寫文件、做表格。

差異 01
以你指定的資料夾為主要工作範圍
它能讀取那個資料夾的真實檔案，不是你貼一段文字它才知道。實際能讀寫的範圍取決於你設定的權限模式、有無啟用外掛與 Computer Use，不是絕對隔離。
差異 02
會產出檔案，不是只給你一段字
它會真的在你電腦裡生出 Excel、Word、PDF——不是截圖、不是說明文字，是可以直接打開用的檔案。
差異 03
能照你給的規則重複做
規則寫一次，下個月同一件事一句話就重跑。這是這門課後半部「把流程做成 Skill」的核心。
(03)

一條責任邊界——全課都會回扣這一句

Codex 不知道自己做不到。規則沒講清楚，它會把錯的東西「很順地」做出來，而且一路狂奔不會停。

!
凡是它產出的，最後都要人看過——做得順不等於做得對。這條線後面每個單元都會回扣（不得補值、寄出要人按、覆寫前確認）。

這也是本課和「炫技型 AI 課」的核心差異：我們要的是「會用、也守得住底線」，而不是讓你盲目交出去。

(05)

觀察單：看 demo 時心裡記三件事

不用寫下來給誰，是給你自己建立判斷：

自我觀察
  • 它是真的改了你的檔，不是嘴上說說。
  • 它有沒有自己亂補不存在的資料？（後面 CH2 會專門練怎麼防）
  • 如果這是真的帳，哪一步你會想先攔下來、自己核一下再用？

然後寫一句話：你最想讓 Codex 幫你做掉的一件月結雜務是什麼？（這句之後每個單元都會用得到。）

學會的標準
  • 你能用自己的話講出 Codex 跟 ChatGPT 的 3 個差異。
  • 你能指出剛剛 demo 裡，哪一步是「要人把關」的（例：採用 Excel 前要先抽查金額）。

兩件都答得出來，就可以進 CH1-2 把 Codex 裝起來。

(07)

常見誤區——先破除，避免後面踩雷

本單元學員尚未上機，這三條是認知誤區，在你開始設定之前先了解：

誤區 1
把 Codex 當「換皮 ChatGPT」
現象：以為它只會回字，把測試檔跟正式月結檔丟在同一個資料夾給它玩。
原因：沒意識到 Codex 會真的改、覆寫資料夾裡的檔案。
解法：正式檔永遠留一份副本不動；練習、測試用獨立資料夾。這是 CH1-3 建「專案資料夾」要強調的。
誤區 2
看到 demo 成果「以後直接交出去」
現象：覺得一句話就生出 Excel，跳過覆核直接用。
原因：把「做得順」當成「做得對」。Codex 文字流暢、表格漂亮，不代表金額對、沒漏列。
解法：建立反射——它產出、你覆核。金額抽查、總額對回來源，是你的責任，不是它的。
誤區 3
「我不會 coding 用不了」
現象：聽到「AI 助理、會寫程式」就先卻步。
原因：誤以為要懂技術、打指令。
解法：本課全程用 Codex 桌面 App、講中文指令即可，不碰終端機、不寫一行程式。
(08)

確認你學到了什麼

Q1：Codex 和 ChatGPT 最關鍵的差異是哪一個？
查看解析
正確答案是 B。Codex 最關鍵的差異在於它會動手做事——在你指定的資料夾裡讀檔、改檔、產出真實的 Excel/Word/PDF，而不只是回答文字。速度、費用、語言都不是核心差異。
Q2：demo 裡 Codex 把 6 份報帳單彙整成一張 Excel。如果這是真的月結帳，你拿去用之前，哪一步一定要先自己把關？
查看解析
正確答案是 C。把關要點有三：
  • 抽查金額與分類是否正確（Codex 可能引用錯範圍、分類錯）
  • 確認它沒有自己亂補不存在的資料（缺值應停而不是硬填）
  • 總額對得回原始來源（沒漏列、沒重複加總）
覆核是你的責任，不是 Codex 的。做得順不等於做得對。
