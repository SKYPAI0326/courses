# 弄一下工作室_6月月結 — 素材包

這是 Codex 基礎使用課程的主線情境素材包。把整個資料夾下載、解壓後，當成你的「工作資料夾」載入 Codex 專案使用（CH1-3 教）。

## 內容

| 檔案 | 說明 |
|------|------|
| `AGENTS.md` | 操作守則範本（命名規則 + 5 條硬規則）。Codex 開專案會先讀它。 |
| 報帳單 ×6（`.docx` / `.xlsx`，命名雜亂） | 各部門傳來的費用報帳，格式與命名不一致——CH2-1 練批次改名與彙整。其中一份內容沒寫清楚部門，用來示範「缺資料即停」。 |
| `供應商_A來源.csv` | 採購系統匯出的供應商主檔（含統編、付款帳戶、地址、聯絡人）。 |
| `供應商_B來源.csv` | 財務備注的供應商清單（欄位較少）。與 A 來源有重複、也有衝突（同統編不同付款帳戶、同統編不同聯絡人、同名不同統編）——CH2-2 練合併去重與主檔檢查。 |
| `transactions.csv` | 6 月交易明細。內含：空金額、空核銷人、異常大金額、統編格式異常、同發票號不同廠商（複合鍵練習）、疑似重複報支——CH2-2 練 Checks 工作表。 |

## 提醒

- 這些是**練習用輸入資料**，數字與廠商均為虛構。
- 開始前先把整包**複製一份備份**，再讓 Codex 動工（改檔名是覆寫動作）。
- Codex 產出的彙整表、Checks 結果，金額與分類**一律要你自己抽查**，不要直接採用。
