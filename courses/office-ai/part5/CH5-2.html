Part 5 行銷文案 AI 實戰
單元 5.2

EDM 與活動文案：從主旨到 CTA 一條龍

行銷信的開信率平均只有 20%，主旨寫得好不好決定了 80% 的成敗。讓 AI 幫你測試多版主旨，再一次生出完整信件。

學完這節，你會
掌握 EDM 四段結構：主旨→開頭→內文→CTA
用 AI 快速產出活動宣傳文與產品介紹文
學會針對不同受眾分群調整文案語氣
(01)

EDM 四段結構：主旨決定生死

一封行銷信（EDM）的命運在收件者點開之前就決定了——主旨行是唯一能讓人從信箱裡停下來的機會。好的 EDM 結構是：吸睛主旨 → 開頭建立關聯 → 內文給好處 → CTA 只有一個動作。

01
主旨行 — 20 字內的戰場
手機信箱只顯示前 15-20 個字。主旨要有「好處」或「好奇」，不要有「廣告感」。
請幫我產出 5 個 EDM 主旨行，主題是[活動/產品]，目標是讓[受眾]想點開。每個主旨不超過 20 字，風格分別是：好奇型、好處型、急迫型、故事型、數字型。
02
開頭 — 3 秒內建立「跟我有關」
不要用「親愛的會員您好」開頭，直接切入讀者的痛點或場景。
開頭第一句直接描述收件者可能遇到的問題或場景，不要用問候語。例如：「每到月底就為報告焦頭爛額？」
03
內文 — 好處優先，功能其次
讀者不在意你的產品有什麼功能，他們在意「用了之後我會怎樣」。先說好處，再說怎麼做到。
內文用「好處→方法→證據」的順序：先說結果（「省下 3 小時」），再說怎麼做（「用我們的模板」），最後給證據（「已有 500 人使用」）。
04
CTA — 只給一個動作
「立即報名」「領取免費模板」「查看優惠」——選一個就好。CTA 越多，轉換越低。
CTA 按鈕文字要具體，不要用「了解更多」。改用「領取免費模板」「立即搶位」「查看完整方案」。

STEP 2 · 試試看
兩版貼進 Gemini，比較哪一版的主旨你自己會想點開。

幫我寫一封促銷活動的 EDM

版本 B（結構化 + 受眾 + 主旨測試）
請幫我寫一封行銷 EDM，推廣公司年度特賣活動。 受眾：過去一年有購買紀錄的老客戶 語氣：親切、有專屬感，讓他覺得這是 VIP 才有的優惠 結構： 1. 先給我 3 個主旨行選項（每個 20 字內，分別用好處型、急迫型、好奇型） 2. 開頭不用問候語，直接切入「你去年買的 [產品] 現在有專屬回饋」 3. 內文：專屬折扣（8 折）+ 限時 3 天 + 免運 4. CTA：「立即查看我的專屬優惠」 全文 200 字內。
提示：B 版的關鍵是「受眾」和「專屬感」——同樣是特賣，對老客戶講「你的專屬回饋」比「全館特賣」有效 3 倍。
STEP 3 · 對照一下
A 版問題：AI 寫出一封「通用促銷信」——主旨是「年度特賣活動」，開頭「親愛的會員您好」，讀者看到第一眼就會歸類為廣告信，直接略過。

B 版關鍵：指定受眾（老客戶）讓 AI 用「回饋」而非「促銷」的角度；要求多版主旨讓你可以 A/B 測試；「不用問候語」避免廣告感；「專屬」「VIP」的語氣讓讀者感覺被重視。

EDM 口訣：主旨要測試、開頭切痛點、好處放前面、CTA 只一個。四個步驟都在 Prompt 裡交代清楚，AI 一次到位。
(02)

活動宣傳文與產品介紹文

除了 EDM，行銷人員最常需要寫的就是活動宣傳文（活動頁、報名頁、海報文字）和產品介紹文（產品頁、DM、型錄）。兩種文案的重點不同：活動文要「製造參加動機」，產品文要「讓人想要擁有」。

01
活動宣傳文模板
重點是：為什麼要來、來了會得到什麼、怎麼報名。三個問題回答完就是一篇好的活動文。
請幫我寫一篇活動宣傳文： 活動名稱：[名稱] 時間地點：[時間] / [地點] 目標受眾：[誰適合參加] 活動亮點：[3 個重點，例如：免費、限額、有實作] 語氣：[熱情邀請/專業權威/輕鬆有趣] 結構：標題（15 字內）→ 一句話說明為什麼要來 → 三個亮點 → 報名方式 字數：200 字內
02
產品介紹文模板
產品文的核心是「功能轉好處」——不要說「AI 驅動」，要說「幫你省下每天 30 分鐘」。
請幫我寫一段產品介紹文： 產品名稱：[名稱] 目標客群：[誰會買] 核心功能：[列出 3 個功能] 語氣：[專業可信/親切易懂/高端質感] 結構： - 一句話定位（這個產品是什麼、給誰用） - 三個好處（不是功能，是用了之後的改變） - 一句社會證明（已有 N 人使用/媒體報導/客戶推薦） - CTA 字數：150 字內
!
受眾分群技巧：同一個產品，對不同受眾說不同的話。在 Prompt 裡加一句「目標受眾是 [新客戶/老客戶/決策者]」，AI 會自動調整語氣和重點。新客戶需要「為什麼選你」，老客戶需要「這次有什麼新的」，決策者需要「ROI 多少」。

STEP 3 · 對照一下
上班族版重點：「省時間」「每天少加班 30 分鐘」「不需要技術背景」——他們在意的是「我能不能馬上用」。

HR 版重點：「提升團隊效率」「降低培訓成本」「可追蹤學習進度」——HR 在意的是「能不能解決組織問題」和「怎麼跟老闆報告成效」。

分群口訣：個人用戶問「對我有什麼好處」，企業用戶問「ROI 是多少」。Prompt 裡的「目標客群」一改，AI 的整個敘事角度就會跟著調整。
(03)

帶走三份行銷文案模板

5.2-A · EDM 完整信件 Prompt
請幫我寫一封行銷 EDM： 受眾：[老客戶/新訂閱者/潛在客戶] 目的：[推廣活動/產品上市/節日優惠] 語氣：[親切有專屬感/專業權威/輕鬆有趣] 結構： 1. 主旨行：給我 3 個選項（20 字內，分別用好處型、急迫型、好奇型） 2. 開頭：不用問候語，直接切入讀者痛點或場景 3. 內文：好處→方法→證據（200 字內） 4. CTA：一個明確動作 + 按鈕文字
5.2-B · 活動宣傳文 Prompt
請幫我寫一篇活動宣傳文： 活動名稱：[名稱] 時間地點：[時間] / [地點] 目標受眾：[誰適合參加] 活動亮點：[3 個重點] 語氣：[熱情邀請/專業權威/輕鬆有趣] 結構：標題（15 字內）→ 為什麼要來（一句）→ 三個亮點 → 報名方式 字數：200 字內
5.2-C · 產品介紹文 Prompt
請幫我寫一段產品介紹文： 產品名稱：[名稱] 目標客群：[誰會買] 核心功能：[列出 3 個功能] 語氣：[專業可信/親切易懂/高端質感] 結構：一句話定位 → 三個好處（不是功能，是改變）→ 社會證明 → CTA 字數：150 字內

常見錯誤對照

X 錯誤版
幫我寫一封行銷信（不說受眾、不說目的）
OK 改良版
幫我寫一封 EDM 給過去一年有購買紀錄的老客戶，推廣年度特賣，語氣要有專屬感，CTA 是「查看我的專屬優惠」
X 錯誤版
EDM 主旨：「年度特賣活動通知」
OK 改良版
用 Prompt 生成 5 個主旨選項，分別測試好處型「省 30% 的機會只到週五」、好奇型「你上次買的那個，有新消息」
X 常見情況
CTA 同時放「了解更多」「立即購買」「追蹤我們」三個按鈕
OK 正確做法
一封信一個 CTA。如果目的是報名，就只放「立即報名」；如果目的是瀏覽，就只放「查看完整方案」
