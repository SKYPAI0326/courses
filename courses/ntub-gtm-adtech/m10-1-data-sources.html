MODULE 10 · SECTION 10.1

Looker Studio 資料源 + 計算欄位

連接 GA4 / Google Ads / Meta Ads / Sheets 四大資料源——並用計算欄位算 ROAS、Blended ROAS、處理 NULL 陷阱。

為什麼這很重要 · WHY THIS MATTERS
Looker Studio 是免費 + 雲端 + 易分享——台灣大多中小型客戶的 Dashboard 首選。
STEP BY STEP · 本節重點
1
1 四大資料源連接
GA4：原生連接器，直接選 property。Google Ads：原生連接器，選 Ads 帳戶。Meta Ads：沒有原生連接器——用 Supermetrics 或 Windsor.ai（付費）或 CSV 匯出。Google Sheets：原生，適合放 LTV / 手動客戶資料。
Meta 沒原生連接是 Looker Studio 最痛點。付 Supermetrics（~$50/月）最穩；零預算用 CSV 定期手動上傳。
2
2 資料模型：Blend vs Join
Blend：類似 SQL LEFT JOIN，用共同維度（日期、campaign name）合併多個資料源。適合『同一時間軸，不同平台的支出與轉換』。Data Blending最多 5 個資料源同時。
Blending 需要維度名稱完全一致——『Campaign』vs『campaign_name』算不同維度。資料源端先命名統一。
3
3 計算欄位：ROAS
建 Blended Data 含 GA4（revenue）+ Ads（cost）+ Meta（spend）。建計算欄位：ROAS = revenue / (cost + spend)。加在 Scorecard 或表格的 metric 欄位。
Looker 的 CASE WHEN 與 SQL 類似。試 CASE WHEN (cost+spend) = 0 THEN 0 ELSE revenue/(cost+spend) END 避免除零錯。
4
4 NULLIF 與 Blended ROAS
MER（Blended ROAS）：total_revenue / NULLIF(total_ad_spend, 0)。NULLIF 把 0 變 null 避免除零。total_revenue 可以是『GA4 總營收』包含有機與直接流量。
Blended ROAS 看起來比平台 ROAS 低——這是對的（分母更大）。老闆懂的話拿這個數字談預算最有力。
5
5 資料刷新頻率 vs 成本
Looker 免費版：GA4/Ads 即時（15 分鐘快取）；Meta 透過 Supermetrics 通常 1 小時。超大資料量可以用 BigQuery 接 Looker——查詢速度更快但 BQ 有 storage + query 費用。小型客戶免費版夠。
Looker 的快取是 Dashboard-level，不是 query-level——同一個儀表板多人同時看不會多查。
6
實戰案例｜把 7 個資料源縫進一張報表
某 D2C 品牌每週要看：GA4（站內）、Meta Ads、Google Ads、LINE LAP、TikTok Ads、Shopify（訂單）、客服 Zendesk（退貨率）。原本是 7 張 Excel 每週人工合併 4 小時。改用 Looker Studio + BigQuery作為中繼層：所有源接到 BQ → 在 BQ 建 view → Looker 接 view。人工時間從 4 小時 → 0（自動每日更新），且數據有版本控制、錯了能重跑。
Looker Studio 直接接多個 connector 會踩『Blend 限制』（最多 5 個、無法複雜 JOIN）。量大一定走 BigQuery 當中繼——BQ 儲存費便宜（$0.02/GB/月）、query 費有每月 1 TB 免費額度、複雜 JOIN/UNION 隨你寫。
完成驗證 · VERIFY
Looker Studio 連成功 GA4 + Ads 兩個資料源，建 1 個 Blend、3 個計算欄位（ROAS / AOV / CVR），輸出截圖存證。
常見問題 · TROUBLESHOOT
Looker Studio 和 Data Studio 是同一個東西嗎？
是。2022 年改名。文件和網址有時還會出現 Data Studio 字樣。
BigQuery 一定要嗎？
不一定。< 100 萬列 / 日資料量 Looker 直連 GA4 夠用。過了才值得 BQ 中轉。
Meta 連接器要付錢嗎？
Supermetrics 約 $50–150/月（看 account 數）。Windsor.ai 類似。零預算用 CSV 匯出 + Sheets + Looker 也可以，但沒人 Schedule 就是手動。
