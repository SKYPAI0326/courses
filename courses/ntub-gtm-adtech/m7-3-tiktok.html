MODULE 7 · SECTION 7.3

TikTok Pixel + Events API

九大標準事件 + Advanced Matching + Events API 後端補——TikTok 的追蹤邏輯其實 80% 像 Meta。

為什麼這很重要 · WHY THIS MATTERS
TikTok Ads 在台灣年輕客群 CPM 最便宜——電商一定要試。Pixel 觀念跟 Meta 非常像，上手快。
STEP BY STEP · 本節重點
1
1 TikTok Pixel 九大標準事件
Browse：ViewContent、Search。Engage：AddToCart、Checkout、AddToWishlist。Convert：PlaceAnOrder、CompletePayment、Contact、SubmitForm、CompleteRegistration、Subscribe。比 Meta 多了『下單 vs 付款』分開追的概念。
PlaceAnOrder（下單）vs CompletePayment（付款完成）：COD（貨到付款）或分期付款時兩者不同時間點。精細追蹤有用。
2
2 GTM Tag：TikTok 官方 Template
GTM Community Template Gallery 搜 'TikTok Pixel' → 官方（由 TikTok 發布）template → 安裝。Base Tag 綁 All Pages + Pixel Code。Event Tag 選 TikTok event name + 帶 value / currency / content_id。
有官方 template 是 TikTok 比 LINE 好上手的主因。用 template 省下自訂 HTML 的坑。
3
3 Advanced Matching：類似 Meta
base Tag 設定中可勾 Advanced Matching → 提供 email/phone/external_id → TikTok 自動 hash 送出。配合 user login 狀態下 push dataLayer 傳過來。EMQ 概念同 Meta。
TikTok 的 Match Quality 指標叫『Event Health』分數 1–5，要 ≥ 3。
4
4 Events API：後端補
TikTok 的 Events API = Meta CAPI 的翻版。Webhook 訂單建立 → Make.com 或 PHP → https://business-api.tiktok.com/open_api/v1.3/event/track/ 送 CompletePayment。需要先在 TikTok Business Center 申請 Access Token。
event_id 去重的邏輯 100% 跟 Meta 一樣——可以共用前端的 event_id 欄位。
5
5 驗證：Events Manager Test Events
TikTok Events Manager → 選 Pixel → Test Events → 產生測試 URL → 用這個 URL 打開你的站 → 操作購買 → 即時看到事件 + 所有參數。
TikTok 的 Test Events 只在特定測試 URL 下有效（不是直接 LocalWP URL）——容易被忽略。
6
實戰案例｜TikTok Events API 救回 Z 世代電商
某潮牌電商目標受眾 18-24 歲，70% 用 iOS 且裝 adblocker。TikTok Pixel 事件覆蓋率只有 48%。導入 TikTok Events API（EAPI）後覆蓋率 → 89%、Match Quality 從 5.3 → 7.6、ROAS 從 2.1 → 3.4。關鍵：EAPI 要帶 event_id 跟 Pixel 去重、ttclid（TikTok click id，從 URL 取）強化歸因。
TikTok Events API 跟 Meta CAPI 概念幾乎一樣，工程師一次學會可以複用。但 TikTok 的 ttclid 只保留 30 天有效、且 URL 上只出現一次——要用 GTM 第一次捕捉時寫入 cookie，後續事件從 cookie 讀。
完成驗證 · VERIFY
TikTok Events Manager Test Events 看到 PlaceAnOrder + CompletePayment 事件，value 與 currency 正確，Event Health ≥ 3。
常見問題 · TROUBLESHOOT
沒 TikTok for Business 帳戶可以先試嗎？
不能。要先申請（免費），才能建 Pixel 與 Events。新帳戶通常 24 小時內審核通過。
TikTok 的 content_id 要用 SKU 還是 ID？
用 SKU（跟 Meta content_ids 一致）。TikTok 也有商品目錄（Catalog）動態廣告，對應 SKU 才能匹配。
TikTok Pixel 吃很多流量，會不會被 Chrome 警告？
Chrome 2024 起對 third-party cookie 更嚴——TikTok 像其他平台一樣逐步改第一方方案（Events API + Pixel enhanced）。短期無解，長期靠 Events API。
