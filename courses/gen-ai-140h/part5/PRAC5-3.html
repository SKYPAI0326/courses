實例演練 · 作品集

個人作品展示首頁

把你在這門課做的所有工具集中到一個首頁。這不只是個人網站的起點，更是你未來求職、接案、分享時最有力的一張名片。

產出目標 · DELIVERABLE
收納至少 3 個工具的個人作品展示首頁（index.html）
每個工具有標題、一句話說明和連結
首頁已部署，有公開網址
Task Overview

任務說明

你現在手上有幾個已經部署或可以部署的工具頁面。這次任務的目標是建立一個「入口首頁」，讓訪客一眼看到你做了什麼，然後點進去試用。頁面不需要複雜——能說清楚每個工具的用途就夠了。

用下方的工具輸入你每個工具的資訊，它會自動生成對應的 HTML 卡片程式碼，你只需要複製貼入你的首頁即可。

🔑
作品集首頁的關鍵：每個工具用「一句話說清楚能解決什麼問題」，不要用技術描述。例如「可以把你的會議逐字稿自動整理成行動清單」比「使用 NLP 的會議摘要工具」更好。
1
列出你要展示的工具
把課程中完成的工具頁列出來（PRAC4-1 ~ PRAC4-4、PRAC5-1 v2，以及任何自己做的工具）。選 3 ~ 6 個最完整的。
2
用下方工具逐一輸入工具資訊
每個工具填入名稱、一句話說明和連結。工具會幫你累積成 HTML 卡片清單。
3
複製生成的 HTML，建立首頁
點「生成首頁 HTML」，把完整的 HTML 複製到一個新檔案 portfolio.html（或 index.html）。用瀏覽器開啟確認樣式正確。
4
推送並部署
把首頁加入你的 GitHub repo，git commit 後 push。如果 PRAC5-2 的部署已設好，平台會自動重新部署。

延伸案例

作品集首頁的排版與 SEO 練過之後，把自己接下來要應徵或接案的真實作品集完整做一次。重點在「三秒內讓人看懂你是誰、會什麼」，而不是塞滿你做過的每個東西。

案例 03 · 自由工作者
轉職前端的第 90 天，作品集首頁要投 3 家公司
難度 ★★☆ 約 50 分鐘

你上班是行銷，晚上自學前端九十天，手上累積四個專案：一個電商商品頁 clone、一個個人記帳工具、一個幫朋友咖啡店做的菜單頁、一個本課程的 Gemini 工具。下週要投三家公司（兩家新創、一家數位代理商），HR 平均只會看作品集首頁十秒。你要決定放哪些、怎麼排。

原始資料｜四個專案簡述
1. 電商商品頁 clone（跟著 YouTube 教學做的，視覺像 Apple 官網，但邏輯都抄） 2. 個人記帳工具（純 localStorage，沒後端，自己每天在用，六個月紀錄） 3. 咖啡店菜單頁（真的上線了，店主有在用，週末平均 80 個訪客，Google Analytics 截圖有） 4. Gemini 匯率即時比對工具（本課程作業，整合真實 API，有 loading / error 處理） 目標職缺： - 新創 A：Junior 前端，要 React - 新創 B：全端，要 Node.js + 一點前端 - 代理商：接案型前端，要能獨立交付整頁
你的任務
  1. 判斷四個專案的取捨：放幾個、順序怎麼排、每個專案主打什麼賣點
  2. 為每個選上的專案寫一句「hook 標題」（不超過 20 字）與三個 bullet 點
  3. 寫首頁 hero 區的一句 tagline（告訴看的人「你是誰、會什麼、想做什麼」）
評估指標
取捨誠實
跟教學 clone 的不放第一，真實上線、自己有在用的往前推
賣點對應
不同職缺可能強調不同專案，對應 JD 關鍵字而不是自我感動
證據具體
用數字說話：80 個訪客、六個月資料、loading/error 都有處理
十秒可懂
hero tagline 一眼看出定位，不是「熱愛前端、持續精進」這種空話
參考 Prompt（建議先自己寫再看）
點擊展開參考答案
【角色】你是幫轉職工程師看作品集的資深前端，看過很多「塞滿 clone 專案的首頁」，會直接指出哪些該砍。 【任務】針對下方四個專案與三個目標職缺，做兩件事： 1. 排序取捨：哪幾個留、哪幾個砍、為什麼 2. 針對留下的每個專案，寫 hook 標題（≤20 字）+ 三個 bullet 點，bullet 用「動詞開頭 + 具體數字或技術」 最後輸出首頁 hero 區 tagline 三個版本：偏新創 A、偏新創 B、偏代理商。 【限制】 - 教學 clone 若沒有自己加東西就砍掉，不要為了湊四個留下來 - bullet 不要寫「使用 HTML/CSS/JS」這種廢話，要寫「解決了什麼問題」 - tagline 不准出現「熱愛」「充滿熱情」「持續學習」 - tagline 要說清楚「想要什麼職位 / 什麼類型的合作」 【輸出格式】 三段：取捨判斷、專案介紹卡片（標題 + 三 bullet）、tagline 三版。
變體挑戰
A改成只投代理商一家，重寫排序——把「咖啡店菜單頁」放第一的話，hook 怎麼寫
B為「個人記帳工具」想三個能量化的指標（連續使用天數、資料筆數、節省時間），嵌進 bullet
CHR 只給你 30 秒口頭介紹，把首頁 tagline 改寫成口說版本（不能超過 50 字）
