44 個 AI 趣味應用

每個 Part 配 4–8 個能在課堂上 demo 的趣味玩具，加上 3 個全課萬用彩蛋。涵蓋圖像／影片／音訊／3D／AR／即時對話／網頁遊戲，多數免註冊、開瀏覽器就玩。標星越多越炸場。最後一區「自己出題」帶學員寫提示詞，現場用 LLM 生出客製玩具。

講師注意：標示「2026 易過時」的工具會頻繁改版或退場（Bolt / v0 / Lovable / Suno / Action Figure 等）。上課前一週請帶課程助教重新試過一輪，確認連結與功能。

DIY

自己出題：用一句提示詞生成趣味體驗

13 組提示詞

前面 44 個是別人做好的玩具。這一區反過來：讓學員自己對 ChatGPT／Claude／Gemini（或本頁的 Poe、AI Studio）下提示詞，現場生出獨一無二的小玩具。複製範本貼上、改成班級主題，就是一份客製彩蛋。設計提示詞抓三要素：說清楚成品形式（單一 HTML 檔／純文字對話）、給具體主題（班級、公司、單元名）、定互動規則（怎麼玩、何時結束）。

貼給 ChatGPT／Claude／Gemini★★★★
① 一句話生小遊戲
請用單一 HTML 檔（CSS 與 JavaScript 都寫在同一個檔案內，不要用任何外部套件）做一個「貪食蛇」網頁小遊戲。要求：方向鍵控制、吃到食物加分、撞牆或撞到自己就結束並顯示分數、配色用米白底搭深灰線條。完成後把完整程式碼一次貼給我。
把 Part 5 vibe coding 縮進一格，學員換主角與配色就有自己的版本
貼給 ChatGPT／Claude／Gemini★★★★★
② 互動心理測驗產生器
我想做一個趣味心理測驗，主題是「你是哪一種辦公室 AI 使用者」。請設計 6 道情境題、每題 4 個選項，並用單一 HTML 檔做成可點選作答的網頁；答完後依選項統計顯示 4 種人格結果之一，每種附一段幽默解析。程式碼請一次給完。
