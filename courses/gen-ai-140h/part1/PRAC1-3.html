這頁的目的：展示「用 Prompt 讓 AI 生成可用工具」的完整流程。你不需要會寫 JavaScript，只需要學會描述你要什麼。
STEP 1 · 確認你的需求

在要求 AI 生成工具之前，先回答三個問題：

1這個工具要解決什麼問題？（例：每次開會都超時，需要有人幫忙盯時間）
2使用者要輸入什麼、看到什麼？（例：輸入議程，看到倒數計時）
3有哪些限制或特殊需求？（例：不需要後端，純 HTML 單一檔案，支援手機瀏覽器）
STEP 2 · 產生工具的 Prompt 範本
// 生成工具的通用 Prompt 結構 你是一位前端開發工程師，擅長用純 HTML/CSS/JavaScript 製作實用的單頁工具。 請幫我製作一個【工具名稱】，功能如下： - 使用者可以輸入【輸入欄位說明】 - 系統會計算/處理並顯示【輸出結果說明】 - 【其他功能，例如：可以重置、可以複製結果】 技術要求： - 純 HTML 單一檔案，不使用外部 JavaScript 套件 - 支援手機瀏覽器，介面簡潔 - 不需要後端或資料庫 樣式要求： - 使用這組 CSS 變數：--c-bg:#f5f3ee; --c-a4:#6b7fa3; - 字型：Google Fonts Noto Sans TC // 可選：附上你的品牌色或設計偏好

延伸案例

示範的計時器與試算器看懂之後，換你把日常會遇到的雜事做成工具。重點不是會寫 HTML，而是能把需求拆到 AI 能一次生出可用成品。

案例 01 · 教育
補習班小老師的單字抽考工具
難度 ★☆☆ 約 40 分鐘

你是國中英語補習班的兼職助教，每週要抽考學生本週背的 30 個單字。目前用紙本隨機挑 10 題，學生常抱怨題目重複。你想要一個 HTML 頁面，丟入單字清單就能自動出題、計分、顯示錯的單字。

原始資料｜本週單字清單
apologize - 道歉 environment - 環境 decision - 決定 recognize - 認出 vegetable - 蔬菜 necessary - 必要的 experience - 經驗 immediately - 立刻 opportunity - 機會 explanation - 解釋 （省略其餘 20 字，結構相同）
你的任務
  1. 先列出工具需求：出題數、方向（中翻英/英翻中）、計分規則、錯題顯示方式
  2. 寫 Prompt 讓 AI 產出單檔 HTML（含 CSS + JS），可直接雙擊打開用
  3. 實測：貼入單字清單、做完一輪，確認計分與錯題顯示正常
評估指標
資料可替換
單字清單放在一個 array，下週換題只改那一段
隨機不重複
同一輪內題目不重複，且分布均勻
即時回饋
答完顯示分數 + 錯題清單，可直接給學生看
零依賴
單檔離線可跑，不需要網路或套件
參考 Prompt（建議先自己寫再看）
點擊展開參考答案
【角色】你是熟悉前端教學情境的工程師，擅長用最少程式碼做出補習班老師能直接用的單檔工具。 【任務】產生一個單檔 HTML（含 CSS + JS 內嵌），功能為國中英語單字抽考。 【資料結構】 在檔案頂部宣告一個 words 陣列，每筆 {en, zh}，目前有 30 筆（我會自己替換內容）。 範例：const words = [{en:'apologize', zh:'道歉'}, ...]; 【功能】 1. 開始畫面：輸入題目數（預設 10）、方向選項（中翻英 / 英翻中 / 隨機混合），按「開始」 2. 作答畫面：一次顯示一題，文字輸入答案，Enter 送出或按按鈕。答對綠色 / 答錯紅色並顯示正解，2 秒後自動下一題 3. 結果畫面：顯示 X/10 分、列出所有錯題（題目｜你的答案｜正確答案），提供「再考一次」按鈕 【限制】 - 單一 .html 檔，雙擊可用，不需要網路 - 同一輪內題目不重複 - 英翻中容許答案的同義詞（可簡單以「答案包含正確翻譯」判斷，不用太嚴格） - 視覺風格簡潔、米白底，手機也能用 - 不引入任何外部函式庫 【輸出格式】 直接輸出完整 HTML 程式碼，從 開始，不需要任何說明文字。
變體挑戰
A改為可上傳 CSV 的版本（讓助教不用改程式就能換題庫）
B增加「錯題本」功能：把錯過的單字寫入 localStorage，下次優先抽考
C新增計時模式：每題 10 秒、全輪倒數，逾時自動判錯

案例 02 · 自由工作者
接案設計師的工時對帳單產生器
難度 ★★☆ 約 45 分鐘

你是 SOHO 平面設計師，每月月底要給 3–5 個客戶對帳。你每天用手機記事本隨手記「4/3 A 客戶 logo 修 1.5h」，月底要手動換算成 PDF 發票附件。你要一個工具：貼入流水帳文字、按一下就生出分客戶彙總＋可列印的對帳單。

原始資料｜一個月的流水帳
4/1 A 客戶 Logo 初稿 3h 4/2 A 客戶 Logo 修改 1.5h 4/3 B 客戶 DM 排版 2h 4/5 A 客戶 名片設計 2.5h 4/8 C 客戶 包裝提案 4h 4/10 B 客戶 DM 第二版 1h 4/15 C 客戶 包裝定稿 3h 4/20 A 客戶 社群圖 6 張 3h 4/25 B 客戶 緊急改稿 0.5h （時薪 A=1200 / B=1000 / C=1500）
你的任務
  1. 拆需求：輸入格式（每行一筆）、時薪表、輸出內容、列印樣式
  2. 寫 Prompt 產出單檔 HTML，含貼文字區、時薪設定、彙總表、可列印樣式
  3. 實測：貼入流水帳、設好時薪、Cmd+P 檢查列印 PDF 樣子
評估指標
容錯解析
日期、客戶代號、時數可接受格式變化（中文/英文日期）
客戶分組
自動依客戶分組並小計時數、金額
列印版面
@media print 下隱藏輸入區，只剩對帳表可直出 PDF
金額正確
小計、總計無四捨五入誤差，可以直接對款
參考 Prompt（建議先自己寫再看）
點擊展開參考答案
【角色】你是熟悉自由接案工作流程的前端工程師，懂得做出「讓非工程師也能用」的單檔工具。 【任務】產生一個單檔 HTML，功能為 SOHO 設計師的月度工時對帳單產生器。 【功能】 1. 上方：時薪設定區（可新增/刪除客戶，每個客戶有代號與時薪） 2. 中間：大 textarea 讓使用者貼入流水帳，格式範例 `4/1 A 客戶 Logo 初稿 3h` 3. 下方：按「產生對帳單」後顯示彙總表，依客戶分組： - 每筆明細：日期 | 項目 | 時數 | 小計金額 - 每組小結：總時數、總金額 - 最底部：本月總時數、總收入 4. 提供「複製為 Markdown」與「列印 / 存成 PDF」兩個按鈕 【解析規則】 - 支援日期格式：4/1、04/01、4 月 1 日 - 客戶代號在日期之後的第一個 A-Z 或中文字串 - 時數格式：數字＋h（可接受小數，如 1.5h） - 項目=去掉日期/客戶代號/時數之後的中間文字 - 無法解析的行集中顯示在最上方「未解析」區，不中斷整體運作 【限制】 - 單檔，不依賴外部套件 - @media print 時只顯示對帳單，隱藏輸入區 - 金額計算用整數相乘後除以 10，避免浮點誤差 - 介面米白底、無印風、無漸層 【輸出格式】 直接輸出完整 HTML 程式碼。
變體挑戰
A改為每個客戶可設定不同稅率（如 B 要開發票 +5%），彙總表自動顯示含稅/未稅
B加上「連續三個月彙總」功能：可貼三個月的流水帳，輸出月對月趨勢
C把時薪設定寫入 localStorage，下次打開不用重設
