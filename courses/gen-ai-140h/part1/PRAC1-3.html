這頁的目的：展示「用 Prompt 讓 AI 生成可用工具」的完整流程。你不需要會寫 JavaScript，只需要學會描述你要什麼。
STEP 1 · 確認你的需求

在要求 AI 生成工具之前，先回答三個問題：

1這個工具要解決什麼問題？（例：每次開會都超時，需要有人幫忙盯時間）
2使用者要輸入什麼、看到什麼？（例：輸入議程，看到倒數計時）
3有哪些限制或特殊需求？（例：不需要後端，純 HTML 單一檔案，支援手機瀏覽器）
STEP 2 · 產生工具的 Prompt 範本
// 生成工具的通用 Prompt 結構 你是一位前端開發工程師，擅長用純 HTML/CSS/JavaScript 製作實用的單頁工具。 請幫我製作一個【工具名稱】，功能如下： - 使用者可以輸入【輸入欄位說明】 - 系統會計算/處理並顯示【輸出結果說明】 - 【其他功能，例如：可以重置、可以複製結果】 技術要求： - 純 HTML 單一檔案，不使用外部 JavaScript 套件 - 支援手機瀏覽器，介面簡潔 - 不需要後端或資料庫 樣式要求： - 使用這組 CSS 變數：--c-bg:#f5f3ee; --c-a4:#6b7fa3; - 字型：Google Fonts Noto Sans TC // 可選：附上你的品牌色或設計偏好
STEP 3 · 迭代改進的常見需求
加入驗證
「請幫我加入輸入驗證：如果數量欄位填入負數或非數字，顯示錯誤提示。」
加入匯出功能
「請幫我在結果區加一個「複製為純文字」按鈕，格式化後可直接貼入 Email。」

「請幫我用 localStorage 儲存使用者上次填入的資料，下次開啟自動帶入。」
