## 使用建議

1. **上課前一週**讀該 Part 指引，選出「必做」vs「選做」組件配合實際時數
2. **每個 MP 開講前**先念「帶領要點」一遍，學員方向比速度重要
3. **冷場 / 卡住時**翻到「常見卡點」段，講師不必即興想解法
4. **示範答案語氣**是完整可朗讀句子，可直接背下當成 live demo 腳本
5. **Peer activity** 前 5 分鐘一定要示範一組，學員才知道要幹嘛
6. **Quiz 綜合任務**評分用各 Part 文末 rubric，保持跨班一致
