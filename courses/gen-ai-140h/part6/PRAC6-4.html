PRAC 6.4 · 1.5 小時

Streaming 回應改造

把一個「等待型」AI 工具升級為「打字機效果」串流回應。閱讀 Before/After 程式碼差異，體驗兩種 UX 的感知差距。

任務目標
理解非 Streaming 與 Streaming 的程式碼差異，知道後端需要設定哪些 HTTP Headers、前端如何接收 SSE 事件，並完成 Part 6 的最後驗收清單。
1
比較 Before / After 程式碼
非 Streaming 版本等模型生成完才一次送出，Streaming 版本設定 Content-Type: text/event-stream 並逐 chunk 送出。兩版本的後端改動不超過 10 行，但使用者體驗差距明顯。
2
體驗兩種 UX 的感知差距
點擊下方「同時開始」，觀察「等待型」和「串流型」在相同模擬延遲下的視覺差異。注意哪個版本讓你感覺 AI 回應更快——即使實際完成時間相同。
感知速度 ≠ 實際速度。串流讓使用者在等待期間有「進度感」，這是讓 AI 應用感覺更好用的最低成本優化之一。
3
記下三個關鍵改動點
Streaming 升級需要改動三個地方：① 後端 HTTP Headers（text/event-stream）、② 後端逐 chunk 寫出（res.write 而非 res.json）、③ 前端接收方式（EventSource 或 fetch ReadableStream）。
4
完成 Part 6 驗收清單
這是 Part 6 的最後一個 PRAC。完成下方清單後，你已掌握 API 呼叫、RAG、Agent 工具鏈、Streaming 四大核心技術，可以開始構建完整的 AI 應用。
程式碼差異：Before → After

後端 Express 路由的改造：

Before（非 Streaming）
app.post('/api/ask', async (req, res) => { const msg = await client.messages.create({ model: 'claude-haiku-4-5-20251001', max_tokens: 1024, messages: [{ role: 'user', content: req.body.prompt }] }); res.json({ result: msg.content[0].text }); });
After（Streaming）
app.post('/api/stream', async (req, res) => { res.setHeader('Content-Type', 'text/event-stream'); res.setHeader('Cache-Control', 'no-cache'); res.setHeader('Connection', 'keep-alive'); const stream = await client.messages.stream({ model: 'claude-haiku-4-5-20251001', max_tokens: 1024, messages: [{ role: 'user', content: req.body.prompt }] }); for await (const chunk of stream) { if (chunk.type === 'content_block_delta') res.write(`data: ${chunk.delta.text}\n\n`); } res.write('data: [DONE]\n\n'); res.end(); });

前端接收方式（EventSource）：

// 前端：用 fetch 接收 Streaming（支援 POST） const res = await fetch('/api/stream', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ prompt: userInput }) }); const reader = res.body.getReader(); const decoder = new TextDecoder(); let output = ''; while (true) { const { done, value } = await reader.read(); if (done) break; const text = decoder.decode(value); text.split('\n').forEach(line => { if (line.startsWith('data: ') && line !== 'data: [DONE]') { output += line.slice(6); outputEl.textContent = output; // 即時更新 DOM } }); }
