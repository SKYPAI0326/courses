Prompt Library — 55 條手機可用 Prompt

創業 AI 實戰手冊完整索引。每條都能手機開著、直接複製、貼進 ChatGPT / Gemini / NotebookLM 就跑，不需改英文、不需學框架。

上方用類別按鈕篩（EF 效率 / MK 行銷 / CS 客服 / RS 研究 / TK 思考 / NB NotebookLM 專用 / WF 跨工具工作流），或直接搜關鍵字。每張卡片點「展開完整 Prompt」→ 按「複製」→ 貼到 AI 工具框裡，把方括號裡的空填起來即可。

EF-01
會議錄音 → 摘要＋待辦
EF 效率 ChatGPT Gemini

開完會錄音丟進去，三分鐘換一份逐字稿、5 點重點摘要、還有誰要做什麼的待辦清單。

展開完整 Prompt
結構示範（綠框即填入欄位）把會議錄音文字貼進去，三分鐘換出逐字稿＋待辦
請幫我把這段會議內容整理成三部分：

[貼上會議錄音轉文字或逐字稿]

1. 逐字稿：保留發言順序，刪除口語贅詞（嗯、那個、對啊）
2. 會議摘要：5 個 bullet points 以內，每點一句話
3. 待辦事項：格式為「誰 / 做什麼 / 何時前」

若某些資訊不完整（例如聽不清楚時間），標註 [待確認]，不要自行補充。

用繁體中文台灣用語。
填入你的內容
EF-02
拍照報價單 → Email 確認信
EF 效率 ChatGPT Gemini

收到客戶或供應商的報價單，拍照 → AI 萃取欄位 → 直接產出回覆 Email 草稿。

展開完整 Prompt
結構示範（綠框即填入欄位）拍照上傳報價單後填四格，直接生確認信
這張照片是一份 [報價單 / 合約 / 發票 / 收據]。請執行以下步驟：

1. 辨識所有文字內容
2. 整理成結構化資料（欄位：項目、單價、數量、小計、聯絡人、日期）
3. 模糊或看不清處標註 [無法辨識]，不要猜測
4. 根據結構化資料，草擬一封確認 Email

Email 要求：
- 收件者：[對方公司的聯絡人]
- 我的身份：[你的名字 / 公司]
- 目的：確認上述報價內容並詢問下一步流程
- 語氣：正式但不僵硬
- 150 字內

[把所有事情條列出來，不用排序]

請幫我：

2. 建議今天的執行順序（上午、下午、明天）
3. 指出哪些事可以授權、延後或取消（並說明理由）

不要建議我「休息」或「運動」這類生活建議。只處理工作事項。

EF-06
把想法變提案大綱
EF 效率 ChatGPT Gemini

腦中有個想法想提給合夥人或投資人，AI 幫你把口水話變成 5 段式提案大綱＋指出最弱的地方。

展開完整 Prompt
結構示範（綠框即填入欄位）想法講得亂沒關係，AI 幫你長成 5 段提案
我有一個想法：[用口語描述你的想法，可以亂講、可以重複]

請幫我：

1. 萃取出這個想法的「核心主張」（一句話）
2. 整理成 5 段式提案大綱：
   - 要解決什麼問題
   - 為什麼現在做
   - 具體方案是什麼
   - 預期成果
   - 需要什麼資源
3. 指出這個想法最弱的地方（被問會答不出來的點）
4. 列出三個我應該先回答的關鍵問題

不要加我沒講過的內容。如果某段資訊不夠，標註 [需補充]。

使用建議
Prompt 裡的方括號欄位務必填滿，空著 AI 只會給你範本文。每條 Prompt 都設計成「把你的話補進去就能跑」的結構，不需改任何英文。WF 類是組合招，每個步驟對應前面單張卡片的編號，照順序做就好。
資安提醒
三條絕對紅線：不貼客戶個資、不貼帳號密碼、不貼未公開的合約條款。即使是免費版 AI 也會把你的輸入拿去訓練模型。有疑慮的資料先去識別化（改代號、遮住金額尾數）再貼進去。
