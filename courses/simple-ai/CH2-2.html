(02)

品牌定位一句話

用 TK-02 從功能、情感、對立三個角度各榨出一句定位，再挑一句你家客人真的會點頭的留下來。

學習成果
能填完 TK-02 四格輸入，讓 AI 產出三版本定位句
能區分功能 / 情感 / 對立三種切角，說出各自適合放在哪裡
能用三個問題（客人點頭？做得到？篩得掉？）挑出最終定位
(01)

「用心沖煮每一杯」——這句話有什麼問題？

為什麼這很重要
爛定位不是錯的，是四海皆準的。一句話能讓人記住你是誰，也能讓你區隔得了隔壁那家——這就是定位的唯一作用。

咖啡館老闆上週末幫自己寫了一頁「關於我們」放在 IG 個人頁。他寫的是：「我們是一家用心經營的精品咖啡館，堅持優質豆源、專業沖煮、溫暖服務，希望給你最棒的咖啡體驗。」

這段話有什麼問題？沒問題，但也沒用。把「咖啡館」換成「拉麵店」「手搖店」「美容院」，一個字都不用改也通。客人看完三秒就忘了，因為他腦袋裡沒有任何一個鉤子可以掛住你。

這就是爛定位：四海皆準、誰都可以說、說了等於沒說。

好定位長什麼樣？試著比較這三句：

(A)
「不追 trend 的固定菜單，每款都做到 80 分以上」
事實鉤子——你腦袋裡立刻有畫面：什麼人會去、為什麼不去隔壁那家。
(B)
「給加班族一杯像家的咖啡，7 點就開」
情緒畫面——時間、對象、感受一句話全交代了。
(C)
「台北少數三種沖煮手法都能現場切換的老闆」
差異訊號——這句話同時篩掉不是你客群的人。

這三句你只要聽一次，就能在腦袋裡生出一張很具體的畫面。今天這 20 分鐘，我們用 CH2-1 做好的品牌聲音摘要跑一次 TK-02，讓 AI 同時給你三個不同角度的定位句，然後你挑一個能真正過你家客人點頭關的留下來。

📱
手機操作重點：品牌聲音摘要如果還在 CH2-1 的 NotebookLM 裡，直接從 Notebook 的「Notes」頁面長按複製；要跨 App 貼到 ChatGPT 的話，複製一次就好，不要來回切。
(02)

三個觀念，剩下時間全部留給實作

這 4 分鐘把三個觀念講清楚。

01
品牌定位一句話
一句話讓人立刻記得你「是誰、賣什麼給誰、跟別人差在哪」，30 字內。不是口號、不是廣告詞，是你跟朋友解釋你家在做什麼時，那個最短版本。
02
三角度（功能 / 情感 / 對立）
同一家店可以從三種切角被記得：功能（你提供了什麼別人沒有的具體東西）、情感（客人走進你家會感受到什麼）、對立（你刻意不是誰、反對什麼主流做法）。三個角度沒有誰比較對，只有「哪個最適合你家現在這批客人」。
03
為什麼要三版本不是一版本
AI 給你一版你會全盤接受；AI 給你三版你會開始比較，而比較的過程就是你自己把定位想清楚的過程。AI 的角色是幫你擴展選項，不是幫你決定答案。
⚠️
為什麼「優質 / 用心 / 專業」這類詞等於沒講？因為所有同業都這樣寫。你說「用心」，隔壁也說「用心」，客人根本沒辦法用這個字區分你們。定位的作用是把你從一堆「差不多的店」裡拉出來，泛用形容詞做不到這件事。所以 TK-02 的最後一行寫：「不要產出『優質』『專業』『用心』這類泛用形容詞。」——這條限制是整則 Prompt 的命脈。
(03)

TK-02 操作示範：咖啡館老闆跑一遍

我現場用咖啡館老闆的資料跑一次 TK-02，你看完馬上自己跑一遍。

輸入素材（接續 CH2-1 的 NotebookLM 產出）：

【品牌聲音摘要 — 咖啡館 A（CH2-1 產出）】 - 核心詞彙：單品、手沖、固定菜單、老闆挑豆、8 折熟客、下午客少 - 語氣基調：平實、不炫技、略帶自嘲 - 客戶最常讚美的 3 件事： 1. 「拿鐵奶泡比連鎖店扎實」 2. 「老闆會記得我上次點什麼」 3. 「菜單不變很安心，不用每次研究」 - 客戶最常抱怨的 2 件事： 1. 「下午常常找不到位子」 2. 「沒有 IG 拍照點」 - 不會出現在品牌溝通裡的詞：網美、打卡、限定、賦能

TK-02 Prompt 原文（建議工具：延用 CH2-1 的 NotebookLM，或 ChatGPT 把上面這份摘要貼進去當背景資料）：

我需要你協助我萃取品牌定位，濃縮成一句話。 我的業務：[一兩句話描述你做什麼] 我的客戶：[誰會買、他們為什麼買] 我與競爭對手的差異：[你有什麼別人沒有的] 我不想服務的客戶：[例：殺價型 / 要 24 小時回覆的] 請做三件事： 1. 先指出我上面哪些資訊太模糊，列出需要補充的問題 2. 等我補充後，產出 3 個不同角度的定位句： A. 功能導向（強調你提供什麼） B. 情感導向（強調客戶感受到什麼） C. 對立導向（明說你「不是」誰） 3. 每句控制在 30 字內 不要產出「優質」「專業」「用心」這類泛用形容詞。

五零件拆解：

身份
隱含的「品牌顧問 / 定位教練」（TK-02 靠任務描述帶出身份，沒明說但 AI 會抓到）
任務
萃取品牌定位，產出三角度版本
格式
A / B / C 三句，每句 30 字內
限制
不用「優質 / 專業 / 用心」這類泛用詞
退路
資訊模糊時先列補充問題，不硬產

咖啡館老闆填完的版本：

我的業務：台北單店咖啡館，開店 2 年，只賣單品手沖 + 義式拿鐵，菜單固定不換。 我的客戶：週間下午來寫東西 / 開會的一人上班族；固定菜單讓他們「不用研究」。 我與競爭對手的差異：老闆自己挑豆、記得熟客上次點什麼、菜單不變；不做季節限定、不追 IG 拍照點。 我不想服務的客戶：把我們當 IG 打卡點的、要求特調的、殺價比菜單價低的。

預期輸出示例（三版本定位）：

【A 功能導向】 「台北少數 2 年不改菜單的單品店，老闆挑豆、老闆記你」 【B 情感導向】 「下午來寫東西的那杯咖啡，不用研究菜單，老闆記得你上次點的」 【C 對立導向】 「不追 IG 拍照點、不做季節限定，只服務每兩週要來一次的你」

看到這三句的差別了嗎？A 放的是事實鉤子，適合對新客解釋。B 放的是情緒畫面，適合 IG 置頂介紹。C 放的是分眾篩子，適合店門口篩掉不是你客群的人。

卡關備案 3 條：

B1
AI 產出太抽象
例：「讓咖啡成為生活的陪伴」。解法：把限制詞追加「也不要『陪伴 / 生活 / 溫度 / 品味』這類詞」，再跑一次。限制越具體，AI 越會往實話走。
B2
三版本讀起來都差不多
通常是「我與競爭對手的差異」那格寫太模糊。把那格改成具體名詞 / 數字（例：不是「菜單不變」，而是「主菜單 2 年沒動過」），再跑一次就會拉開差距。
B3
NotebookLM 免費配額用盡
把 CH2-1 產出的品牌聲音摘要整段複製，貼到 ChatGPT 的對話開頭當背景資料，再貼 TK-02。結果不會比 NotebookLM 差太多，因為真正的資料已經在文字裡。
(04)

現在換你：8 分鐘跑一輪 TK-02

這段 8 分鐘我不講話，你跟著做。

1
把 CH2-1 的品牌聲音摘要打開（2 分鐘）
如果你 CH2-1 已經建好 NotebookLM 品牌 Notebook，點進去，把 NB-01 的輸出整段複製。還沒有的話，用咖啡館老闆的模板當背景資料練手感也可以——重點是跑過一次流程。
2
填 TK-02 四格（2 分鐘）
在手機記事本或 ChatGPT 輸入框填完：我的業務 / 我的客戶 / 我與競爭對手的差異 / 我不想服務的客戶。最後一格不能空——寫得出不要誰，才寫得出要誰。
3
把 TK-02 整段貼進 AI，跑三版本（2 分鐘）
建議工具：NotebookLM（延用 CH2-1 的 Notebook）或 ChatGPT（把 CH2-1 的摘要先貼一次，再貼 TK-02）。跑完如果 AI 先問你補充問題，老實補充，不要跳過。
4
三選一 + 問自己三個問題（2 分鐘）
不要急著選最漂亮的那句。依序問：① 這句話是我家客人會點頭的嗎？② 這句話是我做得到的嗎（不是「想做」，是「現在就在做」）？③ 這句話能把不適合的客人篩掉嗎？選好後，貼在手機備忘錄置頂——CH2-3 產社群貼文時會直接用到它。

完成標準（self-check）：

✓
TK-02 四格全部有內容，「不想服務的客戶」不是空的
✓
AI 給了三版本，每句 30 字內，沒有「優質 / 專業 / 用心」這類詞
✓
已挑 1 句當最終版，並寫下另外兩句不選的理由（一行）
✓
最終版已存入手機記事本或 IG 簡介，方便 CH2-3 直接呼叫
📱
手機操作重點：三版本產出後，馬上長按複製整段，貼到手機記事本或 Apple Notes 存起來；不要只留在 AI 對話裡，下次 session 過期就找不到了。
(05)

確認你學會了，不是聽懂了

自己回答下面三個問題，有一個答不上來就回頭補。

問題一
你選的那句定位，跟隔壁同行能不能互換？把「咖啡館」換成「拉麵店」「美容院」，這句還通嗎？如果通，代表你挑的那句太泛，回去選另一版或再跑一次 TK-02 把差異欄位寫具體點。
問題二
你選的那句，最近 5 個老客人看了會點頭嗎？實體上可以真的傳給其中一兩個熟客問：「我這樣形容我家店，你覺得像不像？」他說「像」就過關；他說「咦？」就要想是客群不對還是定位不對。
問題三
你選的那句話，能讓你篩掉想避開的客人嗎？如果這句話寫在店門口 / IG 簡介，會不會讓你不想服務的那群人自動少來？會就過關；不會就代表這句話還是太通用。

這三題都點頭，你這 20 分鐘就成立了。接下來 CH2-3 我們會拿這句定位當輸入，一次產一週七則社群貼文——所以先把它存好，別搞丟。

(06)

試跑包需求

現場教學要備好下列素材，少一樣學員就會卡住。

P1
TK-02 Prompt 全文
含五零件拆解標註、填空提示、禁用詞黑名單。填空提示：「不想服務的客戶」不能空；具體名詞 / 數字越多，三版本差異越大。
P2
咖啡館 A 品牌聲音摘要（假資料）
接續 CH2-1 的 NotebookLM 輸出：核心詞彙清單、語氣基調、客戶讚美 Top 3、抱怨 Top 2、禁用詞清單。
P3
TK-02 填空範例（四格已填）
咖啡館老闆版本：業務 / 客戶 / 差異 / 不想服務——供講師現場投影示範。
P4
NotebookLM / ChatGPT 雙路徑截圖
兩路徑各 2 張截圖：NotebookLM 內貼 TK-02 的流程 vs ChatGPT 先貼品牌聲音摘要再貼 TK-02 的流程。
P5
預期輸出示例（三版本）
A 功能：「台北少數 2 年不改菜單的單品店，老闆挑豆、老闆記你」；B 情感：「下午來寫東西的那杯咖啡，不用研究菜單，老闆記得你上次點的」；C 對立：「不追 IG 拍照點、不做季節限定，只服務每兩週要來一次的你」。每句附適合用在哪裡一行註解。
(07)

常見錯誤 3 條

01
三版本都偏功能，沒有情感或對立角度
原因：「我與競爭對手的差異」欄只寫了功能性事實，沒提供情緒線索或對立訊號。AI 只能從你給的素材擴展，給不出你沒放進去的東西。
解法：把 CH2-1 NotebookLM 產出的「客戶最常讚美的話」原句貼進差異欄，再追加一行「我反對的業界做法」。對立訊號有了，C 版本才產得出來。
02
AI 產出讀起來很美，但老闆自己念都卡
原因：TK-02 只禁了「優質 / 專業 / 用心」，沒禁「陪伴 / 溫度 / 淬鍊 / 相遇 / 生活」這類文青但空洞的詞。AI 為了「不踩禁用詞」會往過度抒情的方向飄。
解法：把限制詞追加為「不要『陪伴 / 溫度 / 淬鍊 / 相遇 / 品味 / 生活 / 時光』這類抽象名詞；只用具體可驗證的事實詞（例：2 年、30 字、熟客、下午）」，重跑一次。
03
挑了最威的那句，但實際上做不到
原因：「問自己三件事」的第二題（這句話是我做得到的嗎）沒有認真答。學員挑「AI 產出最威的那句」而不是「自己實際做得到的那句」——對外承諾超過實際服務，客人第二次就會踩到失望雷。
解法：三版本到手後，逐句標註「我現在真的做得到 / 還沒到位」。做不到的直接淘汰，不管它聽起來多好。定位是承諾，不是廣告。
(08)

檢核題

Q1 — TK-02 為什麼要一次產三個不同角度的定位句，而不是直接叫 AI 給一句最好的？
A. 因為免費版 AI 單次產出不準，要多版本互相對照
B. 因為三版本可以分別放在不同通路（IG、店門口、簡報）
C. 因為比較三個角度的過程，是老闆自己把定位想清楚的過程；AI 幫你擴展選項，不幫你決定答案
D. 因為品牌定位本來就需要功能、情感、對立三個都有，缺一不可
查看解析
C 最接近本單元核心。A 是工具論但不是本質原因；B 雖然成立但屬於延伸應用，不是根本理由；D 錯在「三個都要有」——實際上你只會選一個當主定位，另外兩個是參考選項，不是「缺一不可」。
Q2 — 你跑完 TK-02，AI 給你這三句定位：A 功能：「嚴選單一產區精品豆，職人手沖每一杯」；B 情感：「在城市裡，給你一杯像家的咖啡」；C 對立：「不跟連鎖店拚速度，只服務願意等 5 分鐘手沖的熟客」。你現在要挑一句印在名片背面，你怎麼選？
查看預期答案要點
1. A 和 B 先被淘汰：A 有「嚴選」「職人手沖」——業界幾乎每家精品咖啡都這樣寫，無法區分你（通用性測試失敗）；B 有「像家的咖啡」「城市裡」這類泛用情感詞，把「咖啡」換成「茶館 / 餐酒館 / 書店」一樣通（隔壁可互換測試失敗）。
2. C 最可能中選：「不跟連鎖店拚速度」是對立訊號，「願意等 5 分鐘手沖的熟客」是具體客群描述，兩者都是其他同業不太敢直白寫的——能把「趕時間、要立刻拿走」的客人篩掉，對雙方都好。
3. 選 C 前要過第二關（做得到嗎）：如果你實際上希望擴張外帶、做上班族商務客，C 版本會自我設限不適合；要先確認「服務願意等的熟客」真的是你當下策略，再選它。如果不是，應該回頭把差異欄寫具體點重跑一次，而不是硬挑現有三版本。
