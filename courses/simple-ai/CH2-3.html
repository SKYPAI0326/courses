社群內容產線

週日晚上 20 分鐘，用三則 Prompt 一條龍備好下週七則貼文骨架、一則的 A/B 標題、一支 30 秒短影音腳本。

學習成果
能用 MK-01 一次產出一週七則社群貼文骨架
能用 MK-02 為同一則貼文產出 A/B 版標題（三組切入點各不同）
能用 MK-03 把任一則貼文改寫為含開頭鉤子、中段鋪陳、結尾 CTA 的 30 秒短影音腳本
(01)

週日晚上十點，那個固定旁白

為什麼這很重要
社群貼文的難題不是「會不會寫」，是每週都得從零想。三則 Prompt 把「週日晚上備好下週內容」變成固定的 20 分鐘流程，不用再每天晚上盯著空白框發呆。

週日晚上十點，咖啡館老闆坐在店裡收完銀、洗完杯子，想起一件事：下週一早上要發 IG。他打開 IG，看著空白的發文框，腦袋也是一片空白。上週發了新拿鐵、前天發了門口的貓、再上週發了……想不起來上週發了什麼。滑一下別家咖啡館的帳號：人家一週五則、排版整齊、每則都有主題。自己呢？想到什麼發什麼，一週頂多兩則，常常發完自己都覺得「這是在發什麼」。

「下週七天，每天一則，現在想不出來。」這句話是老闆每個週日晚上的固定旁白。今天這 25 分鐘，要解掉的就是這個固定旁白。用三則 Prompt，接著上一堂（CH2-2）你剛寫好的品牌定位，從頭做到尾。

💡
為什麼不是一條 Prompt 全包？因為一週七則、A/B 標題、30 秒腳本是三種不同形狀的東西，一條 Prompt 塞進去，AI 會四不像。先用 MK-01 把骨架鋪出來，看哪一則最有感，再用 MK-02 打磨標題，最後用 MK-03 改寫短影音。三件事分開做，每件事都做得好。
(02)

關鍵概念：骨架、鉤子、A/B 標題、短影音三幕

骨架 vs 成稿
核心概念
這一節產出的不是「可以直接發」的完稿，是每則貼文的四件套。
✓主題（這則在講什麼）
✓開頭吸睛句（讓人停下滑動的鉤子）
✓核心訊息（150 字內，含具體產品/場景）
✓CTA（行動呼籲：留言 / 私訊 / 點連結）
一週節奏感
內容策略
七則不是亂發，每天有自己的角色，避免七天都在推「來喝咖啡」。
✓週一打招呼 / 週二產品 / 週三客戶
✓週四幕後 / 週五週末推薦 / 週六互動
✓週日品牌理念
✓這個節奏是定錨，不是規定
A/B 標題的用意
標題優化
同一則貼文寫兩版不同角度的開頭，逼自己想過兩個切入點，挑一個比較有信心的發出去。
✓不是改字，是改切入點（疑問型 / 衝突型 / 故事型）
✓不是要上廣告 AB test
✓唸出來不尷尬的那組就是對的
短影音三幕結構
影片腳本
30 秒腳本分三塊，每塊對應一個拍攝畫面（shot）。
✓開頭 3 秒鉤子（抓住不滑走）
✓中段 15-20 秒核心訊息鋪陳
✓結尾 3-5 秒具體 CTA
✓沒有這三塊，拍出來是日常 vlog 不是行銷影片
(03)

操作示範：MK-01 → MK-02 → MK-03 一條龍

接著上一堂（CH2-2），我們為咖啡館老闆寫好的品牌定位是：給加班族一杯像家的咖啡。現在打開 ChatGPT（手機免費版就行，手邊如果已經在 CH2-1 建好咖啡館 Notebook，用 NotebookLM 效果會更好）。

1
MK-01 — 一週七則社群貼文骨架
把下方 Prompt 貼進 ChatGPT，只換三個欄位：品牌定位 → 你的那句；目標客戶 → 你想賣給誰；本週主題 → 你下週想主推的東西。按送出，等 20 秒。
幫我產出一週（7 則）Instagram 貼文骨架，為品牌建立內容節奏。 品牌定位：給加班族一杯像家的咖啡。 目標客戶：下班後想喘 30 分鐘的 25-40 歲上班族，以及週末想找個安靜角落的一人客。 本週主題：新品上市（本週主打：肉桂燕麥拿鐵 $140） 七則貼文節奏： - 週一：週一激勵或打招呼 - 週二：產品介紹 - 週三：客戶故事 - 週四：幕後花絮 - 週五：週末推薦 - 週六：互動問答 - 週日：品牌理念或一句話 每則格式： 1. 主題（一句話，我要知道這則在講什麼） 2. 開頭吸睛句（第一句要讓人停下滑動，禁用「大家好」「今天要分享」這類廢話開場） 3. 核心訊息（150 字內，要提到具體產品名、價格或場景） 4. 行動呼籲（留言 / 分享 / 點連結 / 私訊 擇一，要具體） 5. Hashtag（5-8 個，大中小混合，加入在地標籤如 #台北咖啡 #松山區） 風格：親切、有家的感覺、像跟老顧客聊天，不要推銷感。 禁用：限時、必買、錯過終生後悔等誇張行銷詞；不要用驚嘆號超過一次。
2
確認七則骨架的品質
看到 AI 產出的表格，注意四件事：① 每則開頭句是具體場景，不是「大家好」；② 每則都點到品牌定位（加班族 / 像家 / 30 分鐘）；③ 有一則直接寫產品，有一則完全不提產品，不是七則都在推銷；④ 有具體的產品名、具體的價格、具體的場景——這是 Prompt 有寫「要提到具體產品名、價格或場景」的結果。
3
MK-02 — 為一則貼文產 A/B 標題（接著上一個對話貼）
從七則中挑一則「心裡有感覺但開頭句還不到位」的，把下方 Prompt 貼進去，把「週三那則客戶故事」換成你那則的主題+核心訊息。看產出的三組，唸出來，圈一組。
上面週三那則客戶故事，幫我用兩個不同的切入點，各寫一組標題 + 開頭第一句，共三組（6 句）。 每組要標明： - 切入點：疑問型 / 衝突型 / 故事型 擇一 - 標題 + 開頭句（標題 15 字內、開頭句 25 字內） - 這組最可能吸引誰（一句話描述那類讀者） 禁用：驚嘆號、省略號、「你一定要知道」這種老套句。
4
怎麼選 A/B 標題
不是哪組比較「漂亮」，是問自己兩件事：① 哪一句你唸出來不會尷尬？如果這句貼文下面要留你的名字，你會不會想改掉。② 哪一句最像你的目標客群會被勾到？回頭對品牌定位，不是對你自己喜歡。挑一組就好，挑了就用，其他存著下次用。
5
MK-03 — 30 秒短影音腳本（接著貼）
從七則中挑一則最有畫面感的（通常是產品 / 幕後 / 客戶故事），把下方 Prompt 貼進去，把咖啡館的設定換成你的。看產出的腳本——如果可以，週末真的拍一支試試。
把上面週五那則「週末推薦」改寫成 30 秒短影音腳本，可以在 Reels / Shorts 上發。 產品：肉桂燕麥拿鐵 + 司康 週末組合 $180 目標客群：下班後想犒賞自己、週末想獨處的 25-40 歲上班族 影片目的：導購（希望看完的人週末會來店裡點這個組合） 拍攝場景：咖啡館吧台 + 窗邊座位 + 產品特寫（我只有手機一支，沒有攝影團隊） 腳本要求： - 第一句必須在 3 秒內抓住注意力（具體動作或反差，不要「嗨大家好」） - 總長度約 80-100 字（30 秒念得完） - 分 4-5 個鏡頭（shot），每個鏡頭 5-7 秒 - 最後 3 秒要有明確 CTA - 每個鏡頭要寫：【時間】畫面描述 + 口白 風格：口語、像自言自語，不要廣告感。 禁用：「快來買」「限時優惠」「錯過就沒有」這種推銷詞。
📱
手機操作重點：手機上按輸入框右邊的麥克風，直接口述品牌定位和本週主題，比打字快 3 倍。AI 產出的表格如果太長，按產出訊息右下角的分享，存成 PDF 到檔案；或長按→複製到備忘錄。覺得某一則不對味，直接回它：「週四那則太普通，重寫一版，開頭要具體到時間和動作」，不用整串重跑。
(04)

動手練習：跑完你自己的一條龍

用你自己的品牌（店 / 作品 / 副業 / 任何你想經營 IG 的東西）跑完一條龍。

步驟 1
拿出品牌定位
把 CH2-2 你寫好的品牌定位一句話拿出來（沒有就用「我賣 [什麼] 給 [誰]」湊一句）。
步驟 2
跑 MK-01
打開 ChatGPT（或 NotebookLM，如果你 CH2-1 有建 Notebook）。貼 MK-01 Prompt，換三個欄位：品牌定位 / 目標客戶 / 本週主題。送出，拿到七則骨架。
步驟 3
跑 MK-02
從七則中挑一則你最有感的，貼 MK-02 Prompt，拿到三組 A/B 標題，圈一組。
步驟 4
跑 MK-03
從七則中挑一則最有畫面感的（通常是產品或幕後），貼 MK-03 Prompt，拿到 30 秒腳本。
步驟 5
存到手機
把三份產出存到手機備忘錄或截圖存相簿，形成「下週社群原料包」。
✓
完成標準（self-check）：① 七則骨架中至少 5 則提到具體產品名 / 具體價格 / 具體場景；② 七則的開頭句沒有任何一則是「大家好」這類廢開場；③ A/B 標題三組的切入點明顯不同（不是改字，是角度換）；④ 短影音腳本的第一句 3 秒內有具體動作或反差；⑤ 最重要：你讀完七則骨架，問自己「這週我真的會想發嗎」——如果答案是「會」，過關。
(05)

完成驗證：三件東西都在手上了嗎

收尾 15 秒，自己檢查手邊有沒有這三件東西：

本節驗收清單
一張七則骨架表（MK-01 產出）：有七天、每天四件套都不空，且其中有具體產品名 / 價格 / 場景
一則貼文的 A/B 標題（MK-02 產出）：至少兩組切入點明顯不同，已圈出你會用的那組
一支短影音腳本（MK-03 產出）：有 4-5 個鏡頭、第一句 3 秒內有鉤子、結尾有具體 CTA、你看著腳本能想像拍出來長什麼樣

這三件東西就是你下週社群排程的原料。週一早上你要做的只剩：看骨架 → 拍照 → 貼上去。

(07)

常見錯誤：三個最容易踩的雷

錯誤 1：七則骨架都是「陽光正好、來一杯咖啡吧」這種通用句，換任何一家店都適用
原因：Prompt 沒給具體輸入——沒有具體產品名、沒有具體客戶情境、沒有本週主題；或品牌定位欄位填太抽象（「給客戶最棒的體驗」）。
解法：回去把三個欄位補具體——品牌定位要有「誰 / 什麼感受 / 什麼東西」三件、目標客戶要有年齡和情境、本週主題要有具體產品名+價格。然後在 Prompt 最後加一句「重寫，每則都要具體到我的產品名和真實情境」。
錯誤 2：MK-02 產出三組 A/B 標題，三組唸起來都是同一個調性（「你知道X嗎 / 你會X嗎 / 你有X嗎」全是問句）
原因：Prompt 沒強制 AI 切換切入點類型，它偷懶給你三個同類型的變體。
解法：在 Prompt 明確指定三組的切入點各不一樣——「一組疑問型、一組衝突型、一組故事型」，用分類強制它換角度。如果內文太單薄，先把那一則的核心訊息擴寫成 100 字以上再丟給它切標題。
錯誤 3：MK-03 產出的腳本，第一句是「嗨大家好，我是 XX 咖啡館的老闆，今天要跟大家介紹我們的新品」——這句用掉 8 秒，觀眾早在第 2 秒就滑走了
原因：AI 的預設反射是「先自我介紹再進入主題」，這是長影片的邏輯，放到短影音完全不對。Prompt 如果沒特別強調「第一句 3 秒內抓住注意力」或沒給具體範例，AI 就會套長影片模板。
解法：Prompt 明確寫「第一句必須在 3 秒內抓住注意力，禁用『嗨大家好』『我是 XX』『今天要介紹』」，最好再給一句範例（例：「像『禮拜五下午最適合做兩件事』這種具體場景句」）。產出後如果還是看到「嗨大家好」，直接回它：「第一句砍掉重寫，用一個具體動作或反差句代替」。
(08)

自我檢核：兩道題確認你學到了什麼

Q1：MK-01 產出一週七則貼文骨架，每則的「四件套」是哪四件？
顯示答案
正確答案：B。骨架聚焦在文字邏輯，不處理視覺（封面 / 排版那是發文當天才補的事）。四件套是：這則在講什麼（主題）/ 怎麼讓人停下滑動（鉤子）/ 想讓人接收什麼（核心訊息）/ 看完要做什麼（CTA）。C 混進了產品細節（是核心訊息的一部分，不是四件套層級）；D 是視覺設計範疇。
Q2：你用 MK-01 跑完手工皂的一週七則，產出看起來都是「今天天氣好適合洗澡」「週末記得放鬆」這類任何店都能發的通用句。下一步你該怎麼做？
顯示答案
正確答案：C。最有效的兩個動作是：① 補具體輸入——品牌定位加「誰 / 什麼感受 / 什麼產品」、目標客戶加年齡和情境、本週主題加具體產品名+價格；② 下指令重寫——在同一對話追加「這七則太通用，重寫一次，每則都要具體到我的產品名 / 我的真實情境」。如果已在 CH2-1 上傳過產品表+客戶評論到 NotebookLM，改用 NotebookLM 跑 MK-01 效果更貼近。
🚀
下週一你會發什麼？現在就打開 ChatGPT，用上面 MK-01 的 Prompt，填好三個欄位（品牌定位 / 目標客戶 / 本週主題），按送出。60 秒後，你手邊就會有一則「下週一早上可以發」的 IG 貼文骨架。剩下六則、A/B 標題、短影音腳本，等你週日晚上有空再補。社群內容產線的起點，就是這 60 秒。
