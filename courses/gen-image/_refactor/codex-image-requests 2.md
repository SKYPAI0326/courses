## 生圖驗收標準（給 Codex 用）

每張新圖需通過以下 4 項：

1. **版型對應**：商品 / 主標 / 徽章 / CTA 位置與 prompt 描述一致
2. **配色 ≤ 3 色**：沒被 AI 自由加色（特別注意 v31-33 preset 對照組必須 3 色明顯不同）
3. **字階三層**：主標 > 副標 > 內文
4. **必要元素齊全**：prompt 中列的 ≥ 80% 都在
5. **無字體名外洩**：圖中不可出現「Shippori Mincho / Noto Sans / Inter」等字樣
6. **無真實人像**：所有圖必須是商品 / 物件 / 場景、無真實人臉

跑完每張可請 AI 自檢：「對照上方 brief 6 項驗收，這張通過幾項？」期望 ≥ 5/6。
