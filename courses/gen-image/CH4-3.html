(03) · CH 4.3

同款品牌、3 張系列也像同個牌子

3 張廣告放在一起，看起來像同一個品牌出的——這就是系列感。不靠設計師、不靠天生美感，靠一份品牌小卡（3 件事）+ AI 執行 + 3 個問題做跨張檢查。時間：45 分鐘

學習成果
能填寫一份品牌小卡（顏色 / 字體 / 風格詞），讓 AI 每次都吃同份規格
能用同款線稿 + 品牌小卡產出 3 張看起來像同品牌的系列完稿
能用系列檢查 3 問（顏色對 / 字體對 / 看起來像同品牌嗎）跨張驗收
今天結束時，你會交出
  1. 1 份品牌小卡（顏色 hex + 字體名稱 + 風格詞 3-5 個）
  2. 3 張系列完稿（共享同份品牌小卡 + 同骨架）
  3. 1 份系列檢查 3 問結果（顏色對 / 字體對 / 像同品牌嗎）

完成這 3 件就算過關。系列感不靠美感、靠 3 件事鎖死。

(01)

看 — 有品牌小卡 vs 沒品牌小卡

為什麼這很重要
你在 CH4-1 學會了跨產業套用、CH4-2 學會了多變體——你現在能跑很多張了。問題是：多張放在一起，它們看起來像「同一個品牌」嗎？本段讓你先感受有沒有品牌小卡的差異，再教你怎麼填、怎麼用。

弄一下咖啡工作室老闆準備做春夏秋冬 4 季 × 3 張新品廣告 = 12 張的年度行銷系列。他已經在 CH4-1 學會跨業種套用、CH4-2 學會多變體，覺得自己準備好了——但跑出 3 張之後，老闆自己的朋友問：「這 3 張是 3 個不同品牌嗎？」

沒有品牌小卡（各張自由發揮）
春抹茶拿鐵 · 鼠尾草綠 + 奶油白 · 明朝體 · 療癒、手作
夏柚子氣泡 · 黃橘 + 天藍 · 黑體 · 清爽、活力
秋焙茶拿鐵 · 深棕 + 米色 · 手寫感 · 沉穩、職人
3 張放一起像 3 個咖啡店的廣告——綠的、黃的、棕的，每張一個世界。
有品牌小卡（所有張共用一份規格）
春抹茶拿鐵 · 鼠尾草綠 + 奶茶棕 + 奶油白 · 明朝體 + 黑體 · 手作、職人
夏柚子氣泡 · 鼠尾草綠 + 奶茶棕 + 奶油白 · 明朝體 + 黑體 · 手作、職人
秋焙茶拿鐵 · 鼠尾草綠 + 奶茶棕 + 奶油白 · 明朝體 + 黑體 · 手作、職人
3 張放一起像同一個咖啡店 3 個季節新品——配色、字體、語氣全一致，只有商品換。
→
決定性的 3 項一致變量：配色 + 字體 + 道具類型。商品換，但視覺規格不換。

沒有品牌小卡 = 每張都重新決定 = 看起來像不同品牌
有品牌小卡 = 每張吃同份規格 = 看起來像同品牌

(02)

學 — 品牌小卡：3 件事記住就夠

品牌小卡是什麼
品牌小卡不是給設計師看的文件——是給 AI 看的品牌規格書。你寫清楚了，AI 每次跑新廣告都吃這份規格，不用重新猜你要什麼。核心只有 3 件事：顏色、字體、風格詞。完整版分 5 段，但就算只填前 3 段，已經比沒填好非常多。
第 1 件事
顏色
3 個顏色（主色 / 強調色 / 背景色）+ 你不要的顏色。顏色要寫 hex 碼，不要寫「清新色調」——AI 猜不出來你要哪個綠。
第 2 件事
字體
2 種字體（標題用哪個 / 內文用哪個）+ 字體名稱要完整寫出來。「好看的明朝體」讓 AI 猜，「Shippori Mincho」讓 AI 精確執行。
第 3 件事
風格詞
3–5 個情緒關鍵詞說明你的品牌調性，再加一組好 vs 壞的示範句讓 AI 知道邊界在哪。「親切」不夠，「手作、職人、季節感」才夠具體。

完整版還有第 4 段（禁用清單）和第 5 段（主力骨架），填越完整、AI 跑起來越穩。以下逐段說明：

品牌小卡 5 段詳解

第 0 段
風格 preset（1 min）
11 種 preset 選 1：日系 / 韓系 / 美式復古 / 矽谷科技 / 北歐極簡 / 歐陸法式 / 美式海報 / 瑞士排版 / 手繪童趣 / 工業風 / 街頭潮流。

再加：選擇理由（30 字內）+ 1–2 個示範品牌參考（告訴 AI 你想靠近哪個品牌的感覺）。
為什麼重要：preset 決定後 4 段的調性，是整份小卡的底色。一個品牌只能選 1 個 preset——不能「日系為主、偶爾韓系」。
第 1 段
顏色（2 min）
角色Hex用途
主色#______主標、強調重點
強調色#______價格、徽章、次要點綴
背景色#______整體背景、大面積底色
另外列出不要用的顏色（例：純黑 #000、螢光、霓虹）。
為什麼重要：3 色是硬限——超過 3 色就雜亂。不要用的顏色比可以用的更重要——AI 預設會用訓練資料平均配色，有禁用清單才守得住邊界。
第 2 段
字體（2 min）
角色字體名稱字重用途
標題Shippori Mincho700主標、大字堆疊
內文Noto Sans JP400-500副標、說明文字
字階建議：主標 40–56pt / 副標 20–28pt / 內文 14–16pt
為什麼重要：同品牌只用 2 種字體——超過 2 種就雜亂。字體要寫完整名稱（「Shippori Mincho」而非「明朝體」），AI 才能精確執行。
第 3 段
風格詞（2 min）
3–5 個情緒關鍵詞（例：手作、季節感、療癒、職人）

加一組好 vs 壞示範句：
✅ 好：「香り高い抹茶と、なめらかなマスカルポーネのハーモニー。」
❌ 壞：「CP 值爆表！抹茶控必喝神級飲品！」
為什麼重要：風格詞決定 AI 產文案時的遣詞造句。示範句對比給 AI 具體錨點，讓它知道「好 vs 壞」的邊界在哪。
第 4 段
禁用清單（2 min）— 必填，不是選填
視覺禁用（3–5 項）：漸層、純黑、霓虹色、寫實人像……
情境禁用（3–5 項）：促銷轟炸、網路流行梗、假名人代言……
字體禁用（3 項）：Comic Sans 類、過瘦 Hairline 字體、同頁超過 2 種字體
為什麼重要：這是整份品牌小卡最關鍵的一段。AI 預設會從訓練資料平均值取用（充滿漸層、多色、裝飾）——沒有禁用清單 = AI 自由發揮。告訴 AI「不要做什麼」比「做什麼」更重要。
第 5 段
主力骨架（1 min）
標記這個品牌最常用的骨架（骨架 1–5）+ 第二常用骨架 + 選擇理由（30 字內）。
為什麼重要：一個品牌若主力骨架是 4，跑到骨架 5 就破壞辨識度。明確標記後，AI 新產物料時預設從主力骨架出發。
💡
品牌小卡維護建議：填完第一版後，每季 review 一次，特別是禁用清單——用 1–2 個月後你會想加幾條新規則，這很正常。品牌小卡是迭代品，不是一次就完美的文件。
(03)

學 — 品牌繼承 prompt 和系列檢查 3 問

品牌繼承 prompt（5 min）

品牌小卡填好之後，每次跑新廣告都用這個模板——整份小卡貼進去，只改商品相關的 5 項：

品牌繼承 prompt 模板（可直接複製）
吃下以下品牌規格： [貼整份品牌小卡] --- 骨架沿用：{骨架代號，例：骨架 4 商品中心 · 道具環繞} 本次主題： - 商品：{新商品} - 季節 / 場景：{例：夏季限定} - 徽章內容：{例：6/1–7/31 限定} - 價格：{例：Regular ¥580 / Tall ¥630} - 道具：{例：柚子切片、冰塊、氣泡水玻璃杯} 請依以上規格 + 骨架 + 主題產出完稿。 【強制自檢】 產完請自我檢查並回覆： 1. 顏色是否只用了品牌小卡指定的 3 色？（列出實際用色 hex） 2. 字體是否符合小卡指定？（列出實際字體名稱） 3. 禁用清單是否全部遵守？（逐條比對） 若任一項不符，請自動修正重新輸出。
→
3 個關鍵鎖定
整份品牌小卡貼上去（不是摘要，是完整 5 段）
強制自檢讓 AI 自己對照清單，你只做最後判定
商品 / 季節 / 徽章 / 價格 / 道具這 5 項是唯一變量，其他全吃品牌小卡

系列檢查 3 問（5 min）

3 張都跑完之後，在同一個對話裡貼這份檢查，讓 AI 做跨張一致性自檢：

系列檢查 3 問（可直接複製）
請對剛才產出的 3 張系列（A / B / C）做品牌一致性自檢，對照以下規格： [貼品牌小卡] 逐項比對 3 張： 1. 顏色對嗎？3 張是否都只用了品牌小卡指定的 3 色？（列出每張實際用色 hex） 2. 字體對嗎？3 張的字體是否都符合小卡指定？（列出每張實際字體名稱） 3. 看起來像同一個品牌嗎？（對照品牌小卡的禁用清單 + 風格詞，逐張比對） 結論：跨張一致性通過 __/3 問。 若全部 3 問通過 = 可作為品牌系列資產收納。 若 2 問通過 = 指出哪問在哪張不符，建議迭代哪張。 若 ≤ 1 問通過 = 品牌小卡太模糊，需要加強禁用清單再重跑。

這份系列檢查 3 問是為了抓跨張微偏離設計的。肉眼比對容易被單張美感吸走、漏掉跨張的顏色偏差（例如鼠尾草綠 #5a7a5a vs 橄欖綠 #7a8a5a 這種細微差異）。以 AI 自檢為主要依據，加你自己的人工複審為輔。

PM 判定標準
3/3 通過
收為系列資產，進 PRAC4 / CH5-1 反覆使用
2/3 通過
迭代不符那張（保留通過 2 問 + 只修不過項），不要全部重跑
≤ 1/3 通過
回去強化品牌小卡（通常是禁用清單填太少），重填後再跑
(04)

做 — 填品牌小卡 + 跑 3 張系列 + 驗收

15 分鐘，3 個階段
為自家品牌（或假想品牌）建立完整的品牌小卡，跑 3 張系列，做跨張一致性驗收。完成後你就有第一份可以長期重複使用的品牌視覺規格。
階段 1 · 5 min
填品牌小卡
打開品牌小卡填空範本，填自家品牌（或假想品牌）：
  • 第 0 段：風格 preset（11 選 1）
  • 第 1 段：3 個顏色 hex + 禁用色
  • 第 2 段：2 種字體完整名稱
  • 第 3 段：風格詞 3–5 個 + 好 vs 壞示範句
  • 第 4 段：禁用清單（視覺 / 情境 / 字體各 3–5 項）
  • 第 5 段：主力骨架 + 次要骨架

若 5 分鐘填不完 5 段：先填第 1 / 2 / 4 段（最關鍵 3 段），第 3 / 5 段簡化就好。

階段 2 · 7 min
跑 3 張系列
用品牌繼承 prompt 模板，跑 3 張系列：
  • 3 張共用同一個骨架 + 同一份品牌小卡
  • 只換商品 / 季節 / 徽章 / 價格 / 道具這 5 項
  • 範例：春抹茶 / 夏柚子 / 秋焙茶 3 個新品

每張約 2 分鐘（30 秒組 prompt + 1 分鐘 AI 執行 + 30 秒等待）。

展開品牌繼承 prompt 模板（複製即用） 
吃下以下品牌小卡：

[貼入完整品牌小卡]

---

結構沿用：{骨架代號，例：骨架 4 商品中心・道具環繞}

本次主題：
- 商品：{新商品，例：春季限定 抹茶ティラミスラテ}
- 季節 / 場景：{例：夏季限定 / 母親節}
- 徽章內容：{例：6/1-7/31 限定}
- 價格：{例：Regular ¥580 / Tall ¥630}
- 道具暗示：{例：柚子切片、冰塊、氣泡水玻璃杯}

請依以上品牌小卡 + 骨架 + 主題產出 1:1 完稿。

【強制自檢】
產完請自我檢查並回覆：
1. 色票是否只用了品牌小卡指定的 3 色？（列出實際用色 hex）
2. 字族是否符合品牌小卡指定？（列出實際字族）
3. 禁用清單是否全部遵守？（逐條比對）
4. 語氣與品牌小卡是否一致？（用「示範句對比」比對）
階段 3 · 3 min
系列檢查 3 問
3 張都跑完後，貼系列檢查 3 問給 AI，讓它做跨張一致性自檢。

判定：3/3 通過 → 存檔 3 張 + 品牌小卡 = 品牌資產雛形
2/3 通過 → 找出不符問，迭代不合格那張
≤ 1/3 通過 → 回階段 1 補強禁用清單
展開系列檢查 3 問（複製貼回同對話） 
請拿這 3 張（A / B / C）做品牌系列一致性自檢，對照以下品牌小卡：

[貼品牌小卡]

檢核項：
1. 3 張的配色是否都只用了品牌小卡指定的 3 色？（列出實際用色 hex）
2. 3 張的字族是否符合品牌小卡指定？
3. 3 張是否都遵守禁用清單？（逐條比對、逐張比對）
4. 3 張的結構骨架是否一致？（例：都是骨架 4）
5. 3 張的語氣調性是否一致？（對照品牌小卡「示範句對比」）

若任一張偏離，指出「第 {A/B/C} 張在第 {1-5} 項不符，具體是 {...}」。
通過條件：3 張在 5 項全部一致 = 可作為同系列資產收藏。

完成標準（self-check）

  • 填完品牌小卡（至少第 1 / 2 / 4 段，第 3 / 5 可簡化）
  • 跑出 3 張系列（同骨架 + 同品牌小卡 + 3 個不同商品 / 季節）
  • 做過系列檢查 3 問跨張自檢
  • 系列檢查 ≥ 2/3 通過（底線，3/3 最佳）
  • 存檔 3 張系列 + 品牌小卡（PRAC4 會用到）
→
預期差異提醒：你的品牌小卡跟同學完全不同，你的 3 張系列也不同——這是設計。3/3 通過不是目標——2/3 已經比沒有品牌小卡好很多。品牌小卡是迭代品，第一版不完美很正常，用個 1–2 個月後你會想修，這是應該的。
(05)

試跑包需求

本單元不備固定示範圖——學員填自家品牌小卡、跑自己的系列，每個人的結果不同。以下是開課前需備齊的素材。

1
看段｜有品牌小卡 vs 沒品牌小卡對比引導
情境 A 沒品牌小卡（春夏秋 3 張配色不同表）+ 情境 B 有品牌小卡（3 張配色一致表）+ 3 項一致變量說明（文字表格形式）
2
學段｜品牌小卡 5 段詳解 + 2 份範本
品牌小卡 5 段逐一說明 + 填空範本（brand-dna-template.md）+ 弄一下咖啡 + HANAKO 兩個完整示範品牌小卡 + 品牌繼承 prompt 完整版（可複製）+ 系列檢查 3 問完整版（可複製）
3
做段｜3 階段引導
階段 1 填品牌小卡引導清單（5 段 + 簡化版第 1/2/4 段）+ 階段 2 品牌繼承 prompt 模板 + 階段 3 系列檢查 3 問判定標準（3/3 / 2/3 / ≤1/3）
4
學員自備（課前告知）
已完成 CH4-1 + CH4-2 ‧ 對自家品牌（或假想品牌）有基本認識 ‧ AI 平台帳號 ‧ 紙本或電子檔寫品牌小卡
備援配置：學員不熟自家品牌 → 用弄一下咖啡的品牌小卡當假想品牌練習 ‧ 配額爆 → 只跑 2 張系列（系列檢查 3 問仍能跑）‧ 系列檢查 < 2/3 → 現場加 3–5 條禁用項重跑 ‧ 教室時間壓縮至 30 min → 砍看段（改口述），保留學段 + 做段
(06)

案例 — 弄一下咖啡的 12 張年度系列

弄一下咖啡工作室老闆要做春夏秋冬 4 季 × 3 新品 = 12 張年度系列廣告。他試跑春夏秋 3 張發現問題：

  • 3 張結構都對（骨架 4），主題都是咖啡
  • 但朋友問「這是 3 個不同品牌嗎？」——因為配色、字體、道具每張都自由發揮

缺的是一份品牌小卡——品牌的視覺規格書，給 AI 看的。

他填的品牌小卡
preset日系職人
顏色鼠尾草綠 #5a7a5a / 奶茶棕 #8a6a4a / 奶油白 #f5efe3
字體Shippori Mincho 700 / Noto Sans JP 400
風格詞手作、職人、季節感、療癒
禁用漸層、純黑、霓虹、促銷轟炸口號
你在這裡學到的

常見錯誤 3 條

錯誤 01
品牌小卡填得太抽象（只寫「日系風」）
現象
第 0 段寫「日系風」、第 1 段寫「清新色調」、第 2 段寫「好看的字體」、第 3 段寫「親切」。填完感覺有寫，實際上 AI 看不懂。
原因
抽象詞 = AI 的猜謎題。日系風的配色有 10 種可能、字體有 20 種可能——AI 只能猜一個組合。下次跑新系列時 AI 又猜一個，兩次結果不一致。
解法
每段都要具體到「可執行」的程度：
抽象（錯誤）具體（正確）
日系風日系職人（參考品牌：蔦屋書店 / MUJI）
清新色調鼠尾草綠 #5a7a5a + 奶茶棕 #8a6a4a + 奶油白 #f5efe3
好看的字體Shippori Mincho 700 / Noto Sans JP 400
親切手作、職人、季節感、療癒（4 詞）

檢查方法：填完每段，換隔壁同學看——他能不能照你的描述重現同樣的視覺？若他說「我要猜什麼顏色才對」——你的描述不夠具體。

錯誤 02
3 張系列跑完沒做系列檢查 3 問
現象
學員跑 3 張系列，每張個別看都還 OK，存檔。沒跑系列檢查 3 問。兩週後要做第 4 張，打開前 3 張才發現——第 1 張鼠尾草綠主導、第 2 張不小心用了橄欖綠（跟鼠尾草綠不同 hex）、第 3 張徽章形狀變了——3 張已經偏離品牌小卡。
原因
人眼比對跨張偏離容易漏判，特別是顏色相近時（鼠尾草綠 vs 橄欖綠）。系列檢查 3 問專門為了抓跨張微偏離設計的。
解法
3 張系列跑完後，必跑系列檢查 3 問。讓 AI 列出每張實際用色 hex，你才會發現「欸第 2 張的 #7a8a5a 不在品牌小卡指定的 3 色裡」。系列檢查 < 3/3 → 立刻迭代那張，不要存檔。

紀律：系列檢查 3 問是系列化的最後守門員——跳過就等於把偏離的資產存進品牌庫，未來每次新廣告都在累積偏離。
錯誤 03
品牌小卡的禁用清單沒填
現象
第 4 段禁用清單留空或只寫「沒特別禁用」。結果 AI 產系列時，自己加了漸層背景、多了裝飾邊框、用了額外的亮橘色點綴——這些品牌小卡沒禁，AI 合法加上，但跟品牌調性不符。
原因
AI 預設從訓練資料平均值取用（訓練資料裡充滿漸層、多色、裝飾邊框）。沒有禁用清單 = AI 自由發揮。
解法
禁用清單是必填，不是選填。至少列 3 類 × 每類 3–5 項：
  • 視覺禁用：漸層（走平面）、純黑 #000、霓虹色、寫實人像
  • 情境禁用：促銷轟炸、網路流行梗、假名人代言、快速消費感
  • 字體禁用：Comic Sans 類、過瘦 Hairline、同頁超過 2 種字體

心法：告訴 AI「不要做 X」比「做 Y」更重要——它會自己學會做 Y，但你不說它就會做 X。

(08)

檢核題

Q1｜概念驗證 — 品牌小卡的 5 段是什麼？為什麼禁用清單是必填而非選填？
看答案要點

5 段：第 0 段風格 preset（11 選 1）、第 1 段顏色 3 色 + 禁用色、第 2 段字體 2 種 + 字階、第 3 段風格詞 3–5 個 + 示範句對比、第 4 段禁用清單（視覺 / 情境 / 字體）、第 5 段主力骨架 + 次要骨架。（至少命中 5 點算通過）

禁用清單必填的理由：AI 預設從訓練資料平均值取用（充滿漸層、多色、裝飾）。只填「可用清單」，AI 會自由發揮加東西。禁用清單 = 明確劃出不跨的界線，AI 才守。告訴 AI「不要做什麼」比「告訴 AI 做什麼」更重要。

不及格答案：只列 5 段沒說為什麼重要 ‧「禁用清單看情況」（錯誤，必填）‧「品牌小卡就是品牌風格」（太空泛）

Q2｜應用驗證 — 為弄一下咖啡工作室填品牌小卡前 3 段，並組成一個品牌繼承 prompt（跑「夏季柚子美式」新品廣告）。
品牌背景：台北單店手沖咖啡，走職人、手作、季節限定，客群 25–35 歲女性
看答案要點

品牌小卡前 3 段：

  • 第 0 段：日系職人（參考品牌：蔦屋書店 / MUJI）
  • 第 1 段：主色 鼠尾草綠 #5a7a5a / 強調色 奶茶棕 #8a6a4a / 背景色 奶油白 #f5efe3 ‧ 禁用色：純黑、霓虹、高飽和紅橘
  • 第 2 段：標題 Shippori Mincho 700 / 內文 Noto Sans JP 400 ‧ 字階：H1 48–56pt / H2 22–26pt / 內文 14–16pt

品牌繼承 prompt 必要元素：

  • 完整貼上品牌小卡（不是摘要）
  • 骨架：骨架 4 商品中心 × 道具環繞
  • 本次主題：商品夏季柚子氣泡美式 ‧ 季節夏季限定 6/1–7/31 ‧ 道具柚子切片、冰塊、氣泡水玻璃杯
  • 含【強制自檢】區塊（列出 3 個自檢項）

不及格答案：顏色寫「鼠尾草綠 + 奶茶棕」不寫 hex ‧ 字體寫「日系字體」不寫完整名稱 ‧ prompt 沒貼品牌小卡 ‧ 沒有【強制自檢】區塊
