(A3)

Brand Poster 單張品牌海報

單張、可印刷、可貼牆、可放 IG 的「品牌海報」。一張圖傳達一個品牌主張，不是 campaign 也不是產品圖，是「品牌個性的視覺名片」。咖啡店牆面、選物店貼紙、文青品牌週報主圖最常用。

這張卡片你會學到
能判斷何時用 brand poster（與 A1/A2/A4 的差異）
能拆解 6 段結構並產出單張高張力品牌海報
能識別 4 種踩雷（主張不清／構圖太擠／色板亂／字體混搭）
(01)

適用場景

這個模板用在哪
單張、克制、有張力的品牌海報。不是 campaign（沒有時效）、不是 product hero（不講功能）、不是雜誌封面（不講故事）。目的是「一張圖讓人記得這個品牌」。比 A1 更純粹、比 A2 更克制、比 A4 更聚焦。
典型客戶 / 交付物　咖啡店 / 選物店 / 文青品牌、單張接案 NT$3,000-8,000、交付：A3/A4 印刷可用單張 + IG 1:1 版本

四個典型用途：

  • 咖啡店 / 選物店牆面海報（實體店面氛圍營造）
  • 文青品牌每月主視覺（貼 IG 用、輪換系列）
  • 活動 / 講座 / 工作坊宣傳（單張海報傳達主題）
  • 品牌精神週報主圖（電子報週六固定欄位）
對比項本卡片 · A3 Brand Poster 單張品牌海報近似模板
尺寸與比例直立 2:3 或 1:1.4（A4/A3 海報尺寸）A1：橫式 21:9；A2：1 主 + 多延展；A4：書封 2:3
主張數量1 個（短主張句）A2：campaign 訴求語；A4：書封標題
延展性通常單張、不延展A2：必須延展
使用場景實體 + 數位混合A1：純數位；A4：純印刷
文字密度1-2 段、克制A2：訴求語 + sub；A4：標題 + 作者 + 副題
產出難度中（克制感最難拿捏）A1：低；A2：高
💡
一句話判斷：客戶問「店裡牆面要貼什麼」走 A3；問「campaign 主視覺」走 A2；問「官網 hero」走 A1；問「書封 / 雜誌封面」走 A4。A3 在文青品牌最受歡迎——單張海報接案 NT$3000-8000、客戶常成系列訂購。
(02)

模板拆解｜6 段 prompt 結構

brand poster 6 段。核心難度在「克制」——AI 容易把海報塞太多元素、變成「廣告」而不是「品牌海報」：

段段落必問還是可預設填什麼
1brand 品牌定位必問品牌名 + 1 句品牌精神（短句）+ 個性形容詞（克制 / 復古 / 文青 / 有機）
2main statement 主張句必問1 句話、≤ 8 字、英文優先；位置（左上 / 中央 / 右下）
3中央焦點視覺主視覺必問1 個視覺元素（產品 / 抽象圖形 / 自然物 / 字體本身）；不要 2 個以上
4color palette 色板必問品牌色 + 至多 1 個強調色；克制感 ≤ 2-3 色
5typography 字體可預設1 個主字（粗 serif 或粗 sans）+ 1 個輔字（細）；英文優先
6限制固定寫死禁忌：元素超過 3 個、色板超過 3 色、字體超過 2 種、淘寶廣告風、字塞滿
📐
記憶口訣：品牌 → 主張 → 主視覺 → 色 → 字 → 禁。新手最常加太多東西——海報的力量在「留白」。明確寫「畫面留白 ≥ 40%」「anchor 元素 ≤ 1 個」這兩道鎖、海報質感就出來了。
(03)

示範圖｜可複製、可重現

下面這張是 OPEN BEANS 「Slow is Better」品牌週主視覺。海報一張、貼咖啡店牆面、放 IG、印 A4 三用。注意極致的留白與單一 anchor 元素（咖啡杯線稿）。

完整 prompt · OPEN BEANS Slow is Better 品牌海報 適用：ChatGPT GPT-image / Gemini Nano Banana 
請生成一張單張品牌海報（brand poster）：

【尺寸與構圖】直立 2:3、A3 海報比例、克制風格、留白超過 50%。

【品牌】OPEN BEANS（精品咖啡品牌、強調慢咖啡精神）

【主張句】「Slow is Better」（巨大、置中略偏上、襯線粗體、深棕色）
副標：「Single-Origin Coffee · Roasted in Taipei」（細體無襯線、淺棕、置中、距主張 1cm）

【Anchor 主視覺】中央極簡咖啡杯線稿（≤ 3 條黑線、handwritten 感、不超過畫面 15%）

【色板】嚴格 2 色：米白底（#F5F0E8）+ 深棕字（#3A2A1F），可有極淺燙金 accent（用於小字下劃線）。

【字體】主張：襯線粗體（避免新細明體）；副標：無襯線細體；都用英文。

【背景】純米白底、無紋理、無漸變、無雜物。

【限制】嚴禁人物、嚴禁第 4 個視覺元素、嚴禁色板超過 3 色、嚴禁字體超過 2 種、嚴禁鮮艷色、嚴禁中文字、嚴禁淘寶廣告風（避免閃光、星星、徽章等元素）。
📷
讀圖時看這 4 個點：(1) 留白夠不夠（≥ 40%）？(2) anchor 是不是只有 1 個？(3) 色板是不是 ≤ 3 色？(4) 字體層級清不清楚（主張大 / 副標小）？
(04)

改成你的場景｜踩雷與失敗案例

把 prompt 拿來、改 3 個替換槽就變你的版本：

替換槽你的版本
ⓐ 主張句 = ?（≤ 8 字、英文）例：Slow is Better / Less is More / Stay Curious
ⓑ anchor 視覺 = ?（單一元素）例：咖啡杯線稿 / 茶葉 / 麥穗 / 抽象幾何
ⓒ 色板 = ?（≤ 3 色）例：米白＋深棕＋燙金（咖啡）／米白＋深綠＋金（茶）／純白＋黑＋紅（簡約）

最常見的 4 種踩雷：

  1. 塞太多視覺元素　AI 自動加咖啡豆、葉片、蒸氣、杯子、ribbon、星星⋯⋯通通都來。解法：明確寫「single anchor element」「max 1 visual element in center」「whitespace ≥ 50%」三道鎖。
  2. 色板鎖不住　寫了「米白＋深棕」AI 還是給你淡藍 / 淡綠陰影。解法：明確寫「strictly 2-color palette: cream white #F5F0E8 + dark brown #3A2A1F」、「no other colors anywhere」。
  3. 字體混搭過多　標題用 serif、副標用 sans、結果 AI 又自動加第 3 個 script 字。解法：明確寫「only 2 typefaces: bold serif for main + light sans for sub」、避免 script / display / decorative。
  4. 中文海報主張別指望 prompt 解決　AI 生圖工具對中文字體只給新細明體系。A3 的最佳策略：英文版海報用 AI 直接生成（如「Slow is Better」），中文版（「慢，才是更好」「日子緩慢，咖啡專注」）走 後製字體疊圖工作流附錄 在 Figma 疊上「金萱半糖粗體」級別字體。

    實戰提醒：A3 brand poster 是文青客戶最在意「字體調性」的卡片——他們要的就是「金萱／源樣明體」那種台灣文青感。Figma 後製疊字是 A3 的標準流程、不要嘗試一次到位。
⚠️
交付提醒　本卡片的 AI 生成的示範圖不是可直接給客戶的最終交付品。中文商業案一律先用 AI 生出構圖（含英文佔位文字）、再走 後製字體疊圖工作流 在 Figma／Canva 疊上中文、最後校稿確認才交付給客戶。
