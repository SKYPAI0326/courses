CH6 自我練習｜挑模板＋改寫＋出圖

看完 18 張卡片不是終點、能用才是。這頁給 3 個常見接案情境、你獨立判斷該用哪張卡片、改寫 prompt 並跑出第一張圖。完成後對照檢核 4 點驗收。

(01)

情境 1｜手作果醬上架蝦皮

情境難度 · 入門
朋友做手工草莓果醬、要上架蝦皮、預算只夠你接 NT$3,000
客戶有 4 款口味（草莓 / 藍莓 / 芒果 / 鳳梨）、要 4 張白底主圖。寫一份 prompt、產出 4 張一致風格的主圖。
你的任務：(1) 從 18 張卡片中挑 1 張對的、(2) 寫出你的 prompt（標籤建議用英文）、(3) 跑 imagegen 一次到位。
  • 挑對卡片：應該選 B1 純白底電商主圖（不是 B2 lifestyle、不是 B6 飲料瓶身設計）
  • 標籤改用英文：「JAMS by Mom · Strawberry · 200g」（規避中文字體）
  • 4 款用同 prompt 改 1 處（口味名 + 標籤色），保持風格一致
  • 背景純白 #FFFFFF、底部柔光陰影、商品占畫面 60%
(02)

情境 2｜健身教練線上一對一招生 EDM

情境難度 · 中階
健身教練要做招生 EDM、放 LINE 官方帳號連結、預算 NT$8,000
教練要的是「真實落地頁感」、客戶看完想立刻點 CTA 報名。要做 1 張桌機橫版 + 1 張手機直版。
你的任務：(1) 挑卡片、(2) 設計三色板（教練品牌色 + 主底色 + 強調色）、(3) 寫主標 + CTA + 資訊卡的 3 個元素。
  • 挑對卡片：應該選 B5 商品卡 UI 疊加（不是 A1 web banner、不是 C4 founder portrait）
  • 三色板限定：例「深灰 + 純白 + 螢光黃」（運動感）
  • CTA 寫英文：「Get Started」「Start Training」（規避字體限制）
  • 右欄資訊卡寫具體數字：「200+ Active Members · 4.8★」建立信任
(03)

情境 3｜松菸活動雙語視覺

情境難度 · 進階
松菸文創園區辦「Slow Living 慢工作坊」、要做活動主視覺、預算 NT$15,000
活動有外國觀眾、要中英雙語並列。中文「慢工作坊」是品牌核心、英文「Slow Living Workshop」副從。一張海報橫式、放官網與印刷宣傳單。
你的任務：(1) 挑卡片、(2) 安排中英主從關係、(3) 設計後製疊字流程（imagegen 出構圖 + Figma 後製疊金萱體中文）。
  • 挑對卡片：應該選 A6 中英雙語版面（不是 A3 brand poster、不是 A4 editorial cover）
  • 中英主從：中文 70% 視覺重量、英文 30%
  • 字體配對：中文金萱粗 + 英文 Playfair Display（都襯線、調性一致）
  • 後製流程：imagegen 出英文版 + 構圖 → Figma 用金萱字體疊上「慢工作坊」中文
(04)

情境 4｜估工 + 報價 + 客戶期待管控

情境難度 · 商業判斷
客戶丟一個案子來——你怎麼估工、報價、管控期待？
客戶 LINE 來訊：「想做一組品牌的視覺、要包含 IG 主圖、官網 banner、產品包裝、節慶 EDM。預算 NT$30,000、希望 1 週內交付」。你怎麼回？
你的任務：(1) 先判斷這個案子涉及 CH6 的哪幾張卡片、(2) 拆解出每張卡片的工時與單價、(3) 寫一封回覆 LINE，講清楚「能做什麼、不能做什麼、為什麼」。
  • 拆案：客戶要的是 4 種視覺類別 → IG 主圖（B1 + 可能 B2）+ 官網 banner（A1）+ 產品包裝（B4 或 B6）+ 節慶 EDM（A2 或 B5）
  • 估工：照每張卡片的「典型客戶/交付物」工時欄位、合計約 4-6 個工作天（含後製疊字）
  • 報價落差：客戶 NT$30,000 預算 vs 實際合理報價可能 NT$45,000-60,000 → 必須誠實溝通
  • 管控期待：明白告知 imagegen 出構圖、中文文案後製疊字、客戶要先準備品牌資料（logo / 色 / 字 / 文案）
  • 誠實標註：示範圖不是最終交付（已在每張卡片底部警語提示）、客戶會看到 imagegen + 後製分流
實戰參考回覆：「謝謝信任。你列的 4 種視覺，按市場合理價約落在 NT$50,000 左右、1 週稍緊但可做。如預算上限是 NT$30,000，我建議先聚焦 2 個最關鍵的（譬如 IG 主圖 + 包裝），其他預算寬鬆時再做。另外想跟你預告：所有最終交付前我會用 Figma 校稿確認字體與內容，這部分會佔 20-30% 的時間。可以先約 30 分鐘聊聊優先順序嗎？」
(05)

驗收 5 點檢核

完成 4 個情境後、對照下面 5 點檢核：

  • 選對卡片：每個情境的卡片是不是對的（情境 1=B1、情境 2=B5、情境 3=A6、情境 4=多卡組合）
  • 規避字體限制：所有中文 logo / 標題你都安排了「英文佔位文字 + Figma 後製疊字」流程
  • 替換槽完整：每張 prompt 你都改了 3 個替換槽（商品 / 色板 / 場景）、不是照搬
  • 4 踩雷避開：對照該卡片的 4 種踩雷、確認你的 prompt 都避開了
  • 商業期待管控：你的回覆有明確分清「能做」「需後製」「需客戶先準備」三類，且報價合理不過度承諾
✓
4 個情境都過、就具備了「看到接案需求 → 立刻知道用哪些卡片 → 估工報價 → 改 3 處跑出構圖 → 後製疊字交付」這套完整商業工作流。CH6 模板速查站到這裡完成階段性目標、後續實戰累積即可。
