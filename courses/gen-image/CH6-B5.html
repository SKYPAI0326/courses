(B5)

商品卡 UI 疊加｜落地頁 hero 樣機

電商落地頁、產品詳情頁、訂閱頁的 hero 區視覺。一張圖把「商品 + 文案 + CTA + 賣點」四要素整合好，看起來像「真實品牌的網頁截圖」。是 SOHO 接案 NT$10000 級別案子最常見的視覺類型。

這張卡片你會學到
能判斷何時用 UI 疊加樣機（與 B1 白底、B2 生活方式、B3 影棚的差異）
能拆解模板的 6 段 prompt 結構並平衡「商品＋UI 元素」的視覺層次
能識別 4 種常見踩雷（排版擁擠／文案不可讀／徽章太多／模特不可信）並修正
(01)

適用場景

這個模板用在哪
三欄結構（左文案 + 中產品 + 右模特或資訊卡）的電商落地頁 hero 區。一張圖把「商品 + 文案 + CTA + 賣點」全部講完。比 B1 白底多了 UI 排版、比 B2 生活方式多了商業資訊層、比 B3 影棚多了真實落地頁感。視覺像「剛上線的精品電商頁面截圖」。

四個典型用途：

  • 電商詳情頁主視覺 / SKU 卡 hero（蝦皮、momo 高階商品頁、訂閱頁）
  • 品牌訂閱頁落地頁 hero（咖啡訂閱、保養品訂閱、健康食品訂閱）
  • FB／LINE 廣告投放主圖（一張圖講清楚產品 + 價格 + CTA）
  • EDM／電子報 hero banner（精品週報、會員專屬週末 EDM）
對比項本卡片 · B5 UI 疊加樣機近似模板
UI 元素必有：CTA 按鈕、賣點卡、徽章B1：選用；B2：不疊；B3：常疊文案但無按鈕
排版三欄（文案＋商品＋資訊卡）B1：商品居中；B2：商品 + 道具；B3：商品略偏黃金分割
背景奶白漸變或淺色品牌色（仿真網頁背景）B1：純白；B2：真實場景；B3：深色絲絨
商品占比30-40%（留位給 UI）B1：60% 以上；B3：35-50%
真實感目標看起來像「真實網頁截圖」B1：規格清楚；B2：故事感；B3：廣告感
產出難度高（要管 UI 排版 + 商品 + 文字一致）B1：最低；B2 / B3：中
💡
一句話判斷：客戶問「能放官網嗎」走 B5；問「FB／LINE 廣告投放圖」走 B5；問「訂閱頁主視覺」走 B5；問「整張圖要把產品 + CTA + 評價講清楚」走 B5。B5 是 B 組裡技術門檻最高、但接案費也最高的卡片，落地頁主視覺報價可達 NT$15000-30000。
(02)

模板拆解｜6 段 prompt 結構

原模板把一張落地頁 hero 拆成 6 段。核心難度在「三欄排版 + UI 元素統一」——AI 容易把每個區塊獨立做、忽略整體版面平衡：

段段落必問還是可預設填什麼
1layout 整體布局必問三欄結構（左文案＋中商品＋右模特/資訊）；比例（16:9 橫式為主、9:16 行動裝置版）
2brand 品牌必問品牌名 + 副標（建議全英文避字體覆轍）+ 配色（base/primary/強調色三色）
3hero 三欄內容必問左欄：主標 + 副標 + CTA 按鈕（≤ 2 個）；中欄：商品；右欄：模特局部 / 場景 / 資訊卡
4info_overlay 資訊疊加選用賣點卡（≤ 3 個）+ 徽章（≤ 3 個、icon + 短文）+ 評分（5 星 + 訂閱數）
5style 樣式可預設乾淨臨床感、商業級渲染、看起來像真實落地頁截圖、色板嚴格按 base/primary/強調色三色
6限制固定寫死禁忌：排版擁擠、模特表情誇張或假笑、徽章超過 3 個、文案顏色與底圖過於接近、看起來像 PPT
📐
記憶口訣：布局 → 品牌 → 三欄 → UI → 樣式 → 禁。新手最常漏「色板三色限定」——沒明確寫的話 AI 會給你一個漂亮的 hero 但顏色亂七八糟、UI 元素互相打架。三色鎖（base + primary + 強調色）是 B5 的關鍵紀律。
(03)

台灣化範例 ①｜小工作室視角

小工作室 · 接客戶案
OPEN BEANS 訂閱方案落地頁 · 取代客戶官網舊 hero
客戶情境：延續 B1/B2/B3/B4 同一個咖啡品牌、現在要做訂閱制服務上線、官網需要新落地頁 hero 取代舊 banner。預算 NT$15000、要做主圖 + 兩個變體（行動裝置版 9:16 + 桌機版 16:9）。客戶想看「真實電商感」不要設計感太重。
  • 1 布局｜三欄結構：左文案 + 中 OPEN BEANS 豆袋 + 右資訊卡（兩張：產地與評分）
  • 2 品牌｜品牌「OPEN BEANS」、配色深棕主色 + 米白底 + 燙金強調色（三色鎖）
  • 3 三欄內容｜左：「Coffee, Reimagined」+「Single-Origin · Subscribe & Save 15%」+「Start Subscription」CTA；中：豆袋；右上：「Roasted in Taipei」、右下：5 星 + 「4.9 / 1,200+ subscribers」
  • 4 資訊疊加｜底部細條：「Free shipping over NT$1,500 · Cancel anytime · No commitment」
  • 5 樣式｜乾淨精品電商感、看起來像真實落地頁截圖、米白漸變背景
  • 6 限制｜禁人物、禁排版擁擠、禁色板出現額外色、禁中英文同等大小（英文為主導）
為什麼這個情境成立：B5 落地頁 hero 跟 RWD 行動版本是「一份產出、兩種尺寸」、prompt 結構不變、輸出比例改成 9:16 即可。NT$15000 接案能涵蓋「桌機版 + 行動版 + 1 次修改」，客戶看到立刻能上線、續訂機率高。客戶官網改版年費 NT$30000-50000 是常態。
(04)

台灣化範例 ②｜個人接案視角

個人接案 · 落地頁設計顧問
幫個人健身教練做「線上一對一」訂閱方案登錄頁
接案情境：你接到個人健身教練的案子——他想開「線上一對一指導」訂閱服務（NT$2500/月）、需要一張落地頁 hero 用於 FB 廣告投放和 LINE 官方帳號連結。預算 NT$8000、要 2 張版本（桌機橫版 + 手機直版）。
  • 1 布局｜三欄結構：左文案 + 中健身教練半身照（背影或側臉、不要正臉） + 右訓練數據卡
  • 2 品牌｜「IRON COACH」（個人品牌名、英文）、配色：深灰主色 + 純白底 + 螢光黃強調色
  • 3 三欄內容｜左：「Train Smart, Not Hard」+「1-on-1 Online · NT$2,500/mo」+「Get Started」CTA；中：教練側影 + 啞鈴局部；右上：「24/7 Online Q&A」、右下：「200+ Active Members · 4.8★」
  • 4 資訊疊加｜底部細條：「First week free · Cancel anytime · No long contract」
  • 5 樣式｜運動感乾淨、像 Nike Training Club 那種落地頁、深灰漸變背景
  • 6 限制｜禁正臉（保護隱私）、禁排版擁擠、禁螢光黃以外的鮮艷色、禁 PPT 感
接案實務 tip：B5 是接「個人 SOHO 服務型品牌」最好用的卡片——教練、顧問、設計師、心理師、占星師，都需要一張「真實落地頁」感的招攬主圖。關鍵 trick：背影 / 側臉而非正臉（規避 AI 人物臉部不可控 + 客戶隱私）、CTA 文案用英文（規避字體限制 + 顯國際感）、資訊卡寫具體數字（4.8★、200+ 會員）建立信任感。中文版 hero 用 後製疊字流程替換英文文案。

示範圖｜可複製、可重現

下面這張圖是用範例 ① 的 6 段（OPEN BEANS 訂閱方案落地頁 hero）實際跑出來的成品。AI 生圖 160 秒一發到位、三欄排版精準、左欄「Coffee, Reimagined」大字 + 兩個 CTA、中欄豆袋、右欄資訊卡（產地與評分）、底部 banner 完整、視覺像「真實品牌訂閱頁面截圖」。下方就是完整的 prompt——點「複製 prompt」、貼到 ChatGPT 或 Gemini：

完整 prompt · OPEN BEANS 訂閱落地頁 hero 適用：ChatGPT GPT-image / Gemini Nano Banana 
請生成一張電商落地頁 hero 區商品卡疊加樣機（landing page hero with product card overlay），延續 OPEN BEANS 咖啡品牌系列。視覺要看起來像「真實的品牌訂閱頁面截圖」：

【整體布局】橫式 16:9 構圖，三欄結構：左欄文案 + 中欄產品 + 右欄資訊卡。仿真品牌官網落地頁 hero。

【背景】奶白色漸變主底（左淺右略深米色）、整體乾淨像剛上線的精品電商頁面。

【中欄 · 商品】OPEN BEANS 200g 牛皮紙咖啡豆袋（與 B1/B3/B4 同款），深棕袋身配米白標籤，標籤英文「OPEN BEANS · Ethiopia · Single Origin · 200g」。位置畫面正中、占畫面 35%、有自然落地陰影。

【左欄 · 文案區】上下排列：
- 上方大字 headline：「Coffee, Reimagined」（無襯線粗體、深棕、英文）
- 下方副標：「Single-Origin · Subscribe & Save 15%」（細體、淺棕、英文）
- 兩個 CTA 按鈕：「Start Subscription」（深棕填充按鈕、米白文字）+「How It Works」（米白邊框按鈕、深棕文字）

【右欄 · 資訊卡疊加】上下兩個訊息卡：
- 上方卡片：圓形深棕 icon + 「Roasted in Taipei」+「Within 7 days of order」
- 下方卡片：5 顆星 + 「4.9 / 1,200+ subscribers」+「Real reviews」

【底部 banner】橫向細條：「Free shipping over NT$1,500 · Cancel anytime · No commitment」（小字、淺棕、英文）。

【風格】乾淨、精品電商感、商業級渲染像真實落地頁截圖。色板嚴格按照「米白 + 深棕 + 燙金 accent」三色。

【限制】嚴禁人物或人體局部、嚴禁排版擁擠、嚴禁色板出現額外鮮艷色、嚴禁看起來像 PPT、嚴禁中英文同等大小（英文為主導）、嚴禁出現非 OPEN BEANS 的 logo。

四個替換重點（同一份 prompt 改四處就變不同產業落地頁）：

  • 【中欄商品】從咖啡豆袋換成你客戶的商品（保養品瓶、3C 配件、健身器材⋯⋯）
  • 【左欄文案】主標 + CTA 全換、配你的商業模式（訂閱 / 試用 / 一次購買 / 會員制）
  • 【三色板】「米白＋深棕＋燙金」換成你客戶品牌色（科技感→深灰＋藍＋白）
  • 【右欄資訊卡】產地 → 換成適合你商品的證明（製造地 / 認證 / 檢驗合格 / 用戶數）

(06)

改成你的場景

把上面 6 段拿來，只動 3 個替換槽，就變成你自己的 prompt：

替換槽你的版本
ⓐ 三色板＝？（base / primary / 強調色）例：米白＋深棕＋燙金（咖啡）／純白＋深灰＋螢光黃（運動）
ⓑ 左欄文案 = 主標 + 副標 + CTA例：「Coffee, Reimagined」+「Subscribe & Save 15%」+「Start Subscription」
ⓒ 右欄資訊卡 × 2 寫什麼？例：產地卡 + 評分卡（具體數字 4.9 ★ 1,200+）

最常見的 4 種踩雷：

  1. 排版擁擠、訊息打架　三欄塞太多元素（每欄超過 3 個文字塊）AI 會把字疊在一起。解法：左欄最多 4 行（主標 + 副標 + 2 CTA）；右欄最多 2 張卡；底部 banner 1 行。違反就刪減。
  2. 文案顏色與底圖過於接近　米白底配淺米色文字 = 看不到。解法：deep + light 對比（深棕字配米白底 / 米白字配深棕按鈕）；明確寫「文案顏色與底圖必須有強對比」一條鎖。
  3. 徽章太多變廣告噪音　寫了「賣點卡」AI 給你 5-6 個徽章疊在一起、看起來像淘寶廣告。解法：明確寫「資訊卡 ≤ 2 張」「徽章 ≤ 3 個」「icon + 短文 only」、並在限制加「徽章不能比 logo 大」。
  4. 中文 logo 字體別指望 prompt 解決　跟 B1／B2／B3／B4 同一個限制——AI 生圖工具對中文字體只會給「系統預設襯線體」。

    B5 的最佳策略：所有 UI 文案（主標 / 副標 / CTA / 資訊卡）都寫英文（如本卡示範圖的「Coffee, Reimagined / Start Subscription / Roasted in Taipei」）。中文版直接砍掉【三欄內容】裡的英文文案、AI 生圖 只生純圖（商品 + UI 框架但無文字）、再用 後製字體疊圖工作流附錄 在 Figma／Canva 疊上中文。

    實戰提醒：B5 落地頁主圖的中文「立即訂閱」「免費試用」「會員專屬」這些 CTA 文案，是客戶買單的關鍵——客戶會盯著看「按鈕字體有沒有夠醒目」。用 Figma 後製疊上「金萱半糖粗體」級別的中文 CTA 字體，整張 hero 看起來就會是「真的會被點」的真實落地頁，而不是「圖片仿製品」。

失敗案例｜prompt 動了哪一處會壞掉：

  • 把 layout.structure 從「三欄」改成「四欄」 → 訊息打架、視覺重量分散
  • 把 color_palette 加上第 4 種顏色 → 三色鎖被破、整體像 PPT
  • 把 cta_buttons 從 2 個加到 4 個 → 客戶不知道按哪個、轉換率掉
  • 把 right_column.card_2.rating 改成「100% 滿意保證」抽象描述 → 不可信，要具體數字（4.9 ★、1,200+）
  • 把 主標 改成中文「咖啡，重新被定義」 → 字體變新細明體系、整張落地頁掉檔次
  • 把 style.rendering 改成「3D 渲染」 → 失去真實落地頁感、看起來像示意圖
⚠️
交付提醒　本卡片的 AI 生成的示範圖不是可直接給客戶的最終交付品。中文商業案一律先用 AI 生出構圖（含英文佔位文字）、再走 後製字體疊圖工作流 在 Figma／Canva 疊上中文、最後校稿確認才交付給客戶。
