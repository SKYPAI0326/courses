Sticker Set LINE 貼圖組

成套的 LINE 貼圖（8-24 張、不同表情 / 反應）。台灣最好變現的 IP 視覺類別——個人 IP 自製貼圖上架賺被動收入、品牌客戶做專屬會員贈品。重點是「角色一致 + 表情多樣 + 風格穩定」。

這張卡片你會學到
能判斷何時用 sticker set（與其他角色卡差異）
能拆解 6 段並產出風格一致的 8-24 張貼圖
能識別 4 種踩雷（角色不一致／表情單調／風格漂移／構圖不適合 LINE 比例）
(01)

適用場景

這個模板用在哪
成套 LINE 貼圖。8 / 16 / 24 / 40 張一組、共用一個角色、不同表情與反應（OK、謝謝、生氣、笑、晚安）。每張獨立使用、整組看是同一個角色。
典型客戶 / 交付物　個人 IP 經營者 / 品牌會員行銷、套組接案 NT$10,000-30,000、交付：8 / 16 / 24 張貼圖組（PNG 透明底 240×240）

四個典型用途：

  • 個人 IP 上架 LINE 貼圖賺被動收入（上架後可賣 5-10 年）
  • 品牌專屬會員贈品（註冊送 8 張限量貼圖）
  • 活動 / 課程贈送學員（增加學員回饋）
  • 工作室周邊 / 文創商品延伸（貼圖 → 馬克杯 → 帆布袋）
對比項本卡片 · C5 Sticker Set LINE 貼圖組近似模板
數量8 / 16 / 24 / 40 張一組其他卡：單張或少量
角色一致性最關鍵—— 24 張要看出是同一角色其他：通常單一視覺
表情多樣8 種以上情緒/反應其他：通常單一情境
構圖正方形（240×240px）+ 透明底其他：橫式或直立
使用平台LINE / Telegram / WhatsApp其他：通用
變現潛力高（一次設計、有機會長期累積（取決於 IP 經營能力））其他：單次接案
💡
一句話判斷：客戶問「我想做 LINE 貼圖」走 C5；問「品牌吉祥物」用 branding-and-packaging/mascot-brand-kit；問「IP 周邊」用 avatars-and-profile 系列其他卡。C5 是 C 組裡變現潛力最高的卡片——一組 24 張上架可賣 NT$30-60、長期累積被動收入。
(02)

模板拆解｜6 段 prompt 結構

sticker set 6 段。核心難度在「角色一致性」——AI 生成 24 張時、第 1 張跟第 24 張的角色細節（眼睛大小、髮型、衣服）容易漂移：

段段落必問還是可預設填什麼
1character base 角色基底必問物種（人 / 動物 / 物件擬人）+ 體型（圓潤 / 瘦長）+ 顏色 + 個性
2character details 角色細節必問髮型 / 服裝 / 配件（眼鏡 / 帽子）+ 必須一致的特徵
3style 風格必問繪畫風格（手繪 / 像素 / 3D / 厚塗）+ 線條（粗 / 細）+ 顏色（厚塗 / 平塗）
4expressions 表情組合必問8-24 種情緒：OK、謝謝、生氣、開心、累、晚安、加油、哭、笑、害羞⋯⋯
5format 規格必問正方形 240×240px、透明底（PNG）、留 10% 邊緣空白避免裁切
6限制固定寫死禁忌：角色不一致、含現實 logo、含真實人物臉、含露骨 / 暴力、文字超過 3 字
📐
記憶口訣：基底 → 細節 → 風格 → 表情 → 規格 → 禁。新手最常踩「角色漂移」——畫 24 張時 AI 把眼睛大小、髮色慢慢改掉。明確寫「same character throughout, identical hair color, same outfit, consistent eye style」、必要時 prompt 內貼「reference image」。
(03)

示範圖｜可複製、可重現

下面這張是 OPEN BEANS 品牌吉祥物「Mr. Bean」的 8 張貼圖預覽。咖啡豆擬人化角色、圓潤可愛、深棕色身體 + 米白臉、8 種表情。可上架 LINE Creators Market。

完整 prompt · OPEN BEANS Mr. Bean 品牌吉祥物貼圖組 適用：ChatGPT GPT-image / Gemini Nano Banana 
請生成一張 LINE 貼圖組預覽（sticker set preview）：

【整體】4×2 網格排列、共 8 張貼圖、每張正方形、整體像「LINE Creators Market 預覽圖」。

【角色】「Mr. Bean」OPEN BEANS 品牌吉祥物：
- 物種：擬人化咖啡豆（圓潤橢圓形體型）
- 顏色：深棕色身體 + 米白色臉部 + 燙金色「OB」logo 在胸前
- 配件：白色小領結 + 圓眼睛 + 微笑嘴
- 個性：友善、認真、有點害羞

【風格】手繪卡通風、粗線條（≥ 3px）、平塗顏色（不要漸變或厚塗）、線條為深棕色、整體溫暖可愛。

【8 個表情（每張正方形、240×240、透明底）】
1. OK 手勢（豎大拇指）
2. 謝謝（鞠躬感謝）
3. 開心（雙手舉起、笑臉）
4. 累（趴下、呼）
5. 晚安（蓋被睡覺、Z 字）
6. 加油（揮拳）
7. 害羞（臉紅、低頭）
8. 笑（捧腹大笑）

【規格要求】每張 240×240、透明底（白色背景代表透明）、留 10% 邊緣空白。

【角色一致性】所有 8 張角色細節必須完全一致：相同的眼睛大小、相同的領結、相同的身體比例、相同的線條粗細、相同的顏色。

【限制】嚴禁角色細節在不同貼圖之間漂移、嚴禁出現非 OPEN BEANS 的 logo、嚴禁真實人物臉、嚴禁露骨或暴力、嚴禁文字超過 3 字（且文字要用英文如「OK」「Hi」「Thanks」）。
📷
讀圖時看這 4 個點：(1) 8 張角色看起來是同一個嗎（眼睛、領結、身體比例一致）？(2) 表情有沒有夠多樣（OK / 累 / 開心 / 晚安）？(3) 線條粗細統一嗎？(4) 平塗顏色而非漸變嗎（LINE 規格要求）？
(04)

改成你的場景｜踩雷與失敗案例

把 prompt 拿來、改 3 個替換槽就變你的版本：

替換槽你的版本
ⓐ 角色基底 = ?例：擬人化咖啡豆（OPEN BEANS）／擬人化貓（書店品牌）／擬人化豆腐（食品品牌）
ⓑ 風格 = ?例：手繪粗線條（友善）／像素風（懷舊）／3D 厚塗（IP 化）／日系細線（清新）
ⓒ 表情組合 = ?例：8 種日常情緒／16 種完整套組／24 種極致表現（含特殊節慶）

最常見的 4 種踩雷：

  1. 角色不一致　AI 第 1 張畫的眼睛大、第 8 張變小、領結顏色也偷偷變了。解法：明確寫「identical character details across all 8 stickers: same eye size, same outfit, same color palette, no variations」、必要時 prompt 內貼第 1 張當 reference image 跑 image-to-image。
  2. 風格漂移　前 4 張手繪風、後 4 張變 3D 風。解法：明確寫「same drawing style throughout: hand-drawn, thick lines, flat colors, no 3D rendering, no gradient」。
  3. 構圖不適合 LINE 規格　AI 給你的角色貼到貼圖邊緣、被 LINE 自動裁切。解法：明確寫「character centered with 10% margin on all sides」「no elements touching the edge」。
  4. 中文情緒文字一律後製　AI 生圖工具的中文字體限制 + LINE 貼圖文字往往是中文（「OK」「晚安」「加油」）。C5 的最佳策略：用英文佔位文字（OK / Good Night / Cheer Up）跑 AI 生圖、最後在 Figma 用「金萱半糖粗體」疊上中文。

    實戰提醒：C5 sticker set 上架 LINE Creators Market 是長期被動收入——一組 24 張定價 NT$30、若紅了若內容對到目標客群、月入 NT$3,000-10,000 是進階情況（多數新上架的貼圖月入低於 NT$1,000，需累積 IP 與行銷）。關鍵：角色 IP 一致性。建議用 Figma component 做 master character、所有貼圖從它衍生、確保 24 張完全一致。
⚠️
交付提醒　本卡片的 AI 生成的示範圖不是可直接給客戶的最終交付品。中文商業案一律先用 AI 生出構圖（含英文佔位文字）、再走 後製字體疊圖工作流 在 Figma／Canva 疊上中文、最後校稿確認才交付給客戶。
