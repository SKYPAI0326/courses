CH 2.1

商業廣告的 8 種用途版型

同一張產品圖、搭 8 種版型、做出 8 種效果的廣告。你的工作不是拍 8 張圖，是選對版型讓 AI 跑。

學習成果
能說出 8 種版型（A～H）的用途與適用場景，並用一句話描述各版型的感受
能用「目的 → 版型」3 問速查，為任意需求挑出 1 個合適版型
能對照版型結構驗收 AI 產出：主視覺占比對、文字密度對、整體感覺對
今天結束時，你會交出
  1. 8 種版型（A-H）各 1 句感受描述（不必背、靠速查表）
  2. 1 個用「目的→版型」3 問為自己情境挑出的版型 + 理由
  3. 1 份對 AI 產出的驗收結果（占比 / 密度 / 感覺 3 項打勾）

完成這 3 件就算過關。8 版型不必背、會查就好。

(01)

看——同一杯抹茶拿鐵，可以做 8 種廣告

開場情境
弄一下咖啡工作室老闆問你一個問題：「我這款春季抹茶ティラミスラテ、IG 要發、官網要 banner、節慶要禮盒、會員日要促銷⋯⋯我要拍 8 張不同的圖嗎？」答案是：不用。1 張產品圖 + 8 種版型 = 8 種完全不同效果的廣告。

你只要1張產品圖、然後搭 8 種不同的版型——同一個商品、用不同方式排版 + 不同訊息重點 = 8 種完全不同效果的廣告。這就是這個單元要教你的事：學完之後，你拿到任何任務、3 秒查表、選 1 個版型、就能告訴 AI「請用 X 版型」、AI 直接幫你做。

8 個版型 = 8 種不同效果——底層用的是同一張產品圖。你的工作不是「拍 8 張不同產品圖」，是「用 1 張產品圖、搭對版型、做出 8 種廣告」。後面 35 分鐘教你怎麼選版型，不必背 8 種、用 3 問就能挑。

(03)

8 版型詳細卡片——每個版型的 4 件事

每個版型看 4 件事：用來做什麼 / 設計重點 / 適用場景 / 版型結構。看完每張卡片，結尾有「記住這句話」幫你把核心觀念壓進腦子。

A 主視覺版
最吸睛的版型——用來抓住注意力
商品占滿畫面、文字極少，視覺衝擊最強。讓人第一眼就停下來，停留 3 秒決定要不要看下去。
  • 設計重點：畫面最大化（商品 80–90%）、文字只留標題或賣點（10–20%）、留白最少化
  • 適用場景：IG 廣告首圖、廣告主視覺、任何「第一眼留住人」的場合
  • 小技巧：流心/亮點放在黃金視覺位置（畫面中上偏左 1/3）；用光影強化質感
┌─────────────────────────┐ │ │ │ 主角（產品） │ │ 80-90% │ │ │ ├─────────────────────────┤ │ 極少文字（標題/賣點） │ 10-20% └─────────────────────────┘
「畫面越強、訊息才會被看見。」
B 橫幅版
最適合橫式空間——用來快速吸引注意
商品 + 大標並列、橫向延展，文字精簡有力。讓人 1 秒看完重點，知道「這是什麼、為什麼要點進去」。
  • 設計重點：橫向延展、主次分明（主標與畫面主體並列）、資訊精簡到 1-2 行
  • 適用場景：網站 BANNER、廣告橫幅、EDM 標題 / 信件 hero
  • 小技巧：主體放大 + 搭配短標語；視覺高度 70% 以上
┌──────────────┬────────────┐ │ │ 主標題 │ │ 主角 │ │ │ （產品） │ 短標語 │ │ │ │ └──────────────┴────────────┘ 橫向延展 90%+ │ 文字 < 30%
「在橫幅中、1 秒抓住眼球就是關鍵。」
C 社群版
最輕量的版型——讓產品自己說話
留白多、文字少、產品占視覺焦點，呼吸感強，質感高。不打擾用戶滑動節奏，適合日常曝光。
  • 設計重點：文字少（品牌/品名即可）、大量留白、產品占 70%+
  • 適用場景：IG 貼文、Threads 貼文、日常社群不打擾的場合
  • 小技巧：讓流心/主體引導視線；不要塞滿；不適合促銷檔期
┌─────────────────────────┐ │ │ │ 主角（產品） │ │ 70%+ │ │ │ ├─────────────────────────┤ │ 少量文字（品牌/品名） │ 30%- └─────────────────────────┘
「少即是多、讓產品自己說話。」
D 系列版
完整呈現多款——讓客戶比較選擇
3 款並列、差異一目了然。讓猶豫的客戶看清楚差異，加上特色標籤拉高信任感。
  • 設計重點：系列整合（統一風格構圖）、比較引導（並列呈現差異）、加入食材/製程資訊提升信任
  • 適用場景：電商商品頁、IG 限動系列推薦、品牌目錄 / Lookbook
  • 小技巧：每個商品的角度、光線、留白對齊；加「特色標籤」（手工製作/無添加）
┌─────┬─────┬─────┐ │產品1│產品2│產品3│ 主視覺區 70%+ │ │ │ │ ├─────┼─────┼─────┤ │口味 │口味 │口味 │ │特色 │特色 │特色 │ 資訊區 30%- └─────┴─────┴─────┘ 賣點 / 品質保證
「系列感、就是讓喜歡變成全都想要。」
E 禮盒版
質感包裝 × 精緻呈現——用來拉高單價
包裝主視覺、燙金紋飾、深色背景，質感優先於資訊量。讓人感受到「這個值得送禮」的氛圍。
  • 設計重點：包裝與情境營造價值感、質感優先於資訊量、適合節慶/送禮/高單價策略
  • 適用場景：節慶禮盒（中秋、母親節、聖誕）、婚禮/企業禮品、高價商品主視覺
  • 小技巧：拍攝角度低（商品仰望感）；深色背景或絨布感；加金色/燙金元素
┌─────────────────────────┐ │ │ │ 禮盒（外包裝為主角） │ │ 含緞帶、紋飾、標籤 │ │ │ ├─────────────────────────┤ │ 品牌名 + 系列名 │ └─────────────────────────┘
「包裝感、就是溢價的視覺證明。」
F 特寫版
極致特寫 × 感官刺激——用來提升食慾
放大細節與質感（流心、起司拉絲、糖霜），強化視覺衝擊。幾乎零文字，讓畫面說話。
  • 設計重點：細節質感放大、快速激發食慾或感官想像、適合食物/飾品/化妝品
  • 適用場景：IG / Threads 貼文、美食平台（Foodpanda / UberEats）、餐廳菜單視覺
  • 小技巧：拍流動瞬間（巧克力流心、咖啡蒸氣）；光線讓質感發亮（油光、結露）
┌─────────────────────────┐ │ │ │ │ │ 產品極致特寫 │ │ （流心、質感、紋路） │ │ │ │ │ └─────────────────────────┘ 零文字或極少品名
「越近越想吃。」
G 促銷版
強烈訊息 × 行動導向——用來快速轉單
大字優惠 + 對比色，不追求美感，追求轉換率。讓人一眼看到「折扣」「限時」「現在下單」的衝動。
  • 設計重點：突出優惠與行動指令（折扣/限時/限量）、大字 + 對比色加速決策
  • 適用場景：檔期活動（雙 11、母親節、會員日）、FB / IG / LINE 廣告投放、限時優惠 banner
  • 小技巧：數字越大越好（「85 折」比「優惠中」強 10 倍）；限時感（即日起至 X/X）
┌─────────────────────────┐ │ 限時優惠 │ │ 商品名 │ │ 2 盒 85 折 │ 大字優惠 │ 即日起-4/30 │ ├─────────────────────────┤ │ 產品 │ └─────────────────────────┘
「優惠不亮、訊息白給。」
H 生活版
情境氛圍 × 情感共鳴——用來拉近距離
融入生活場景（午後、睡前、辦公），文字溫柔像朋友推薦，不像廣告。建立品牌好感與忠誠度。
  • 設計重點：融入生活場景、情感連結優先、文字柔軟（情境文案/手寫感）
  • 適用場景：品牌日常（IG 動態）、生活提案（季節物語）、內容行銷 / 品牌故事
  • 小技巧：加入人的痕跡（咖啡杯、翻開的書、毛毯一角）；自然光；避免命令句
┌─────────────────────────┐ │ 「午後時光、 │ 情境文案 │ 來杯春の抹茶ラテ。」 │ （手寫感） │ │ │ 產品 + 生活道具 │ 情境主視覺 │ （咖啡、書、餐巾） │ │ │ └─────────────────────────┘
「不賣商品、賣與商品一起的時光。」
(04)

選版型 3 問——拿到任務、3 秒決定用哪個

不必背 8 種版型，用這 3 問挑：

1
這要給誰看？（目標受眾在什麼平台、什麼情緒）
IG 滑動的人（25–35 歲女性）→ A / C / F / H
信箱掃過的人（會員 / 訂閱戶）→ B / G
商品頁瀏覽的人（已有購買意圖）→ D / E
路過廣告看到的人（不認識品牌）→ A / G
受眾的平台決定版面方向：IG = 直式多留白；網站 = 橫式；商品頁 = 多款並列。
2
要他們做什麼？（目標行動）
「想看更多」（建立印象）→ A / C
「記住品牌」（轉發、收藏）→ B / H
「比較選擇」（看清楚差異）→ D
「立刻下單」（轉換）→ G
「想送禮」（高單價）→ E
「想吃 / 想要」（感官激發）→ F
「要他們做什麼」是選版型最關鍵的一問。G 促銷版只為一件事：下單。H 生活版只為一件事：喜歡你。
3
場景與資源？（在哪個平台 / 場合看到）
社群日常（每天發）→ C 為主、F / H 變化
廣告檔期（推新品）→ A 預熱、G 衝刺
節慶送禮（中秋 / 母親節）→ E 為主
電商商品頁（多款）→ D 整合
3 問答完、對照上面速查表選 1 個。不必死記、查表就好。
☑
版型沒有最好、只有最適合。同一個商品在預熱期選 A、在節慶期選 E、在檔期衝刺期選 G——這是三個不同目的的三個正確答案。
(05)

同產品 × 多版型——系列行銷思維

一個商品上市、別只做 1 張——同時做 3–4 個版型搭配行銷漏斗：

行銷階段 用什麼版型 為什麼
預熱期 A 主視覺版 抓注意力、引發期待
上市期 B 橫幅版 + F 特寫版 網站 banner + 社群激發購買慾
推廣期 D 系列版 + H 生活版 多款比較 + 情感連結
衝刺期 G 促銷版 檔期優惠快速轉單
節慶期 E 禮盒版 拉高單價

1 個商品 → 8 張不同版型 → 覆蓋整個行銷漏斗——這就是版型策略的價值。

→
版型不是 OR、是 AND。版型不是「美感選擇題」，是「用途選擇題」。每個版型都有它最該用的場合，不是哪個漂亮就用哪個。

(07)

做——為自己的真實需求挑版型並跑 1 張

1
想 1 個真實任務（3 min）
從以下任選 1 個（或用自己工作的真實需求）：
· 我下週要為自家品牌做 IG 廣告
· 老闆要我做電商商品頁的 banner
· 朋友請我幫忙做節慶禮盒文宣
· 我在做季節限定產品的促銷檔期
寫下：目標受眾（誰會看到）、目標行動（希望他們做什麼）、場景（在哪個平台看到）、商品（賣什麼）。
2
用 3 問挑版型（2 min）
對照速查表 + 3 問、挑 1 個版型。
寫下：我選的版型：______／為什麼選它（30 字內）：______
3
複製對應版型 prompt → 改變數 → 跑（10 min）
不必自己想 prompt 怎麼寫——翻開 8-prompt-examples.md，找到你階段 2 選的版型段（§ A / B / C / D / E / F / G / H 任一段），複製「快速啟動 prompt」，替換變數空格後丟 Gemini 或 ChatGPT。
3 步操作： 1. 翻開 8-prompt-examples.md → 找到你選的版型段 2. 複製「快速啟動 prompt」（版型結構已預填，不必你想） 3. 替換變數空格 → 丟 AI
範例對照（假設你選 G 促銷版做咖啡店「第 2 杯 5 折」）：
變數空格你填什麼
{業種}咖啡店
{商品}春季拿鐵
{主優惠}第 2 杯 5 折
{限時}4/25–4/30 限定
{CTA}點餐優惠
{對比色}黑底 + 黃 + 紅
預期差異：你跑出的圖跟同學不同——這是正常的，AI 每次輸出都會浮動。重點是版型方向對不對，不是細節完不完美。
4
3 件事自檢
拿到 AI 產出後，對照你選的版型結構問自己：
  • 主視覺占比對嗎？（A/F 要 80%+、D 要多商品並列）
  • 文字訊息對嗎？（G 要大字優惠、H 要小字情境）
  • 整體感覺對嗎？（A 是衝擊、C 是質感、G 是急迫）
3 件事都對 → 收。1–2 件不對 → 跟 AI 說「{某項}不夠、請調整」。

完成標準（self-check）

  • 寫下 1 個真實任務的受眾 / 目標 / 場景
  • 用 3 問選 1 個版型 + 寫 30 字選擇理由
  • 跑出 1 張 L4 完稿
  • 對照版型 3 件事自檢、決定收 / 改
—
跑出來不對勁？翻 CH5-1 § 卡關 troubleshooting 速查——8 種常見症狀都有「診斷 + 修正話術」、複製貼回同對話即可修圖。
(08)

常見錯誤——這 3 件事你最容易踩到

錯誤 1
選版型用「哪個漂亮」、不用「哪個適合」
現象
看到 8 版型、覺得 H 生活版最有質感，不論什麼任務都選 H。結果做促銷檔期也用 H、轉換率超低。
原因
H 生活版適合情感連結 / 品牌日常，不適合促銷轉單。版型有各自用途，不是「最漂亮就最好」。
解法
強制 3 問選法：① 受眾在哪個平台 / 情緒？② 要他做什麼動作？③ 預算與時間夠嗎？3 問答完對照表——版型沒有最好、只有最適合。
→
版型沒有最好、只有最適合。
錯誤 2
每張廣告都從零想、不用版型查表
現象
回到工作崗位後忘了 8 版型，每次老闆要新物料又從「我要做一張看起來很厲害的廣告」開始想。每次 2–3 小時，品質還不穩。
原因
8 版型是工具、不是知識——要拿來用才有價值。學完不存進工作流、就沒帶走。
解法
把 8 版型表貼工作桌（或存手機 / Notion）：拿到任務 → 看表 → 3 問選 1 個版型 → 告訴 AI → 3 件事自檢。30 分鐘可完成 1 張。
錯誤 3
1 個商品只做 1 個版型、錯失系列行銷的價值
現象
新品上市只跑 A 主視覺版，覺得「夠了」。錯過了 B 網站 banner、D 商品頁系列、F 特寫吸引食慾、G 檔期促銷、H 生活情境。新品只曝光 1 次、沒覆蓋整個行銷漏斗。
原因
1 張廣告 = 1 個曝光點。1 個商品 × 8 版型 = 8 個曝光點 × 不同情境 × 不同行為呼籲——這才是版型的乘數效應。
解法
新品上市時用「同產品 × 多版型 × 行銷漏斗」一次做 5–8 張：A 預熱 → B+F 上市 → D+H 推廣 → G 衝刺 → E 節慶。版型不是 OR、是 AND。
(09)

檢核題——確認你學會了

以下 2 題確認你對 8 種版型的掌握程度。

Q1（概念驗證｜簡答）
請寫出 8 版型的名稱與一句話用途，並說明：為什麼版型用「目的」分類比用「結構」分類更實用？
展開解答提示
8 版型名稱與用途（至少命中 5 點算通過）：
A 主視覺版：抓住注意力／B 橫幅版：快速傳重點／C 社群版：輕量易讀
D 系列版：完整呈現比較／E 禮盒版：拉高單價／F 特寫版：提升食慾
G 促銷版：快速轉單／H 生活版：情感連結

目的分類比結構分類好用的理由：
學員的工作不是「練排版」，是「達成某個目的」（賣貨 / 建品牌 / 衝轉換）。用「目的」對照、3 秒選 1 個；用「結構」對照、要先學 5 種骨架特徵才能判斷，慢且抽象。目的導向讓你「拿到任務 → 立刻知道用哪個版型」。

加分項：版型沒有最好、只有最適合。

不及格：只列名稱沒說用途；說「都很重要」；用美感判斷哪個版型好。
Q2（應用驗證｜情境題）
主線 A 老闆要為春季抹茶ティラミスラテ做完整行銷，列了 5 個物料。請對 5 個物料各選 1 個版型，並用 30 字內說明理由：
① IG 廣告主圖（首次接觸品牌的人）
② 官網首頁 hero banner（已知品牌的回購客戶）
③ 電商商品頁（3 款口味比較）
④ 母親節禮盒文宣（節慶送禮）
⑤ 5/1 限時優惠 banner（會員下單）
展開解答提示
預期答案（5 個全對 + 理由合理算通過）：

① IG 廣告主圖 → A 主視覺版：首次接觸、要抓注意力、商品占滿畫面
② 官網 hero banner → B 橫幅版：橫向延展、商品 + 主標、回購客戶要快速看到
③ 商品頁系列 → D 系列版：3 款並列、比較選擇、信任感強化
④ 節慶禮盒 → E 禮盒版：拉高單價、包裝感營造價值、適合送禮場景
⑤ 限時優惠 → G 促銷版：大字 85 折、強烈訊息、會員快速下單

加分：提到「同 1 張產品圖、5 個版型、5 個曝光點」（系列行銷思維）；或指出可加做的版型（如 F 特寫版 IG 貼文）

不及格：5 個都選 A；用「我覺得好看」當理由；跳過任一物料。
「不是圖片不夠好、而是版型不夠對。」

8 版型是你未來每次接案的快速啟動鍵——把總表存手機、貼工作桌、印 A4。下次老闆說「幫我做一張廣告」，你不會再從零想——你會問 3 個問題、查表選 1 個、5 分鐘給 AI、剩下時間優化文字與素材。
