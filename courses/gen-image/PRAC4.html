(12) · PRAC 4

品牌系列演練

你是確認系列像同品牌的人。這是 M4 最後一輪演練——為自己建立完整品牌資產（品牌小卡 + 3 張系列）。從跨季節 / 跨產品線 / 跨活動週期 3 種組合挑 1 組，填完品牌小卡 6 段、跑 3 張系列、用系列檢查 3 問確認跨張一致性。今天產出的資產就是 CH5-1 整合實戰的起點。

學習成果
能為自選品牌填完整品牌小卡（6 段全齊），在 8 分鐘內完成自我檢查：顏色有 hex、字體有名稱、禁用清單 ≥ 10 項
能從 3 種演練組合挑 1 組、跑 3 張系列完稿（共享同品牌小卡 + 同骨架，只換主題變量），達到「放一起能被認出是同品牌」的系列感
能用系列檢查 3 問做跨張一致性檢核達 ≥ 4/5，並說出今天產出的品牌資產未來半年可省下多少重複設計時間
(01)

熱身 — 快速決策：選演練組合 + 品牌

定位提醒
你是確認系列像同品牌的人——這是 M4 最後一輪演練，為自己建立完整品牌資產（品牌小卡 + 3 張系列）。5 分鐘內決定好「演練組合 + 品牌」，後面 20 分鐘的演練才能順跑。

步驟 1｜挑 1 個品牌（2 min）

三選一：

  • A 自家品牌：你實際經營或服務的品牌（最有商業價值）
  • B 客戶品牌：你最近接案的客戶品牌（最接近實戰）
  • C 主線 A 弄一下咖啡工作室：課程預設範例（若 A/B 不適用）

步驟 2｜從 3 種演練組合挑 1 組（3 min）

組合情境3 張主題變量適合誰
I 跨季節年度行銷規劃春 / 夏 / 秋 新品（同類商品、換季節）快消品 / 餐飲 / 美妝
II 跨產品線品牌陣列展示主打 A / B / C 3 款不同產品有多產品線品牌 / 手作店
III 跨活動週期行銷日曆新品上市 / 週年慶 / 會員專屬 3 個活動電商 / 服務業 / SaaS

快速判斷：

  • 手上業務一年會跑 4 季行銷 → 選組合 I
  • 品牌有 3+ 核心產品 → 選組合 II
  • 每月都有不同行銷主題 → 選組合 III
  • 都不太符合 → 預設用組合 I（最通用）
i
預期差異：你挑的組合 + 品牌跟同學不會一樣——這是設計。每個人跑自己業務情境才有帶走價值。
(02)

演練 — 3 個階段，20 分鐘跑完

節奏說明
20 分鐘分 3 階段：階段 1 填品牌小卡（8 min）→ 階段 2 跑 3 張系列（10 min）→ 階段 3 系列檢查 3 問（2 min）。每個階段都有硬性要求，超時就壓縮自查項目，不能跳階段。

階段 1｜填品牌小卡 6 段（8 min）

品牌小卡要填 3 件事：顏色（色票）、字體（字族）、風格詞（語氣 + 禁用）。完整 6 段如下：

段內容硬性要求
0風格 preset11 選 1、含示範品牌參考
1顏色（色票）3 色 + 具體 hex + 禁用色
2字體（字族）2 種 + 具體字族名 + 字階
3語氣3-5 詞 + 禁用語氣 + 好 vs 壞示範句
4禁用清單視覺 3-5 + 情境 3-5 + 字體 3
5風格詞（主力骨架）1-5 選 1 + 次要骨架 + 30 字理由

若真的來不及：第 0 / 1 / 2 / 4 段為底線必填；第 3 / 5 段可簡化但不能空。

填完立刻自查：

  • 顏色：色票每個都有 hex（不是「鼠尾草綠」，是 #5a7a5a）
  • 字體：字族有具體名稱（不是「明朝體」，是 Shippori Mincho）
  • 風格詞：禁用清單至少 10 項（視覺 5 + 情境 3 + 字體 2 以上）

階段 2｜跑 3 張系列（10 min，每張 3 min）

依你選的演練組合（I / II / III）跑 3 張系列：

組合第 1 張第 2 張第 3 張
I 跨季節{品牌}春季限定 {新品 A}{品牌}夏季限定 {新品 B}{品牌}秋季限定 {新品 C}
II 跨產品線{品牌}主打 {產品 A}{品牌}主打 {產品 B}{品牌}主打 {產品 C}
III 跨活動週期{品牌}新品上市{品牌}週年慶優惠{品牌}會員專屬活動

每張 3 min 流程：

1
貼品牌繼承 prompt（30 秒）

用以下模板（扣 prompt-templates.md § 類型 4），填入當張主題：

吃下以下品牌小卡：
[貼完整品牌小卡]

---

結構沿用：{你品牌小卡指定的主力骨架}
本次主題：{當張主題、例：春季限定 {新品 A}}
- 商品：{商品描述}
- 季節 / 場景：{如春季限定 / 週年慶}
- 徽章：{限期 / 活動名稱}
- 價格：{含稅價}
- 道具：{對應主題的道具暗示}

請依以上品牌小卡 + 骨架 + 主題產出 1:1 L4 完稿。

【強制自檢】產完請回覆：
1. 配色是否只用品牌小卡指定 3 色？
2. 字族是否符合？
3. 禁用清單是否全遵守？
2
跑 AI（1.5 min）

把 prompt 送出，等 AI 產出。

3
等待產出 + 初步檢查（1 min）

拿到圖後快速確認：AI 強制自檢 3 項是否都是「是」；若有「否」記下來，留系列檢查 3 問處理。

i
核心紀律：3 張都要用同一份品牌小卡 + 同一個骨架——只換主題變量。

預期差異：3 張每次跑結果不同，AI 對「禁用清單」的遵守程度也有波動——通常 80-90% 遵守、10-20% 偷偷加東西（多色 / 多字族 / 漸層）。這是為什麼要跑系列檢查 3 問。

階段 3｜系列檢查 3 問 跨張一致性檢核（2 min）

3 張跑完後，同對話內貼系列檢查 3 問（扣 prompt-templates.md § 類型 5）：

請對剛才的 3 張系列（A / B / C）做品牌一致性自檢、對照以下品牌小卡：
[貼品牌小卡]

逐項跨張比對：
1. 顏色：3 張是否只用品牌小卡指定 3 色？（列每張實際 hex）
2. 字體：3 張字族是否符合？（列每張實際字族）
3. 禁用清單：3 張是否全遵守？（逐條 × 每張）
4. 骨架：3 張是否同骨架？
5. 語氣：3 張調性是否一致？

結論：跨張通過 __/5。
5/5 = 可作為系列資產收納。
3-4/5 = 指出哪張哪項不符、建議迭代哪張。
≤ 2/5 = 品牌小卡需加強。
≥ 4/5 通過 → 本 PRAC 合格，存檔進個人品牌資產庫
3/5 通過 → 迭代 1 張不符的（保留 2 項 + 修 3 項）
< 3/5 通過 → 回階段 1 補強品牌小卡禁用清單，重跑 1 張最差的
(03)

分享 — 互相展示，外部視角驗系列感

互相展示（3 min）

2-3 人一組，每人快速展示：

  • 自家品牌小卡（口述 5 段要點，30 秒）
  • 3 張系列（並排展示，30 秒）
  • 系列檢查 3 問結果（通過幾項，30 秒）
  • 最滿意的 1 張 + 最卡關的 1 張（30 秒）

同學回饋（互相給 30 秒）：

  • 你的 3 張看起來像同品牌嗎？（外部視角檢驗）
  • 品牌小卡有沒有什麼段落可以更具體？
  • 若要再做第 4 張，你建議什麼主題？

老師總結 + M5 預告（2 min）

  • 統計全班系列檢查 3 問通過率 ≥ 4/5 的人數
  • 挑 2-3 個最完整的品牌小卡範例當全班示範
  • CH5-1 預告：下個單元 30 分鐘跑完一次完整閉環——今天產出的品牌小卡 + 3 張系列就是 CH5-1 的起點。全部存檔備份。
(04)

動手練習 — 30 分鐘完整流程

PRAC 4 演練題目

在 30 分鐘內完成完整品牌系列演練：

  • 熱身（5 min）：挑 1 個品牌（A/B/C 三選一）+ 1 個演練組合（I/II/III 三選一）
  • 演練（20 min）：填品牌小卡 6 段（8 min）→ 跑 3 張系列（10 min）→ 系列檢查 3 問跨張一致性（2 min）
  • 分享（5 min）：互相展示 + 同學回饋
預期成果
  • 1 份完整品牌小卡（硬性 6 段，可簡化到 4 段）
  • 3 張系列 L4 完稿（共享同品牌小卡 + 同骨架）
  • 1 份系列檢查 3 問記錄（5 項打勾 + 結論）
  • 1 個外部驗證結果（同學能否辨認為「同品牌」）
  • 1 組可進 CH5-1 整合實戰的品牌資產雛形
完成標準（self-check）
  • 品牌明確（自家 / 客戶 / 假想皆可）
  • 演練組合明確（I / II / III 選 1）
  • 品牌小卡至少 4/6 段填完、全部具體可執行
  • 跑出 3 張系列（共享同品牌小卡 + 同骨架、只換 3 項主題變量）
  • 系列檢查 3 問跨張通過 ≥ 4/5
  • 至少 1 位同學確認「看起來像同品牌」
  • 品牌小卡 + 3 張系列獨立備份（CH5-1 會用）
i
預期差異提醒：你的品牌、組合、品牌小卡、系列跟同學完全不同——這是設計。不追求「跑得最漂亮的系列」，追求「放一起能被認出是同品牌」。系列檢查 3 問 4/5 一致性已合格，5/5 是加分，不強求。今天產出是個人品牌資產庫的雛形——未來做新廣告從這份品牌小卡開始，不再從零。

(05)

常見錯誤 3 條

01
熱身花 10 分鐘決定演練組合（選擇癱瘓）
錯誤現象

學員看到 3 種組合（跨季節 / 跨產品線 / 跨活動週期），開始糾結「我到底該選哪個？」「跨季節會不會太簡單？」「跨產品線我只有 2 款產品怎麼辦？」——結果 10 分鐘過去了，還沒開始填品牌小卡。

原因

選擇癱瘓。演練的重點是「跑通流程」，不是「選最完美組合」。對初學者來說 3 選 1 已經足夠，糾結組合不會讓成果更好。

解法

5 分鐘強制決策：一年做 4 季行銷 → 選 I；品牌有 3+ 產品 → 選 II；每月不同行銷主題 → 選 III；都不符合或選不定 → 預設 I（最通用）。

心法：選哪個都會有收穫——重點是跑完完整流程，體驗 M4 所有技能的整合。

02
品牌小卡填 4 段就跑系列（跳過禁用清單）
錯誤現象

學員覺得時間不夠，填了 0/1/2/3 四段就開始跑系列——跳過第 4 段禁用清單。結果 3 張系列跑完，AI 自由發揮加了：第 1 張加了淡淡漸層背景、第 2 張多了一種輔助色、第 3 張徽章從圓形變成半透明玻璃態。3 張單看都不差，合起來系列檢查 3 問只通過 2/5。

原因

禁用清單是品牌小卡的守門員。沒禁用 = AI 用訓練資料平均值（漸層、多色、裝飾邊框）自由發揮。扣 CH4-3 錯誤 3 的 PRAC 實戰版——在時間壓力下最容易被跳過，但跳過代價最高。

解法

禁用清單是 PRAC4 必填，時間不夠改第 3 / 5 段簡化，不能省第 4 段：視覺禁用至少 5 項、情境禁用至少 3 項、字體禁用至少 2 項，共 10 項 = 底線。

快速寫禁用：對照 CH4-3 教的通用禁用項（漸層 / 純黑 / 霓虹 / 寫實人像 / 快時尚口號 / Comic Sans / 過瘦字體）直接抄，不用創新。

03
3 張系列主題變量設太多（變量失控）
錯誤現象

學員挑組合 I 跨季節，但 3 張主題是：春季抹茶拿鐵（杯裝）/ 夏季柚子蛋糕（盤裝）/ 秋季焙茶禮盒（盒裝）。3 張商品型態完全不同——杯 / 盤 / 盒。結果骨架套不進去，系列檢查 3 問第 4 項「骨架一致性」直接失敗。

原因

變量失控。系列演練的核心是控制變量、換單一維度：組合 I 跨季節 → 商品類別保持一致，只換季節主題（都是飲品類，或都是甜點類）。

解法

3 張主題的商品型態必須相近：

組合 I 範例（飲品）：春抹茶拿鐵 / 夏柚子拿鐵 / 秋焙茶拿鐵（都是拿鐵類）
組合 II 範例（皂類）：玫瑰皂 / 薰衣草皂 / 檜木皂（都是單塊皂）
組合 III 範例（App）：新品上市 / 週年慶 / 會員方案（都是同 App 不同訊息）

心法：演練是控制變量測技能，不是炫技。變量多 = 看不出品牌小卡效果，失去演練意義。

若未通過——看這裡
  • 6-7/10 → 回頭補最弱 2-3 項（通常是禁用清單太空，或 3 張變量失控）
  • < 6/10 → 完整重跑階段 1 + 2（重填品牌小卡 + 重跑 3 張）
Q2｜反思：品牌小卡的長期價值

請回答以下 3 個問題：

  1. 你今天產出的品牌小卡 + 3 張系列，未來半年內可以重複使用多少次？
  2. 若下個月要做第 4 張系列（例：春季 → 冬季），你需要重新填品牌小卡嗎？還是只換主題變量？
  3. 這種「品牌小卡一次寫、多年用」的模式，跟你之前做行銷物料的模式相比，最大的時間節省在哪裡？
預期答案要點（至少命中 3 點算通過）
  1. 重複使用次數：每季新品 × 4 季 = 4 次 / 年，加上週年慶 / 會員活動 / 特殊主題，10-20 張 / 半年都用同份品牌小卡
  2. 下個月做第 4 張：不用重填品牌小卡——品牌小卡是品牌規格，半年內不動。只換主題變量（商品 / 季節 / 徽章 / 價格 / 道具），直接貼同份品牌小卡 + 新主題 → 跑一張 → 系列檢查
  3. 時間節省來源：過去每次從零想（配色 / 字體 / 語氣 / 版面），每張 30-60 分鐘；現在品牌小卡寫一次，後續每張只用 3-5 分鐘（跑 AI + 驗收），節省幅度約 80-90%
  4. 額外價值（加分）：系列一致性自動保證；新員工 / 外包設計師來，給他一份品牌小卡就能接手，不用教品牌風格；品牌辨識度複利——每次曝光都強化相同視覺記憶

不及格的答案：「看情況」／「不確定能不能重複用」／沒算出具體使用次數／「還是要重畫設計」（違反本單元核心）。
