(B4)

包裝展示圖｜禮盒與內容物

禮盒打開、內容物有序攤開的視覺。展示包裝設計本身、同時讓客戶看清「裡面有什麼」。節慶禮盒、限量套裝、發布禮盒、季節禮品最高頻——客戶花的不只是商品錢、是「拆開的儀式感」。

這張卡片你會學到
能判斷何時用包裝展示圖（與 B1 白底、B2 生活方式、B3 影棚 hero 的差異）
能拆解模板的 6 段 prompt 結構，平衡「外盒+內容物」的視覺重量
能識別 4 種常見踩雷（內容物擠在一起／盒子角度怪／logo 被光吃掉／快遞箱廉價感）並修正
(01)

適用場景

這個模板用在哪
主體是一個開蓋禮盒、內容物有序擺放在盒中或盒旁、背景同色調乾淨表面。重點是同時呈現「包裝設計品味」+「裡面有什麼」+「節慶/限量氛圍」。比 B1 多了內容物、比 B2 沒情境、比 B3 不需戲劇光。

四個典型用途：

  • 節慶禮盒主視覺（中秋、年節、母親節、聖誕——電商最高頻的促銷檔期）
  • 限量套裝宣傳圖（聯名、紀念版、發行版套組）
  • 美妝套裝 / 美食禮包 / 數碼套組（百貨櫃、誠品禮品專櫃、選物店常見）
  • 品牌官網禮盒專區 hero（季節更換、輪播主視覺）
對比項本卡片 · B4 包裝展示近似模板
視覺主體外盒 + 內容物並列B1：純單品；B2：商品在情境中；B3：商品為唯一主角
內容物數量3-5 件（盒內或盒旁）B1：嚴禁；B3：≤ 2 個道具；B2：3-5 個生活道具
盒子角度必須開蓋、自然不別扭其他模板：不需要盒
光線45° 頂光 + 環境補光、不要戲劇感B1：正面柔光；B3：戲劇 key+fill+rim
節慶氛圍明確（緞帶、裝飾元素）B1/B2/B3 通常無
使用時機禮盒檔期、限量發行B1：日常上架；B2：日常社群；B3：品牌格調
💡
一句話判斷：客戶問「禮盒怎麼拍」走 B4；問「節慶 EDM 主視覺」走 B4；問「裡面要看到什麼」走 B4。B4 是 B 組裡最賺錢的卡片——禮盒檔期客單價高、客戶願意付設計費，6 張節慶禮盒主圖一年可接 NT$50000+ 的單。
(02)

模板拆解｜6 段 prompt 結構

原模板把一張禮盒展示拆成 6 段。核心難度在「平衡外盒+內容物」——內容物太多會擠、太少會空、角度太規矩會死板：

段段落必問還是可預設填什麼
1package 包裝必問包裝類型（紙盒／鐵盒／木盒／布袋）+ 形狀（長方／正方）+ 外盒色 + 內襯色 + 緞帶 + logo（建議燙金或壓凹）
2contents 內容物必問3-5 件（<3 顯空、>5 開始亂）、各自描述、擺放（盒內對稱 / 盒外散開 / 主商品略前置）
3scene 場景可預設背景表面（亞光石面／木紋／絨布）+ 色調（米白＋盒色點綴）+ 點綴裝飾（小植物葉、灑落顆粒）
4lighting 光線可預設45° 頂光柔光 + 環境補光均勻、不要戲劇光（會讓燙金 logo 反白）
5text_overlay 文案常疊品牌名 + 副標（節慶／限定／2026 系列）。位置：左上或右上留白處
6限制固定寫死禁忌：人物或手、內容物擠在一起、強反光金屬背景、出現非品牌 logo、快遞箱廉價感
📐
記憶口訣：包裝 → 內容物 → 場景 → 光 → 文案 → 禁。新手最常漏「內容物擺放方式」——只寫「3 件」AI 會給你疊在一起或全塞回盒裡。要明確寫「主商品略前置」「對稱排列」「至少 1 件半露在盒外」。
(03)

台灣化範例 ①｜小工作室視角

小工作室 · 接客戶案
OPEN BEANS 中秋禮盒主視覺 · 季節推廣 EDM 一條龍
客戶情境：延續 B1/B2/B3 同一個咖啡品牌，中秋禮盒檔期上市。客戶要做：1 張 EDM 主視覺、3 張不同產地組合的禮盒對照圖、1 張 IG 限時動態素材。預算 NT$10000。一週交付。
  • 1 包裝｜OPEN BEANS 節慶禮盒、深棕牛皮紙硬盒、開蓋狀態、燙金 logo、米白絨布內襯、米白棉麻緞帶蝴蝶結
  • 2 內容物｜3 個 50g 試喝裝豆袋（衣索比亞綠／肯亞橘／瓜地馬拉藍）+ 黃銅量匙 + 品牌折頁卡 + 圓形咖啡濾紙
  • 3 場景｜奶白亞光石面背景、盒外散落 2-3 顆深棕咖啡豆呼應品牌
  • 4 光線｜45° 頂光柔光 + 環境補光均勻、燙金 logo 不被吃掉
  • 5 文字疊加｜畫面右上：「OPEN BEANS」品牌名 + 「Holiday Tasting Set 2026」副標
  • 6 限制｜禁人物、禁內容物擠成一堆、禁強反光金屬背景、禁快遞箱廉價感
為什麼這個情境成立：禮盒檔期客戶決策層級不同（不是上架賣家、是品牌行銷）、預算往往是 B1 的 3-5 倍。同 prompt 結構改「節慶名 + 副標」就可批次產出中秋／聖誕／春節三檔禮盒。客戶看到一致的「品牌世界觀」、續訂率高。
(04)

台灣化範例 ②｜個人接案視角

個人接案 · 限量套裝顧問
幫手工皂品牌做母親節限量禮盒主視覺
接案情境：B1 範例 ② 那個主婦客戶接著想做母親節限量禮盒。3 款皂（玫瑰／薰衣草／檸檬草）+ 麻繩束口袋 + 手寫卡片，總價 NT$680/盒。要一張 EDM 主圖 + 1 張 IG 貼文 + 1 張產品頁 hero。預算 NT$5000、3 天交付。
  • 1 包裝｜素色奶白色硬紙禮盒、開蓋狀態、長方形、表面壓凹「For Mom」英文字、深米色棉繩繫於盒角（不要蝴蝶結、要自然垂落感）
  • 2 內容物｜3 塊手工皂（玫瑰粉／薰衣草紫／檸檬草綠）並排對稱、米白色折頁卡片（手寫風）、一小束乾燥薰衣草（素雅、不喧賓）
  • 3 場景｜亞光奶白石面背景、盒外散落幾片薰衣草花瓣、整體調性自然有機
  • 4 光線｜45° 頂光柔光 + 環境補光、皂身的天然顆粒紋理要看得清楚
  • 5 文字疊加｜畫面左上：「Mother's Day Limited」（無襯線細體、深棕）+「Handmade with Love」副標（襯線細體、淺棕）
  • 6 限制｜禁人物、禁手持、禁過飽和粉紅、禁工業感包裝（要手作溫度）、禁出現非品牌 logo
接案實務 tip：B4 比 B1 接案費可以翻倍（NT$5000 vs NT$3000）、因為「禮盒主視覺」的客戶心理價位高。關鍵 trick：包裝寫「自然不別扭」、內容物寫「至少 1 件半露在盒外」、緞帶寫「自然垂落」（不要蝴蝶結）——這三點直接讓 AI 出來的圖跳脫「淘寶禮盒展示」的 cliché、進入「文青品牌」感。中文「給媽媽」「手作」logo 仍走 後製疊字流程。

示範圖｜可複製、可重現

下面這張圖是用範例 ① 的 6 段（OPEN BEANS Holiday Tasting Set 中秋禮盒）實際跑出來的成品。AI 生圖 86 秒一發到位、3 個小豆袋對稱排列、量匙與品牌卡片有序擺放、燙金 logo 清楚可讀、米白棉麻緞帶自然垂落、右上「OPEN BEANS / Holiday Tasting Set 2026」文案精準。下方就是完整的 prompt——點「複製 prompt」、貼到 ChatGPT 或 Gemini，把產地組合、副標、緞帶顏色一改，就是不同檔期的禮盒主視覺：

完整 prompt · OPEN BEANS 節慶禮盒展示 適用：ChatGPT GPT-image / Gemini Nano Banana 
請生成一張禮盒包裝展示圖（packaging showcase）：

【包裝】OPEN BEANS 節慶限定禮盒。硬紙禮盒、開蓋狀態、長方形橫式擺放。外盒：深棕色牛皮紙質感 + 燙金「OPEN BEANS」品牌字。內襯：奶白色絨布。盒蓋上有米白色棉麻緞帶輕鬆繫於蓋角（不要對稱蝴蝶結、要自然垂落感）。

【內容物（4 件、盒內對稱擺放、主商品略前置）】
- 3 個 50g 試喝裝小豆袋（深棕牛皮紙 + 不同顏色標籤標示產地：衣索比亞綠、肯亞橘、瓜地馬拉藍）並排
- 黃銅手沖咖啡量匙（左下角斜放）
- 米白色折頁品牌卡片（含使用說明、輕薄）
- 一小疊白色圓形咖啡濾紙

【場景】背景奶白色亞光石面、色調米白 + 深棕點綴、盒外散落 2-3 顆深棕咖啡豆呼應品牌（不搶戲）。

【光線】45 度頂光柔光 + 環境補光均勻、商品紋理清楚、燙金 logo 不被吃掉。

【文字疊加】畫面右上留白：品牌「OPEN BEANS」（無襯線細體大字、深棕）+ 副標「Holiday Tasting Set 2026」（襯線細體小字、淺棕）。

【風格】高解析度商業產品攝影、包裝材質紋理（牛皮紙 + 絨布 + 黃銅）真實、色板統一克制不雜亂。

【限制】嚴禁人物或人體局部、嚴禁內容物擠在一起難分辨、嚴禁強反光金屬背景吃 logo、嚴禁出現非 OPEN BEANS 的 logo、嚴禁快遞箱廉價感、嚴禁過飽和背景。

四個替換重點（同一份 prompt 改四處就變不同檔期）：

  • 【內容物】產地組合：衣索比亞／肯亞／瓜地馬拉 → 巴西／哥倫比亞／葉門／印尼
  • 【副標】檔期：「Holiday Tasting Set 2026」→ 「Mid-Autumn Edition」「Christmas Gift Box」「Lunar New Year 2027」
  • 【包裝】緞帶顏色：米白棉麻 → 深綠絨布（聖誕）／酒紅金邊（春節）／粉色細麻（母親節）
  • 【場景】季節點綴：散落咖啡豆 → 月餅模型（中秋）／松枝（聖誕）／梅花枝（春節）

(06)

改成你的場景

把上面 6 段拿來，只動 3 個替換槽，就變成你自己的 prompt：

替換槽你的版本
ⓐ 包裝顏色 + 緞帶＝？例：深棕牛皮紙 + 米白棉麻（自然）／中國紅 + 金色蝴蝶結（春節）
ⓑ 內容物 3-5 件分別是什麼？例：3 試喝豆袋 + 量匙 + 卡片 + 濾紙
ⓒ 檔期副標＝？（英文短句、≤ 5 字）例：「Holiday Tasting Set 2026」「Mid-Autumn Edition」

最常見的 4 種踩雷：

  1. 內容物擠在一起難分辨　寫了「3 件」AI 給你的可能是疊在一起或全塞回盒裡。解法：明確寫「對稱排列」「主商品略前置突出」「至少 1 件半露在盒外」三道鎖；如出現擠成一堆、改寫「each item visible separately」「inter-item spacing 1cm」。
  2. 盒子角度怪、看起來像快遞箱　盒蓋打開角度太大像被踢開、太小看不到內容物。解法：明確寫「盒蓋打開約 100-110 度、自然不別扭」「視角為俯瞰 30 度斜上」；並在限制加「禁快遞箱廉價感」「禁封箱膠帶」。
  3. 燙金 logo 被光吃掉、整盒掉檔次　寫了「戲劇光」AI 給你太強的 spotlight、燙金反白看不清品牌名。解法：B4 不要寫「戲劇主光」「rim light」（那是 B3）、明確寫「45° 頂光柔光 + 環境補光均勻」、並加「燙金 logo 必須清楚可讀」一條鎖。
  4. 中文 logo 字體別指望 prompt 解決　跟 B1／B2／B3 同一個限制——AI 生圖工具對中文字體只會給「系統預設襯線體」。

    B4 的最佳策略：外盒燙金 logo 與右上文案都寫英文（如本卡示範圖的「OPEN BEANS / Holiday Tasting Set 2026」）。中文「中秋限定」「節慶禮盒」「給媽媽的禮物」等情緒性副標——這些往往是禮盒主圖的銷售關鍵——一律走 後製字體疊圖工作流附錄 在 Figma／Canva 疊上。

    實戰提醒：B4 禮盒主圖的「節慶感中文字」往往是客戶買單的關鍵——「中秋」「春節」「母親節」這幾個字用對字體（金萱／文鼎晶熙黑／思源宋體粗）能直接讓客戶覺得「值這個價」。新細明體系會讓整盒掉檔次、客戶會要求重做。建議客戶溝通時就講白：「AI 生圖 只生圖、中文字一律後製」。

失敗案例｜prompt 動了哪一處會壞掉：

  • 把 contents.count 從 4 件加到 7 件 → 內容物擠成一堆、客戶看不清「裡面有什麼」
  • 把 contents.arrangement 移除「主商品略前置突出」 → AI 把所有東西平鋪在盒裡、沒有視覺層次
  • 把 package.ribbon 從「米白棉麻自然垂落」改成「對稱蝴蝶結」 → 變淘寶禮盒 cliché、文青客戶會皺眉
  • 把 lighting 改成 B3 戲劇光 → 燙金 logo 被光吃掉、看不見品牌名
  • 把 scene.background_surface 改成「鮮黃漆面」+ 盒色仍是深棕 → 整體掉檔次、看起來像 30 元商品禮盒
  • 把 限制.avoid 移除「快遞箱廉價感」 → AI 自動加封箱膠帶或瓦楞紋理、整盒沒救
⚠️
交付提醒　本卡片的 AI 生成的示範圖不是可直接給客戶的最終交付品。中文商業案一律先用 AI 生出構圖（含英文佔位文字）、再走 後製字體疊圖工作流 在 Figma／Canva 疊上中文、最後校稿確認才交付給客戶。
