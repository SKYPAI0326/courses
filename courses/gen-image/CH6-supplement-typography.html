後製字體疊圖工作流

當 AI 生圖工具給不了你想要的中文字體（金萱體、思源宋體、台灣文青包裝字）——這份附錄是配套解法。三條工作流：Canva（最簡單）、Figma（最精準）、Photoshop（最傳統）。從 CH6 任何一張涉及中文 logo 的卡片過來，這裡就是你的下一站。

這份附錄你會學到
能判斷哪些字要交給 AI 生圖、哪些字一定要後製疊上
能用 Canva 6 步驟把品牌字體疊到生成圖上（不會 Figma 也能做）
知道台灣常用品牌字體去哪裡取得（免費／付費／訂閱三種來源）
(01)

為什麼一定要後製

前情提要
在 CH6-B2 踩雷 #4 已驗證：AI 生圖工具對中文字體的選擇幾乎只會給「系統預設襯線體」（接近新細明體）。無論 prompt 寫「金萱體」「圓體明朝」「Maru Mincho」「禁用思源宋體」，回來的字體都長得差不多——這是模型訓練資料的固有限制、不是 prompt 寫得不夠。

實務分工原則：

  • AI 生圖工具負責：構圖、光線、人物姿態、道具、商品本身的型體與材質
  • 後製負責：中文 logo、品牌名、slogan、有設計感的標題字、瓶身的中文標籤字
  • 英文小字可賭：簡單英文（如 "Daily Oat 300ml"）有機會 AI 直接給對；但客戶交付前還是要檢查是否拼錯（B2 的 sub-label 就拼成 "Daily Yoy"）
⚖️
什麼時候不必後製：(1) 純白底電商主圖、本來就只要英文標籤（B1 場景）；(2) 純氛圍生活照、商品標籤被刻意虛化；(3) 純插畫風、不寫實的商品擬稿。除這 3 種情境外，**只要客戶實際品牌名要出現在圖上、就一定要後製**。
(02)

三條工作流選一條

依照你最熟悉的工具與輸出需求挑一條。三條都跑得通、產出品質沒太大差異——選你最快上手的那條。

推薦 A · Canva 沒有設計背景的學員首選。瀏覽器就能用、有免費版、字體庫含台灣常用字（Noto Sans TC、Noto Serif TC）。1-2 分鐘搞定。 跳到步驟 →
設計師 B · Figma 已會 Figma 的學員。控制最精細、可上 Adobe Fonts／Google Fonts、適合做品牌系統。免費版功能足夠。 跳到步驟 →
傳統 C · Photoshop 已有 Adobe 訂閱的學員。可上 Adobe Fonts 全套字體（含商業授權）、輸出 PSD 給客戶。一個月 NT$700 起。 跳到步驟 →
(03)

A · Canva 路徑｜6 步驟

最適合給沒有設計背景的客戶或學員。用免費 Canva 帳號 + Google 帳號登入即可。

  1. 新建空白設計　Canva 首頁 → 自訂尺寸 → 輸入跟你 AI 生圖 圖一樣的寬高（例：1024×1280）。提示：尺寸要一致、後續疊字位置才會對。
  2. 上傳生成圖　左側選單 上傳檔案 → 把 AI 跑出來的 png 拖進來 → 拉到畫布鋪滿。鎖定圖層（右鍵 → 鎖定），避免後續誤動。
  3. 遮蓋原本的字　如果 AI 生圖 已經把「日常燕麥」之類的中文字畫上去（B2 那種），用形狀 → 矩形（顏色取標籤背景同色，例：米白 #F5F0E8），蓋掉原字區域。也可用 Canva 的魔法橡皮擦（付費）去除。
  4. 新增文字 + 套品牌字體　左側文字 → 新增標題 → 輸入「日常燕麥」 → 字體選單搜尋「Noto Serif TC」（免費）或「金萱半糖」（如已加購 justfont 字體包）。字級先粗估、後續微調。
  5. 對齊與微調　縮放、旋轉、調行距字距讓字對準原本的標籤位置。Canva 的對齊輔助線（紫色虛線）會自動吸附中心；按住 Shift 拉縮放保持比例。
  6. 下載　右上 分享 → 下載 → 格式選 PNG（高品質）→ 存回 img/ 資料夾、命名加 -final 後綴（例：v45-b2-lifestyle-oat-final.png）。原圖保留作對照。
💡
免費 Canva 已含的台灣常用字：Noto Sans TC（黑體）、Noto Serif TC（襯線／接近金萱風）、TW-Sung、TW-Kai。如果客戶要正式金萱體（justfont 出品），需另購 justfont 訂閱（NT$199/月起、可在 Canva 中載入）。
(04)

B · Figma 路徑｜6 步驟

給已經會 Figma 的設計師等級學員。控制更細、適合做品牌系統（多張圖共用一套字體）。

  1. 新建 Frame　F → 拉一個跟 AI 生圖 圖同尺寸的 Frame（例：1024×1280）。Frame 命名為 B2-final。
  2. 放入生成圖　拖 png 進 Frame → 設成 Frame 的 Fill Image（不是 image layer）。鎖定 Frame fill 防止誤動。
  3. 遮蓋原字　建一個 Rectangle 蓋在原本的中文字位置 → Fill 取色用 Eyedropper 吸標籤底色 → 邊角適度圓角（matching 標籤紙質感）。如果原字本來就在 AI 自己畫的標籤上、直接蓋掉就好。
  4. 加入文字 + 載字體　T → 輸入「日常燕麥」 → 在右側 Font 欄位搜尋字體：免費可用「Noto Serif TC」（Google Fonts 內建）；付費可用 Adobe Fonts 中的「Source Han Serif（思源宋體）」或「金萱字體」（需安裝 justfont）。Figma 自動同步系統字體 + Google Fonts，Adobe Fonts 需先在桌機 Adobe Creative Cloud 啟用後才會出現。
  5. 對齊到 Pixel Grid　按 Cmd+'（Mac）開 Pixel Grid。文字對齊時用 Auto Layout 或數值對齊（X/Y 輸入精確值），確保跟原圖標籤完美貼合。色彩用 Eyedropper 吸原圖暗部、再 -10% lightness 取得「印刷字感」。
  6. 輸出　選中 Frame → 右下 Export → PNG → 2x（高解析度）。輸出後存回 img/ 加 -final 後綴。
💡
Figma 字體優勢：可建 Text Style 一次設定（字體 + 字級 + 行距 + 字距），之後 18 張卡片同一套品牌字直接套用、不用每張重設。Adobe Fonts 內含 justfont 金萱體（含在 Adobe 訂閱 NT$700/月內）、不必另外訂閱 justfont。
(05)

C · Photoshop 路徑｜6 步驟（簡略）

給已有 Adobe 訂閱的學員。如果你只是要疊字、Photoshop 的能力其實過剩——但好處是輸出 PSD 可給客戶後續修改。

  1. 開啟生成圖　File → Open 打開 AI 生圖 png，背景圖層按住 Alt 雙擊解鎖（變一般圖層 Layer 0）。
  2. 遮蓋原字　Spot Healing Brush（污點修復筆刷）或Generative Fill（Adobe Firefly、需訂閱）擦掉原本中文字。也可用矩形選取 + 取樣標籤色填滿。
  3. 新增文字圖層　T → 點擊位置 → 輸入「日常燕麥」。
  4. 套品牌字體　頂部字型選單搜尋「源樣明體」「金萱」「Noto Serif TC」（依你的 Adobe Fonts 啟用狀況）。
  5. 圖層樣式微調　雙擊文字圖層 → Layer Style → 加 Drop Shadow 0-2px（仿印刷字浮凸感）+ Blend Mode 設為 Multiply（讓字跟標籤融合）。
  6. 輸出　File → Export As → PNG，解析度設 100% 或 2x。同時 Save As PSD 留一份原始檔給客戶。

進階技巧：如果商品標籤是曲面（瓶身曲面、罐裝），用 Edit → Transform → Warp 讓文字輕微彎曲貼合曲面、避免「字平貼但商品圓」的違和感。也可用 Free Transform → Distort（按住 Cmd 拖角點）做透視調整。

(07)

完成檢核 + 常見踩雷

疊完字後對著檢核 4 點：

  • 字體對　跟客戶品牌的調性一致（不要文青皂用粗黑、不要科技 SaaS 用書法）
  • 字級對　原圖標籤位置的字級比例合理（標籤 30% 寬、字佔標籤 60%）
  • 位置對　字的中心線跟標籤中心線吻合、字距跟整張視覺重量平衡
  • 不過度修飾　不亂加陰影、發光、漸層；商業包裝字體應該樸素、不喧賓奪主
常見 5 種踩雷
  • 原字沒蓋乾淨　AI 生圖 的中文字疊一層新字後、原字還隱隱透出來。解法：放大 200% 檢查、必要時擴大遮蓋色塊或用Generative Fill。
  • 新字顏色太黑太突兀　AI 生圖 的標籤多是低飽和、新字若用純黑 #000000 會像貼上去的。解法：取原字暗部色 #2C2B28-#3A352A 為主色，融入畫面。
  • 字體授權沒對　商業案用 Google Fonts / 開源字體最安全。jf 系列訂閱期間可用、退訂後不可繼續商用（要看授權條款）。客戶交付前確認。
  • 曲面沒做變形　瓶身、罐裝是曲面但字平貼，視覺會違和。Photoshop Warp、Figma 用Curve plugin 或拉成 vector 路徑彎曲。
  • 輸出解析度降低　Canva 免費版輸出 PNG 若沒勾「高品質」會被縮小、字邊緣糊。解法：升級 Canva Pro 或用 Figma/PS 直出 2x。
