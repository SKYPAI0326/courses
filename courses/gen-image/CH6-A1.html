(A1)

Web Hero Banner 官網首屏

網頁首屏 / 落地頁 / app banner / 投放素材的橫向構圖視覺。一句強訴求語 + CTA + 主視覺、底部安全留白避免裁切。是品牌數位露出最高頻的格式、SOHO 接案 NT$5,000-15,000 級別案子最常見。

這張卡片你會學到
能判斷何時用 web hero banner（與 A2 KV、A3 海報的差異）
能拆解 6 段並安排橫向構圖的視覺重量
能識別 4 種踩雷（訴求語不夠強／CTA 不顯眼／安全區被裁切／主視覺位置不對）
(01)

適用場景

這個模板用在哪
橫向超寬比例（16:9 / 21:9 / 3:1）的網頁 hero 區。左側文案 + 右側主視覺 + 底部安全留白避免被瀏覽器裁切。比 A3 brand poster 多了 CTA、比 A2 KV 不需延展系統、比 B5 UI 疊加更聚焦品牌氛圍。
典型客戶 / 交付物　中型品牌官網 / 訂閱頁、單次接案 NT$5,000-15,000、交付：桌機 16:9 / 21:9 + 行動 9:16 兩比例 banner 圖

四個典型用途：

  • 品牌官網首屏 hero（首頁打開第一眼）
  • 電商落地頁 banner（活動專頁、訂閱專頁）
  • App 頂部活動 banner（推播、首頁活動橫幅）
  • Email marketing banner（EDM hero 區、會員週報）

比例橫向 16:9 / 21:9 / 3:1A3：直立 2:3；A4：直立 2:3
Claim1 句強訴求A3：品牌主張；A2：campaign 訴求語
CTA必有明顯按鈕A3：通常無；A4：通常無
安全留白底部 15% 避免裁切其他模板：較自由
延展性通常單張、不延展A2：必延展多比例
使用場景純數位（網頁 / app / EDM）A3：可印刷
💡
一句話判斷：客戶問「官網首屏」走 A1；問「campaign 系列」走 A2；問「品牌精神海報」走 A3；問「落地頁含表單 + UI」走 B5。A1 是 SOHO 接案最常碰的 A 組卡片。
(02)

模板拆解｜6 段 prompt 結構

web hero banner 6 段。核心難度在「橫向構圖的視覺重量分配」——左欄文案太多、右欄主視覺太空、整體不平衡：

段段落必問還是可預設填什麼
1比例 + 用途必問16:9 / 21:9 / 3:1（依平台）+ 用途（web / app / EDM）
2訴求語 + 副標必問主標 ≤ 8 字（英文優先避字體覆轍）+ 副標 1-2 行說明
3CTA 區必問1-2 個按鈕（主 CTA 填充 + 副 CTA 邊框）；位置：左欄主標下方
4主視覺必問右欄產品 / 模特 / 概念圖；占畫面 30-50%
5色板 + 字體可預設品牌色 + ≤ 3 色強調色；字體 ≤ 2 種
6安全留白固定寫死底部 15% 留白避免裁切；左右各留 5% padding
📐
記憶口訣：比例 → 訴求語 → CTA → 視覺 → 色 → 安全。新手最常忽略「安全留白」——AI 把元素貼到底部邊緣、實際放網頁時被瀏覽器自適應裁切。明確寫「安全區 15% from bottom edge」。
(03)

示範圖｜可複製、可重現

下面這張是 OPEN BEANS 訂閱方案的 web hero banner。21:9 超寬比例、左欄「Coffee, Reimagined.」大字 + CTA「Start Your Subscription」、右欄咖啡豆袋 + 散落咖啡豆 + 後景虛化的烘豆機。可直接放品牌官網首屏。

完整 prompt · OPEN BEANS 訂閱方案 web hero banner 適用：ChatGPT GPT-image / Gemini Nano Banana 
請生成一張 web 首屏 hero banner（網頁頂部主視覺），延續 OPEN BEANS 咖啡品牌系列。橫向 21:9 比例、可直接放進品牌官網首頁頂部：

【整體構圖】橫向超寬、左側 40% 文案區（左對齊）、右側 60% 主視覺區（OPEN BEANS 咖啡豆袋 + 散落咖啡豆 + 暖光氛圍）。底部留 15% 安全留白。

【背景】溫暖深棕漸變（左暗右亮）、絲絨紋理感、像精品咖啡店的牆面光影。色溫暖橘棕。

【左欄文案】（畫面左上區）
- 大字 headline：「Coffee, Reimagined.」（無襯線粗體大字、米白色、英文）
- 副標：「Single-Origin coffee, roasted in Taipei within 7 days of order. Subscribe now.」（細體、淺米色、英文）
- CTA 按鈕：「Start Your Subscription →」（深棕填充、米白文字、圓角）

【右欄主視覺】OPEN BEANS 200g 牛皮紙咖啡豆袋（同 B 組系列）置於畫面右側偏中、占整體畫面 25%、有戲劇性側光、底部散落幾顆深棕咖啡豆、瓶後遠景虛化的烘豆機輪廓（極淡）。

【風格】高級精品咖啡品牌氣質、像 Blue Bottle Coffee 官網 hero 那種視覺。低飽和、溫暖光、雜誌廣告級渲染。

【限制】嚴禁人物、嚴禁中文文字、嚴禁排版擁擠、嚴禁色板出現額外鮮艷色（限米白＋深棕＋暖橘 accent）、嚴禁底部 15% 安全留白被裁切到關鍵元素。
📷
讀圖時看這 4 個點：(1) 橫向構圖左右視覺重量平衡嗎？(2) CTA 按鈕夠不夠搶眼（不會被主視覺吃掉）？(3) 底部 15% 安全留白有沒有保留（沒有元素貼底）？(4) 主標是不是 ≤ 8 字夠精煉？
(04)

改成你的場景｜踩雷與失敗案例

把 prompt 拿來、改 3 個替換槽就變你的版本：

替換槽你的版本
ⓐ 比例 + 用途 = ?例：21:9 web hero / 16:9 EDM banner / 3:1 app 活動 banner
ⓑ 主標 + CTA = ?（英文優先）例：「Coffee, Reimagined」+「Start Subscription」
ⓒ 主視覺 = ?例：商品（咖啡豆袋）／模特（半身、不見正臉）／概念圖（抽象幾何）

最常見的 4 種踩雷：

  1. Claim 不夠強　「我們提供高品質的精品咖啡訂閱服務」這種長句沒人看。解法：主標 ≤ 8 字、英文優先、動詞驅動（Reimagined / Discover / Slow Down）。
  2. CTA 不夠顯眼　AI 給你一個米白邊框米白底的 CTA、跟背景融合。解法：明確寫「primary CTA filled with brand color, white text, rounded corners, prominent size」。
  3. 安全區被裁切　AI 把咖啡豆袋底部貼到畫面底邊、實際 RWD 縮小時被裁掉。解法：明確寫「all key elements within central 70%, bottom 15% reserved as safe area」。
  4. 中文主標一律後製　AI 生圖工具的中文字體限制。A1 的最佳策略：英文版用 AI 直接生成、中文版（「咖啡，重新被定義」）走 後製字體疊圖工作流附錄。

    實戰提醒：A1 是 A 組裡最高頻的接案類型——客戶官網每年改版至少 1 次、且常需多種比例（桌機 21:9 + 行動 9:16）。建議跟客戶簽「年度官網視覺套餐」，每季 1 張、客戶單價可拉到 NT$50,000+。
⚠️
交付提醒　本卡片的 AI 生成的示範圖不是可直接給客戶的最終交付品。中文商業案一律先用 AI 生出構圖（含英文佔位文字）、再走 後製字體疊圖工作流 在 Figma／Canva 疊上中文、最後校稿確認才交付給客戶。
