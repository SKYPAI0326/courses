CH6 模板速查站

CH6 · 模板速查站

CH1-CH5 教技法與工作流；CH6 是「做完想用時來查」的速查素材庫。18 張模板卡片 + 1 張後製字體附錄、按品牌行銷／商品電商／內容創作三組分類。卡關處從這裡找對應卡片、複製 prompt、改三處替換槽就能用。

這個章節怎麼用
CH1-CH5 學完後遇到實際接案、先看卡片找對應模板
每張卡片附完整 prompt、複製貼到 ChatGPT / Gemini 即可
中文 logo / 文案一律走後製字體疊圖附錄、不要硬靠 prompt
(01)

怎麼選對的卡片

三組分類邏輯
三組對應台灣本地接案三類客戶：A 組品牌行銷給有品牌經營需求的客戶（公司／自媒體／文化機構）；B 組商品電商給有實體商品要上架的客戶（蝦皮／momo／PChome／Pinkoi）；C 組內容創作給個人經營者（IG／Threads／Substack／LINE 貼圖）。看客戶屬於哪一類、再看每張卡片的「適用場景」段判斷。

A 組 · 品牌行銷

A1官網首屏 Web Hero Banner A2主視覺 Campaign KV A3品牌海報 Brand Poster A4雜誌封面 Editorial Cover A5品牌識別板 Brand Identity Board A6中英雙語版面 Bilingual Layout

B 組 · 商品電商

B1純白底電商主圖 B2生活方式情境圖 B3影棚高質感 hero B4包裝展示圖 B5商品卡 UI 疊加 B6飲料瓶身標籤設計

C 組 · 內容創作

C1便當格資訊圖 Bento Grid C2對比資訊圖 C3步驟教學圖 C4創辦人肖像 Founder Portrait C5LINE 貼圖組 C6穿搭多格拼貼 Lookbook Grid
⚠️
重要警語　CH6 所有卡片的 AI 生成的示範圖不是最終交付品。中文商業案件交付前，務必經 Figma／Canva／Photoshop 後製確認字體、確認標籤拼寫、確認商業合規。AI 生圖工具給你構圖與英文佔位文字、後製給你「能交付給客戶」的最終品。
📐 配套附錄　所有涉及中文 logo / 文案的卡片，最後都會引用這份 → 後製字體疊圖工作流（Canva / Figma / Photoshop 三條 6 步驟工作流 + 台灣常用字體取得管道）。AI 生圖工具對中文字體有固有限制、後製疊字是 CH6 的標準收尾。
(02)

每張卡片的結構拆解

所有 18 張模板卡片用同一套結構拆解、學員看一張會用所有：

區塊內容讀取重點
Hero標題 + 副標 + 學習成果判斷這張卡片是不是你要的
(01) 適用場景典型用途 + 對比近似模板確認卡片邊界、避免誤用
(02) 模板拆解6 段 prompt 結構表學會結構、之後寫 prompt 不從零開始
(03) 示範圖 + prompt實際成品 + 完整可複製 prompt + JSON 進階版直接複製、改替換槽即可用
(04) 改寫 + 踩雷3 個替換槽 + 4 種常見錯誤避免新手陷阱
💡
B1-B6 卡片是結構拆解完整版（6 區塊、含兩個情境範例）。A 組與 C 組是精簡版（4 區塊、不含獨立情境），但保留所有複製貼上必要的 prompt 與踩雷內容。設計取捨：B 組是學員最先學的、教學深度高；A／C 組是查閱用、結構精簡讓查找更快。
📖
學習路徑建議　第一次學 CH6，先讀完B 組 6 張完整版卡片（B1→B6）建立結構拆解直覺；之後 A 組與 C 組（精簡版）當查閱用即可。B 組學會了、A/C 組看 prompt 就能複製。
